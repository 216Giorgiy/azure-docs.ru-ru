Следующие ограничения применяются к "Сетка событий Azure" с системными разделами и настраиваемые разделы *не* домены событий.

| Ресурс | Ограничение |
| --- | --- |
| Количество пользовательских разделов на подписку Azure | 100 |
| Количество подписок на события на раздел | 500 |
| Скорость публикации пользовательского раздела (входящий трафик) | 5000 событий на раздел в секунду |

Только домены событий применяются следующие ограничения.

| Ресурс | Ограничение |
| --- | --- |
| Разделы в домене событий | 1000 (во время действия общедоступной предварительной версии) |
| Подписки на события на раздел в домене | 50 (во время действия общедоступной предварительной версии) |
| Подписки на события область домена | 50 (во время действия общедоступной предварительной версии) |
| Публикация скорость для домена событий (входящий трафик) | 5000 событий в секунду (во время действия общедоступной предварительной версии) |
