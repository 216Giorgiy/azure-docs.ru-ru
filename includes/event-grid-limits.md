####

Следующие ограничения применяются к пользовательским и системным разделам Сетки событий, а *не* к доменам событий.

| Ресурс | Ограничение |
| --- | --- |
| Количество пользовательских разделов на подписку Azure | 100 |
| Количество подписок на события на раздел | 500 |
| Скорость публикации пользовательского раздела (входящий трафик) | 5000 событий на раздел в секунду |

####

Ниже указаны ограничения, применяемые только к доменам событий.

| Ресурс | Ограничение |
| --- | --- |
| Количество разделов на домен событий | 1000 (во время действия общедоступной предварительной версии) |
| Количество подписок на события на раздел в домене | 50 (во время действия общедоступной предварительной версии) |
| Количество подписок на события области домена | 50 (во время действия общедоступной предварительной версии) |
| Скорость публикации для домена событий (входящий трафик) | 5000 событий в секунду (во время действия общедоступной предварительной версии) |
