> [!IMPORTANT]
> Для содержимого этой статьи необходим классический Azure CLI. Текущая версия Azure CLI не поддерживает функции, описанные в этой статье, поэтому требуется классический интерфейс командной строки. 
> 
> Классический интерфейс командной строки можно установить параллельно с современным Azure CLI, но мы рекомендуем использовать Azure CLI для всех новых сценариев и развертываний с поддержкой данной функции. Чтобы установить классический интерфейс командной строки, обратитесь к статье [Установка классического Azure CLI](/cli/azure/install-classic-cli), а для установки текущей версии интерфейса командной строки обратитесь к статье [Установка Azure CLI](/cli/azure/install-azure-cli).
