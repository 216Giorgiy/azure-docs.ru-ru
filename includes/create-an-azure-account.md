# Создание учетной записи Azure и включение предварительных версий компонентов

Для развертывания приложений в Azure необходимо использовать учетную запись. В данной статье описаны способы создания бесплатной пробной учетной записи и включения предварительных версий компонентов Azure, в том числе служб BizTalk, HDInsight и службы архивации Azure, для этой учетной записи.

## Создание учетной записи Azure

1.  Откройте веб-обозреватель и перейдите к <a href="http://www.windowsazure.com">http://www.windowsazure.com</a>.
2.  При наличии подписки на MSDN, перейдите к странице <a href="/ru-ru/pricing/member-offers/msdn-benefits-details/" target="_blank">Преимущество Azure для подписчиков MSDN</a> и выполните инструкцию по созданию учетной записи, активировав свои преимущества подписчика MSDN.

   В противном случае перейдите к странице <a href="/ru-ru/pricing/free-trial/" target="_blank">создания бесплатной пробной учетной записи</a> и выполните инструкцию по созданию пробной учетной записи. Для подтверждения личности требуется указать номер кредитной карты и номер мобильного телефона, но счет выставляться не будет.

<h2><a id="enable"></a>Включение предварительных версий компонентов Azure</h2>

1.  Перейти к [https://account.windowsazure.com/](https://account.windowsazure.com/) и выполните вход с использованием своей учетной записи Azure.
2.  Щелкните **Предварительные версии компонентов**, чтобы просмотреть доступные предварительные версии.<br />
    ![открыть вкладку предварительных версий компонентов][1]
3.  Найдите компонент, который требуется включить, и щелкните **Попробовать сейчас**.<br />
    ![выбрать предварительную версию компонента][2]
4.  Выберите свою подписку и установите флажок.<br />
    ![выбрать подписку][3]
