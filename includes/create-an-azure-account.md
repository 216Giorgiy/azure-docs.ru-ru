# Создание учетной записи Azure и включение предварительных версий компонентов

Для развертывания приложений в Azure необходима учетная запись. В данной статье описаны способы создания бесплатной пробной учетной записи и включения предварительных версий компонентов Azure, в том числе служб BizTalk, HDInsight и службы архивации Azure, для этой учетной записи.

## Создание учетной записи Azure

1.  Откройте веб-браузер и перейдите по адресу <a href="http://www.windowsazure.com">http://www.windowsazure.com</a>
2.  Если у вас есть подписка на MSDN, перейдите к странице <a href="/ru-ru/pricing/member-offers/msdn-benefits-details/" target="_blank">Преимущество Azure для подписчиков MSDN</a> и выполните указания по созданию учетной записи, активировав свои преимущества подписчика MSDN.

   В противном случае перейдите к странице <a href="/ru-ru/pricing/free-trial/" target="_blank">создания бесплатной пробной учетной записи</a> и выполните указания по созданию пробной учетной записи. Для подтверждения личности понадобится указать номер кредитной карты и номер мобильного телефона, но счет выставляться не будет.

<h2><a id="enable"></a>Включение предварительных версий компонентов Azure</h2>

1.  Перейдите по адресу [https://account.windowsazure.com/](https://account.windowsazure.com/) и войдите с помощью своей учетной записи Azure.
2.  Щелкните **Предварительные версии компонентов** для просмотра доступных предварительных версий.<br />
    ![open preview features tab][1]
3.  Найдите компонент, который требуется включить, и щелкните **Попробовать сейчас**.<br />
    ![select a preview feature][2]
4.  Выберите свою подписку и установите флажок.<br />
    ![select subscription][3]
