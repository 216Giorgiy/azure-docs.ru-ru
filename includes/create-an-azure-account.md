# Создание учетной записи Azure и включение предварительных версий компонентов

Для развертывания приложений в Azure необходима учетная запись. В данной статье описаны способы создания бесплатной пробной учетной записи и включения предварительных версий компонентов Azure, в том числе служб BizTalk, HDInsight и службы архивации Azure, для этой учетной записи.

## Создание учетной записи Azure

1.  Откройте браузер и перейдите на страницу <a href="http://www.windowsazure.com">http://www.windowsazure.com</a>.
2.  Если у вас есть подписка на MSDN, перейдите на страницу <a href="/pricing/member-offers/msdn-benefits-details/" target="_blank">Преимущество Azure для подписчиков MSDN</a>, создайте учетную запись, следуя приведенным указаниям, и получите преимущества подписчика MSDN.

   Или перейдите на страницу <a href="/pricing/free-trial/" target="_blank">создания бесплатной пробной учетной записи</a> и создайте пробную учетную запись, следуя приведенным указаниям. Для подтверждения личности требуется указать номер кредитной карты и номер мобильного телефона, но счет выставляться не будет.

<h2><a id="enable"></a>Включение предварительных версий компонентов Azure</h2>

1.  Перейдите по адресу [https://account.windowsazure.com/](https://account.windowsazure.com/) и войдите в систему, используя свою учетную запись Azure.
2.  Щелкните **Предварительные версии компонентов**, чтобы просмотреть доступные предварительные версии.<br />
    ![open preview features tab][1]
3.  Найдите компонент, который требуется включить, и щелкните **Попробовать сейчас**.<br />
    ![select a preview feature][2]
4.  Выберите свою подписку и установите флажок.<br />
    ![select subscription][3]
