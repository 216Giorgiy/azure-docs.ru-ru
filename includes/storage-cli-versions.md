## <a name="versions-of-the-azure-cli"></a>Версии Azure CLI

Сейчас доступны две версии интерфейса командной строки Azure (CLI):

* [Azure CLI](../articles/storage/common/storage-azure-cli.md). Текущая версия Azure CLI для модели развертывания с помощью Resource Manager.
* [Классический Azure CLI](../articles/storage/common/storage-azure-cli-nodejs.md). Более старая версия Azure CLI, предназначенная только для классической модели развертывания.
