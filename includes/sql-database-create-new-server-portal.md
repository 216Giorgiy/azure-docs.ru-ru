## Создание логического сервера базы данных SQL Azure
Следуйте приведенным ниже инструкциям, чтобы создать на портале Azure логический сервер базы данных SQL Azure.

1. Если вы еще этого не сделали, подключитесь к [порталу Azure](http://portal.azure.com).
2. Щелкните **Создать**, **Данные+хранилище** и найдите элемент **База данных SQL**.
   
      ![База данных SQL 1](./media/sql-database-get-started/sql-database-new-database-1.png)
3. Щелкните **База данных SQL**, чтобы открыть колонку "База данных SQL".
   
      ![Новый логический сервер](./media/sql-database-get-started/sql-database-new-database-2.png)
4. Укажите значения для следующих свойств сервера:
   
   * имя сервера;
   * учетные данные администратора сервера для входа;
   * Пароль
   * "Подписка" — только если у вас несколько подписок.
   * группа ресурсов (новая или существующая);
   * Расположение
     
        ![New logical server](./media/sql-database-create-new-server-portal/sql-database-create-new-server-portal-4.png)
5. Щелкните **Создать**. В области уведомлений вы увидите, что развертывание началось.
   
      ![Новый логический сервер](./media/sql-database-create-new-server-portal/sql-database-create-new-server-portal-5.png)
6. Прежде чем переходить к следующему шагу, дождитесь завершения развертывания.
   
       ![Новый логический сервер](./media/sql-database-create-new-server-portal/sql-database-create-new-server-portal-6.png)
