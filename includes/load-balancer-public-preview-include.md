Эта общедоступная предварительная версия Load Balancer "Стандартный" может не отличаться таким же уровнем доступности и надежности, как общедоступный выпуск. Эта функция не поддерживается, может иметь ограничения и быть доступной не во всех расположениях Azure. Актуальные сведения о доступности и состоянии этой функции см. на странице [обновлений для Azure Load Balancer](https://azure.microsoft.com/en-us/updates/?product=load-balancer).
