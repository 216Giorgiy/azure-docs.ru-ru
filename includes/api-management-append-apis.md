## <a name="append-other-apis"></a>Добавление других интерфейсов API

API-интерфейс может состоять из других API-интерфейсов, предоставляемых различными службами, включая спецификацию OpenAPI, SOAP API, функцию приложений API Службы приложений Azure, Функции Azure, Azure Logic Apps и Azure Service Fabric.

![Импорт API](./media/api-management-append-apis/import.png)

Чтобы добавить другой API к существующему, выполните следующие действия. После импорта другого API операции добавляются к текущему API.

1. Перейдите к экземпляру службы управления API на портале Azure.
2. В разделе **Управление API** выберите **Интерфейсы API**.
3. Выберите многоточие (**…**) рядом с API, к которому нужно добавить другой API.
4. В раскрывающемся меню выберите **Импорт**.
5. Выберите службу, из которой следует импортировать API.
