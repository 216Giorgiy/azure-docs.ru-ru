На странице **Глобальные сети (WAN)** выберите созданную глобальную сеть. В меню слева перейдите на страницу **Начало работы**. На этой странице отображается порядок, согласно которому создаются ресурсы виртуальной глобальной сети. Для создания функциональной глобальной сети необходимо выполнить шаги 1–4. Сайты и центры можно создавать в любом порядке, так как они не зависят друг от друга. При этом их нужно создать, прежде чем их связывать.

Необходимые действия: 

1. Создание сайтов
2. Создание центров
3. Связывание сайтов с центрами
4. Загрузка файла
5. Подключение центров с помощью виртуальной сети (необязательно для некоторых конфигураций)
