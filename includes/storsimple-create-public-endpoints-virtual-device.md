#### <a name="to-create-public-endpoints-on-the-virtual-device"></a>Создание общедоступных конечных точек на виртуальном устройстве

1. Войдите на классический портал Azure.
2. Щелкните **Виртуальные машины**, а затем выберите виртуальную машину, которая используется как виртуальное устройство.
3. Нажмите кнопку **Конечные точки**. На странице **Конечные точки** перечислены все конечные точки виртуальной машины.
4. Щелкните **Добавить**. Откроется диалоговое окно **Добавить конечную точку** . Щелкните стрелку для продолжения.
5. В поле **Имя** введите имя конечной точки **WinRMHttps**.
6. В поле **Протокол** укажите **TCP**.
7. В поле **Общий порт**введите номера порта, которые хотите использовать для подключения.
8. В поле **Частный порт** введите **5986**.
9. Чтобы создать конечную точку, поставьте галочку.

После создания конечной точки вы можете просмотреть ее сведения, чтобы определить общедоступный виртуальный IP-адрес. Запишите этот адрес.
