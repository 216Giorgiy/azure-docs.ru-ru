
1. В Visual Studio щелкните правой кнопкой мыши проект и выберите **Опубликовать** > **Опубликовать веб-сайт** > **Веб-приложения Microsoft Azure**.

2. Войдите с учетными данными Azure и выберите свою службу в списке **Существующие веб-приложения** (к ее названию может быть добавлен суффикс -code). Visual Studio загрузит параметры публикации прямо из Azure. Чтобы завершить, нажмите кнопку **Опубликовать**.
