Ниже перечислены ограничения на использование и другие ограничения для службы Azure Active Directory (Azure AD).

| Категория | Ограничения |
| --- | --- |
| Каталоги | Один пользователь может принадлежать максимум к 500 каталогам Azure AD как член или гость.<br/>Один пользователь может создать максимум 20 каталогов. |
| Домены | Можно добавить не более 900 управляемых доменных имен. Если вы настраиваете все домены для федерации с локальной службой Active Directory, в каждый каталог можно добавить до 450 доменных имен. |
| Объекты |<ul><li>Пользователи выпуска Free службы Azure Active Directory могут создать в одном каталоге не более 500 000 объектов.</li><li>Пользователь без прав администратора может создать не более 250 объектов.</li></ul> |
| Расширения схемы |<ul><li>Расширения типа String не должны превышать 256 знаков. </li><li>расширения типа Binary не должны превышать 256 байт;</li><li>в один объект можно записать не более 100 значений расширений (для всех типов и всех приложений);</li><li>Дополнить однозначными атрибутами типов String или Binary можно только сущности User, Group, TenantDetail, Device, Application и ServicePrincipal.</li><li>Расширения схемы доступны только в предварительной версии API Graph 1.21. Приложению должен быть предоставлен доступ на запись для регистрации расширения.</li></ul> |
| ПРИЛОЖЕНИЯ |Владельцами одного приложения могут быть не более 100 пользователей. |
| Группы |<ul><li>Владельцами одной группы могут быть не более 100 пользователей.</li><li>Участниками одной группы в Azure Active Directory может быть любое число объектов.</li><li>Число участников в группе, которых можно синхронизировать из локальной службы Active Directory в Azure Active Directory с помощью Azure AD Connect, ограничено 50 тысячами.</li></ul> |
| Панель доступа |<ul><li>Для пользователей с лицензиями на Azure AD Premium или Enterprise Mobility Suite количество приложений, отображаемых на панели доступа для одного конечного пользователя, не ограничено.</li><li>Каждый пользователь с лицензией для выпуска Free или Basic Azure Active Directory может видеть на панели доступа не более 10 плиток приложения (примеры: Box, Salesforce или Dropbox). Это ограничение не действует для учетной записи администратора.</li></ul> |
| Отчеты | В любом отчете можно просматривать и загружать не более 1000 строк. Любые дополнительные данные будут усечены. |
| Административные единицы | Объект может входить не более чем в 30 административных единиц. |
