## Создание учетной записи хранения Azure

Для использования хранилища Azure вам потребуется учетная запись хранения. Выполнив следующие шаги, вы создадите учетную запись хранения. (Для создания учетной записи хранения можно также воспользоваться клиентской библиотекой управления службами Azure или непосредственно самим интерфейсом управления службами [API REST].)

1.  Выполните вход на [портал управления Azure].

2.  В нижней части области навигации щелкните **СОЗДАТЬ**.

	![\+Создать][plus-new]

3.  Последовательно выберите **СЛУЖБЫ ДАННЫХ**, **ХРАНИЛИЩЕ** И **БЫСТРО СОЗДАТЬ**.

	![Диалоговое окно "Быстрое создание"][quick-create-storage]

4.  В качестве адреса URL введите имя поддомена для URI учетной записи хранения. Записи могут содержать от 3 до 24 строчных букв и цифр. Это значение станет именем узла в универсальном коде ресурса (URI), который используется для обращения к ресурсам больших двоичных объектов, очередей и таблиц для подписки.

5.  Выберите регион или территориальную группу, где требуется расположить хранилище. При использовании хранилища в приложении Azure выберите тот же самый регион, в котором будет развернуто приложение.

6. Дополнительно можно выбрать тип репликации, необходимый для вашей учетной записи. По умолчанию используется геоизбыточная репликация, предоставляющая максимальную надежность. Дополнительные сведения о вариантах репликации см. в разделах [Варианты избыточности хранилища Azure](http://msdn.microsoft.com/library/azure/dn727290.aspx) и [Блог группы хранилища Azure](http://blogs.msdn.com/b/windowsazurestorage/).

6.  Щелкните **СОЗДАТЬ УЧЕТНУЮ ЗАПИСЬ ХРАНЕНИЯ**.
