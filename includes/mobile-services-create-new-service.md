Выполните следующие действия для создания новой мобильной службы.

1.  Выполните вход в [портал управления][портал управления].

2.  В нижней части области навигации щелкните **+СОЗДАТЬ**.

    ![][0]

3.  Разверните окна **Среда выполнения приложений** и **Мобильная услуга**, нажмите **Создать**.

    ![][1]

    Откроется диалоговое окно **Новая мобильная служба**.

4.  На странице **Создать мобильную службу** выберите **Создать бесплатную базу данных SQL размером 20 МБ**, выберите среду выполнения **Node.js**, введите имя дочернего домена для новой мобильной службы в поле **URL-адрес** и ждите проверки имени. После завершения проверки имени нажмите кнопку со стрелкой вправо, чтобы перейти к следующей странице.

    ![][2]

    Откроется страница **Указать параметры базы данных**.

    <div class="dev-callout">

    **Примечание.**
    В рамках данного учебника вы создадите новый экземпляр базы данных SQL и сервер. Эту новую базу данных можно потом использовать и администрировать ее, как и любой другой экземпляр базы данных SQL. Если в этом регионе у вас уже имеется база данных в качестве новой мобильной службы, вы можете выбрать **Использовать существующую базу данных** и выбрать эту базу данных. Из-за затрат на увеличение пропускной способности и более длительных задержек не рекомендуется использовать базу данных в другом регионе.

    </div>

5.  В поле **Имя** введите имя новой базы данных, затем введите **Имя для входа в систему**, которое является именем входа администратора для нового сервера базы данных SQL, введите и подтвердите пароль и нажмите кнопку "Проверить", чтобы завершить процесс.

    ![][3]

    <div class="dev-callout">

    **Примечание.**
    Если введенный вами пароль не соответствует требованиям в отношении минимального размера или не совпадает, выводится предупреждение.
    Рекомендуется записать имя пользователя и пароль администратора; эта информация понадобится, чтобы использовать в будущем экземпляр базы данных SQL или сервер.

    </div>

Вы создали новую мобильную службу, которая может использоваться вашими мобильными приложениями.
