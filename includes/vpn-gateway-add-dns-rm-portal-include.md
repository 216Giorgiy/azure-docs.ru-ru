1. На странице **Параметры** виртуальной сети щелкните **DNS-серверы**, чтобы открыть колонку **DNS-серверы**.

    ![Добавление DNS-сервера](./media/vpn-gateway-add-dns-rm-portal/add_dns_server.png "Add DNS Server")

  - **DNS-серверы** — выберите **Пользовательский**.
  - **Добавить DNS-сервер** — укажите IP-адрес DNS-сервера, который нужно использовать для разрешения имен.

2. Добавив DNS-серверы, нажмите кнопку **Сохранить** в верхней части колонки.
