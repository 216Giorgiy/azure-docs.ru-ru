## Создание концентратора событий

1. Перейдите на [классический портал Azure][] и нажмите кнопку **СОЗДАТЬ** в нижней части экрана.

2. Последовательно щелкните элементы **Службы приложений**, **Служебная шина**, **Концентратор событий** и **Быстрое создание**.

	![][1]

3. Введите имя для концентратора событий, выберите нужный регион и щелкните **Создать новый концентратор событий**.

	![][2]

4. Если в указанном регионе не выбрать существующее пространство имен, портал создаст его автоматически (обычно ему присваивается имя в виде ***имя\_концентратора\_событий*-ns**). Щелкните это пространство имен (в этом примере — **eventhub-ns**).

	![][3]

5. В нижней части страницы щелкните **Сведения о подключении**. Нажмите кнопку копирования (см. рисунок ниже), чтобы скопировать строку подключения **RootManageSharedAccessKey** в буфер обмена. Сохраните эту строку. Она вам понадобится позже.

	![][4]

Теперь концентратор событий создан, и у вас есть строки подключения, необходимые для отправки и приема событий.
