Для доступа к службе хранилища Azure требуется подписка Azure. Если у вас еще нет подписки, вы можете [создать бесплатную учетную запись Azure](https://azure.microsoft.com/free/?WT.mc_id=A261C142F), прежде чем начинать работу.

Доступ к хранилищу Azure осуществляется с помощью учетной записи хранения. Если у вас еще нет учетной записи хранения, ее можно без усилий создать с на [портале Azure](https://portal.azure.com/#create/Microsoft.StorageAccount-ARM). Учетную запись хранения также можно создать с помощью Azure PowerShell или Azure CLI. Инструкции по созданию учетной записи хранения см. в статье [Создайте учетную запись хранения](../articles/storage/common/storage-quickstart-create-account.md).
