При использовании облачного устройства StorSimple учитывайте следующие рекомендации по безопасности:

* Защита облачного устройства обеспечивается в рамках подписки Microsoft Azure. Это означает, что если при использовании облачного устройства подписка Azure будет скомпрометирована, данные, сохраненные на облачном устройстве, также будут уязвимы.
* Открытый ключ сертификата, используемый для шифрования данных, сохраненных в StorSimple, безопасно предоставляется на портале Azure, а закрытый ключ хранится на облачном устройстве StorSimple. На облачном устройстве StorSimple и открытый, и закрытый ключи хранятся в Azure.
* Облачное устройство размещено в центре обработки данных Microsoft Azure.
