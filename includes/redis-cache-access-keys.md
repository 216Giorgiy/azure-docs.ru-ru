### <a name="retrieve-host-name-ports-and-access-keys-by-using-the-azure-portal"></a>Получение имени узла, сведений о портах и ключей доступа с помощью портала Azure

Чтобы подключаться к экземпляру предложения "Кэш Redis для Azure", клиентам кэша нужны имя узла, сведения о портах и ключ кэша. Некоторые клиенты могут ссылаться на эти элементы с помощью незначительно различающихся имен. Эти данные можно получить на портале Azure.

#### <a name="to-retrieve-the-access-keys-and-host-name"></a>Получение ключей доступа и имени узла

1. Чтобы получить ключи доступа с помощью [портала Azure](https://portal.azure.com), перейдите к своему кэшу и выберите **Ключи доступа**. 

    ![Ключи для Кэша Redis](media/redis-cache-access-keys/redis-cache-keys.png)

2. Чтобы получить имя узла и сведения о портах, выберите **Свойства**.

    ![Свойства Кэша Redis](media/redis-cache-access-keys/redis-cache-hostname-ports.png)
