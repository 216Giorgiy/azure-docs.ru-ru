Выполните следующие действия для создания новой мобильной службы.

1.  Выполните вход в [портал управления][].

2.  В нижней части области навигации щелкните **+СОЗДАТЬ**.

    ![][ ]

3.  Разверните окна **Среда выполнения приложений** и **Мобильная услуга**, нажмите **Создать**.

    ![][1]

    Откроется диалоговое окно **Новая мобильная служба**.

4.  На странице **Создать мобильную службу** щелкните **Создать бесплатную базу данных SQL размером 20 МБ**, выберите среду выполнения **.NET**, введите имя поддомена для новой мобильной службы в текстовое поле **URL-адрес** и подождите проверки имени. После завершения проверки имени нажмите кнопку со стрелкой вправо, чтобы перейти к следующей странице.

    ![][2]

    Откроется страница **Указать параметры базы данных**.

    <div class="dev-callout"> 
	<b>Примечание.</b> 
	<p>В рамках данного учебника вы создадите новый экземпляр базы данных SQL и сервер. Эту новую базу данных можно потом использовать и администрировать ее, как и любой другой экземпляр базы данных SQL. Если в этом регионе у вас уже имеется база данных в качестве новой мобильной службы, вы можете выбрать <strong>Использовать существующую базу данных</strong> и выбрать эту базу данных. Из-за затрат на увеличение пропускной способности и более длительных задержек не рекомендуется использовать базу данных в другом регионе.</p></div>

5.  В поле **Имя** введите имя новой базы данных, затем введите **Имя для входа в систему**, которое является именем входа администратора для нового сервера базы данных SQL, введите и подтвердите пароль и нажмите кнопку "Проверить", чтобы завершить процесс.

    ![][3]

    <div class="dev-callout"> 
	<b>Примечание.</b> 
	<p>Если введенный вами пароль не соответствует требованиям в отношении минимального размера или не совпадает, выводится предупреждение. <br/>Рекомендуется записать имя пользователя и пароль администратора; эта информация понадобится, чтобы использовать в будущем экземпляр базы данных SQL или сервер.</p> 
	</div>

Вы создали новую мобильную службу, которая может использоваться вашими мобильными приложениями.
