Откройте Unity, а затем откройте проект в папке `Unity`.

Откройте **Build Settings** (Параметры сборки), выбрав пункты **File** -> **Build Settings** (Файл > Параметры сборки).

В разделе **Платформа** выберите **iOS**.

Выберите **Switch Platform** (Изменить платформу), чтобы изменить платформу на **iOS**.

![Параметры сборки Unity](./media/spatial-anchors-unity/unity-ios-build-settings.png)

Закройте окно **Build Settings** (Параметры сборки).

### <a name="download-and-import-the-unity-arkit-plugin"></a>Скачивание и импорт модуля Unity ARKit

Скачайте модуль [Unity ARKit Plugin v2.0.0](https://bitbucket.org/Unity-Technologies/unity-arkit-plugin/get/v2.0.0.zip) и извлеките содержимое архива.

Скопируйте содержимое папки `Assets` из извлеченной папки Unity ARKit Plugin в папку `Assets` примера.
