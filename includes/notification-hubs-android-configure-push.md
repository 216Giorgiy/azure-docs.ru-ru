

1. Войдите на [портал управления Аzure](https://manage.windowsazure.com/) и щелкните **+СОЗДАТЬ** в нижней части экрана.

2. Последовательно щелкните **Службы приложений**, **Service Bus**, **Концентратор уведомлений** и **Быстрое создание**.

3. Введите имя для концентратора уведомлений, выберите нужный регион и нажмите кнопку **Создать новый концентратор уведомлений**.

   	![Задание свойств центра уведомлений](./media/notification-hubs-android-configure-push/notification-hub-create-from-portal2.png)

4. В разделе **Служебная шина** щелкните недавно созданное пространство имен (обычно это ***имя_концентратора_уведомлений*-ns**).

5. В своем пространстве имен откройте расположенную вверху вкладку **Центры уведомлений** и выберите недавно созданный центр уведомлений.

6. Перейдите на расположенную сверху вкладку **Настройка**, введите значение **Ключ API**, полученное в предыдущем разделе, а затем нажмите кнопку **Сохранить**.

   	![Задание ключа GCM API на вкладке "Настройка"](./media/notification-hubs-android-configure-push/notification-hub-configure-android.png)

7. Откройте расположенную сверху вкладку **Панель мониторинга** и нажмите кнопку **Просмотреть строку подключения**. Запишите значения двух строк подключения.

Концентратор уведомлений теперь настроен для работы с GCM. Также у вас есть строки подключения, с помощью которых вы можете зарегистрировать приложение для получения уведомлений и отправки push-уведомлений.
