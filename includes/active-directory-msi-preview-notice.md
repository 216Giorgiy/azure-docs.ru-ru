|Удостоверение управляемой службы (MSI) сейчас доступно как предварительная версия функции Azure Active Directory, предоставляемая только для ознакомительных целей. Так как MSI-файл развертывается в регионы глобально, его отображение в вашем регионе может занять некоторое время. Прежде чем начинать, ознакомьтесь с [известными проблемами](/azure/active-directory/msi-known-issues). См. дополнительные сведения о [дополнительных условиях использования предварительных выпусков Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).|
