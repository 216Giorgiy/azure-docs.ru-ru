Журнал аудита содержит все операции записи (PUT, POST, DELETE), выполняемые с ресурсами. Он не содержит операции чтения (GET). Журналы аудита можно использовать для поиска ошибки при устранении неполадок, а также для наблюдения за тем, как пользователь в вашей организации изменяет ресурс.

При работе с журналами аудита следует помнить о двух важных ограничениях.

1. Журналы аудита сохраняются только в течение 90 дней.
2. Вы можете отправить запрос на получение данных за период не более 15 дней.
