


Используйте преимущества пакета Microsoft HPC и служб вычислений и инфраструктуры Azure для создания облачного кластера высокопроизводительных вычислений (HPC) и управления им. [Пакет HPC](https://technet.microsoft.com/library/jj899572.aspx) — это бесплатное решение Майкрософт для высокопроизводительных вычислений, созданное на основе технологий Microsoft Azure и Windows Server, которое поддерживает широкий диапазон рабочих нагрузок высокопроизводительных вычислительных систем. Облачный кластер пакета HPC предоставляет администратору кластера или независимому поставщику программного обеспечения (ISV) гибкую, масштабируемую платформу для выполнения приложений для ресурсоемких вычислений. Он также может снизить инвестиции в локальный вычислительный кластер. Чтобы изучить другие варианты выполнения высокопроизводительных вычислений и пакетных рабочих нагрузок в Azure, ознакомьтесь с [техническими материалами по пакетным и высокопроизводительным вычислениям](../articles/batch/big-compute-resources.md).
