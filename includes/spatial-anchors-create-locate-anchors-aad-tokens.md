### <a name="azure-active-directory-authentication"></a>Аутентификация Azure Active Directory

Azure пространственных привязки также позволяет проверять подлинность с помощью маркеров Azure Active Directory пользователя приложений. Например можно использовать токены AAD для интеграции с ASA. Если предприятия сохраняет пользователей в Azure Active Directory, вы можете указать маркер AAD пользователя в пакете SDK Azure пространственных привязки. Это позволит вам осуществлять проверку подлинности непосредственно к службе ASA для учетной записи, которая входит в том же клиенте AAD.
