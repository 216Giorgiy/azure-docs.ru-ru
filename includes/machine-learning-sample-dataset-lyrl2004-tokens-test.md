Характеристики тестовых примеров в наборе данных новостей RCV1-V2 Reuters. Набор данных содержит 781 тыс. новостных статей с идентификаторами (первый столбец набора данных). Для каждой статьи выполнен анализ по лексемам, стоп-словам и однокоренным словам. Набор данных предоставлен Дэвидом Д. Льюисом (David. D. Lewis).
