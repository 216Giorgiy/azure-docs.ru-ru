На предыдущем шаге вы создали ресурсы Azure в группе ресурсов. Если вы считаете, что в будущем эти ресурсы вам не понадобятся, их можно удалить, удалив группу ресурсов.
 
1. Выполните следующую команду, которая покажет, содержит ли группа ресурсы, которые нужно сохранить:

  ```azurecli
  az group show --name myResourceGroup
  ```

2. Если отображаются только те ресурсы, которые необходимо удалить, выполните следующую команду:
 
  ```azurecli
  az group delete --name myResourceGroup
  ```
