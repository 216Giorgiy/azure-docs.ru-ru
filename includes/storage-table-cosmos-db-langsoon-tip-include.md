> [!TIP]
> Сведения в этой статье касаются хранилища таблиц Azure. Но теперь есть расширенная версия хранилища таблиц в виде API таблиц Azure Cosmos DB. Для нее предусмотрены оптимизированные для пропускной способности таблицы, глобальное распределение и автоматическое вторичное индексирование. См. дополнительные сведения об [API таблиц Azure Cosmos DB](https://aka.ms/premiumtables). Язык программирования, рассматриваемый в статье, еще не поддерживается расширенной версией, но его поддержка будет добавлена в будущем.
