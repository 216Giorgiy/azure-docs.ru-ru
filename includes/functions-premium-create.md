1. Перейдите на портал Azure.
2. Добавьте новый ресурс, выбрав **создать ресурс** кнопку на портале Azure.
3. Выберите **Безсерверное приложение-функция**.
4. Введите все данные в параметры создания и настройте вашей **текущему плану размещения и** для **план службы приложений**:
    
    ![Создайте план функции приложений](./media/functions-premium-create/create-function-app-plan.png)  

    1. Выберите уровень плана (EP) уровня "премиум" для плана обслуживания. 
        ![Выберите план "премиум"](./media/functions-premium-create/hosting-plan.png)  

    1. Проверьте параметры завершенных плана.  
        ![Готовое приложение службы плана](./media/functions-premium-create/create-app-service-plan.png)  
    
1. Проверить параметры приложения и плана и выберите **создать**.

    ![Готовое приложение службы плана](./media/functions-premium-create/create-function-app.png)
