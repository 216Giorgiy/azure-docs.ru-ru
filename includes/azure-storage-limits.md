| Максимальная скорость исходящих данных<sup>3</sup> на одну учетную запись хранения (регионы США) | 20 Гбит/с для хранилищ RA-GRS и GRS, 30 Гбит/с для хранилищ LRS и ZRS<sup>4</sup> |

| Максимальная скорость исходящих данных<sup>3</sup> на одну учетную запись хранения (регионы за пределами США) | 10 Гбит/с для хранилищ RA-GRS и GRS, 15 Гбит/с для хранилищ LRS и ZRS<sup>4</sup> |

<sup>1</sup>Сюда входят учетные записи хранилища уровней "Стандартный" и "Премиум". Если требуется более 200 учетных записей хранения в заданном регионе, обратитесь с запросом в [службу поддержки Azure](https://azure.microsoft.com/support/faq/). Команда разработчиков службы хранилища Azure рассмотрит вашу аргументацию и может утвердить до 250 учетных записей хранения для заданного региона. 

<sup>2</sup> Если требуются расширенные ограничения для учетной записи хранения, обратитесь в службу [поддержки Azure](https://azure.microsoft.com/support/faq/). После рассмотрения запроса группа разработчиков службы хранилища Azure может расширить ограничения в индивидуальном порядке. Учетные записи хранения для хранилища больших двоичных объектов и общего назначения поддерживают по запросу увеличенную емкость, входящий и исходящий трафик и частоту запросов. Новые максимальные значения для учетных записей хранилища больших двоичных объектов см. в [объявлении об учетных записей хранения большего размера и большего масштаба](https://azure.microsoft.com/blog/announcing-larger-higher-scale-storage-accounts/).

<sup>3</sup> Ограничения действуют только для исходящих и входящих данных учетной записи. Под *входящими данными* подразумеваются все данные (запросы), отправляемые в учетную запись хранения. *исходящими данными* понимаются все данные (отклики), получаемые из учетной записи хранения.  

<sup>4</sup> Параметры избыточности службы хранилища Azure:
* **RA-GRS.** Геоизбыточное хранилище с доступом для чтения. Если параметр RA-GRS включен, входящие целевые объекты для вторичного расположения идентичны целевым объектам для первичного расположения.
* **GRS.** Геоизбыточное хранилище. 
* **ZRS.** Хранилище, избыточное в пределах зоны.
* **LRS.** Локально избыточное хранилище.
