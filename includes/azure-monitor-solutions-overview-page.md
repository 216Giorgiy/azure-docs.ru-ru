Данные, собранные при помощи решения для мониторинга, доступны на странице **Обзор** в Azure Monitor на портале Azure. Откройте ее в меню **Azure Monitor**, щелкнув **Больше** в разделе **Аналитика**. Каждое решение представлено в виде плитки. Чтобы получить дополнительные сведения о данных, собранных при помощи этого решения, щелкните плитку.
