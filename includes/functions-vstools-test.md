Основные инструменты службы Функции Azure позволяют запускать проекты функций Azure на локальном компьютере разработчика. Вам будет предложено установить эти инструменты при первом запуске функции из Visual Studio.  

Чтобы протестировать функцию, щелкните правой кнопкой мыши узел проекта в **обозревателе решений** и выберите **Отладка** > **Запустить новый экземпляр**. Если будет предложено, примите запрос от Visual Studio на скачивание и установку основных инструментов службы Функции Azure (CLI).  Кроме того, вам может понадобится включить исключение брандмауэра, чтобы разрешить инструментам обрабатывать HTTP-запросы.
