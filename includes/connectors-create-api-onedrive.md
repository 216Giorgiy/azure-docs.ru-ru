### Предварительные требования
- Учетная запись [OneDrive](http://OneDrive.com) 

Чтобы использовать учетную запись OneDrive в приложении логики, вы должны авторизовать приложение логики для подключения к вашей учетной записи OneDrive. К счастью, это можно легко сделать из приложения логики на портале Azure.

Ниже приведены инструкции по авторизации приложения логики для подключения к учетной записи OneDrive.

1. Чтобы создать подключение к OneDrive, в конструкторе приложений логики в раскрывающемся списке выберите параметр **Показать управляемые API Microsoft**, а затем введите в поле поиска *OneDrive*. Выберите триггер или действие: ![](./media/connectors-create-api-onedrive/onedrive-1.png)
2. Если вы ранее не создавали подключения к OneDrive, вам будет предложено ввести учетные данные OneDrive. Эти учетные данные будут использоваться для авторизации приложения логики, чтобы оно могло подключиться и получить доступ к данным вашей учетной записи OneDrive: ![](./media/connectors-create-api-onedrive/onedrive-2.png)
4. Укажите имя пользователя и пароль своей учетной записи OneDrive, чтобы авторизовать приложение логики: ![](./media/connectors-create-api-onedrive/onedrive-3.png)   
3. Разрешите подключение к OneDrive: ![](./media/connectors-create-api-onedrive/onedrive-4.png)   
5. Подключение создано, и теперь вы можете перейти к другим действиям в приложении логики. ![](./media/connectors-create-api-onedrive/onedrive-5.png)
