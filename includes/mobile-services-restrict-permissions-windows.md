
1. В обозревателе сервера Visual Studio последовательно разверните узел **Azure**, раздел **Мобильные службы** и свою мобильную службу.

2. Щелкните правой кнопкой мыши таблицу **TodoItem**, выберите пункт **Изменить разрешения**, установите для всех разрешений значение **Только прошедшие проверку пользователи** и нажмите кнопку **Применить**. Благодаря этому только прошедший проверку пользователь сможет выполнять все операции с таблицей _TodoItem_.

3. Щелкните правой кнопкой мыши проект клиентского приложения, а затем последовательно выберите пункты **Отладка** и **Запустить новый экземпляр**. Убедитесь, что после запуска приложения возникает необработанное исключение с кодом состояния 401 ("Отсутствует авторизация").

	Это происходит, потому что приложение пытается получить доступ к мобильным службам как пользователь, не прошедший проверку подлинности, а таблица *TodoItem* теперь требует выполнения проверки подлинности.

Далее приложение будет обновлено таким образом, что оно станет производить аутентификацию учетных данных пользователей, прежде чем запрашивать ресурсы из мобильной службы.
