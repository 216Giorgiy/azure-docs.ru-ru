[Пакет SDK для службы "Речь"](~/articles/cognitive-services/speech-service/speech-sdk.md) Microsoft Cognitive Services предоставляет способ для распознавания **намерений из речи** и поддерживается в [службе "Распознавание речи" (LUIS)](https://www.luis.ai/home) Cognitive Services.

1. Создайте конфигурацию речи с помощью ключа подписки LUIS и [региона](~/articles/cognitive-services/speech-service/regions.md#intent-recognition) в качестве параметров. В документации по службе ключ подписки LUIS называется **ключом конечной точки**. Ключ разработки LUIS использовать нельзя. (См. примечание далее в этом разделе.)

1. Создайте распознаватель намерений на основе конфигурации речи. Предоставьте конфигурацию звука, если требуется выполнять распознавание из источника, отличного от микрофона по умолчанию (например, звукового потока или аудиофайла).

1. Получите модель распознавания речи, основанную на вашем идентификаторе **AppId**. Добавьте необходимые намерения.

1. При необходимости свяжите события для асинхронной операции. Распознаватель затем вызывает обработчики событий при получении промежуточного и конечного результатов (включая намерения). Если не привязать события, приложение получает только конечный результат расшифровки.

1. Запустите распознавание намерений говорящего. Для краткого распознавания, например распознавания команды или запроса, используйте метод `RecognizeOnceAsync()`. Этот метод возвращает первое распознанное высказывание. Для длительного распознавания используйте метод `StartContinuousRecognitionAsync()`. Привяжите события к результатам асинхронного распознавания.

Для сценариев распознавания намерений, которые используют пакет SDK для службы "Речь", ознакомьтесь со следующими фрагментами кода. Замените значения в примере своим собственным ключом подписки LUIS (ключом конечной точки), [регионом подписки](~/articles/cognitive-services/speech-service/regions.md#intent-recognition) и идентификатором **AppId** для модели намерений.

> [!NOTE]
> В отличие от других служб, поддерживаемых пакетом SDK для службы "Речь", распознавание намерений требует наличия конкретного ключа подписки (ключа конечной точки LUIS). Сведения о технологии распознавания намерений см. на [веб-сайте LUIS](https://www.luis.ai). Сведения о том, как получить **ключ конечной точки**, см. в разделе [Создание ключа конечной точки LUIS](https://docs.microsoft.com/azure/cognitive-services/LUIS/luis-how-to-azure-subscription#create-luis-endpoint-key).
