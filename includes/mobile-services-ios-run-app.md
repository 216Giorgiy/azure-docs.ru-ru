


Последний раздел учебника — построение и выполнение нового приложения.

1. Перейдите в расположение, где были сохранены сжатые файлы проекта, извлеките файлы на компьютере и откройте файл проекта с помощью Xcode.

   	![](./media/mobile-services-ios-run-app/mobile-xcode-project.png)

2. Нажмите кнопку **Запуск**, чтобы создать проект и запустить приложение в эмуляторе iPhone, используемом по умолчанию для этого проекта.

3. В приложении введите содержательный текст, например _Работа с учебником_, и щелкните значок "плюс" (**+**).

   	![](./media/mobile-services-ios-run-app/mobile-quickstart-startup-ios.png)

   	Запрос POST отправляется в новую мобильную службу, размещенную в Azure. Данные из запроса вставляются в таблицу TodoItem. Элементы, хранящиеся в таблице, возвращаются мобильной службой, а данные отображаются в списке.

	>[WACOM.NOTE]Код, который обращается к вашей мобильной службе для запроса и вставки данных, можно просмотреть в файле TodoService.m.</p>
