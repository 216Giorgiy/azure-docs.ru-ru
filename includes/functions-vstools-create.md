С помощью шаблона проекта функций Azure в Visual Studio можно создать проект, который затем можно опубликовать в приложении-функции в Azure. позволяющее группировать функции в логические единицы и упростить развертывание и совместное использование ресурсов, а также управление ими.   

Щелкните правой кнопкой мыши узел проекта в **обозревателе решений**, а затем щелкните **Добавить** > **Новый элемент**. Щелкните **Функция Azure** в диалоговом окне.

В диалоговом окне **Новый проект** разверните узел **Visual C#** > **Облако**, выберите **Функции Azure**, укажите **имя** проекта и нажмите кнопку **ОК**. Имя приложения-функции должно быть допустимым в качестве пространства имен C#, поэтому не используйте символы подчеркивания, дефисы и другие символы, не являющиеся буквенно-цифровыми.
