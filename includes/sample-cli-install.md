

Чтобы запустить этот пример, установите последнюю версию [Azure CLI 2.0](https://docs.microsoft.com/cli/azure/install-azure-cli). Перед началом выполните команду `az login`, чтобы создать подключение к Azure.

Этот пример работает в оболочке Bash. Сведения о вариантах выполнения скриптов Azure CLI в клиенте Windows см. в статье [Установка Azure CLI 2.0 в Windows](/cli/azure/install-azure-cli-windows).
