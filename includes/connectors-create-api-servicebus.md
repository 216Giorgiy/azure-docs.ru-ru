### Предварительные требования

- Учетная запись [служебной шины](https://azure.microsoft.com/services/service-bus/)  


Перед использованием учетной записи служебной шины в приложении логики необходимо авторизовать приложение логики для подключения к учетной записи служебной шины. Это легко сделать в приложении логики на портале Azure.

Ниже приведены указания по авторизации приложения логики для подключения к учетной записи служебной шины.
1. Чтобы создать подключение к служебной шине, в конструкторе приложений логики в раскрывающемся списке выберите параметр **Показать управляемые API Microsoft**, а затем введите в поле поиска *Служебная шина*. Выберите триггер или действие, которые хотите использовать: ![](./media/connectors-create-api-servicebus/servicebus-1.png)  
2. Если вы ранее не создавали подключения к служебной шине, вам будет предложено ввести учетные данные служебной шины. Эти учетные данные будут использоваться для авторизации приложения логики, чтобы оно могло подключиться и получить доступ к данным вашей учетной записи служебной шины. ![](./media/connectors-create-api-servicebus/servicebus-2.png)  
3. Подключение создано, и теперь вы можете перейти к другим действиям в приложении логики: ![](./media/connectors-create-api-servicebus/servicebus-3.png)
