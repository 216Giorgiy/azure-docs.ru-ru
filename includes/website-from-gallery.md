Azure Marketplace предоставляет широкий спектр популярных веб-приложений, разработанных нами, сторонними компаниями и группами разработки программного обеспечения с открытым исходным кодом. Веб-приложения, созданные в Azure Marketplace, не требуют наличия специального программного обеспечения. Все, что вам нужно, - это браузер для подключения к [порталу Azure](http://go.microsoft.com/fwlink/?LinkId=529715). 

С помощью этого учебника вы научитесь:

- создавать новые веб-приложения в Azure Marketplace;

- развертывать веб-приложения с помощью портала Azure.
 
Вы создадите блог WordPress, который использует шаблон по умолчанию. На иллюстрации ниже показано готовое приложение:


![Wordpress blog][13]

>[AZURE.NOTE] Если вы хотите начать работу со службой приложений Azure еще до создания учетной записи Azure, перейдите на страницу [ознакомительной версии службы приложений](http://go.microsoft.com/fwlink/?LinkId=523751). Там вы сможете сразу создать свое первое веб-приложение, правда срок его службы будет ограничен. Никаких кредитных карт и обязательств.

## Создание веб-приложения на портале

1. Войдите на портал Azure.

2. Откройте Azure Marketplace. Для этого щелкните значок **Marketplace** или в левом нижнем углу панели мониторинга щелкните **Создать**, выберите **Интернет и мобильные устройства**, а затем внизу окна - **Azure Marketplace**.
	
	<!--todo:![Create New][5]-->
	
3. В окне Marketplace выберите **Веб-приложения**.

4. Щелкните значок **Scalable WordPress**.

	<!--todo:![WordPress from list][7]-->
	
5. Прочитав описание приложения WordPress, нажмите кнопку **Создать**.

6. Поочередно выберите вкладки **ВЕБ-ПРИЛОЖЕНИЕ**, **БАЗА ДАННЫХ** и **ПОДПИСКА** и на каждой из них заполните обязательные поля:
	
- Введите URL-адрес по своему выбору.	
- Выберите ближайший к вам регион.

	<!--todo:![configure your app][8]-->

7. Определив параметры веб-приложения, нажмите кнопку **Создать**.

## Запуск веб-приложения WordPress и управление им
	
1. В колонке **Веб-приложения** щелкните свое новое веб-приложение. Откроется панель мониторинга.

	<!--todo:![launch dashboard][10]-->

2. На странице **Основные сведения** выберите ссылку в столбце **URL-адрес**. Откроется страница приветствия для вашего веб-приложения.

	<!--todo:![site URL][11]--> 

3. Если вы еще не установили WordPress, введите необходимые сведения о конфигурации WordPress и щелкните **Установить WordPress**. Настройка завершится, и откроется страница входа в веб-приложение.

	<!--todo:![login to WordPress][12]-->

4. Щелкните **Вход** и введите свои учетные данные.  
	<!--todo:screenshot-->

5. Откроется веб-приложение WordPress, похожее на то, что показано на рисунке ниже.    

	<!--todo:![your WordPress site][13]-->
