Коллекция предоставляет широкий спектр популярных веб-приложений, разработанных корпорацией Майкрософт, сторонними компаниями и группами разработки программного обеспечения с открытым исходным кодом. Веб-приложения, созданные из коллекции, не требуют установки никакого программного обеспечения, кроме браузера, используемого для подключения к порталу управления Azure.

В этом учебнике вы узнаете следующее:

-   Как создать новый сайт с помощью коллекции.

-   Как развернуть сайт с помощью портала Azure.

Вы создадите блог WordPress, который использует шаблон по умолчанию. На следующем рисунке показано завершенное приложение:

## Создание веб-сайта на портале

1.  Войдите на [портал управления Azure][портал управления Azure].

2.  Щелкните значок **Создать** в нижнем левом углу панели мониторинга.

    ![Создать][Создать]

3.  Щелкните значок **Веб-сайт** и выберите **Из коллекции**.

    ![Создание из коллекции][Создание из коллекции]

4.  Найдите и щелкните значок WordPress в списке и нажмите кнопку **Далее**.

    ![WordPress из списка][WordPress из списка]

5.  На странице **Настройка приложения** введите или выберите значения для всех полей:

-   Введите имя URL-адреса по вашему выбору.
-   Установите флажок **Создать новую базу данных MySQL** в поле **База данных**.
-   Выберите ближайший к вам регион.

    ![настройка приложения][настройка приложения]

1.  Нажмите кнопку **Далее**.

2.  На странице **Создать новую базу данных** можно указать имя для новой базы данных MySQL или использовать имя по умолчанию. Выберите ближайший к вам регион в качестве места размещения. Поставьте флажок в нижней части экрана, чтобы согласиться с условиями использования ClearDB для вашей размещенной базы данных MySQL. Затем установите флажок для завершения создания сайта.

    ![создание базы данных][создание базы данных]

После нажатия кнопки **Готово** Azure инициирует операции создания и развертывания. Во время создания и развертывания веб-сайта состояние этих операций отображается в нижней части страницы веб-сайтов. После выполнения всех операций, когда сайт успешно развернут, появляется конечное сообщение о состоянии.

## Запуск сайта WordPress и управление им

1.  Щелкните новый сайт на странице **Веб-сайты**, чтобы открыть панель мониторинга для сайта.

    ![запуск панели мониторинга][запуск панели мониторинга]

2.  На странице управления **Панель мониторинга** прокрутите страницу вниз и выберите ссылку в левой части окна в разделе **URL-адрес сайта**, чтобы открыть начальную страницу сайта.

    ![URL-адрес сайта][URL-адрес сайта]

3.  Введите необходимые сведения о конфигурации, запрошенные WordPress, и щелкните **Установить WordPress**, чтобы завершить настройку и открыть страницу входа на веб-сайт.

    ![вход на WordPress][вход на WordPress]

4.  Выполните вход на новый веб-сайт WordPress, введя имя пользователя и пароль, указанные вами на странице **приветствия**.

5.  У вас будет новый сайт WordPress, схожий с сайтом, показанным ниже.

    ![ваш сайт WordPress][Блог WordPress]
