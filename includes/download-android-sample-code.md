Данное руководство построено на приложении Android **GetStartedWithData**. Его пользовательский интерфейс совпадает с интерфейсом приложений Android в проекте быстрого запуска мобильных служб. Отличие заключается в том, что элементы, добавляемые в список, хранятся локально в памяти. Поэтому будет добавлен код, необходимый для сохранения данных в хранилище.


1. Загрузите репозиторий mobile-services-samples с ресурса gitHub, щелкнув <a href="https://github.com/RickSaling/mobile-services-samples/tree/futures" target="blank">эту ссылку</a> и выбрав **Download ZIP** (Загрузить ZIP-файл).

3. Распакуйте загруженный файл и запишите его расположение.

2. В Eclipse щелкните **Файл**, затем щелкните **Импорт**, разверните элемент **Android**, щелкните **Существующий код Android в рабочую область**, а затем нажмите кнопку **Далее**. 

 	![](./media/download-android-sample-code/mobile-services-import-android-workspace.png)

3. Нажмите кнопку **Обзор** и укажите расположение распакованного репозитория gitHub. Перейдите в каталог GetStartedWithData и выберите каталог Android. Нажмите **ОК** и убедитесь, что для проекта GetStartedWithData установлен флажок. Если необходимо скопировать проект в рабочую область (рекомендуется), установите флажок **Скопировать проекты в рабочую область**. Наконец, нажмите кнопку **Готово**. 

 	![](./media/download-android-sample-code/mobile-services-import-android-project.png)

	При этом файлы проекта импортируются в текущую рабочую область.
 
>[AZURE.NOTE] Если вам нужен образец файла, используемого в предыдущей версии пакета SDK для мобильных служб для Android, его можно найти [здесь][GitHub].
