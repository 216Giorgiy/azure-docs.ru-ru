Этот учебник основан на [приложении GetStartedWithData][приложении GetStartedWithData], которое является приложением Android. Пользовательский интерфейс этого приложения совпадает с интерфейсом приложения, созданного при быстром запуске мобильных служб Android. Отличие заключается в том, что добавленные элементы хранятся локально в памяти. Будет добавлен код, необходимый для хранения данных в хранилище.

1.  Скачайте пример приложения `GetStartedWithData` и распакуйте файлы на свой компьютер.

2.  В Eclipse нажмите кнопку **Файл**, затем щелкните **Импорт**, разверните **Android**, щелкните пункт **Существующий код Android в рабочую область**, а затем нажмите кнопку **Далее.**

    ![][0]

3.  Нажмите кнопку **Обзор**, перейдите в расположение развернутых файлов проекта, нажмите кнопку **ОК** и убедитесь, что отмечен проект TodoActivity. Если необходимо скопировать проект в рабочую область, установите флажок **Скопировать проекты в рабочую область**. Наконец, нажмите кнопку **Готово**.

    ![][1]

    При этом файлы проекта импортируются в текущую рабочую область.
