Количество метрик и событий, используемых в приложении (то есть на ключ инструментирования), ограничено. 

Ограничения зависят от выбранного [ценового плана](https://azure.microsoft.com/pricing/details/application-insights/).

| **Ресурс** | **Ограничение по умолчанию** | **Примечание.**
| --- | --- | --- |
| Общий объем данных в день | 100 ГБ | Можно уменьшить, задав ограничение. Если вам необходимо больше, отправьте письмо по адресу AIDataCap@microsoft.com 
| Бесплатный объем данных (в месяц)<br/> (ценовой план "Базовый") | 1 GB | Дополнительный объем оплачивается погигабайтно.
| Хранение данных | 90 дней | Для [поиска](../articles/application-insights/app-insights-diagnostic-search.md), [аналитики](../articles/application-insights/app-insights-analytics.md) и [обозревателя метрик](../articles/application-insights/app-insights-metrics-explorer.md).
| Хранение подробных результатов [многошагового теста доступности](../articles/application-insights/app-insights-monitor-web-app-availability.md#multi-step-web-tests) | 90 дней | Подробные результаты каждого шага.
| Количество имен [свойств](../articles/application-insights/app-insights-api-custom-events-metrics.md#properties) и [метрик](../articles/application-insights/app-insights-api-custom-events-metrics.md#properties)<sup>2</sup> | 200 | 
| Длина имен свойств и метрик | 150 |
| Длина строки значения свойства | 8192 |
| Количество уникальных значений для свойств<sup>3,4</sup> | 100 | Если > 100 = >, свойство нельзя использовать в качестве фильтра в обозревателе метрик.
| Длина сообщения трассировки и исключения | 10 000 |
| Количество [тестов доступности](../articles/application-insights/app-insights-monitor-web-app-availability.md) для одного приложения  | 10 |

1. Все эти числа относятся к одному ключу инструментирования.
2. Имена метрик определяются в TrackMetric и в параметре измерения других вызовов Track*(). Имена метрик являются глобальными для каждого ключа инструментирования.
3. Свойства можно использовать для фильтрации и группирования, только если на каждое свойство приходится менее 100 уникальных значений. Если количество уникальных значений превышает 100, свойство по-прежнему можно искать, но его больше нельзя использовать для фильтрации или группирования.
4. Максимальное количество уникальных значений для стандартных свойств, таких как Request Name или Page URL, в неделю — 1000. После превышения максимального количества уникальных значений (1000 уникальных значений) дополнительные значения отмечаются как «Другие значения». Исходные значения можно по-прежнему использовать для полнотекстового поиска и фильтрации.
