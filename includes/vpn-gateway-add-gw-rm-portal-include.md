1. На портале перейдите к разделу **Создать**, затем щелкните **Сеть**. Выберите **Шлюз виртуальной сети** из списка.

	![Шлюз](./media/vpn-gateway-add-gw-rm-portal-include/creategw250.png)

2. В колонке **Создание шлюза виртуальной сети** в поле **Имя** укажите имя шлюза. Это имя должно отличаться от имени подсети шлюза. Это имя объекта шлюза.
 
3. Затем выберите виртуальную сеть, в которой вы хотите развернуть этот шлюз. Щелкните стрелку, чтобы открыть колонку **Выбор виртуальной сети**, и выберите виртуальную сеть. Чтобы виртуальная сеть появилась в списке, она должна иметь допустимую подсеть шлюза.

4. Выберите общедоступный IP-адрес. Щелкните стрелку, чтобы открыть колонку **Выбор общедоступного IP-адреса**. Щелкните **Создать**, чтобы открыть колонку **Создание общедоступного IP-адреса**. Введите имя общедоступного IP-адреса. Обратите внимание, что это не запрос IP-адреса. IP-адрес назначается динамически. Скорее, это имя объекта IP-адреса, которому будет назначен адрес. Нажмите кнопку **ОК**, чтобы сохранить изменения.

5. В поле **Тип шлюза** выберите тип шлюза, который указан для конфигурации.

6. В поле **Тип VPN** выберите тип VPN, который указан для конфигурации.

7. Проследите, чтобы в поле **Подписка** отображалась правильная подписка.

8. Группа ресурсов в поле **Группа ресурсов** зависит от выбранной виртуальной сети.

9. Убедитесь, что в поле **Расположение** отображается расположение, в котором находится и группа ресурсов, и виртуальная сеть.

10. Вы можете выбрать параметр **Закрепить на панели мониторинга**, если хотите, чтобы шлюз отображался на панели мониторинга. Щелкните **Создать**, чтобы создать шлюз. Плитка "Развертывание виртуального сетевого шлюза" появится на панели мониторинга. Создание шлюза может занять до 45 минут. Множество процессов происходит в фоновом режиме. Вам понадобится обновить страницу портала, чтобы увидеть готовое состояние.

	
	![Шлюз](./media/vpn-gateway-add-gw-rm-portal-include/deployvnetgw150.png)

11. После создания шлюза вы можете просмотреть назначенный ему IP-адрес в разделе сведений о виртуальной сети на портале. Шлюз будет отображаться как подключенное устройство. Щелкните подключенное устройство (шлюз виртуальной сети), чтобы получить дополнительные сведения.
