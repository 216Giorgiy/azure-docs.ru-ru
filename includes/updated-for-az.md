> [!NOTE]
>
> Эта статья недавно была изменена и теперь имеет сведения о командах к новому модулю Az Azure PowerShell. Он является новым модулем PowerShell для Azure, совместимый с версиями PowerShell 5.x и 6. AzureRM продолжит получать обновления устранения критических проблем, но новые функции будут доступны только в модуле Az.
>
> * Дополнительные сведения о модуле Az с дальнейшей поддержкой модуля AzureRM см. в статье [Знакомство с новым модулем Az для Azure PowerShell](/powershell/azure/new-azureps-module-az).
> * Модуль Az уже доступен на Cloud Shell с автоматически включенной совместимостью с AzureRM.
>   Инструкции по установке модуля Az см. в статье [Установка модуля Azure PowerShell](/powershell/azure/install-az-ps).
> * Чтобы включить команды совместимости с AzureRM, запустите `Enable-AzureRmAlias` после установки.
>   Дополнительные сведения о совместимости см. в статье [Миграция с AzureRM на Az для Azure PowerShell](/powershell/azure/migrate-from-azurerm-to-az).
