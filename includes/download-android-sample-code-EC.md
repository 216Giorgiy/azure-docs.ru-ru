Данный учебник построен на приложении Android **GetStartedWithData**. Его пользовательский интерфейс совпадает с интерфейсом приложений Android в проекте быстрого запуска мобильных служб. Отличие заключается в том, что элементы, добавляемые в список, хранятся локально в памяти. Будет добавлен код, необходимый для хранения данных в хранилище.


1. Загрузите репозиторий `mobile-services-samples`с ресурса gitHub, щелкнув <a href="https://github.com/RickSaling/mobile-services-samples/tree/futures" target="blank">здесь</a>, а затем нажав кнопку **Загрузить ZIP-файл**.

3. Распакуйте загруженный файл и запишите его расположение.

2. В Eclipse нажмите кнопку **Файл**, затем щелкните **Импорт**, разверните **Android**, щелкните пункт **Существующий код Android в рабочую область**, а затем нажмите кнопку **Далее.**

 	![](./media/download-android-sample-code/mobile-services-import-android-workspace.png)

3. Нажмите кнопку **Обзор** и укажите расположение распакованного репозитория gitHub. Перейдите к каталогу `GetStartedWithData` и выберите каталог `Android`. Нажмите кнопку **ОК** и убедитесь, что установлен флажок для проекта `GetStartedWithData`. Если необходимо скопировать проект в рабочую область (рекомендуется), установите флажок **Копировать проекты в рабочую область**. Наконец, нажмите кнопку **Готово**.

 	![](./media/download-android-sample-code/mobile-services-import-android-project.png)

	При этом файлы проекта импортируются в текущую рабочую область.
 
>[AZURE.NOTE]Чтобы получить образец файла, который использовался в предыдущей версии SDK мобильных служб для Android, загрузите его [здесь][GitHub].
