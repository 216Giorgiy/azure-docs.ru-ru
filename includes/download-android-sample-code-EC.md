Этот учебник создан на основе приложения Android **GetStartedWithData**. Его пользовательский интерфейс совпадает с интерфейсом приложений Android в проекте быстрого запуска мобильных служб. Отличие заключается в том, что элементы, добавляемые в список, хранятся локально в памяти. Будет добавлен код, необходимый для хранения данных в хранилище.


1. Скачайте репозиторий `mobile-services-samples` с сайта gitHub, щелкнув <a href="https://github.com/RickSaling/mobile-services-samples/tree/futures" target="blank">здесь</a> и нажав кнопку **Скачать ZIP-файл**.

3. Распакуйте загруженный файл и запишите его расположение.

2. В Eclipse щелкните **Файл**, затем щелкните **Импорт**, разверните элемент **Android**, щелкните **Существующий код Android в рабочую область**, а затем нажмите кнопку **Далее**. 

 	![](./media/download-android-sample-code/mobile-services-import-android-workspace.png)

3. Нажмите кнопку **Обзор** и укажите расположение распакованного репозитория gitHub. Перейдите к каталогу `GetStartedWithData` и выберите каталог `Android`. Нажмите кнопку **OK** и убедитесь, что для проекта `GetStartedWithData` установлен флажок. Если необходимо скопировать проект в рабочую область (рекомендуется), установите флажок **Скопировать проекты в рабочую область**. Наконец, нажмите кнопку **Готово**. 

 	![](./media/download-android-sample-code/mobile-services-import-android-project.png)

	При этом файлы проекта импортируются в текущую рабочую область.
 
>[AZURE.NOTE] Если вам нужен пример файла, используемого в предыдущей версии пакета SDK для мобильных служб для Android, его можно найти [здесь][GitHub].
