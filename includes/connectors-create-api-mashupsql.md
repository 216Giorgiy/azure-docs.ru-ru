### Предварительные требования
- Доступ к базе данных SQL Server

Чтобы использовать учетную запись Mashup SQL в приложении логики, вы должны авторизовать приложение логики для подключения к вашей учетной записи Mashup SQL. К счастью, это можно легко сделать из приложения логики на портале Azure.

Ниже приведены инструкции по авторизации приложения логики для подключения к учетной записи Mashup SQL.

1. Чтобы создать подключение к Mashup SQL, в конструкторе приложений логики в раскрывающемся списке выберите параметр **Показать управляемые API Microsoft**, а затем введите в поле поиска *Mashup SQL*. Выберите триггер или действие: ![Mashup SQL — шаг 1](./media/connectors-create-api-mashupsql/mashupsql-1.png)
2. Если вы ранее не создавали подключения к Mashup SQL, вам будет предложено ввести учетные данные Mashup SQL. Эти учетные данные будут использоваться для авторизации приложения логики, чтобы оно могло подключиться и получить доступ к данным вашей учетной записи Mashup SQL: ![Mashup SQL — шаг 2](./media/connectors-create-api-mashupsql/mashupsql-2.png)
5. Подключение создано, и теперь вы можете перейти к другим действиям в приложении логики. ![Mashup SQL — шаг 3](./media/connectors-create-api-mashupsql/mashupsql-3.png)
