Вы можете создать очереди хранилища Azure с помощью **обозревателя сервера** Visual Studio.

![Большие двоичные объекты обозревателя сервера][Image1]

1. В меню **Вид** выберите **Обозреватель серверов**.
2. В обозревателе сервера разверните узел **Azure** для своей подписки, разверните узел **Служба хранилища** и узел для учетной записи хранения, указанной вами в подключенной службе хранилища Azure.
3. Выберите узел **Очереди** и в контекстном меню выберите пункт **Создать очередь**.
4. Введите имя контейнера и щелкните **ОК**.   

По умолчанию новый контейнер закрытый, и вам необходимо указать ключ доступа к хранилищу, чтобы загрузить BLOB-объекты из этого контейнера. Если вы хотите сделать файлы в контейнере общедоступными, выберите контейнер в **обозревателе сервера** и нажмите клавишу `F4`, чтобы открыть окно **Свойства**. Задайте для параметра **Общий доступ на чтение** значение **Большой двоичный объект**. Любой пользователь в Интернете может видеть большие двоичные объекты в открытом контейнере, но изменить или удалить их можно только при наличии ключа доступа.
