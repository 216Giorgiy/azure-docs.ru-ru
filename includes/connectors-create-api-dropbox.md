### Предварительные требования
- Учетная запись [Dropbox](https://www.Dropbox.com/) 

Чтобы использовать учетную запись Dropbox в приложении логики, вы должны авторизовать приложение логики для подключения к вашей учетной записи Dropbox. К счастью, это можно легко сделать из приложения логики на портале Azure.

Ниже приведены инструкции по авторизации приложения логики для подключения к учетной записи Dropbox.

1. Чтобы создать подключение к Dropbox, в конструкторе приложений логики в раскрывающемся списке выберите параметр **Показать управляемые API Microsoft**, а затем введите в поле поиска *Dropbox*. Выберите триггер или действие: ![Dropbox — шаг 1](./media/connectors-create-api-dropbox/dropbox-1.png)
2. Если вы ранее не создавали подключения к Dropbox, вам будет предложено ввести учетные данные Dropbox. Эти учетные данные будут использоваться для авторизации приложения логики, чтобы оно могло подключиться и получить доступ к данным вашей учетной записи Dropbox: ![Dropbox — шаг 2](./media/connectors-create-api-dropbox/dropbox-2.png)
3. Укажите имя пользователя и пароль своей учетной записи Dropbox, чтобы авторизовать приложение логики: ![Dropbox — шаг 3](./media/connectors-create-api-dropbox/dropbox-3.png)   
4. Авторизуйте приложение логики для использования вашей учетной записи Dropbox. ![Dropbox — шаг 4](./media/connectors-create-api-dropbox/dropbox-4.png)
5. Подключение создано, и теперь вы можете перейти к другим действиям в приложении логики. ![Dropbox — шаг 5](./media/connectors-create-api-dropbox/dropbox-5.png)
