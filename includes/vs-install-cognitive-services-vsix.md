## <a name="install-the-cognitive-services-vsix-extension"></a>Установка расширения VSIX служб Cognitive Services

1. Открыв веб-проект в Visual Studio, выберите вкладку **Подключенные службы**. Эта вкладка расположена на странице приветствия, которая появляется при открытии нового проекта. Если вкладка не отображается, щелкните **Подключенные службы** в своем проекте в обозревателе решений.

   ![Снимок экрана: вкладка "Подключенные службы"](./media/vs-install-cognitive-services-vsix/Connected-Services-Tab.PNG)

1. Прокрутите список до конца и выберите **Find more services** (Найти другие службы).

    ![Снимок экрана: ссылка Find more services (Найти другие службы)](./media/vs-install-cognitive-services-vsix/Find-More-Services.PNG)
 
    Откроется диалоговое окно **Расширения и обновления**.

1. В диалоговом окне **Расширения и обновления** выполните поиск по фразе **Cognitive Services**, а затем скачайте и установите пакет VSIX служб Cognitive Services.

   ![Снимок экрана: диалоговое окно "Расширения и обновления"](./media/vs-install-cognitive-services-vsix/install-cognitive-services-vsix.PNG)

   Для установки расширения необходимо перезапустить интегрированную среду разработки (IDE).

2. Перезапустите Visual Studio. Расширение будет установлено при закрытии Visual Studio и станет доступно при следующем запуске интегрированной среды разработки.
