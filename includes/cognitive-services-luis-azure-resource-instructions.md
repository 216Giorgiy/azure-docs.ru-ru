Если вы не знакомы со службами Cognitive Services или не работали с Azure, создайте временный ключ [здесь](https://azure.microsoft.com/try/cognitive-services/). Если у вас есть учетная запись Azure, создайте новый ресурс на портале. Запросы конечной точки API распознавания речи поддерживают два типа ресурсов Azure: **API распознавания речи** или ресурс **Cognitive Service**. Имя ресурса требуется, когда вы [присваиваете](../articles/cognitive-services/luis/luis-how-to-azure-subscription.md) ключ приложению.
