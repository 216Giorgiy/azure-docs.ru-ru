## <a name="intent-recognition-from-a-file-with-events"></a>Распознавание намерений из файла с помощью событий

В следующем фрагменте кода показано непрерывное распознавание намерений на языке по умолчанию (en-US). Этот код позволяет получить доступ к дополнительной информации, например к промежуточным результатам. Входные данные берутся из звукового файла. Поддерживаемый формат — одноканальный (монофонический) WAV- или PCM-файл с частотой дискретизации 16 кГц.
