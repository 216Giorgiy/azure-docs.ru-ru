
1. Откройте [портал Azure]. Щелкните **Просмотреть все** > **Мобильные приложения** и выберите только что созданную серверную часть приложения. В параметрах мобильного приложения щелкните **Быстрый старт** > **Android**. В разделе **Настройка клиентского приложения** щелкните **Загрузка**. Загрузится полный проект Android для приложения, предварительно настроенный для подключения к серверу. Откройте проект при помощи Android Studio, импортируйте проект (Eclipse ADT, Gradle и т. п.).

2. Нажмите кнопку **Запуск приложения**, чтобы создать проект и запустить приложение в симуляторе Android.

3. В приложении введите содержательный текст, например _Работа с руководством_, и нажмите кнопку "Добавить". Это отправляет запрос POST на ранее развернутый внутренний сервер Azure. Сервер вносит данные из запроса в таблицу TodoItem SQL и возвращает сведения о сохраненных элементах в мобильное приложение. В мобильном приложении эти данные отображаются в списке.
