## <a name="a-namesetupdevenvaset-up-the-development-environment"></a><a name="setupdevenv"></a>Настройка среды разработки
Для начала настройте среду разработки, установив [пакет SDK Azure для Visual Studio 2013][Azure SDK for Visual Studio 2013].

Если у вас не установлена служба Visual Studio, вместе с пакетом SDK будет установлена служба Visual Studio Express for Web.

> [!NOTE]
> В зависимости от того, сколько зависимостей пакета SDK уже имеется на компьютере, установка пакета SDK может занять определенное время, от нескольких минут до получаса или более.
> 
> 

[Azure SDK for Visual Studio 2013]: http://go.microsoft.com/fwlink/?LinkID=324322
