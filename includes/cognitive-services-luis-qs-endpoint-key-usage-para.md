Для удобства в рамках этого краткого руководства используется ключ разработки. Этот ключ предназначен для разработки модели, но позволяет выполнить небольшое количество (1000) запросов к конечной точки. Когда все будет готово для выполнения большего количества запросов к конечной точки в тестовой, промежуточной или рабочей среде, создайте ресурс для **Интеллектуальной службы распознавания речи** на портале Azure и назначьте его приложению LUIS на портале LUIS.
