Дополнительные сведения о дисках в виртуальных машинах Azure см. в разделе [Сведения о дисках виртуальных машин Azure][Сведения о дисках виртуальных машин Azure].

## <span id="attachempty"></span></a>Практическое руководство. Присоединить пустой диск

Подключение пустого диска — это более простой способ добавления диска данных, поскольку Azure автоматически создает VHD-файл и сохраняет его в учетной записи хранения.

1.  Нажмите кнопку **Виртуальные машины**, а затем выберите соответствующую виртуальную машину.

2.  На панели команд нажмите **Присоединить** и выберите **Присоединить пустой диск**.

    ![Присоединить пустой диск][Присоединить пустой диск]

    Откроется диалоговое окно **Присоединение пустого диска**.

    ![Присоединить новый пустой диск][Присоединить новый пустой диск]

3.  В разделе **Имя файла** примите автоматически созданное имя или введите описательное имя. Диск данных, который создается из VHD-файла, всегда использует автоматически созданное имя.

4.  В поле **размер** введите размер диска данных.

5.  Установите флажок, чтобы присоединить пустой диск данных.

    Диск данных отобразится в списке на панели мониторинга виртуальной машины.

    ![Пустой диск данных успешно подключен][Пустой диск данных успешно подключен]

## <span id="attachexisting"></span></a>Практическое руководство. Подключение существующего диска

Для подключения существующего диска требуется VHD-файл в учетной записи хранения. Для загрузки VHD-файла учетной записи хранения используйте командлет [Добавить AzureVhd][Добавить AzureVhd]. После создания и отправки VHD-файла можно подключить его к виртуальной машине.

1.  Нажмите кнопку **Виртуальные машины**, а затем выберите соответствующую виртуальную машину.

2.  На панели команд нажмите кнопку **Присоединить**, а затем выберите **Присоединить диск**.

    ![Присоединение диска данных][Присоединение диска данных]

    Откроется диалоговое окно **Присоединить диск**.

    ![Введите сведения о диске данных][Введите сведения о диске данных]

3.  Выберите диск данных, который требуется присоединить к виртуальной машине.
4.  Установите флажок, чтобы присоединить диск данных к виртуальной машине.

    Диск данных отобразится в списке на панели мониторинга виртуальной машины.

    ![Диск данных успешно подключен][Диск данных успешно подключен]

> [WACOM.NOTE]
>  После добавления диска данных необходимо войти в систему на виртуальной машине и инициализировать диск, чтобы виртуальная машина могла использовать его для хранения данных.
