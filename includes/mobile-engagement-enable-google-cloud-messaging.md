
### <a name="create-a-google-firebase-project-with-api-key"></a>Создание проекта Google Firebase с ключом API
> [!NOTE]
> Для выполнения этой процедуры необходима учетная запись Google с проверенным электронным адресом. Чтобы создать учетную запись Google, перейдите по ссылке <a href="http://go.microsoft.com/fwlink/p/?LinkId=268302" target="_blank">accounts.google.com</a>.
> 
> 

1. Перейдите к [консоли Firebase](https://console.firebase.google.com/) и войдите в нее с помощью учетной записи Google.
2. Нажмите кнопку **Создать новый проект**. Вы также можете щелкнуть **или импортировать проект Google**, чтобы импортировать имеющийся проект. 
3. Если вы выбрали создание нового проекта, укажите **название проекта** и выберите **страну или регион**.
4. На странице проекта нажмите кнопку **параметров** (значок шестеренки в верхнем левом углу). В появившемся меню щелкните **Настройки проекта**.  
5. В верхней части щелкните **Cloud Messaging**. 
6. Запишите **ключ сервера** и **идентификатор отправителя**, которые понадобятся далее в *файле манифеста Android*.
