
Выполните следующие действия, чтобы присоединить диск данных.

1. На [портале управления Azure)][AzurePreviewPortal] щелкните **Виртуальные машины**, а затем выберите только что созданную виртуальную машину (**testwinvm**).

2. На панели команд нажмите кнопку **Присоединить**, а затем — **Присоединить пустой диск**.
	
	Откроется диалоговое окно **Присоединение пустого диска к виртуальной машине**.


3. Значения **Имя виртуальной машины**, **Расположение хранилища** и **Имя файла** уже определены. Достаточно ввести размер диска. Введите **5** в поле **Размер**.

	![Присоединить пустой диск][Image2]

	**Примечание.** Все диски создаются из VHD-файла в хранилище Azure. Можно ввести имя для добавляемого в хранилище VHD-файла, однако Azure создаст имя диска автоматически.

4. Установите флажок, чтобы присоединить диск данных к виртуальной машине.

5. Щелкните имя виртуальной машины для отображения панели мониторинга. Это позволит убедиться, что диск данных был успешно присоединен к виртуальной машине.

	Число дисков для данной виртуальной машины теперь составляет 2. Присоединенный диск отображается в таблице "Диски".

	![Присоединить пустой диск][Image3]

	После присоединения диска данных к виртуальной машине диск отключен и не инициализирован. Чтобы использовать его для хранения данных, необходимо предварительно войти в систему на виртуальной машине и инициализировать этот диск.

##Подключитесь к виртуальной машине с использованием удаленного рабочего стола и выполните программу установки
1. После подготовки виртуальной машины на портале управления щелкните **Виртуальные машины**, а затем щелкните новую виртуальную машину. Отображаются сведения о вашей виртуальной машине.	

2. В нижней части страницы щелкните **Подключиться**. Откройте RPD-файл с использованием программы "Удаленный рабочий стол Windows" (*%windir%\system32\mstsc.exe*).	

3. В диалоговом окне **Безопасность Windows** введите пароль для учетной записи **Администратор**. (Может быть выведен запрос на подтверждение учетных данных виртуальной машины.) При первом входе в эту виртуальную машину может возникнуть необходимость дождаться завершения нескольких процессов, включая настройку рабочего стола, обновления Windows и выполнение задач начальной настройки Windows. После подключения к виртуальной машине с помощью удаленного рабочего стола Windows виртуальная машина будет работать аналогично любому другому компьютеру.

4. После входа в систему на виртуальной машине откройте **Диспетчер сервера** В левой области разверните узел **Хранилище**, а затем щелкните **Управление дисками**.

	![Диспетчер сервера][Image4]

5. Откроется окно **Инициализировать диск**.  Нажмите кнопку **ОК**.

	![Инициализировать диск][Image5.0]

6. Щелкните правой кнопкой мыши область выделения места на диске 2, нажмите кнопку **Создать простой том**, а затем завершите работу мастера со значениями по умолчанию.

	![Создать простой том][Image6]

	Диск подключен и готов для использования с новой буквой диска.

	![Инициализация выполнена успешно][Image7]
