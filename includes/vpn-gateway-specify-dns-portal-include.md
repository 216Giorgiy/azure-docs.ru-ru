1. На странице **Параметры** виртуальной сети щелкните **DNS-серверы**, чтобы открыть страницу **DNS-серверы**.

    ![Указание DNS-сервера](./media/vpn-gateway-specify-dns-portal-include/specify-dns-server.png "Specify a DNS Server")

   - **DNS-серверы**. Выберите **Пользовательский**.
   - **Добавить DNS-сервер**. Введите IP-адрес DNS-сервера, который нужно использовать для разрешения имен.

2. Добавив DNS-серверы, нажмите кнопку **Сохранить** в верхней части страницы.
