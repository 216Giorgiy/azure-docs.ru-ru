Подписаться на размещенную в Azure полностью управляемую базу данных MongoDB можно в магазине Azure. Для этого выполните следующие действия.

1. Выполните вход на портал управления Azure.
2. Нажмите кнопку **Создать**. ![Создать][button-new]
3. Выберите **Магазин**. ![Магазин][button-store]
4. Выберите MongoLab. Вы найдете нас в категории **Службы приложений**, а также в категории **Все**. ![MongoLab][entry-mongolab]
5. Нажмите кнопку **Далее**. ![Далее][button-next] Отобразится запись магазина MongoLab. ![NewMongoLab][screen-newmongolab]
6. Выберите желаемый вариант **Подписки**.
7. Введите **Имя** для вашей базы данных. Имя может содержать только алфавитно-цифровые символы, тире, точки и знаки подчеркивания. MongoLab также требует, что это имя было уникальным, поэтому, возможно, потребуется повторно отправить запрос, если данное имя уже занято.
8. Выберите нужный элемент **Регион**
9. Нажмите кнопку **Далее**. ![Далее][button-next]
10. Просмотрите сведения о вашей покупке в магазине и нажмите кнопку **Купить** для подтверждения. ![Далее][button-purchase]  
11. Кнопка отображения состояния на панели инструментов предоставляет сведения о состоянии подготовки. ![ProgressButton][button-progress] После завершения подготовки появляется сообщение об успешном выполнении. ![SuccessMessage][message-success]

Поздравляем! MongoLab только что подготовил базу данных MongoDB в выбранном вами регионе Azure. Теперь у вас есть доступ к пользовательскому интерфейсу управления и круглосуточной поддержке без перерывов и выходных.
