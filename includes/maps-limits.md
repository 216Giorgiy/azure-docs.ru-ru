Ниже приведено ограничение использования для службы Azure Maps **ценовой категории S0**. Ограничение использования зависит от ценовой категории. 

| Ресурс                              | Ограничение для ценовой категории S0 |
|---------------------------------------|:---------------------:|
| Максимальная частота запросов для подписки |   50 запросов в секунду  |


Ниже указан максимальный объем данных для службы данных Azure Maps. Служба данных Azure Maps доступна только в **ценовой категории S1**.

| Ресурс                              | Ограничение |
|---------------------------------------|:---------------------:|
| Максимальный объем данных |   50 MB  |

Дополнительные сведения о ценовых категориях Azure Maps см. на [странице цен на эту службу](https://azure.microsoft.com/pricing/details/azure-maps/).
