> [!IMPORTANT]
> Для выполнения действий в этой статье необходим классический Azure CLI. Это более старая версия Azure CLI, предназначенная только для классической модели развертывания Azure.
>
> Классический CLI можно установить параллельно с современным Azure CLI, но мы рекомендуем использовать Azure CLI для всех новых сценариев и развертываний.
> Чтобы установить классический интерфейс командной строки, обратитесь к статье [Установка классического интерфейса командной строки Azure (Azure Classic CLI)](/cli/azure/install-classic-cli), а для установки текущей версии интерфейса командной строки обратитесь к статье [Установка Azure CLI](/cli/azure/install-azure-cli).
