## <a name="create-an-iot-hub"></a>Создание центра IoT
Создайте центр Интернета вещей, к которому будет подключено приложение, имитирующее устройство. Ниже показано, как это сделать с помощью портала Azure.

[!INCLUDE [iot-hub-create-hub](iot-hub-create-hub.md)]

После создания Центра Интернета вещей найдите важные данные для подключения к нему устройств и приложений. 

1. Когда Центр Интернета вещей будет создан, щелкните на портале Azure новую плитку для центра, чтобы открыть соответствующее окно свойств. Запишите **имя узла**, а затем щелкните **Политики общего доступа**.
   
    ![Новое окно центра Интернета вещей][4]
1. В колонке **Политики общего доступа** выберите политику **iothubowner**, а затем скопируйте и запишите строку подключения в окне **iothubowner**. Дополнительные сведения см. в разделе [Контроль доступа][lnk-access-control] в руководстве разработчика для Центра Интернета вещей Azure.
   
    ![Политики общего доступа][5]
