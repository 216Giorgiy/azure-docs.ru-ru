В службе Azure Backup используется два типа хранилища: резервное хранилище и хранилище служб восстановления. Сначала появилось хранилище архивации. Затем было создано хранилище служб восстановления для поддержки расширенных развертываний Resource Manager. Мы советуем использовать развертывания Resource Manager, если нет необходимости в классическом развертывании. К окончанию 2017 г. все резервные хранилища были преобразованы в хранилища служб восстановления.

> [!NOTE]
> Резервные хранилища не могли защитить решения, развернутые с помощью Resource Manager. Но вы можете защитить серверы и виртуальные машины, развернутые с помощью классической модели, используя хранилище служб восстановления.
