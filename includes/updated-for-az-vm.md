> [!NOTE]
>
> Эта статья была изменена, и теперь в ней находятся сведения о новом [модуле Az Azure PowerShell](/powershell/azure/new-azureps-module-az). AzureRM продолжит получать обновления устранения критических проблем, но новые функции будут доступны только в модуле Az.
> * Модуль Az уже доступен на Cloud Shell с включенной совместимостью с AzureRM.
> * Чтобы установить модуль Az локально, см. статью [Установка модуля Azure PowerShell](/powershell/azure/install-az-ps).
> * Если модуль Az был установлен локально, можете включить совместимость AzureRM путем запуска командлета `Enable-AzureRmAlias`.
