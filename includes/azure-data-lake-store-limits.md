Хранилище озера данных Azure — это крупномасштабный репозиторий корпоративного уровня для рабочих нагрузок анализа больших данных. Data Lake Store позволяет сохранять данные с любым размером, типом и скоростью приема в одном месте для эксплуатационной и исследовательской аналитики. Объем данных, которые можно хранить в учетной записи Data Lake Store, не ограничен.

| **Ресурс** | **Ограничение по умолчанию** | **Комментарии** |
| --- | --- | --- |
| Максимальное количество учетных записей Data Lake Store на подписку в регионе. |10 | Обратитесь в службу поддержки, чтобы запросить увеличение этого максимального значения. |
| Максимальное число списков ACL для доступа на файл или папку |32 | Это строгое ограничение. Используйте группы для управления доступом с меньшим количеством записей |
| Максимальное число списков ACL по умолчанию для доступа на файл или папку |32 | Это строгое ограничение. Используйте группы для управления доступом с меньшим количеством записей |
