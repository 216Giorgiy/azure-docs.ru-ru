После регистрации приложения в APNS и настройки проекта необходимо настроить мобильную службу для интеграции с APNS.

1.  В Keychain Access щелкните правой кнопкой мыши новый сертификат, нажмите кнопку **Экспорт**, назовите файл QuickstartPusher, выберите формат **.p12**, а затем щелкните **Сохранить**.

    ![][ ]

Запишите имя файла и расположение экспортируемого сертификата.

> [WACOM.NOTE] В этом учебнике описывается создание файла QuickstartPusher.p12. Имя файла и расположение могут отличаться.

1.  Выполните вход на [портал управления Azure][], щелкните элемент **Мобильные службы**, а затем щелкните свое приложение.

    ![][1]

2.  Перейдите на вкладку **Push** (Push-уведомления) и щелкните **Upload** (Отправить).

    ![][2]

    Откроется диалоговое окно отправки сертификата.

3.  Щелкните **File** (Файл), выберите файл экспортированного сертификата QuickstartPusher.p12, введите пароль в поле **Password** (Пароль), убедитесь, что выбран правильный режим в поле **Mode** (Режим), щелкните значок галочки, а затем нажмите кнопку **Save** (Сохранить).

    ![][3]

    > [WACOM.NOTE] В этом учебнике используются сертификаты разработки.

Ваша мобильная служба теперь настроена для работы с APNS.
