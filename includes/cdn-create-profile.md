**Создание нового профиля сети CDN**

1. На [портале Azure](https://portal.azure.com) слева вверху выберите **Создать ресурс**.
    
2. В колонке **создания** выберите **Интернет и мобильные устройства**, а затем **CDN**.
   
    Появится колонка **Профиль CDN**.
   
    ![Новый профиль сети CDN](./media/cdn-create-profile/new-cdn-profile-include.png)
3. В поле **Имя** введите уникальное имя профиля CDN.
    
4. В поле **Подписка** выберите нужную подписку для этого профиля CDN.
   
5. В поле **Группа ресурсов** выберите или создайте группу ресурсов. Дополнительные сведения о группах ресурсов см. в статье [Общие сведения об Azure Resource Manager](../articles/azure-resource-manager/resource-group-overview.md#resource-groups).
    
6. Для **расположения группы ресурсов** выберите расположение Azure, в котором будут хранится сведения о профиле CDN. Это расположение не влияет на расположение конечных точек сети CDN.
    
7. Выберите категорию в поле **Ценовая категория**. Сравнение функций, доступных в каждой ценовой категории, см. в разделе [Компоненты Azure CDN](../articles/cdn/cdn-overview.md#azure-cdn-features).
   
    ![Выбор ценовой категории CDN](./media/cdn-create-profile/cdn-choose-sku-include.png)

8. При необходимости установите флажок **Создать конечную точку CDN**, чтобы отобразить параметры для создания конечной точки во время создания профиля. Дополнительные сведения см. в разделе [Создание новой конечной точки сети CDN](../articles/cdn/cdn-create-new-endpoint.md#create-a-new-cdn-endpoint).
   
9. Если нужно сохранить созданный профиль на панели мониторинга, чтобы его было проще найти, установите флажок **Закрепить на панели мониторинга**.
    
10. Нажмите кнопку **Создать**, чтобы создать профиль.
