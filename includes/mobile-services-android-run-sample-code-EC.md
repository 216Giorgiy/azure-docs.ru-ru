1. В обозревателе решений разверните **GetStartedWithData**, **src** и **.com.example.GetStartedWithData**, затем изучите файл ToDoActivity.java.

   	![](./media/download-android-sample-code/mobile-eclipse-project.png)

   	Обратите внимание на комментарии `//TODO`, указывающие на действия, которые необходимо выполнить, чтобы приложение работало с вашей мобильной службой.

2. В меню **Запуск** щелкните кнопку **Запуск**, затем выберите **Приложение Android** для запуска проекта.

	> [AZURE.NOTE]Можно запустить этот проект с помощью Android-телефона или с помощью эмулятора Android. Чтобы работать с Android-телефоном, необходимо загрузить USB-драйвер для конкретного телефона.
	>
	> Чтобы запустить проект в эмуляторе Android, необходимо определить как минимум одно виртуальное устройство на платформе Android (AVD). Для создания этих устройств и управления ими используйте диспетчер AVD.

3. В приложении введите содержательный текст, например _Завершение работы с учебником_, и нажмите кнопку **Добавить**.

   	![](./media/download-android-sample-code/mobile-quickstart-startup-android.png)

   	Обратите внимание, что сохраненный текст хранится в памяти и отображается в списке ниже.
