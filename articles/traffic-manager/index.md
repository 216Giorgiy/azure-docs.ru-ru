#<a name="traffic-manager-documentation"></a>Документация по диспетчеру трафика

Диспетчер трафика Azure позволяет управлять распределением пользовательского трафика между конечными точками службы в разных центрах обработки данных.  Диспетчер трафика использует службу доменных имен (DNS) для перенаправления клиентских запросов к наиболее подходящей конечной точке в зависимости от метода маршрутизации трафика и работоспособности конечных точек, обеспечивая доступность и скорость реагирования, необходимые для крупномасштабных приложений.
