# Начало работы с расширенными настройками Azure API Management

Руководства в этом разделе предлагают руководство по расширенным настройкам и задачам API Management.

-   [Настройка портала разработчика в Azure API Management][] — это руководство описывает способы изменения вида и поведения портала разработчика Azure API Management для его соответствия корпоративному дизайну.
-   [Как использовать инспектор API для трассировки вызовов в Azure API Management][] — это руководство описывает, как использовать инспектор API для трассировки и сбора информации о вызовах операций.
-   [Как создать и задать расширенные настройки продукта в Azure API Management][] — это руководство демонстрирует, как задавать некоторые расширенные настройки продукта, включая ограничения скорости и политики квот, и приводит демонстрацию вызова операций до превышения ограничения скорости.
-   [Как делегировать регистрацию пользователей и подписку продуктов][] — это руководство описывает, как использовать существующий веб-сайт для выполнения входа и регистрации разработчиков и подписки на продукты, а не встроенные функции портала разработчика.
