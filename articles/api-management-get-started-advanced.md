# Начало работы с расширенными настройками Azure API Management

Руководства в этом разделе предлагают руководство по расширенным настройкам и задачам API Management.

-	[Настройка портала разработчика в управлении Azure API][]. В данном учебнике показано, как изменить внешний вид и работу портала разработчика в соответствии с требованиями своего бренда.
-	[Как с помощью инспектора API отслеживать вызовы в управлении Azure API][]. Этот учебник показывает, как с помощью инспектора API отслеживать и собирать информацию о вызовах операций.
-	[Как создать и настроить расширенные настройки продукта в управлении Azure API][]. Этот учебник показывает, как настроить некоторые расширенные настройки продукта, включая ограничение скорости и политики квот, а также показывает демонстрацию вызова операций, пока не будет превышено ограничение скорости.
-	[Как делегировать пользователю регистрацию и подписку на продукт][]. Этот учебник показывает, как пользоваться существующим веб-сайтом для реализации функционала входа и выхода и подписки на продукты по сравнению с использованием встроенной функциональности портала разработчика.
