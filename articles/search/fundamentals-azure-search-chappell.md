# Введение в Поиск Azure #

**Сводка:** сведения о том, как можно использовать Поиск Azure — полностью управляемую и размещенную облачную службу поиска в Azure — для реализации расширенных возможностей поиска в веб-приложениях и мобильных приложениях. Поиск — это стиль по умолчанию для взаимодействия с данными в Интернете. Его использование все чаще ожидается во всех ситуациях приложений.

**Оглавление:**

В это документе вы узнаете следующее:

* Что предоставляет Поиск Azure
* Приступая к работе с индексом
* Выполнение поиска в Поиске Azure
* Цены и масштабирование

В этом документе приведен обзор технологии, а также поддерживаемые ею функции и сценарии. Он не является подробным техническим руководством.
