# Отправка запросов в индекс службы поиска Azure с помощью обозревателя поиска на портале Azure
> [AZURE.SELECTOR]
- [Overview](search-query-overview.md)
- [Search Explorer](search-explorer.md)
- [Fiddler](search-fiddler.md)
- [.NET](search-query-dotnet.md)
- [REST](search-query-rest-api.md)

**Обозреватель поиска** — это средство для создания запросов на портале Azure. Обозреватель позволяет создавать запросы к индексу службы поиска Azure без необходимости написания кода. Он подключается к любому индексу в службе и предоставляет поле поиска для ввода поисковых строк и выражений. Допустимый синтаксис запроса создается на основе введенных данных. Результаты отображаются на странице портала.

Обозреватель поиска идеально подходит для изучения синтаксиса запросов, отправки нерегулярных и ситуативных запросов и уточнения выражения запроса до его добавления в код. Для работы с обозревателем у вас должна быть служба поиска Azure и индекс. Сведения о том, как создать службу поиска и индекс, см. в статьях [Создание службы поиска Azure на классическом портале Azure](search-create-service-portal.md) и [Импорт данных в службу поиска Azure с помощью портала](search-import-data-portal.md).

## Открытие обозревателя поиска

1. Войдите на [портал Azure](https://portal.azure.com).

2. Откройте служебную панель службы поиска Azure. Найти панель мониторинга можно несколькими способами.
	- На навигационной панели щелкните **Home** (Главная). На главной странице отображаются плитки для всех служб, на которые вы подписались. Щелкните плитку, чтобы открыть панель мониторинга службы.
	- На навигационной панели последовательно выберите **Просмотреть все** > **Фильтровать по** > **Службы поиска** и найдите в списке нужную службу поиска.

3. На панели мониторинга службы отобразится панель команд, на которой будет команда **Проводник поиска**.

  	![][1]

4. Щелкните **Проводник поиска**, чтобы открыть колонку обозревателя.
5. Задайте индекс и версию API. Обозреватель поиска автоматически подключится к первому индексу в списке. Чтобы подключиться к другому индексу, щелкните **Изменить индекс**. Команда **Задать версию API** позволяет выбрать предварительную или общедоступную версию. Синтаксис некоторых запросов доступен только для просмотра.
6. Если вы следовали инструкциям, приведенным в статье о [начале работы со службой поиска Azure](search-get-started-portal.md), и создали индекс на основе данных, предоставленных Геологической службой США о штате Род-Айленд, с помощью следующего поискового запроса можно убедиться, что в обозревателе поиска возвращаются те же три результата: `roger williams +school -building`

Обратите внимание, что синтаксис запроса создается автоматически по мере ввода запроса.
