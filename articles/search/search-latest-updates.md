#Обзор последнего обновления Поиска Azure#

Поиск Azure — это размещенная облачная служба поиска в Microsoft Azure. Она свободно доступна на условиях соглашения об уровне обслуживания (SLA), обеспечивающего доступность в течение 99,9 % времени, для конфигураций [API REST службы поиска версии 2015-02-28](https://msdn.microsoft.com/library/azure/dn798935.aspx) или [пакета SDK для .NET](https://msdn.microsoft.com/library/azure/dn951165.aspx).

##Развертывание компонентов и управление их версиями

Компоненты выпускаются отдельно или совместно с помощью [API REST](https://msdn.microsoft.com/library/azure/dn798935.aspx), [пакета SDK для .NET](http://go.microsoft.com/fwlink/?LinkId=528216) или панели мониторинга службы на [портале Azure](https://portal.azure.com). Существует несколько версий библиотеки .NET и API-интерфейсов REST. Старые API-интерфейсы сохраняют работоспособность по мере развертывания новых компонентов. Дополнительные сведения о нашей политике управления версиями см. в статье [Управление версиями службы поиска](https://msdn.microsoft.com/library/azure/dn864560.aspx).

Компоненты предварительной и общедоступной версий привязаны к версии API той же категории.

- Компоненты версии для предварительного просмотра являются экспериментальными и могут изменяться и даже удаляться до перехода в общедоступную версию. Компоненты версии для предварительного просмотра всегда доступны в [предварительной версии интерфейса API REST](search-api-2015-02-28-preview.md) и в некоторых случаях в [пакете SDK для .NET](http://go.microsoft.com/fwlink/?LinkId=528216). В документации по компонентам всегда описано, как получить доступ к рассматриваемому компоненту.
- Компоненты общедоступной версии являются стабильными и вряд ли будут изменены в будущем. Любые изменения компонентов общедоступной версии объявляются как критические изменения.

Компоненты, которые исключительно принадлежат порталу или основаны на инструментах, могут изменяться со временем и не классифицируются как компоненты предварительной или общедоступной версии.
