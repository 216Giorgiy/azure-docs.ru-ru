# Запросы в службе поиска Azure
> [AZURE.SELECTOR]
- [Overview](search-query-overview.md)
- [Search Explorer](search-explorer.md)
- [Fiddler](search-fiddler.md)
- [.NET](search-query-dotnet.md)
- [REST](search-query-rest-api.md)

Использующие службу поиска Azure пользовательские решения со встроенными функциями службы поиска должны содержать код, который создает и отправляет поисковые запросы в службу поиска для обработки.

Методы для выполнения запроса можно создать с помощью пакета SDK для .NET или интерфейса REST API. Для предварительного тестирования или ознакомления можно использовать такие средства, как встроенный обозреватель поиска на портале Azure (для отправки запросов к индексу) или Fiddler (для отправки любых допустимых запросов к службе поиска).

Посетите ресурс с [видеороликами, руководствами, демонстрациями и примерами кода](search-video-demo-tutorial-list.md), чтобы просмотреть список примеров и пошаговых руководств, в которых описывается создание запросов в коде. Почти каждый используемый нами пример содержит код, который отправляет запрос в службу поиска.
