# Запросы в службе поиска Azure
> [AZURE.SELECTOR]
- [Overview](search-query-overview.md)
- [Fiddler](search-fiddler.md)
- [.NET](search-query-dotnet.md)
- [REST](search-query-rest-api.md)

Пользовательские решения со встроенными функциями поиска, использующими службу поиска Azure, должны содержать код, который создает поисковый запрос и отправляет его в службу поиска для обработки.

Методы для выполнения запроса можно создать с помощью пакета SDK для .NET или интерфейса REST API. Для предварительного тестирования или просмотра можно отправить запросы в службу, используя такие средства, как Fiddler.
