# Публикация приложения в RemoteApp

После создания облачной или гибридной коллекции нужно опубликовать приложения или ресурсы, которые должны быть доступны пользователям. Образы шаблонов, входящие в подписку, по умолчанию включают только некоторые приложения. Чтобы открыть доступ к другим приложениям, их необходимо опубликовать.
 
На вкладке **Публикация** портала нажмите кнопку **Опубликовать**. Вы можете добавить приложение из меню "Пуск" образа шаблона или указать путь, по которому оно установлено в образе. При использовании меню "Пуск" выберите нужное приложение в списке. Если указывается путь к приложению, введите его имя и путь. Используйте в пути переменные, например "%systemdrive%" вместо "c:".

**Примечание.** Если вы хотите добавить приложение из меню "Пуск", необходимо заранее *добавить это приложение в меню "Пуск" в образе шаблона*. В противном случае RemoteApp распознает только те приложения, которые *добавлены* в меню "Пуск". Если вы забыли добавить приложение в меню "Пуск" при создании шаблона, добавьте приложение по его пути (или создайте образ шаблона заново, что потребует больше усилий).
