# Настройка предупреждений о выставлении счета для подписок Microsoft Azure

Вас заботит, сколько денег уходит ежемесячно на подписку Azure? Если вы являетесь администратором учетной записи для подписки Azure, с помощью службы оповещений Azure можно создать пользовательские предупреждения, которые помогут отслеживать и контролировать действия по выставлению счетов для учетных записей Azure.

Данная служба предназначена для предварительного просмотра, поэтому первое, что вам нужно сделать, — это зарегистрироваться. Для этого пройдите по ссылке на <a href="https://account.windowsazure.com/PreviewFeatures">страницу предварительных функций</a> на портале управления учетными записями Azure.

## Укажите адресатов электронной почты и пороговое значение для отправки предупреждений

Получив по электронной почте подтверждение о том, что для вашей подписки включена служба выставления счетов, откройте страницу <a href="https://account.windowsazure.com/Subscriptions">Подписки</a> на портале учетной записи. Выберите интересующую вас подписку и щелкните **Оповещения**.

![][Image1]

Затем нажмите **Добавить оповещение**, чтобы создать первое из них. Всего для каждой подписки можно настроить до пяти оповещений с различными пороговыми значениями и до двух адресатов электронной почты.

![][Image2]

При добавлении предупреждения присвойте ему уникальное имя, укажите пороговое значение расхода и выберите адреса электронной почты, куда будет отправляться сообщение. При настройке порогового значения можно выбрать либо **Общий счет**, либо **Денежный кредит** из списка **Оповещение для**. Для общего счета предупреждение отправляется, когда затраты по подписке превышают пороговое значение. Для денежного кредита предупреждение отправляется тогда, когда сумма денежного кредита стала ниже заданного предела. Денежные кредиты обычно применяются для бесплатных пробных версий и подписок, связанных с учетными записями MSDN.

![][Image3]

Azure поддерживает любой адрес электронной почты, но не проверяет их работоспособность, поэтому дважды перепроверьте правильность адреса.

## Проверка настроенных предупреждений

После настройки предупреждения они появятся в центре учетных записей. Кроме того, там будет показано, сколько предупреждений еще можно добавить. Показывается дата и время отправки каждого предупреждения, его тип (по общему счету или по денежному кредиту) и настроенное предельное значение. Время показывается в 24-часовом формате (UTC), дата в формате "гггг-мм-дд". Щелкните значок "плюс" рядом с оповещением в списке, чтобы изменить его. Чтобы удалить, щелкните значок с корзиной.
