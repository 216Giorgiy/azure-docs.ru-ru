# Уровень поиска Azure (шаблоны архитектуры Azure)

[Инфраструктура интеграции облачных платформ (CPIF)](azure-architectures-cpif-overview.md) предоставляет инструкции по интеграции рабочей нагрузки для адаптации приложений в облачном решении корпорации Майкрософт.

Инфраструктура интеграции облачных платформ описывает способы, которые следует использовать организациям, клиентам и партнерам для разработки и развертывания предназначенных для облака рабочих нагрузок с помощью гибридной облачной платформы и возможностей управления Azure, System Center и Windows Server.

Шаблон **уровня поиска Azure** является частью области **основания**, подробно описанной в документе об архитектуре CPIF.

##  Уровень поиска Azure

Шаблон разработки уровня поиска Azure содержит сведения о компонентах и службах Azure, необходимых для предоставления поисковых услуг, обеспечивающих предсказуемый уровень производительности и высокую доступность во всех географических регионах, а также предоставляет шаблон архитектуры для использования поиска Azure при доставке решений поиска. Поиск Azure — это поиск как услуга, которая встроена в Microsoft Azure и позволяет разработчикам внедрять возможности поиска в приложения без необходимости развертывать, управлять или поддерживать службы инфраструктуры. Цель этого шаблона — предоставить воспроизводимое решение для использования в различных ситуациях и разработки.

## Обзор шаблона архитектуры 

В этом документе описан основной шаблон для использования службы поиска Azure в двух вариациях, чтобы продемонстрировать диапазон архитектуры службы. Основной шаблон состоит из поиска Azure и окружающих служб Azure и предназначен для руководства по созданию сквозных проектов. Вариации шаблона, в частности шаблоны общей службы и параллелизма, также включены в этот раздел в качестве инструкций в зависимости от различных требований, соглашений об уровне обслуживания (SLA) и других конкретных условий.
