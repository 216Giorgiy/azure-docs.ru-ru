# Обращение в службу поддержки при возникновении проблем с оплатой или подпиской на Azure

В этой статье содержится информация, необходимая для отправки запроса в службу поддержки Azure.

>[AZURE.NOTE]Доступ к центру учетных записей предоставляется только учетной записи администратора учетных записей Azure.

1. Перейдите на веб-сайт [службы поддержки Azure](https://azure.microsoft.com/support/) и щелкните **Получить поддержку**. Можно также отправить обращение с портала Microsoft Azure или из центра учетных записей Azure.

 * Чтобы отправить запрос в службу поддержки на [портале Azure](https://manage.windowsazure.com/), щелкните имя своей учетной записи и выберите «Обратиться в службу поддержки Майкрософт».
 * Чтобы отправить запрос в службу поддержки в [центре учетных записей Azure](https://account.windowsazure.com/Subscriptions), щелкните имя своей подписки и выберите «Обратиться в службу поддержки Майкрософт».

2. В окне **Создание запроса в службу поддержки** выберите подписку, для которой нужно перенести данные и службы.

3. В поле **Тип поддержки** выберите **Выставление счетов**.

4. Выберите регион и язык и нажмите кнопку **Создать запрос**.

5. На следующей странице в поле **Тип проблемы** выберите вариант **Моя подписка отображается как отключенная**. Если вам нужна помощь с обращением в службу поддержки, щелкните **Чат со службой поддержки**, чтобы связаться с ее представителем.

6. Подтвердите свои контактные сведения, введите контактный номер телефона и нажмите кнопку **Продолжить**.

7. На следующей странице введите такие сведения:

 * В поле **Сведения о проблеме** укажите название инцидента для своего запроса, а затем заполните текстовые поля и введите дополнительные сведения о запросе.
 * В поле **Степень серьезности** выберите вариант, отражающий степень серьезности вашей проблемы. Дополнительные сведения о типах серьезности см. в таблице [Серьезность инцидентов в Microsoft Azure](http://support.microsoft.com/gp/AzureSevDetails).
 * С помощью средства **Отправка файлов** прикрепите к запросу связанную с ним документацию.

8. Если вы принимаете соглашение о предоставлении услуг Майкрософт, нажмите кнопку **Отправить**.

Вы свяжетесь в представителем службы поддержки Azure в соответствии с условиями этого соглашения.
