# Обращение в службу поддержки при возникновении проблем с оплатой или подпиской Azure
В этой статье содержится информация, необходимая для отправки запроса в службу поддержки Azure.

1. Войдите на [портал Azure](https://portal.azure.com/). Выберите **Справка + поддержка**>**Новый запрос в службу поддержки**.
   
    ![кнопка справки и поддержки](./media/billing-how-to-create-billing-support-ticket/helpandsupport.png)
2. В колонке **Основные сведения** выберите тип проблемы, подписку и план поддержки.
3. В колонке **Проблема** введите следующие сведения:
   
   * **Уровень серьезности**. Выберите вариант, отражающий степень серьезности вашей проблемы. Дополнительные сведения о типах серьезности см. в таблице [Серьезность инцидентов в Microsoft Azure](http://support.microsoft.com/gp/AzureSevDetails).
   * **Тип проблемы**. Выберите тип проблемы.
   * **Сведения о проблеме**. Укажите название инцидента для своего запроса, а затем заполните текстовые поля и введите дополнительные сведения о запросе.
   * **Интервал времени** (необязательно). Если возможно, укажите дату, время и часовой пояс последнего появления ошибки.
   * С помощью средства **Отправка файлов** прикрепите к запросу связанную с ним документацию.
4. В колонке **Контактные данные** проверьте свои контактные данные и предпочтительный способ связи и нажмите кнопку **Создать**.

Вы свяжетесь в представителем службы поддержки Azure в соответствии с условиями этого соглашения.

Можно также отправить запрос на поддержку с веб-сайта службы поддержки Azure или из центра учетных записей Azure.

* Чтобы отправить запрос на поддержку с [веб-сайта службы поддержки Azure](https://azure.microsoft.com/support/options/), щелкните **Получить поддержку**.
* Чтобы отправить запрос на поддержку из [центра учетных записей Azure](https://account.windowsazure.com/Subscriptions), выберите подписку, а затем щелкните **Обратиться в службу поддержки Майкрософт**.
  
  > [!NOTE]
  > Доступ к центру учетных записей предоставляется только учетной записи администратора учетных записей Azure. Чтобы узнать, кто является администратором учетной записи для подписки, см. инструкции в разделе [Часто задаваемые вопросы](billing-subscription-transfer.md#faq).
