# Что делать в случае отключения подписки Azure?

Подписка Azure может быть отключена по следующим причинам:

### Достигнута предельная сумма расходов

Дополнительные сведения о предельной сумме расходов см. в статье [Предельная сумма расходов Azure](https://azure.microsoft.com/pricing/spending-limits/). В этой статье рассказывается, что при достижении максимального ежемесячного ограничения на использование по вашему предложению служба будет отключена до окончания месяца выставления счета. После этого приводятся инструкции по удалению предельной суммы расходов.

### Просрочена оплата счета

Решение этой проблемы см. в статье [Почему вы получили уведомление о том, что имеется задолженность по вашей подписке Azure?](../billing-azure-subscription-past-due-balance.md)

### Подписка Azure отменена

Чтобы снова активировать подписку, необходимо создать запрос в службу поддержки. Создание запроса в службу поддержки о включении подписки Azure описывается в статье [Как создать запрос в службу поддержки по вопросам подписки и выставления счетов в Azure](../billing-how-to-create-billing-support-ticket.md).
