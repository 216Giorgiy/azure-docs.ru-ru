# Что делать в случае отключения подписки Azure?

Подписка Azure может быть отключена по следующим причинам:

### Достигнута предельная сумма расходов

Дополнительные сведения о предельной сумме расходов см. в статье [Предельная сумма расходов Azure](https://azure.microsoft.com/pricing/spending-limits/). В этой статье рассказывается, что при достижении максимального ежемесячного ограничения на использование по вашему предложению служба будет отключена до окончания месяца выставления счета. После этого приводятся инструкции по удалению предельной суммы расходов.

### Просрочена оплата счета

Решение этой проблемы см. в статье [Почему вы получили уведомление о том, что имеется задолженность по вашей подписке Azure?](billing-azure-subscription-past-due-balance.md)

### Подписка Azure отменена

Чтобы повторно активировать подписку, необходимо создать запрос в службу поддержки. Войдите на [портал Azure](https://portal.azure.com/), выберите **Справка + поддержка** > **Новый запрос в службу поддержки**.

![кнопка справки и поддержки](./media/billing-how-to-create-billing-support-ticket/helpandsupport.png)

Если у вас нет доступа к порталу, отправьте запрос на поддержку с веб-сайта службы поддержки Azure или из центра учетных записей Azure.

 * Чтобы отправить обращение в службу поддержки с [веб-сайта службы поддержки Azure](https://azure.microsoft.com/support/options/), щелкните **Получить поддержку**.
 * Чтобы отправить обращение в службу поддержки из [центра учетных записей Azure](https://account.windowsazure.com/Subscriptions), выберите подписку, а затем щелкните **Обратиться в службу поддержки Майкрософт**.

Подробнее об этом см. в разделе [Обращение в службу поддержки при возникновении проблем с оплатой или подпиской на Azure](billing-how-to-create-billing-support-ticket.md).
