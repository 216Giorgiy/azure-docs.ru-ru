# <a name="how-to-manage-pricing-tier-of-your-azure-maps-account"></a>Как управлять ценовой категорией для учетной записи Azure Maps

Вы можете управлять ценовой категорией своей учетной записи Azure Maps на портале Azure. Создав [учетную запись](https://azure.microsoft.com/free/?WT.mc_id=A261C142F), можно просмотреть или изменить ее ценовую категорию.

Дополнительные сведения о выборе подходящей ценовой категории см. в статье [Выбор подходящей ценовой категории в Azure Maps](https://docs.microsoft.com/azure/azure-maps/choose-pricing-tier).

## <a name="view-pricing-tier"></a>Просмотр ценовой категории

Чтобы увидеть выбранную ценовую категорию, выберите пункт **Ценовая категория** в меню параметров.

![Просмотр выбранной ценовой категории](./media/how-to-manage-pricing-tier/view-pricing-tier.png)

## <a name="change-pricing-tier"></a>Смена ценовой категории

Создав учетную запись Azure Maps, вы можете повышать и понижать ценовую категорию для нее. Для этого выберите пункт **Ценовая категория** в меню параметров и щелкните ценовую категорию, которая не выделена. Нажмите кнопку сохранения, чтобы сохранить выбор ценовой категории.

![Смена ценовой категории](./media/how-to-manage-pricing-tier/change-pricing-tier.png)

> [!Note]
> При повышении или понижении ценовой категории для учетной записи Azure Maps не требуется создавать новые ключи подписок.
