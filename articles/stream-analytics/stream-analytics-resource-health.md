# <a name="stream-analytics-resource-health-page-in-the-azure-portal"></a>Использование страницы "Работоспособность ресурсов" в Azure Stream Analytics

На странице **Работоспособность ресурсов** на портале Azure представлены сведения о работоспособности заданий Azure Stream Analytics, а также практические рекомендации по устранению проблем. 

Страница **Работоспособность ресурсов** поможет вам быстрее определить основную причину проблемы (приложение или событие) в Azure. Так вы сможете понять, когда нужно отправить запрос в службу поддержки для получения помощи.

## <a name="troubleshooting-workflow"></a>Процесс устранения неполадок

Для просмотра сведений о работоспособности ресурсов заданий Stream Analytics перейдите на страницу **Работоспособность ресурсов**. 

![Устранение неполадок Stream Analytics — доступ](media/stream-analytics-resource-health/stream-analytics-access-troubleshooting.png)

Следующая блок-схема может помочь вам определить, что стоит искать и когда следует обратиться в службу.

![Устранение неполадок Stream Analytics — карта](media/stream-analytics-resource-health/stream-analytics-troubleshooting-map.png)
