# <a name="stream-analytics-resource-health-blade"></a>Колонка работоспособности ресурсов Stream Analytics

В колонке **Работоспособность ресурсов** на портале Azure представлены сведения о работоспособности заданий Azure Stream Analytics, а также практические рекомендации по устранению проблем. 

Колонка **Работоспособность ресурсов** поможет вам быстрее определить основную причину проблемы (приложение или событие в Azure). Так вы сможете понять, когда нужно отправить запрос в службу поддержки для получения помощи.

## <a name="troubleshooting-workflow"></a>Процесс устранения неполадок

Для просмотра сведений о работоспособности ресурсов заданий Stream Analytics выберите колонку **Работоспособность ресурсов**. 

![Устранение неполадок Stream Analytics — доступ](media/stream-analytics-resource-health/stream-analytics-access-troubleshooting.png)

Следующая блок-схема может помочь вам определить, что стоит искать и когда следует обратиться в службу.

![Устранение неполадок Stream Analytics — карта](media/stream-analytics-resource-health/stream-analytics-troubleshooting-map.png)
