# <a name="connect-with-windows-7-and-windows-10"></a>Подключение с помощью Windows 7 и Windows 10

> Касается Windows 7 и Windows 10.

Загружаемый клиент доступен, обеспечивающий доступ к ресурсам Апробация виртуального рабочего стола Windows с устройств под управлением Windows 7 и Windows 10.

## <a name="install-the-client"></a>Установка клиента

[Скачайте](https://go.microsoft.com/fwlink/?linkid=2068602) и установка клиента на локальный компьютер. Это требует прав администратора.

## <a name="subscribe-to-a-feed"></a>Подпишитесь на веб-канал

Получите список ресурсов, доступных для вас на локальном Компьютере, подписавшись на канал, предоставленный администратором

Чтобы подписаться на веб-канал:

1. Запустите клиент в список всех приложений, искать удаленного рабочего стола.
2. Выберите рабочую или учебную учетную запись на главной странице. / Будет осуществлена подписка имеет значения по умолчанию, URL-адрес канала.
   - Можно также указать другой URL-адрес, при необходимости с **вместо этого используйте URL-адрес** параметр.
3. Войдите с помощью учетной записи пользователя.

> [!NOTE]
> Если ваш администратор настроил обнаружение электронной почты, клиент будет автоматически искать веб-канала URL-адрес, если ввести адрес электронной почты.

После успешной проверки подлинности, вы увидите список ресурсов, доступных пользователю.

Ресурсы можно запустить одним из двух способов.

- Перейдите на главную страницу клиента и выберите ресурс.
- Запустите ресурс обычным образом другие приложения из меню "Пуск".
  -  Можно также выполнить поиск приложения в строке поиска.

## <a name="update-the-client"></a>Обновление клиента

Если доступна новая версия клиента, вы будете получать уведомления клиентом и центром поддержки Windows. Выберите уведомление, чтобы начать процесс обновления.
