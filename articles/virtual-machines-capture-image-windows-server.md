#Как записать образ виртуальной машины под управлением Windows Server#

Чтобы упростить создание виртуальных машин, можно воспользоваться готовыми образами из коллекции образов. Можно также записать и использовать собственные образы для создания настраиваемых виртуальных машин. Образ — это файл виртуального жесткого диска (VHD), который используется в качестве шаблона для создания виртуальной машины. Образ — это шаблон, так как в отличие от работающей виртуальной машины, у него нет конкретных параметров, таких как имя компьютера и параметры учетной записи пользователя. Если нужно создать несколько виртуальных машин с одинаковыми настройками, можно записать образ настроенной виртуальной машины и использовать его в качестве шаблона.


1. Подключитесь к виртуальной машине, выполнив действия, описанные в разделе [Как войти в систему на виртуальной машине, работающей под управлением Windows Server][].

2.	Откройте окно командной строки с правами администратора.


3.	Перейдите в каталог `%windir%\system32\sysprep` и выполните команду sysprep.exe.


4. 	Откроется диалоговое окно **Программа подготовки системы**.


	В разделе **Действие по очистке системы** выберите **Запуск при первом включении компьютера (OOBE)** и убедитесь, что установлен флажок **Обобщить**. Дополнительные сведения об использовании Sysprep см. в разделе [Как использовать Sysprep: введение][].

5.	В разделе **Параметры завершения работы** выберите **Завершение работы**.

6.	Нажмите кнопку **ОК**.

7.	Программа Sysprep завершит работу виртуальной машины, в результате чего статус машины на [Портале управления](http://manage.windowsazure.com) будет изменен на **Остановлена**.


8.	Нажмите кнопку **Виртуальные машины**, а затем выберите виртуальную машину, которую требуется настроить.

9.	На панели команд щелкните **Запись**.

	Откроется диалоговое окно **Запись образа виртуальной машины**.

10.	В поле **Имя образа** введите имя нового образа.

11.	Прежде чем добавить образ Windows Server в набор настраиваемых образов, его необходимо обобщить, запустив программу Sysprep, как описано в предыдущих шагах. Нажмите кнопку **Я запустил Sysprep на виртуальной машине**, чтобы указать, что это выполнено.

12.	Установите флажок, чтобы записать образ. При записи образа виртуальной машины эта машина будет удалена.

	Новый образ станет доступен в разделе **Образы**.

	При создании виртуальной машины с помощью метода **Из коллекции** можно использовать записанный образ, щелкнув пункт **Мои образы** на странице **Выберите образ**.
