# <a name="microsoft-azure-storage-explorer-087-preview"></a>Microsoft Azure Storage Explorer 0.8.7 (предварительная версия)
## <a name="overview"></a>Обзор
В этой статье содержатся заметки о выпуске предварительной версии Azure Storage Explorer 0.8.7.

[Microsoft Azure Storage Explorer (предварительная версия)](./vs-azure-tools-storage-manage-with-storage-explorer.md) — это изолированное приложение, позволяющее легко работать с данными из службы хранилища Azure на платформе Windows, macOS и Linux.

## <a name="azure-storage-explorer-087-preview"></a>Azure Storage Explorer 0.8.7 (предварительная версия)
### <a name="download-azure-storage-explorer-087-preview"></a>Скачивание Azure Storage Explorer 0.8.7 (предварительная версия)
- [Azure Storage Explorer 0.8.7 (предварительная версия) для Windows](https://go.microsoft.com/fwlink/?LinkId=708343)
- [Azure Storage Explorer 0.8.7 (предварительная версия) для Mac](https://go.microsoft.com/fwlink/?LinkId=708342)
- [Azure Storage Explorer 0.8.7 (предварительная версия) для Linux](https://go.microsoft.com/fwlink/?LinkId=722418)

### <a name="new-updates"></a>Новые обновления
* Можно выбрать способ разрешения конфликтов в начале сеанса обновления, скачивания или копирования в окне **Действия**.
* Наведите указатель на вкладку, чтобы увидеть полный путь к ресурсу службы хранилища.
* При выборе вкладки она синхронизируется с расположением в левой части области навигации.

### <a name="fixes"></a>Исправления
* Исправлено. Storage Explorer теперь относится к доверенным приложениям в macOS.
* Исправлено. ОС Ubuntu 14.04 снова поддерживается.
* Исправлено. Иногда элемент пользовательского интерфейса "Добавление учетной записи" мигал при загрузке подписок.
* Исправлено. Иногда не все ресурсы службы хранилища указывались в левой части области навигации.
* Исправлено. Иногда на панели операций отображались пустые действия.
* Исправлено. Теперь сохраняется размер окна в предыдущем закрытом сеансе.
* Исправлено. Возможность открыть несколько вкладок для одного ресурса с помощью контекстного меню.

### <a name="known-issues"></a>Известные проблемы
* Быстрый доступ работает только с элементами подписки. В этом выпуске не поддерживаются локальные ресурсы или ресурсы, подключенные с помощью ключа или маркера SAS.
* Чтобы перейти к целевому ресурсу с помощью быстрого доступа, может потребоваться несколько секунд. Это зависит от количества ваших ресурсов.
* При одновременной передаче более трех групп больших двоичных объектов или файлов могут возникнуть ошибки.
* Служба поиска обрабатывает поисковые запросы приблизительно по 50 000 узлам. После достижения этого порога может снизиться производительность или могут возникать необработанные исключения.
* При первом использовании Storage Explorer в macOS может появиться несколько запросов, предлагающих указать разрешение пользователя на доступ к цепочке ключей. Рекомендуется выбрать **Всегда разрешать**, чтобы эти запросы больше не появлялись.

## <a name="previous-releases"></a>Предыдущие выпуски
### <a name="features"></a>Функции
#### <a name="main-features"></a>Основные функции
* Версии для macOS, Linux и Windows.
* Вход для просмотра учетных записей хранения, сгруппированных по подписке:
    * использование рабочей учетной записи, учетной записи Майкрософт, двухфакторной проверки подлинности и т. д.;
    * настройка параметров прокси-сервера и управление ими;
    * удаление учетных записей при выходе.
* Подключение к учетным записям хранения с помощью:
    * имени и ключа учетной записи;
    * пользовательских конечных точек (включая Azure для Китая);
    * универсального кода ресурса (URI) SAS для учетных записей хранения.
* Поддержка Azure Resource Manager и классического хранилища.
* Создание ключей SAS для больших двоичных объектов, контейнеров больших двоичных объектов, очередей, таблиц или файловых ресурсов.
* Подключение к контейнерам больших двоичных объектов, очередям, таблицам или файловым ресурсам с помощью ключа подписанного URL-адреса.
* Управление хранимыми политиками доступа для контейнеров больших двоичных объектов, очередей, таблиц или файловых ресурсов.
* Локальное хранилище для разработки в эмуляторе хранения (только Windows).
* Создание и удаление контейнеров больших двоичных объектов, очередей или таблиц.
* Просмотр контейнеров больших двоичных объектов $logs и таблиц $metrics.
* Поиск конкретных больших двоичных объектов, очередей, таблиц или файловых ресурсов.
* Прямые ссылки на учетные записи хранения либо контейнеры, очереди, таблицы или файловые ресурсы для совместного использования и удобного доступа к ресурсам.
* Переименование контейнеров больших двоичных объектов, таблиц, файловых ресурсов.
* Возможность управлять правилами CORS и настраивать их.
* Простое копирование первичного и вторичного ключей для учетных записей хранения.
* Проверки MD5 при передаче и скачивании для обеспечения целостности и согласованности данных.
* Поиск контейнеров больших двоичных объектов, таблиц, очередей, файловых ресурсов или учетных записей хранения с помощью поля поиска.
* Теперь вы можете закрепить наиболее часто используемые службы в области быстрого доступа для упрощения навигации.
* Теперь можно открыть несколько редакторов на разных вкладках. Одним щелчком можно открыть временную вкладку. Двойным щелчком можно открыть постоянную вкладку. Можно также щелкнуть временную вкладку, чтобы сделать ее постоянной.
* Значительное улучшение производительности и стабильности при передаче и скачивании, особенно заметное для больших файлов на быстрых компьютерах.
* Мы повторно представляем расширенный поиск в заданных областях. Также мы добавили понятие определения области. Введите путь к узлу с помощью значка для наведенного указателя, щелкните правой кнопкой мыши и выберите пункт "Искать от этого места" или вручную определите область этого узла. Определив область, можно добавить условие поиска в конец пути, чтобы начать поиск с этого узла.
* Мы добавили различные темы: "Светлая" (по умолчанию), "Темная", "Высокая контрастность черного" и "Высокая контрастность белого".
* Выберите "Изменить" > "Темы", чтобы изменить параметры темы.
* В среде Linux теперь является обязательной 64-разрядная операционная система.
* Мы обновили наш логотип!
#### <a name="blobs"></a>BLOB-объекты
* Просмотр больших двоичных объектов и перемещение по каталогам.
* Передача, скачивание, удаление и копирование больших двоичных объектов и папок.
* Открытие и просмотр содержимого больших двоичных объектах с текстом и изображениями.
* Просмотр и изменение свойств и метаданных больших двоичных объектов.
* Поиск больших двоичных объектов по префиксу.
* Создание и прекращение аренды для больших двоичных объектов и контейнеров больших двоичных объектов.
* Перетаскивание файлов для передачи.
* Переименование больших двоичных объектов и папок.
* Теперь в контейнерах больших двоичных объектов можно создавать пустые "виртуальные" папки.
* Можно изменять свойства больших двоичных объектов и файлов.
#### <a name="tables"></a>Таблицы
* Просмотр и запрос сущностей с помощью ODATA или использование конструктора запросов для создания сложных запросов.
* Добавление, изменение и удаление сущностей.
* Импорт и экспорт содержимого таблицы в формате CSV (включая экспорт результатов запроса).
* Настройка порядка столбцов.
* Возможность сохранения запросов.
#### <a name="queues"></a>Очереди
* Просмотр 32 последних сообщений.
* Добавление, удаление из очереди и просмотр сообщений.
* Очистка очереди.
* Мы дали вам возможность решать, следует ли кодировать и декодировать очереди сообщений.
#### <a name="file-shares"></a>Файловые ресурсы
* Просмотр файлов и перемещение по каталогам.
* Передача, скачивание, удаление и копирование файлов и папок.
* Просмотр свойств файла.
* Переименование файлов и каталогов.

### <a name="bug-fixes"></a>Исправления ошибок
* Исправлено. Проблема с блокированием экрана.
* Исправлен. Усилена безопасность.

### <a name="known-issues"></a>Известные проблемы
* Служба поиска обрабатывает поисковые запросы приблизительно по 50 000 узлам. После достижения этого порога может снизиться производительность или, если служба установлена в macOS, могут появляться запросы на повышенные привилегии.
* Панель "Параметры учетной записи" может запросить повторный ввод учетных данных для фильтрации подписок.
* При переименовании больших двоичных объектов (по отдельности или в переименованном контейнере больших двоичных объектов) не сохраняются моментальные снимки. Все прочие свойства и метаданные больших двоичных объектов, файлов и сущностей при переименовании сохраняются.
* Azure Stack в настоящее время не поддерживает файлы, поэтому попытка развернуть узел **Files** приведет к ошибке.
* При установке в Linux 14.04 требуется обновить или установить новую версию gcc. Чтобы установить новую версию, выполните следующие действия.

```
sudo add-apt-repository ppa:ubuntu-toolchain-r/test
sudo apt-get update
sudo apt-get upgrade
sudo apt-get dist-upgrade
```
