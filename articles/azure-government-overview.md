## <a name="Overview"></a>Обзор

Правительственные Azure - это государственных сообщества облако (GCC) поддерживают стратегического государственных сценариев, которые требуют скорость, масштаб, безопасности, соответствия и рентабельность для организаций правительства США.   Она была разработана по результатам доставки программного обеспечения, безопасности, соответствия и элементы управления в других облачных предложениях Майкрософт, как открытый Azure, Office 365, GCC Office 365, Microsoft CRM Online д огромного опыта корпорации Майкрософт. 

Кроме того, правительственные Azure разработан с учетом более высокий уровень безопасности и соответствия требуется для рабочих нагрузок США государственного сектора регистр, выделенных, найден в нормам, например United States Federal риска и авторизации управления программы (FedRAMP), отдел из защиты корпоративных облачной службы Broker (ECSB), Джастис атаках сведения службы (CJIS) политики безопасности и здравоохранения HIPAA и отчетность HIPAA (HIPAA).     

Ниже приведен сводное представление облако Azure государственных инфраструктуры, структуры, служб и платформ, доступные для государственных организаций построение гибридных облачных решений для достижения своих целей.  Как ниже некоторые службы доступны только в режиме предварительного просмотра, см. в разделе [области страницы](http://azure.microsoft.com/regions/#services) как самые даты перечислены службы, которые обычно доступны.

![][2]

Правительственные Azure включает основные компоненты инфраструктуры как служба (IaaS) и платформы как услуга (PaaS).  Это включает инфраструктуру, сети, хранилища, управления данными, управления удостоверениями и многих других служб.  Правительственные Azure поддерживает замечательные возможности, открытый клиентов Azure используется как синхронные географической репликации данных и автоматического масштабирования. Майкрософт было определено как лидера в обоих <a href="https://www.gartner.com/doc/2575715/magic-quadrant-cloud-infrastructure-service" target="_new">IaaS</a> и <a href="https://www.gartner.com/doc/2645317/magic-quadrant-enterprise-application-platform" target="_new">PaaS<a/>, ведущих отраслевых аналитиков.

Помимо надежных служб и функций открытый Azure, правительственные Azure предоставляет несколько функций для обеспечения США сущностей государственных свои данные безопасности, предоставляя:

- **Экземпляр физической и сетевой изоляцией** - среды государственных Azure является полностью отдельный экземпляр из Microsoft Azure открытый и только используемые полное организаций правительства США и поставщикам решений.

- **Безопасность, конфиденциальность и соответствия** - Корпорация Майкрософт реализовала его надежную инфраструктуру управления безопасности, конфиденциальности и соответствия, а также дополнительные строгие элементы управления в соответствии с чем выше уровень уровней воздействия ECSB и CJIS в требования.

- **Хранение данных** - среды государственных Azure использует 2 центры обработки данных более чем 500 километров друг от друга. Все данные о клиентах управляемых хранятся в центрах обработки данных континентальной США (CONUS)

- **Персонал США** - Администраторы и операторы всех правительственных Azure, экранированная населения США.

- **Управление идентификаторами** - удостоверений в среде Azure государственных является отдельный экземпляр Azure Active Directory.

- **Соответствие** - Корпорация Майкрософт постоянно приобретение соответствуют Ведение тщательное и изменение федеральных, состояние и локальные требованиям актов FedRAMP, CJIS, ECSB и HIPAA для правительство США облачных решений.

- **Облако интеграции** - государственных Azure предоставляет интегрированную среду с Office 365 правительством поддержка единым входом облачных служб и улучшенных служб, таких как 1 ТБ OneDrive дискового пространства.

Правительственные Azure также позволяет организациям поддерживать свои существующие вложения в технологии и использовать преимущества облачных служб.  Поскольку государственных Azure представляет собой платформу для взаимодействия облака, с продуктами и технологии организации могут создавать приложения, они более откройте с нуля.  Агентства можно выбрать средства, службы, операционная система, архитектура и платформ, включая Windows, Linux, Oracle, SharePoint, .NET, Java, PHP и Node.js, для их облачных решений. Гибкость платформы Azure государственных позволяет новых форм между агентство совместной работы, разработки приложений и интеграции.  

Заинтересованы в облачных службах организаций правительства США можно быть уверенным, что правительственные Azure предоставляет огромного масштаб и методы строгой безопасности в зависимости от изменяющихся потребностей. 







## <a name="Features"></a>Функции, доступные в Microsoft Azure государственных
Azure Goverment в настоящее время имеет следующие службы, доступные в регионах штата US GOV IOWA и US GOV VIRGINIA:

- Виртуальные машины
- Облачные службы
- Хранилище
- Active Directory
- Планировщик
- Виртуальная сеть
- База данных SQL

Доступны и другие службы, и их список будет расширяться.  Для наиболее актуального списка служб, см. в разделе [области страницы](http://azure.microsoft.com/regions/#services) которого будут выделены каждой доступной области и их службы.  

В данный момент Azure Government поддерживают центры обработки данных US GOV Iowa и US GOV Virginia.  На указанной выше странице регионов можно посмотреть доступные в данный момент центры обработки данных и службы.
