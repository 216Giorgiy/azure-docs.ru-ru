# Создайте свое первое решение с использованием Поиска Azure

## Скоро

Мы просим вас подождать, поскольку мы наносим последние штрихи для образца приложения, демонстрирующего возможности Поиска Azure. Оно станет доступно через пару дней. Когда оно будет готово, вы сможете загрузить код примера и следовать инструкциям, чтобы запустить приложение в службе поиска Azure.

Однако несколько шагов можно пройти уже сегодня. Начните заранее, загрузив несколько элементов и настроив службу:

-   Visual Studio 2013 Express для Web
-   Настройка предварительной версии Поиска Azure

### Загрузка Visual Studio 2013 Express для Web

Образец приложения создан для Visual Studio 2013. Если у вас уже установлена версия Visual Studio 2013, используйте ее. В противном случае можно загрузить бесплатную версию с сайта корпорации Майкрософт.

Откройте [Центр загрузки][], чтобы загрузить Visual Studio 2013 Express для Web.

### Настройка бесплатной службы поиска

Для обучения рекомендуем пользоваться бесплатной версией службы. Это ограниченная версия Поиска, доступная всем подписчикам без дополнительной платы. Если у вас нет подписки, вы можете зарегистрироваться для бесплатного пробного использования.

Откройте раздел [Настройка поиска на портале предварительной версии Azure][] для получения дополнительной информации по настройке службы.

В инструкциях приведено больше шагов, чем вам нужно будет выполнить. Можно остановиться после создания бесплатной службы. Вам потребуется адрес URL службы и ключ API для работы с учебником.
