# <a name="overview-of-azure-time-series-insights"></a>Общие сведения об Azure Time Series Insights

Azure Time Series Insights — это полностью управляемая облачная служба с хранилищем, аналитикой и компонентами визуализации, невероятно упрощающая прием, хранение, изучение и анализ миллиардов событий одновременно. Time Series Insights дает вам глобальное представление о данных, позволяя быстро проверить решение Интернета вещей и избежать дорогостоящих простоев для критически важных устройств. Это стало возможным благодаря обнаружению скрытых тенденций, аномалий и анализу их первопричин практически в реальном времени. Time Series Insights принимает данные временных рядов из брокеров событий (например, Центра Интернета вещей или концентратора событий), индексирует их и удаляет с учетом настроенной политики хранения. Пользователи используют данные для бизнес-сценария в интуитивно понятном интерфейсе или REST API для запросов.

## <a name="primary-scenarios"></a>Основные сценарии

* Проверка и мониторинг решений Интернета вещей за несколько минут
* Интуитивная визуализация и анализ данных Интернета вещей в масштабе
* Ускорение анализа первопричин и обнаружения аномалий
* Создание глобального представления нескольких устройств, производства и данных

## <a name="key-capabilities-and-benefits"></a>Основные возможности и преимущества

* **Простое начало работы.** Azure Time Series Insights работает невероятно быстро и для этой службы не требуется предварительная подготовка данных, поэтому вы можете подключиться к миллиарду событий в Центре Интернета вещей Azure или в концентраторе событий за несколько минут. После подключения вы сможете мгновенно визуализировать данные счетчиков и установить с ними взаимодействие, чтобы быстро проверить свое решение Интернета вещей. Решение Time Series Insights интуитивно понятное и невероятно простое в использовании. Вы можете взаимодействовать с данными, не написав ни одной строки кода.  Так как Time Series Insights предоставляет область, где продвинутые пользователи смогут создавать подробные текстовые запросы в свободной форме, а новички — исследовать данные по выделению и щелчку, вам не придется осваивать новый язык программирования.

* **Получение аналитических сведений за несколько секунд в практически реальном времени.** Эффективнее используйте данные временных рядов благодаря таким компонентам Time Series Insights, как хранилище, анализ и визуализация, которые теперь доступны в одном расположении. Служба Time Series Insights принимает сотни миллионов событий датчиков в день с задержкой в одну минуту, что позволяет быстро реагировать на изменения. Time Series Insights позволяет подробнее узнать о данных датчика, быстро определять тенденции и аномалии, чтобы вы могли легко анализировать их сложные первопричины и избегать дорогостоящих простоев. Включив перекрестную корреляцию между данными реального времени и хронологическими данными, Time Series Insights позволяет пользователям выявлять скрытые тенденции в своих данных.

* **Создание собственных решений.** Внедряйте данные Azure Time Series Insights в имеющиеся приложения или создавайте настраиваемые решения с помощью REST API службы Time Series Insights.  Кроме того, создавать персонализированные представления и обмениваться ими стало очень просто. Теперь пользователи могут с легкостью изучать ваши открытия.

* **Масштабируемость.** Служба Time Series Insights поддерживает работу в масштабе Интернета вещей. В предварительной версии она может принимать от 1 события до 100 миллионов, срок хранения которых составляет 31 день. Благодаря Time Series Insights пользователи могут визуализировать и анализировать динамические потоки данных практически в режиме реального времени, а также большие объемы хронологических данных. В дальнейшем процент входящего трафика и сохранения клиентов будет увеличен, чтобы гарантировать прогресс в темпе развития организации.

## <a name="taxonomy-of-time-series-insights"></a>Таксономия Time Series Insights

* **Среда.** Среда — это ресурс Azure с входящим трафиком и объемом хранилища.  Клиенты подготавливают среды через портал Azure, используя требуемую емкость.
* **Источник событий.** Источник событий является производным от брокеров событий, таких как концентраторы событий Azure.  Time Series Insights подключаются напрямую к источникам событий, принимая поток данных, при этом у пользователей нет необходимости писать код. Сейчас Time Series Insights поддерживает концентраторы событий Azure и Центра Интернета вещей Azure. В будущем будут добавлены дополнительные источники событий.
* **Ссылочные данные.** Time Series Insights предоставляют пользователям возможность объединить данные временных рядов со ссылочными данными.  Ссылочные данные могут включать метаданные об устройствах или другие статические данные, которые изменяются относительно редко. Time Series Insights объединяют ссылочные данные и потоки данных, позволяя пользователям визуализировать и анализировать эти данные практически в реальном времени.
