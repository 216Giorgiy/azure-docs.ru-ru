# <a name="quickstart-configure-your-iot-solution"></a>Краткое руководство. Настройка решения Интернета вещей

> [!IMPORTANT]
> Центр безопасности Azure (ASC) для Интернета вещей сейчас предоставляется в общедоступной предварительной версии.
> Эта предварительная версия предоставляется без соглашения об уровне обслуживания и не рекомендована для использования рабочей среде. Некоторые функции могут не поддерживаться или их возможности могут быть ограничены. Дополнительные сведения см. в статье [Дополнительные условия использования предварительных выпусков Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).

В этой статье объясняется, как выполнить начальную настройку решения для обеспечения безопасности Интернета вещей с помощью ASC для Интернета вещей. 

## <a name="asc-for-iot"></a>ASC для Интернета вещей

ASC для Интернета вещей обеспечивает эффективную комплексную безопасность для решений Интернета вещей на основе Azure.

С помощью ASC для Интернета вещей можно отслеживать все ваше решение Интернета вещей на одной панели мониторинга, где отображаются все ваши устройства Интернета вещей, платформы Интернета вещей и серверные ресурсы в Azure.

После включения в Центре Интернета вещей ASC для Интернета вещей автоматически идентифицирует другие службы Azure, также подключенные к вашему Центру Интернета вещей и связанные с вашим решением Интернета вещей.

В дополнение к автоматическому обнаружению связей можно также выбрать другие ресурсы Azure, которые следует пометить как часть решения Интернета вещей.
Выбранные параметры позволяют добавлять целые подписки, группы ресурсов или отдельные ресурсы.

После определения всех связей ресурсов ASC для Интернета вещей использует Центр безопасности Azure, чтобы предоставлять рекомендации по безопасности и оповещения для этих ресурсов.

## <a name="add-azure-resources-to-your-iot-solution"></a>Добавление ресурсов Azure в решение Интернета вещей

Чтобы добавить новый ресурс в решение Интернета вещей, сделайте следующее. 

1. Откройте **Центр Интернета вещей** на портале Azure. 
2. Щелкните и откройте **Ресурсы** в разделе **Безопасность** в меню слева. 
3. Щелкните **Добавление ресурсов**.
4. Выберите ресурсы, принадлежащие вашему решению Интернета вещей.
4. Щелкните **Добавить**. 

Поздравляем! Вы добавили новый ресурс в решение Интернета вещей.

ASC для Интернета вещей теперь отслеживает недавно добавленные ресурсы, а также отображает соответствующие рекомендации по безопасности и оповещения, как часть вашего решения Интернета вещей.
