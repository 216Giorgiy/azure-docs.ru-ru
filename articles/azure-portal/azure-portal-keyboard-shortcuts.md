# <a name="azure-portal-keyboard-shortcuts"></a>Сочетания клавиш портала Azure
## <a name="open-hubs"></a>Открыть центры
| Клавиша | Действие |
| --- | --- |
| Нет |Уведомления |
| / |Обзор служб |
| A |Активные сеансы |
| B |Выставление счетов |
| C |Создание |

## <a name="navigation"></a>Навигация
| Клавиша | Действие |
| --- | --- |
| H |Начальная панель мониторинга Azure |
| F |Первая открытая колонка |
| J |Ранее открытая колонка |
| K |Следующая открытая колонка |
| L |Последняя открытая колонка |
| - |Свернуть колонку или восстановить из развернутого состояния |
| + |Развернуть колонку или восстановить из свернутого состояния |
| # |Закрыть текущую колонку |
| ¦ |Закрыть все колонки |
| , |Переместить фокус на командную строку |
| . |Переместить фокус на панель навигации |
| ? |Просмотреть сочетания клавиш |
