# Ограничения DocumentDB для предварительного выпуска
В таблице ниже описаны ограничения и параметры применения дисковых квот для DocumentDB в предварительном выпуске. В большинстве случаев ограничения применяются либо с намерением получить ваши отзывы или на основании текущих ограничений емкости ресурсов. Если вы хотите уменьшить ограничения, позвоните нам, и мы сделаем все от нас зависящее в рамках публичной оферты.    



|объект |Дисковая квота на (стандартная оферта для предварительного выпуска)|
|-------|--------|
|Учетные записи базы данных     |5
|Количество баз данных в учетной записи базы данных     |100
|Количество пользователей в учетной записи базы данных (для всех баз данных) |500,000
|Количество разрешений в учетной записи базы данных (для всех баз данных)   |2,000,000
|Объем хранилища вложений в учетной записи базы данных      |2 ГБ
|Максимальное количество единиц емкости в учетной записи базы данных       |5
|Количество коллекций в единице емкости      |3
|Минимальный выделенный объем хранилища для коллекции, содержащей по крайней мере один документ    |3,3 ГБ
|Минимальная выделенная пропускная способность для коллекции, содержащей по крайней мере один документ |667 единиц запросов

|Максимальное количество уникальных путей в коллекции       |100
