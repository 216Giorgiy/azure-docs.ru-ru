# Ограничения DocumentDB для предварительного выпуска
В следующей таблице указаны ограничения и квоты, применяемые к DocumentDB на этапе предварительной версии. В большинстве случаев ограничения применяются либо с намерением получить ваши отзывы или на основании текущих ограничений емкости ресурсов. Если у вас возникнет необходимость увеличить указанные пределы, позвоните нам, и мы сделаем все от нас зависящее в рамках ограничений публичной оферты.    

|Сущность |Квота (стандартное предложение для предварительной версии)|
|-------|--------|
|Учетные записи базы данных     |5
|Число баз данных на учетную запись базы данных     |100
|Число пользователей на учетную запись базы данных - для всех баз данных |500 000
|Число разрешений на учетную запись базы данных - для всех баз данных |2 000 000
|Хранилище вложений на учетную запись базы данных      |2 ГБ
|Максимальное число единиц емкости на учетную запись базы данных       |50
|Число коллекций на единицу емкости     |3
|Минимальное выделенное хранилище на коллекцию хотя бы с 1 документом    |3,3 ГБ

|Максимальное число единиц запросов в секунду на коллекцию   |2000

|Подготавливаемое хранилище документов на единицу емкости |10 ГБ
