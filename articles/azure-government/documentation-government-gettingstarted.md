#  Приступая к работе с Azure для государственных организаций

##  Общие рекомендации для клиентов Azure для государственных организаций

Так как большая часть доступных в настоящее время технических материалов исходит из того, что приложения разрабатываются для глобальной службы, а не для службы Microsoft Azure для государственных организаций, важно ознакомить разработчиков с основными отличиями приложений, разработанных специально для размещения в этой службе.

- Во-первых, есть различия в работе служб и функций, то есть определенные функции в некоторых регионах глобальной службы могут оказаться недоступными в Azure Government.

- Во-вторых, для функций, доступных в Azure Government, есть изменения в процессе конфигурации по отношению к глобальной службе. Таким образом, необходимо изучить примеры кода, конфигурации и действия, необходимые для построения и выполнения приложений в среде облачных служб Azure Government.

- В-третьих, следует ознакомиться с документацией технических служб Azure для государственных организаций, доступной на этом сайте. Она содержит информацию об области действия Azure для государственных организаций, а также инструкции и рекомендации по клиентским данным, подлежащим регулированию или контролю.

Чтобы получать дополнительные сведения и обновления, подпишитесь на <a href="https://blogs.msdn.microsoft.com/azuregov/">блог Microsoft Azure для государственных организаций. </a>

##  ITAR и Azure для государственных организаций

Информация на этом сайте предназначена для клиентов Azure для государственных организаций — пользователей, у которых есть обязательства согласно правилам международной торговли оружием (ITAR) и которые с помощью служб Azure для государственных организаций планируют обрабатывать, хранить или передавать информацию, подлежащую регулированию. На этом сайте описываются возможности служб Microsoft Azure для государственных организаций и предоставляются общие рекомендации, применимые для пользователей ITAR. Перед добавлением данных, подлежащих контролю согласно ITAR, в подписку Azure для государственных организаций, следует ознакомиться с возможностями Azure для государственных организаций. Если у вас возникнут вопросы, обратитесь в службу поддержки учетных записей.

Актуальные сведения о службах Azure для государственных организаций, соответствующих нормам и нормативам ITAR, см. на <a href="http://www.microsoft.com/ru-RU/TrustCenter/Compliance/default.aspx/">странице нормативных требований центра управления безопасностью Microsoft Azure</a>. Кроме того, могут быть доступны дополнительные службы Майкрософт, но они не входят в число служб Azure для государственных организаций. Службы Azure для государственных организаций также позволяют использовать различные дополнительные ресурсы, приложения или службы, предоставляемые сторонними разработчиками или корпорацией Майкрософт на основании отдельных условий использования и политик конфиденциальности. Вы несете ответственность за проверку условий дополнительных предложений (например, предложений Marketplace) на предмет того, что они удовлетворяют требованиям по соответствию нормам ITAR.

<a href="https://azure.microsoft.com/ru-RU/features/gov/"> Служба Azure </a> для государственных организаций доступна компаниям, занимающимся обработкой данных, к которым применяются государственные нормы и требования, например ITAR (если использование этой службы требуется для соблюдения норм). Клиенты Azure для государственных организаций проходят проверку на соответствие требованиям. Эта проверка включает в себя подтверждение того, что вы являетесь производителем, экспортером или посредником в торговле предметами военного снабжения согласно ITAR. Для этого нужна регистрация в Государственном департаменте США или спонсорский договор с государственным учреждением, которое предъявляет определенные требования к обработке данных.

Если у вас есть вопросы о требованиях служб Azure для государственных организаций, вам следует обратиться к своей службе поддержки учетных записей.
