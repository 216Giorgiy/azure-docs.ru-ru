#<a name="azure-government-documentation"></a>Документация по Azure для государственных организаций

Microsoft Azure для государственных организаций предоставляет облачную платформу, основанную на фундаментальных принципах обеспечения безопасности, конфиденциальности и управления, соответствия требованиям и прозрачности. Субъекты государственного сектора получают физически изолированный экземпляр Microsoft Azure, который использует службы обеспечения соответствия требованиям и безопасности мирового уровня, которые критически важны для всех систем и приложений правительства США, построенных на основе данной архитектуры.
