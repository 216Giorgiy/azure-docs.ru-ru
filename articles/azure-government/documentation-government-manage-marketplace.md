# <a name="azure-marketplace-for-government"></a>Azure Marketplace для государственных организаций
Azure Marketplace доступен для Azure для государственных организаций с обновленным списком образов от издателей Marketplace. 

## <a name="variations"></a>Варианты
Ниже приведены рекомендации по использованию Azure Marketplace для государственных организаций.

* Поддерживаются только образы с использованием собственных лицензий (BYOL). Любые образы, требующие транзакций приобретения в Azure Marketplace, недоступны.
* Сейчас доступна только часть образов, имеющихся в общедоступном магазине. Список доступных образов см. [здесь](../azure-government-image-gallery.md). 
* Перед подготовкой образа администратору предприятия нужно включить возможность совершать покупки в Marketplace для подписки Azure.
  * Войдите на портал как администратор предприятия.
  * Перейдите к разделу *Управление*.
  * В разделе *Сведения о регистрации* щелкните значок карандаша возле позиции *Azure Marketplace*.
  * Установите переключатель *Включено/Отключено* соответствующим образом.
  * Нажмите кнопку *Сохранить*

> [!NOTE]
> Дополнительные сведения о предоставлении доступа к образам в Azure для государственных организаций см. в [рекомендациях по адаптации от партнеров](documentation-government-manage-marketplace-partners.md).
> 
> 

### <a name="step-1"></a>Шаг 1
Запустите Marketplace.

![замещающий текст](./media/government-manage-marketplace-launch.png)  

### <a name="step-2"></a>Шаг 2
Просмотрите различные продукты, чтобы найти нужный.

В описании продукта издатель Marketplace указывает список сертификатов, который поможет вам сделать правильный выбор. 

![замещающий текст](./media/government-manage-marketplace-service.png)

### <a name="step-3"></a>Шаг 3.
Выберите продукт или образ.

![замещающий текст](./media/government-manage-marketplace-image.png)

### <a name="step-4"></a>Шаг 4.
Запустите процесс создания и введите необходимые параметры для развертывания.

![замещающий текст](./media/government-manage-marketplace-deployment.png)

> [!NOTE]
> В раскрывающемся списке "Расположение" отображаются только расположения Azure для государственных организаций.
> 
> 

### <a name="step-5"></a>Шаг 5
Просмотрите сведения о ценах.

![замещающий текст](./media/government-manage-marketplace-pricing.png)

### <a name="step-6"></a>Шаг 6
Выполните все шаги и нажмите кнопку "ОК", чтобы начать процесс подготовки.
