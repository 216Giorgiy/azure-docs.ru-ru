# Функция поиска в Operational Insights
Основой службы Microsoft Azure Operational Insights является функция поиска, позволяющая объединять и сопоставлять данные компьютера из нескольких источников в существующей среде. Функция поиска поддерживает пакеты аналитики, которые объединяют важные показатели по устранению проблем для каждой из затронутых областей.


## Поиск данных
Функцию **поиска** в службе Azure Operational Insights можно использовать для поиска необходимых данных. 

![Image of the Search tile](./media/operational-insights-search/overview-search.png)

На странице **Поиск** можно создать запрос, а затем во время поиска отфильтровать результаты с помощью элементов управления аспектами. Можно создать сложные запросы для преобразования и фильтрации результатов, а также для формирования соответствующих отчетов.

![Image of the Search page](./media/operational-insights-search/search-01.png)

Общие запросы поиска отображаются на большинстве страниц поиска. В консоли Operational Insights можно щелкать плитки или переходить к другим элементам, чтобы просмотреть сведения о них с помощью функции **Поиск**.
Чтобы сузить область результатов, можно воспользоваться элементами управления аспектами или фильтрами. Ввод с возможностью автозаполнения помогает понять синтаксис и быстро возвращает результаты.
