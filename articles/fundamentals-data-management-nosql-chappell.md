# Управление данными: Основные сведения о технологиях NoSQL в Azure

**Сводка:** Azure предлагает широкий диапазон вариантов для рабочих нагрузок NoSQL. Но определение различий между всеми доступными вариантами может оказаться сложной задачей. В этом документе описываются технологии NoSQL в Azure, и как управление в них отличается от реляционной модели управления данными.

Реляционные технологии уже давно являются главным подходом для работы с данными. Однако при работе с большим количеством различных типов данных, что сегодня происходит все чаще, возникают новые сложности по управлению этими объемами. Реляционная технология не всегда подходит для управления этими данными. Это руководство содержит пошаговые инструкции по вариантам с разъяснениями относительно возможностей каждого из них и потенциальных причин его выбора.

* Реляционные технологии Azure: Быстрый обзор

	* Аналитика больших объемов данных: HDInsight
