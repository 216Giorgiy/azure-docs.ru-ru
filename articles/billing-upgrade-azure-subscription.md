# <a name="upgrade-your-free-azure-subscription-to-pay-as-you-go"></a>Обновление бесплатной подписки Azure до подписки с оплатой по мере использования
[Бесплатную пробную версию](https://azure.microsoft.com/free/) или подписку [Microsoft Imagine](https://azure.microsoft.com/offers/ms-azr-0144p/) можно обновить до [подписки с оплатой по мере использования](https://azure.microsoft.com/offers/ms-azr-0003p/) на портале учетных записей Azure. 

1. Войдите на [портал учетных записей Azure](https://account.windowsazure.com/subscriptions) и перейдите на вкладку **Подписки**.
2. В разделе состояния подписки выберите баннер **Обновить**.
   
    ![Не забудьте обновить подписку с бесплатной пробной версии до версии с оплатой по мере использования](./media/billing-upgrade-azure-subscription/billpage.png)
3. Подтвердите обновление.
   
    ![Подтверждение обновления подписки](./media/billing-upgrade-azure-subscription/Upgrade.png)

При обновлении бесплатной пробной версии до подписки с оплатой по мере использования будут действовать все преимущества, которыми вы не успели воспользоваться в бесплатной пробной версии, такие как период ее предоставления 30 дней или кредит 200 долларов на службы Azure, пока одно из них не будет исчерпано. После этого подписка обновится.
