# <a name="use-cost-management-reports"></a>Использование отчетов службы "Управление затратами"

В этой статье описано, как использовать различные отчеты службы "Управление затратами" на портале Cloudyn. Большинство отчетов Cloudyn интуитивно понятны и имеют однородный внешний вид. Общие сведения об отчетах Cloudyn см. в статье [Интерпретация отчетов о затратах](understading-cost-reports.md). В статье также описаны различные параметры и поля, используемые в отчетах.

## <a name="cost-analysis-reports"></a>Отчеты по анализу затрат

В отчетах по анализу затрат отражены данные о выставлении счетов поставщиками облачных служб. С помощью отчетов можно группировать и подробно изучить различные сегменты данных, перечисленные в файле выставления счетов. Отчеты обеспечивают детализированную навигацию по сведениям о стоимости в необработанных данных выставления счетов поставщиков.

Отчеты с анализом затрат не группируют затраты по тегам. Отчеты, основанные на тегах, доступны только в отчетах о распределении затрат, заданных после создания модели затрат с использованием диаграммы комплексного распределения затрат.

### <a name="actual-cost-analysis"></a>Анализ фактических затрат

Анализ фактических затрат показывает основные расходы, включая текущие расходы и единовременные платы.

 Используйте анализ фактических затрат для:

- анализа и мониторинга фактических затрат в заданном интервале времени;
- планирования предупреждения о пороговом значении;
- анализа стоимости виртуальных счетов и возвратных платежей.

#### <a name="to-use-the-actual-cost-analysis-report"></a>Использование отчета по анализу фактических затрат

Как минимум выполните следующие действия. Можно также использовать другие параметры и поля.

1. Выберите диапазон дат.
2. Выберите фильтр.

Можно щелкнуть правой кнопкой мыши результаты отчета, чтобы развернуть их и просмотреть более подробную информацию.

![Пример отчета анализа фактических затрат](./media/use-reports/actual-cost-analysis.png)

### <a name="actual-cost-over-time"></a>Фактические затраты за период времени

Отчет по фактическим затратам за период времени — это стандартный отчет по анализу затрат, распределяющий затраты в определенном временном разрешении. В отчете показаны расходы с течением времени, позволяющие отслеживать тенденции и обнаруживать несоответствия стоимости. В этом отчете показаны основные затраты, в том числе текущие расходы и единовременные расходы на зарезервированный экземпляр в течение выбранного периода времени.

Используйте отчет по фактическим затратам за период времени, чтобы:

- отслеживать тенденции затрат по времени;
- находить несоответствия стоимости;
- найти все вопросы, связанные с затратами, которые связаны с Amazon Web Services.

#### <a name="to-use-the-actual-cost-over-time-report"></a>Использование отчета по фактическим затратам за период времени

Как минимум выполните следующие действия. Можно также использовать другие параметры и поля.

- Выберите диапазон дат.

Например, вы можете выбирать группы для просмотра их стоимости с течением времени, а затем добавить фильтры, чтобы сузить результаты.

![Пример отчета по фактическим затратам за период времени](./media/use-reports/actual-cost-over-time.png)



### <a name="amortized-cost-reports"></a>Отчет по амортизированной стоимости

В этом наборе отчетов по амортизированной стоимости представлены преобразованные в линейный вид тарифы на обслуживание, не связанные с использованием, или одноразово оплачиваемые расходы, равномерно распределенные по времени в течение их срока действия.

Например, одноразовая плата может включать:

- годовую плату за поддержку;
- годовую плату за обеспечение безопасности;
- плату за покупку зарезервированных экземпляров;
- некоторые элементы Azure Marketplace.

В файле выставления счетов единовременные сборы характеризуются, когда даты начала и завершения использования службы (или метка времени) имеют одинаковые значения. Затем Cloudyn распознает их как единовременные сборы, которые могут быть амортизированы. Другие службы с оплатой по мере использования с потреблением по запросу не могут быть амортизированы.

Чтобы показать амортизированные затраты, просмотрите приведенный ниже рисунок примера по фактическим затратам за период времени. В примере показан пик затрат (23 августа). Это может показаться аномалией по сравнению с обычными суточными расходами. При анализе первопричин и перемещения в базе данных определилось, что это затраты на ежегодное резервирование APN службы AWS, которые представляют собой разовую плату, выставленную в этот день. Вы можете увидеть, как эти затраты амортизируются в следующем разделе.

![Пример отчета по фактическим затратам за период времени с демонстрацией единовременных затрат](./media/use-reports/actual-amort-example.png)

#### <a name="to-use-the-amortized-cost-over-time-report"></a>Чтобы использовать отчет по амортизации затрат за период времени, сделайте следующее:

Как минимум выполните следующие действия. Можно также использовать другие параметры и поля.

1. Выберите диапазон дат.
2. Выберите службу и поставщик.

Выполняя предыдущий пример, можно видеть, что одноразовая стоимость теперь амортизируется на следующем изображении:

На предыдущем рисунке показана амортизированная стоимость резервирования APN за период времени. В этом отчете указывается одноразовая амортизация платы и стоимость APN в качестве приобретения годового резервирования. Стоимость APN распределяется равномерно ежедневно, как 1/365 зарезервированных начальных трат.

## <a name="cost-allocation-analysis-reports"></a>Отчеты по анализу распределения затрат

Отчеты по анализу распределения затрат станут доступны после создания модели затрат с помощью диаграммы комплексного распределения затрат. Cloudyn обрабатывает данные затрат и выставления счетов и сопоставляет их с потреблением и тегами облачных учетных записей. Чтобы сопоставить данные, Cloudyn требуется доступ к данным об использовании. Учетные записи, для которых отсутствуют учетные данные, помечены как ресурсы без категории.

### <a name="cost-analysis-report"></a>Отчет анализа затрат

В отчете по анализу затрат предоставляется аналитика о потреблении облачных ресурсов и затратах за выбранный интервал времени. Политики, заданные в диспетчере распределения затрат, используются в отчете по анализу затрат.

Как Cloudyn рассчитывает этот отчет?

Cloudyn гарантирует, что выделение обеспечивает целостность каждой связанной учетной записи за счет сопоставления учетных записей. Сопоставление гарантирует, что учетная запись, которая не использует определенную службу, не понесет никаких затрат за эту выделенную для нее службу. Расходы, вычисленные в этой учетной записи, остаются в этой учетной записи и не рассчитываются политиками распределения. Например, у вас может быть 5 связанных учетных записей. Если только три из них используют службы хранения, то стоимость услуг хранения распределяется только по тэгам в трех учетных записях.

 Используйте отчет по анализу затрат для:

- отображения вычисленного представления всего развертывания в течение определенного периода времени;
- просмотра затрат по категориям тегов на основе политики, созданной в модели затрат.

#### <a name="to-use-the-cost-analysis-report"></a>Чтобы использовать отчет анализа затрат:

1. Выберите диапазон дат.
2. Добавьте теги (при необходимости).
3. Добавьте группы.
4. Выберите модель затрат, созданную ранее.

### <a name="cost-over-time-report"></a>Отчет по затратам за период времени

В отчете по затратам за период времени отображаются расходы за период времени, поэтому можно определять тенденции и обнаруживать несоответствия в развертывании. Это, по существу, показывает затраты, распределенные в течение определенного периода. В этот отчет включены основные затраты, в том числе текущие расходы и единовременные расходы за зарезервированный экземпляр, которые расходуются в течение выбранного периода времени. Политики, заданные в Cost Manager 360°, можно использовать в этом отчете.

Используйте отчет по затратам за период времени, чтобы:

- просматривать изменения с течением времени, которые влияют на изменения с одного дня (или диапазона дат) до следующего;
- анализировать затраты с течением времени для конкретного экземпляра;
- понять, почему произошло увеличение затрат для определенного экземпляра.

#### <a name="to-use-the-cost-over-time-report"></a>Чтобы использовать отчет по затратам за период времени:

1. Выберите диапазон дат.
2. Добавьте теги (при необходимости).
3. Добавьте группы.
4. Выберите модель затрат, созданную ранее.
5. Выберите фактические расходы или расходы на амортизацию.
6. Выберите, следует ли применять правила распределения для просмотра необработанных данных об оплате или для пересчета стоимости в представлении Cloudyn.
