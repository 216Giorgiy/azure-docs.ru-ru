# <a name="app-service-on-linux-documentation"></a>Документация по службе приложений на платформе Linux
В настоящее время служба приложений Azure на платформе Linux доступна в виде общедоступной предварительной версии. Она позволяет клиентам выполнять веб-приложения изначально на платформе Linux. Это обеспечивает лучшую совместимость определенных типов приложений и упрощает процесс переноса готовых веб-приложений, размещенных на платформе Linux, в службу приложений Azure.

Клиенты могут создавать веб-приложения на платформе Linux только в рамках плана службы приложений с поддержкой Linux. В рамках этого плана можно размещать только веб-приложения на базе Linux. 

> Цена на общедоступную предварительную версию службы приложений на платформе Linux на 50 % ниже обычной цены. Она рассчитывается на основе количества часов вычислений, потребленных в рамках используемого плана службы приложений.
> 
> На этапе общедоступной предварительной версии соглашение об уровне обслуживания не предоставляется.
