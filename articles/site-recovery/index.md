# <a name="site-recovery-documentation"></a>Документация по Site Recovery

Служба Azure Site Recovery помогает реализовать стратегию непрерывности бизнес-процессов и аварийного восстановления (BCDR), выполняя репликацию локальных виртуальных машин и физических серверов. Серверы и виртуальные машины можно реплицировать из основного локального центра обработки данных в Azure или в дополнительный центр обработки данных. При сбоях основного сайта выполняется отработка отказа на дополнительный сайт. Это обеспечивает доступность приложений и рабочих нагрузок.
