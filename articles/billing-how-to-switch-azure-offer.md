# Переключение подписки Azure на другое предложение

## Руководство по переходу на другое предложение

> [AZURE.VIDEO switch-to-a-different-azure-offer]

1.	Войдите в [Центр управления учетной записью Azure](https://account.windowsazure.com/Subscriptions).
2.	Выберите подписку, которую вы хотите изменить.
3.	Щелкните **Переключиться на другое предложение**. [Если эта кнопка не отображается](#where-is-the-button), узнайте причину.

	![siwtchbutton](./media/billing-how-to-switch-azure-offer/switchbutton.png)
	
4.	Появятся все предложения, на которые можно переключить вашу подписку. Список будет зависеть от того, с какими группами связана ваша учетная запись, а также от вашего текущего предложения. Дополнительные сведения о каждом предложении можно получить, щелкнув расположенную под ним ссылку **Подробнее**. Щелкните нужное предложение, чтобы перейти к следующему шагу.

	![selectoffer](./media/billing-how-to-switch-azure-offer/selectoffer.png)
5.	При переходе на некоторые предложения может появляться уведомление о последствиях перехода. Внимательно ознакомьтесь с этим списком и выполните указанные действия, прежде чем продолжить.

	![thingstonote](./media/billing-how-to-switch-azure-offer/thingstonote.png)
6.	Подписку можно переименовать. По умолчанию ей присваивается имя нового предложения. Чтобы завершить процедуру, щелкните **Переключить предложение**.

	![confirmpage](./media/billing-how-to-switch-azure-offer/confirmpage.png)
7.	Готово! Подписка переведена на новое предложение.

## Часто задаваемые вопросы

### Какие предложения доступны для переключения?

См. список [доступных предложений, на которые можно перейти](#what-you-need-to-know).

### Почему не отображается кнопка?

Кнопка **Переключиться на другое предложение** может не отображаться в перечисленных ниже случаях.

- Еще не завершился ваш первый цикл выставления счетов; необходимо дождаться формирования первого счета, и после этого вы сможете изменить предложение.

- У вас нет права на изменение предложения. Проверьте список [доступных предложений, на которые вы можете переключиться](#what-you-need-to-know), чтобы убедиться, что необходимое вам предложение поддерживается.

### Повлияет ли смена предложения на мое обслуживание?

Служба продолжает работать без простоя. Однако предложение, на которое вы переключаетесь, может включать ограничения, например, некоторые предложения запрещают использование в рабочей среде. В этом случае рабочие ресурсы необходимо переместить в другую подписку.

### Повлияет ли смена предложения на администраторов службы? 

На связанных с подпиской пользователей переключение не влияет.

### Как изменится выставление счетов при смене предложения?

В день смены предложения будет составлен счет на все непогашенные задолженности. После этого счет за подписку будет выставляться по тарифам нового предложения. Дата взыскания годовой абонентской платы изменится в соответствии с датой смены предложения. Данные по использованию ресурсов и платежам до смены предложения не сохраняются, поэтому рекомендуем скачать их перед сменой тарифа.

> [AZURE.NOTE] Из-за ограничений, связанных с выставлением счетов, предложение нельзя изменить до окончания первого цикла выставления счетов после создания подписки.

### Можно ли использовать этот механизм для перехода с Web Direct на предложение [Поставщик облачных решений](https://partner.microsoft.com/Solutions/cloud-reseller-overview) (CSP) или [Соглашение Enterprise](https://azure.microsoft.com/pricing/enterprise-agreement/) (EA)?

Этот механизм можно использовать только для смены предложения Web Direct или перехода на это предложение. Для перевода имеющейся подписки на предложение "Соглашение Enterprise" попросите своего администратора регистрации добавить вашу учетную запись в соглашение Enterprise. После этого вы получите приглашение по электронной почте. Как только вы примете приглашение согласно инструкциям, ваши подписки будут автоматически переведены в соглашение Enterprise. Перевести действующую подписку Web Direct на CSP в настоящее время нельзя.
