# Создание пользовательского образа лаборатории для разработки и тестирования из файла VHD

## Обзор

После [создания лаборатории для разработки и тестирования](devtest-lab-create-lab.md) вы сможете [добавить в эту лабораторию виртуальные машины](devtest-lab-add-vm-with-artifacts.md). При создании виртуальной машины необходимо указать *базовый образ* виртуальной машины, который может представлять собой *пользовательский образ* или *образ из Marketplace*. В этой статье будет показано, как создать пользовательский образ из VHD-файла. Обратите внимание, что для выполнения всех шагов в этой статье вам потребуется действительный VHD-файл.

## Создание пользовательского образа

1. Войдите на [портал Azure](https://portal.azure.com).

1. Выберите **Обзор**, а затем в списке выберите **Лаборатории для разработки и тестирования**.

1. В списке лабораторий найдите нужную лабораторию.

1. Появится колонка **Параметры** для выбранной лаборатории.

1. В колонке **Параметры** выберите **Custom images** (Пользовательские образы).

    ![Параметр "Пользовательские образы"](./media/devtest-lab-create-template/lab-settings-custom-images.png)

1. В колонке **Custom images** (Пользовательские образы) выберите **+ Custom Image** (+ пользовательский образ).

    ![Добавить пользовательский образ](./media/devtest-lab-create-template/add-custom-image.png)

1. Введите имя пользовательского образа. Это имя отображается в списке базовых образов при создании новой виртуальной машины.

1. Введите описание пользовательского образа. Это описание отображается в списке базовых образов при создании новой виртуальной машины.

1. Выберите **VHD-файл**.

1. Если у вас есть VHD-файл, который отсутствует в списке, добавьте его, следуя инструкциям в разделе [Upload a VHD file](#upload-a-vhd-file) (Отправка VHD-файла), а затем вернитесь к этому руководству.

1. Выберите нужный VHD-файл.

1. Выберите **ОК**, чтобы закрыть колонку **VHD-файл**.

1. Нажмите **Конфигурация ОС**.

1. На вкладке **Конфигурация ОС** выберите **Windows** или **Linux**.

1. Если выбран вариант **Windows**, с помощью флажка укажите, будет ли запущена программа *Sysprep*.

1. Нажмите кнопку **ОК** закройте колонку **Конфигурация ОС**.

1. Нажмите кнопку **ОК**, чтобы создать пользовательский образ.

1. Перейдите к разделу [Дальнейшие действия](#next-steps).

##Отправка VHD-файла

Чтобы добавить новый пользовательский образ, у вас должен быть доступ к VHD-файлу.

1. В колонке **VHD-файл** выберите **Upload a VHD file using PowerShell** (Отправить VHD-файл с помощью PowerShell).

    ![Отправить образ](./media/devtest-lab-create-template/upload-image-using-psh.png)

1. В следующей колонке будут отображены инструкции по изменению и запуску сценария PowerShell, который отправляет VHD-файл в вашу подписку Azure. **Примечание.** В зависимости от размера VHD-файла и скорости подключения этот процесс может занять длительное время.

##Дальнейшие действия

После добавления пользовательского образа для виртуальной машины необходимо [добавить виртуальную машину в лабораторию для разработки и тестирования](./devtest-lab-add-vm-with-artifacts.md).
