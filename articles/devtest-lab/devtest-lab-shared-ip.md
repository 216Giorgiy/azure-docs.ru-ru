# <a name="understand-shared-ip-addresses-in-azure-devtest-labs"></a>Общие IP-адреса в Azure DevTest Labs

Azure DevTest Labs использует общие IP-адреса, чтобы минимизировать количество общедоступных IP-адресов, необходимых для доступа к отдельным виртуальным машинам лаборатории.  В этой статье описывается принцип работы общих IP-адресов и варианты их настройки.

## <a name="shared-ip-setting"></a>Настройка общих IP-адресов

Когда создается лаборатория, она размещается в подсети виртуальной сети.  При создании этой подсети параметру **Enable shared public IP** (Включить совместное использование общедоступного IP-адреса) по умолчанию задается значение *Yes* (Да).  При такой конфигурации создается общедоступный IP-адрес для всей подсети.  После создания лаборатории этот параметр можно отключить, выполнив действия, описанные в [этой статье](devtest-lab-configure-vnet.md).

![Создание подсети лаборатории](media/devtest-lab-shared-ip/lab-subnet.png)

На всех виртуальных машинах, созданных в этой лаборатории, по умолчанию используется общий IP-адрес.  При создании виртуальной машины этот параметр отображается в колонке **Дополнительные параметры** в разделе **Настройка IP-адреса**.

![Создание виртуальной машины](media/devtest-lab-shared-ip/new-vm.png)

Когда в подсеть добавляется виртуальная машина с включенным общим IP-адресом, общедоступному IP-адресу назначается TCP-порт, выполняющий переадресацию на порт RDP на виртуальной машине.  

## <a name="using-the-shared-ip"></a>Использование общего IP-адреса

Для подключения к удаленному рабочему столу на виртуальной машине в RDP-клиенте введите IP-адрес или полное доменное имя, после него поставьте двоеточие, а затем укажите номер порта.  Например, на приведенном ниже рисунке таким адресом RDP для подключения к виртуальной машине является `doclab-lab13998814308000.centralus.cloudapp.azure.com:51686`.  Также можно нажать кнопку **Подключить** на портале Azure и скачать предварительно настроенный RDP-файл.

![Пример виртуальной машины](media/devtest-lab-shared-ip/vm-info.png)
