#Где я могу найти сведения о преимуществах и ценах на службы Azure?

Microsoft Azure позволяет быстро разворачивать нужные для вашей деятельности инфраструктуры и службы. Вы можете выполнять приложения из ОС Windows и Linux в 19 региональных центрах обработки данных Azure, предлагаемых в рамках соглашений об уровне обслуживания корпоративного класса. Дополнительные сведения о преимуществах служб Azure см. в разделе [Сведения о предложении Microsoft Azure](https://azure.microsoft.com/support/legal/offer-details/). Дополнительные сведения о ценах см. на странице [Сведения о ценах на Azure](https://azure.microsoft.com/pricing/).
