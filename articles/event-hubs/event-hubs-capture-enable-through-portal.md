# Включение записи концентраторов событий с помощью портала Azure

Запись можно настроить при создании концентратора событий с помощью [портала Azure](https://portal.azure.com). Чтобы включить запись, нажмите кнопку **Вкл.** в колонке **Создание концентратора событий** на портале. Затем настройте учетную запись хранения и контейнер, щелкнув в колонке раздел **Контейнер**. Так как для функции записи концентраторов событий и хранилища используется проверка подлинности с взаимодействием между службами, указывать строку подключения к хранилищу не нужно. Средство выбора ресурсов автоматически выбирает универсальный код ресурса (URI) для вашей учетной записи хранения. При использовании Azure Resource Manager этот универсальный код ресурса необходимо явным образом указать как строку.

Продолжительность времени окна по умолчанию составляет 5 минут. Минимальное значение равно 1, а максимальное — 15. Диапазон **окна размера** составляет 10–500 МБ.

## Добавление функции записи в существующий концентратор событий

Запись можно настроить в существующих концентраторах событий, расположенных в соответствующих пространствах имен. Эта функция недоступна в пространствах имен **обмена сообщениями** и в пространствах имен **смешанного типа**. Чтобы включить запись в существующем концентраторе событий или изменить параметры записи, щелкните пространство имен. Откроется колонка **Основные компоненты**. Затем щелкните концентратор событий, для которого необходимо включить запись или изменить ее параметры. Наконец, щелкните в отобразившейся колонке раздел **Свойства**, как показано на следующем рисунке.
