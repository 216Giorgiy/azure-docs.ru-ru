# Получение данных телеметрии и событий приложения в нужном масштабе
 
Современные приложения в ходе свой работы генерируют большой объем информации. Будь то действия пользователя в мобильном приложении или данные о переходах пользователей на веб-сайте, эти данные телеметрии являются ценной информацией как для долгосрочной аналитики, так и для наблюдения за работоспособностью системы в режиме реального времени или поддержки принятия решений. Концентраторы событий Microsoft Azure предоставляют средства, необходимые для отслеживания и обработки этой информации.
