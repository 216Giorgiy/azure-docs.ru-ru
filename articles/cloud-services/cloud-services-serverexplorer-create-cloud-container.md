# Использование Visual Studio для создания контейнера облачной службы

Чтобы просмотреть **обозреватель сервера** в Visual Studio, в панели меню выберите **Представление**, **Обозреватель сервера**. 

1.  Разверните узел **Azure**

2.  Если пользователь не авторизован, автоматически появится диалоговое окно с запросом на вход. Следуйте инструкциям для входа в подписку Azure

3.  Щелкните правой кнопкой мыши узел **Облачная служба** и выберите **Создать облачную службу...**

4.  Появится диалоговое окно **Создание облачной службы**. Заполните поля **Подписка**, **Имя**, **Регион или территориальная группа** и нажмите кнопку **Создать**, чтобы создать пустой контейнер облачной службы

## Дальнейшие действия

Теперь, имея контейнер облачной службы, можно [создать и выполнить развертывание](cloud-services-how-to-create-deploy.md) облачной службы.
