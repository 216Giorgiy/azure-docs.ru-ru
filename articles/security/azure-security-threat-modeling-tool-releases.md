# <a name="threat-modeling-tool-releases"></a>Выпуски Threat Modeling Tool

Microsoft Threat Modeling Tool выпущено в виде бесплатного приложения Windows, щелкните [здесь](https://aka.ms/threatmodelingtool), чтобы скачать. Этот механизм доставки позволяет нам отправлять новейшие усовершенствования и исправления клиентам каждый раз при запуске средства, что упрощает обслуживание и использование.

## <a name="system-requirements"></a>Требования к системе

- Поддерживаемые операционные системы
  - Microsoft Windows 10
- Дополнительные требования
  - Для получения обновлений средства и шаблонов требуется подключение к Интернету.
