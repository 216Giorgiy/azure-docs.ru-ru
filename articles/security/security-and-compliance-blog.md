# Блог по безопасности и соответствию требованиям Azure

[Блог группы по безопасности и обеспечению соответствия Azure](https://blogs.msdn.microsoft.com/azuresecurity/) содержит все актуальные сведения о безопасности и соответствии в Azure.

Кроме того, в этом блоге публикуются важные обновления технических статей, посвященных [центру безопасности Azure](https://azure.microsoft.com/documentation/services/security-center/).

Помимо технических сведений в этом блоге публикуются объявления о материалах по безопасности и обеспечению соответствия, появляющихся в блоге Azure.

Обязательно заглядывайте в блог группы по безопасности и обеспечению соответствия Azure, чтобы быть в курсе последних нововведений.
