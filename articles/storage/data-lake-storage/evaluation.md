# <a name="guidance-for-azure-data-lake-storage-gen1-customers"></a>Руководство для клиентов хранилища Azure Data Lake поколения 1

Хранилище Azure Data Lake поколения 2 (предварительная версия) предлагает собственный набор функций хранилища BLOB-объектов, который включает в себя многоуровневое хранение данных, управление жизненным циклом и улучшенные функции безопасности сети. Кроме того, хранилище Azure Data Lake поколения 1 имеет совместимую с Hadoop файловую систему.

Если вы являетесь существующим клиентом хранилища Azure Data Lake поколения 1, корпорация Майкрософт рекомендует оценить все новые и улучшенные возможности хранилища Azure Data Lake поколения 2 и поделиться своими впечатлениями об [использовании предварительной версии](https://aka.ms/adlsgen2signup). Кроме того, хранилище Azure Data Lake поколения 1 будет по-прежнему поддерживаться. Хранилища Azure Data Lake поколения 1 и поколения 2 могут существовать параллельно, так что ваши ресурсы и приложения, использующие версию хранилища Azure Data Lake поколения 1 продолжат работать, но не будут обновляться. Вы можете разработать план миграции для перехода на версию хранилища Azure Data Lake поколения 2 по своему усмотрению.

Если сейчас вы рассматриваете для приобретения хранилище Azure Data Lake поколения 1, корпорация Майкрософт рекомендует использовать предварительную версию хранилища Azure Data Lake поколения 2 и оценить предлагаемые в ней новые и улучшенные возможности. Аналогичным образом, если вы являетесь существующим клиентом хранилища BLOB-объектов, корпорация Майкрософт рекомендует оценить целесообразность использования файловой системы, совместимой с Hadoop, для аналитических потребностей вашей компании.
