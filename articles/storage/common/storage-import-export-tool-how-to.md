# <a name="using-the-azure-importexport-tool"></a>Использование средства импорта и экспорта Azure 

Инструмент импорта и экспорта Azure (WAImportExport.exe) используется для создания заданий службы импорта и экспорта Azure и управления ими, что позволяет передавать большие объемы данных в хранилище BLOB-объектов Azure или из него.

Данный документ предназначен для самой последней версии средства импорта и экспорта Azure. Сведения об использовании средства версии 1 см. в разделе [Использование средства импорта и экспорта Azure версии 1](storage-import-export-tool-how-to-v1.md).

В этих статьях показано, как использовать средство для выполнения следующих задач.  

- Установка и настройка средства импорта и экспорта.
- Подготовка жестких дисков для задания, импортирующего данные с дисков в хранилища BLOB-объектов Azure.
- Просмотр состояния задания с помощью файлов журнала копирования. 
- Исправление задания импорта. 
- Исправление задания экспорта. 
- Устранение неполадок средства импорта и экспорта Azure, возникающих при его использовании.
