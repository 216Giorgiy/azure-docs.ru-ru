#<a name="storage-documentation"></a>Документация по хранилищу

Хранилище Azure — это облачное решение для хранения современных приложений, обеспечивающее устойчивость, доступность и масштабируемость для удовлетворения потребностей клиентов.  Хранилище очень масштабируемое, гибкое и доступное в глобальном масштабе. Кроме того, оно автоматически балансирует нагрузку с учетом трафика.  Служба хранилища Azure включает хранилище BLOB-объектов, таблиц, очередей и файлов, обеспечивая потребности приложений в ресурсах.
