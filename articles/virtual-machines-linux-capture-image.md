# Как записать образ виртуальной машины под управлением Linux ##

Можно использовать образы из коллекции образов для создания виртуальных машин или записать и использовать собственные образы для создания настраиваемых виртуальных машин. Образ — это файл виртуального жесткого диска (VHD), который используется в качестве шаблона для создания виртуальной машины. Образ — это шаблон, так как в отличие от работающей виртуальной машины, у него нет определенных параметров, таких как имя хоста и параметры учетной записи пользователя. Если нужно создать несколько виртуальных машин с одинаковыми настройками, можно записать образ настроенной виртуальной машины и использовать его в качестве шаблона.

1. Подключитесь к виртуальной машине, выполнив действия, описанные в разделе [Как войти в систему на виртуальной машине, работающей под управлением Linux][].

2. В окне SSH введите следующую команду, а затем введите пароль для учетной записи, которую вы создали на виртуальной машине:  Обратите внимание, что выходные данные `waagent` могут незначительно отличаться в зависимости от версии этой служебной программы:

	`sudo waagent -deprovision`

	![Отзыв виртуальной машины](./media/virtual-machines-linux-capture-image/LinuxDeprovision.png)


3. Чтобы продолжить, введите **y**.

	![Отзыв виртуальной машины выполнен успешно](./media/virtual-machines-linux-capture-image/LinuxDeprovision2.png)

4. Введите **Exit**, чтобы закрыть SSH-клиент.

5. На [портале управления](http://manage.windowsazure.com) выберите виртуальную машину и щелкните **Завершение работы**.

	![Завершение работы виртуальной машины](./media/virtual-machines-linux-capture-image/ShutdownVM.png)

6. Щелкните **Да**, чтобы подтвердить свое согласие с выставлением счетов за виртуальную машину, даже если она не работает.

7. Когда виртуальная машина остановлена, в командной строке щелкните **Записать**.

	![Запись образа виртуальной машины](./media/virtual-machines-linux-capture-image/CaptureVM.png)

	Отображается диалоговое окно **Запись виртуальной машины**.
	
	![Введите сведения о записи](./media/virtual-machines-linux-capture-image/CaptureLinux.png)

8.	В поле **Имя образа** введите имя нового образа.

9.	Все образы Linux должны *отзываться* путем выполнения команды `waagent` с параметром отзыва `-deprovision`. Щелкните **Я выполнил команду отзыва на виртуальной машине**, чтобы указать, что операционная система подготовлена к тому, чтобы стать образом.

10.	Щелкните галочку, чтобы записать образ.

	Новый образ станет доступным в разделе **Образы**. Виртуальная машина удаляется после записи образа.

	![Успешная запись образа](./media/virtual-machines-linux-capture-image/CaptureSuccess.png)

	При создании виртуальной машины с помощью метода **Из коллекции** можно использовать записанный образ, щелкнув пункт **Мои образы** на странице **Выберите операционную систему виртуальной машины**.
