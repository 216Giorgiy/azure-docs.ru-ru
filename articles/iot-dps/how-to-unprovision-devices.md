# <a name="how-to-unprovision-devices-enrolled-by-your-provisioning-service"></a>Как отменить подготовку устройств, зарегистрированных в службе подготовки

Вам может понадобиться отменить подготовку устройств, которые были подготовлены через службу подготовки устройств. Например, если устройство продано, перемещено в другой Центр Интернета вещей, утеряно, украдено или скомпрометировано. 

Как правило, процесс отмены подготовки устройства состоит из двух шагов:

1. Отменить доступ устройства к службе подготовки. В зависимости от того, на какой период времени требуется запретить доступ (временно или навсегда), или, в случае использования механизма аттестации X.509, от иерархии имеющихся групп регистраций, может потребоваться отключить или удалить запись регистрации. 
 
   - Сведения об отмене доступа к устройству с помощью портала см. в статье [Процедура отмены доступа устройств к службе подготовки на портале Azure](how-to-revoke-device-access-portal.md).
   - Сведения о том, как отменить доступ устройства программными средствами, используя один из пакетов SDK службы подготовки, см. в статье [Как управлять регистрациями устройств с помощью пакетов SDK службы подготовки устройств Azure](how-to-manage-enrollments-sdks.md).

2. Отключить или удалить запись устройства в реестре удостоверений для Центра Интернета вещей, в котором она была подготовлена. Дополнительные сведения см. в разделе [Управление удостоверениями устройства](https://docs.microsoft.com/en-us/azure/iot-hub/iot-hub-devguide-identity-registry#disable-devices) в документации по Центру Интернета вещей Azure. 

Точные действия по отмене подготовки устройства будут зависеть от используемого механизма аттестации.

Устройства, использующие аттестацию доверенного платформенного модуля или X.509 с самозаверяющимся сертификатом (не цепочкой сертификатов), подготовлены с помощью отдельной записи регистрации. Для этих устройств можно удалить запись, чтобы окончательно отменить доступ устройства к службе подготовки, или отключить запись, чтобы временно отменить доступ, а затем удалить или отключить устройство в реестре удостоверений через Центр Интернета вещей, для которого оно было подготовлено.

При использовании аттестации X.509 устройства также можно подготовить через группу регистраций. Для групп регистраций настраивается сертификат для подписи (промежуточный или корневой сертификат ЦС), который используется для контроля доступа. Доступ к службе подготовки получают устройства с этим сертификатом в цепочке сертификатов. Дополнительные сведения о группах регистраций и сертификатах X.509, а также их применении в службе подготовки см. в разделе [Сертификаты X.509](concepts-security.md#x509-certificates). 

Список устройств, подготовленных через группу регистраций, можно найти в подробных сведениях о группе регистраций. Это простой способ понять, для какого Центра Интернета вещей подготовлено каждое устройство. Чтобы просмотреть список устройств, сделайте следующее: 

1. Войдите на портал Azure и щелкните **Все ресурсы** в меню слева.
2. Щелкните службу подготовки в списке ресурсов.
3. В службе подготовки выберите **Manage enrollments** (Управление регистрациями), а затем — вкладку **Enrollment Groups** (Группы регистраций).
4. Щелкните группу регистраций, чтобы открыть ее.

   ![Просмотр записи группы регистраций на портале](./media/how-to-unprovision-devices/view-enrollment-group.png)

Есть два сценария работы с группами регистраций:

- Чтобы отменить подготовку всех устройств, подготовленных через группу регистраций, необходимо сначала отключить группу регистраций, чтобы ее сертификат для подписи попал в список запрещенных. Затем можно использовать список подготовленных устройств для этой группы регистраций, чтобы отключить или удалить каждое устройство из реестра удостоверений соответствующего Центра Интернета вещей. Когда вы отключите или удалите все устройства из соответствующих Центров Интернета вещей, можно при необходимости удалить группу регистраций. Однако следует помнить, что если вы удалите группу регистраций, но для сертификата для подписи выше в цепочке сертификатов одного или нескольких устройств останется включенная группа регистраций, эти устройства можно зарегистрировать повторно. 
- Чтобы отменить подготовку одного устройства из группы регистраций, необходимо сначала создать отключенную отдельную регистрацию для сертификата конечного объекта (устройства). Таким образом вы отмените доступ к службе подготовки для этого устройства, тогда как для других устройств, имеющих в цепочке сертификат для подписи группы регистраций, доступ будет разрешен. Затем можно использовать список подготовленных устройств в сведениях о группе регистраций, чтобы найти Центр Интернета вещей, для которого было подготовлено устройство, и отключить или удалить его из соответствующего реестра удостоверений. Не удаляйте отдельную отключенную регистрацию устройства. Так вы сможете повторно зарегистрировать устройство через группу регистраций.
