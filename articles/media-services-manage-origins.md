# <span id="managemediaservicesorigins"></span></a>Управление конечными точками потоковой передачи в учетной записи служб мультимедиа

Службы мультимедиа позволяют добавить множество конечных точек потоковой передачи в учетную запись и настроить их. С каждой учетной записью служб мультимедиа связана по крайней мере одна конечная точка потоковой передачи **по умолчанию**.

> [WACOM.NOTE] Конечные точки потоковой передачи раньше назывались источниками (Origins).

## Добавление и удаление конечных точек потоковой передачи

1.  Войдите на [Портал управления][Портал управления] и щелкните элемент **Службы мультимедиа**. Затем щелкните имя службы мультимедиа.
2.  Откройте страницу КОНЕЧНЫЕ ТОЧКИ ПОТОКОВОЙ ПЕРЕДАЧИ.
3.  В нижней части страницы нажмите кнопку ДОБАВИТЬ или УДАЛИТЬ. Обратите внимание, что конечную точку потоковой передачи по умолчанию удалить нельзя.
4.  Нажмите кнопку ЗАПУСК для запуска конечной точки.
5.  Щелкните имя конечной точки потоковой передачи, чтобы настроить ее.

    ![Origin page][Origin page]

## Настройка конечной точки потоковой передачи

На вкладке НАСТРОЙКА можно выполнять следующие настройки:

1.  Задать максимальный период кэширования, который будет указан в заголовке управления кэшем в ответах HTTP. Это значение не будет переопределять максимальное значение кэша, заданное явным образом в контенте BLOB-объекта.

2.  Указать IP-адреса, которым будет разрешено подключение к опубликованной конечной точке потоковой передачи. Если IP-адреса не указаны, можно подключаться с любого IP-адреса.

3.  Определить настройку для проверки подлинности заголовка подписи Akamai.

    ![Configure origin][Configure origin]

    Конфигурация на этой странице будет применена только к конечным точкам потоковой передачи, у которых есть хотя бы один зарезервированный модуль. Для резервирования зарезервированных единиц потоковой передачи по требованию перейдите на вкладку МАСШТАБ. Дополнительные сведения о зарезервированных единицах см. в разделе [Масштабирование служб мультимедиа][Масштабирование служб мультимедиа].
