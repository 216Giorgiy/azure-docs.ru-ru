# Примеры

[Скачивание примера общедоступной предварительной версии Power BI Embedded](http://go.microsoft.com/fwlink/?LinkID=761493)

Чтобы помочь вам приступить к работе, мы создали пример веб-приложения. Вы можете скачать его и прочитать сопутствующую статью [Get started with Microsoft Power BI Embedded sample](power-bi-embedded-get-started-sample.md). В примере показано, как:

  -	подготовить содержимое;
      - создать рабочую область;
      - импортировать PBIX-файл;
      - обновить строки подключения и указать учетные данные для наборов данных;

  - безопасно внедрить отчет.
