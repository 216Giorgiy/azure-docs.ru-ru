# REST API Power BI Embedded

С помощью **Microsoft Power BI Embedded** можно использовать REST API, чтобы программными средствами разрабатывать приложения, которые интегрируют **отчеты Power BI** в ваши приложения.

Чтобы узнать о REST API **Power BI Embedded**, ознакомьтесь с [операциями REST Power BI](https://msdn.microsoft.com/library/mt711493).
