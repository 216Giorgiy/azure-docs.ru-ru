# Типичные сценарии использования Power BI Embedded

Служба **Microsoft Power BI Embedded** предоставляет большинство существующих функций API Power BI как часть службы Azure. С помощью этих функций вы можете разрабатывать свои приложения. Вы также сможете программно подготавливать, разрабатывать и развертывать необходимые ресурсы и содержимое Power BI. Дополнительные сведения см. в статье [Начало работы с Microsoft Power BI Embedded (предварительная версия)](power-bi-embedded-get-started.md).

Ниже приведено несколько сценариев использования **Power BI Embedded** в приложениях.

- Полностью интерактивные отчеты, которые были созданы в Power BI Desktop, можно внедрить в приложение с помощью новой модели аутентификации приложений посредством маркера.
- Независимые поставщики программного обеспечения и его пользователи создают приложения для клиентов, которые могут использовать службу **Power BI Embedded** и пакет SDK для Power BI, чтобы внедрять интерактивные отчеты. Дополнительные сведения см. в статье [Начало работы с Microsoft Power BI Embedded (предварительная версия)](power-bi-embedded-get-started.md).
- Разработчик может использовать платформу визуализации Power BI для создания пользовательских визуализаций, которые можно применять в собственном приложении. См. раздел [Создание пользовательского визуального элемента с помощью средств разработчика (предварительная версия)](https://powerbi.microsoft.com/custom-visuals/).
