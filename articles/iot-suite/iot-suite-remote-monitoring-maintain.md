# <a name="troubleshoot-and-remediate-device-issues"></a>Устранение неполадок на устройстве

В этом руководстве показано, как использовать в решении страницу **Обслуживание** для диагностики и устранения неполадок на устройстве. Для демонстрации этих возможностей в руководстве используется сценарий приложения Интернета вещей Contoso.

Contoso тестирует новый **прототип** устройства в поле. Оператор Contoso во время тестирования замечает, что **прототип** устройства неожиданно активирует на панели мониторинга уведомление о температуре. Теперь следует изучить реакцию на событие неисправного **прототипа** устройства.

Из этого руководства вы узнаете, как выполнять такие задачи:

>[!div class="checklist"]
> * Использование страницы **Обслуживание** для изучения уведомления.
> * Вызов метода устройства, чтобы устранить неполадки.

## <a name="prerequisites"></a>Предварительные требования

Для работы с этим руководством нужен развернутый экземпляр решения для удаленного мониторинга в подписке Azure.

Если решение для удаленного мониторинга еще не развернуто, выполните инструкции из руководства [Развертывание предварительно настроенного решения для удаленного мониторинга](iot-suite-remote-monitoring-deploy.md).

## <a name="use-the-maintenance-dashboard"></a>Использование панели мониторинга обслуживания

На странице**Панель мониторинга** появились непредвиденные уведомления о температуре, поступающие из правила, связанного с **прототипом** устройства:

![Уведомления на панели мониторинга](media/iot-suite-remote-monitoring-maintain/dashboardalarm.png)

Чтобы выяснить причину неполадки, щелкните параметр**Explore Alarm** (Просмотреть уведомление) рядом с уведомлением:

![Просмотр уведомления на информационной панели](media/iot-suite-remote-monitoring-maintain/dashboardexplorealarm.png)

Теперь можно просмотреть список уведомлений на странице **Обслуживание**:

![Список уведомлений на странице обслуживания](media/iot-suite-remote-monitoring-maintain/maintenancealarms.png)

Чтобы отобразить подробные сведения об уведомлении, выберите его в списке **Уведомления**. В представлении показаны такие сведения:

* Время активации уведомления.
* Сведения о состоянии устройств, связанных с уведомлением.
* Данные телеметрии с устройства, связанного с уведомлением.

![Сведения об уведомлениях](media/iot-suite-remote-monitoring-maintain/maintenancealarmdetail.png)

Чтобы подтвердить уведомление, выберите **Alarm occurrences** (Вхождения уведомлений) и выберите **Acknowledge** (Подтвердить). Таким образом вы дадите другим операторам знать, что вы ознакомились с уведомлением и работаете над ним.

![Подтверждение уведомлений](media/iot-suite-remote-monitoring-maintain/maintenanceacknowledge.png)

В списке вы увидите **прототип** устройства, отвечающий за уведомление о температуре:

![Список устройств, вызвавших уведомление](media/iot-suite-remote-monitoring-maintain/maintenanceresponsibledevice.png)

## <a name="remediate-the-issue"></a>Устранение проблемы

Для устранения проблемы с **прототипом** устройства, необходимо вызвать метод **DecreaseTemperature** на устройстве.

Для работы с устройством выберите его в списке устройств и нажмите **Расписание**. Модель устройства **Модуль** указывает три метода, которые должно поддерживать устройство:

![Методы, которые поддерживает устройство](media/iot-suite-remote-monitoring-maintain/maintenancemethods.png)

Выберите **DecreaseTemperature** и задайте имя задания **DecreaseTemperature**. Нажмите кнопку **Применить**:

![Создание задания по уменьшению температуры](media/iot-suite-remote-monitoring-maintain/maintenancecreatejob.png)

Для отслеживания состояния задания на странице **Обслуживание** выберите раздел **Состояние системы**. В представлении **Состояние системы** отследите все задания и вызовы методов в решении:

![Мониторинг задания по уменьшению температуры](media/iot-suite-remote-monitoring-maintain/maintenancerunningjob.png)

Чтобы просмотреть сведения о конкретном задании или вызове метода, выберите его в списке в представлении **Состояние системы**:

![Просмотреть сведения о задании](media/iot-suite-remote-monitoring-maintain/maintenancejobdetail.png)
