# <a name="integrate-sim-data-in-the-remote-monitoring-solution"></a>Интеграция данных SIM-карты в решении удаленного мониторинга

## <a name="overview"></a>Обзор
Устройства Интернета вещей часто подключаются к облаку с помощью SIM-карты, которая позволяет им отправлять потоки данных из любого расположения. Решение для удаленного мониторинга Интернета вещей Azure позволяет интегрировать данные управления SIM-картой, чтобы операторы также могли отслеживать работоспособность устройства через данные, предоставленные этой SIM-картой. Решение удаленного мониторинга обеспечивает готовую интеграцию с Интернетом вещей Telefonica, позволяя клиентам, использующим платформу подключения Интернета вещей, синхронизировать данные о подключении SIM-карт устройств с решениями. Это решение можно расширить для поддержки других поставщиков телефонной компании через репозиторий GitHub.
Из этого руководства вы узнаете, как выполнять такие задачи:
* интегрировать данные SIM-карты в решение удаленного мониторинга;
* просмотр телеметрии в реальном времени;
* просмотреть данные SIM-карты. 

## <a name="telefonica-iot-integration-setup"></a>Настройка интеграции Интернета вещей Telefonica

### <a name="prerequisites"></a>предварительным требованиям
Чтобы синхронизировать данные о подключении в решение удаленного мониторинга Azure, выполните такие задачи:

1.  Заполните форму запроса [на сайте Telefonica](https://iot.telefonica.com/contact), выберите параметр **удаленного мониторинга Azure** и укажите ваши контактные данные.
2.  Telefonica активирует вашу учетную запись. 
3.  Если вы еще не являетесь клиентом Telefónica и хотите воспользоваться этой или другими услугами IoT Connectivity Cloud Ready, посетите [сайт Telefonica](https://iot.telefonica.com/contact) и выберите параметр **подключения**.

### <a name="telefonica-sim-setup"></a>Программа установки SIM Telefonica
Связь Telefónica SIM и идентификатора двойника устройства Azure будет основываться на свойстве "псевдоним" SIM-карты Интернета вещей Telefónica. 

Перейдите на [портал платформы подключения Интернета вещей Telefónica](https://m2m-movistar-es.telefonica.com/), выберите инвентаризацию SIM-карты, затем выберите SIM-карту и укажите для свойства "псевдоним" каждой SIM-карты выбранный идентификатор двойника устройства. 

Эту задачу также можно выполнить в режиме массового обслуживания (см. руководства пользователя по платформе подключения Интернета вещей Telefónica).

![Обновление Telefonica](media/iot-suite-remote-monitoring-telefonica/telefonica_site.png)

Чтобы подключить устройство к решению удаленного мониторинга, ознакомьтесь с руководствами, где описывается использование [C](iot-suite-connecting-devices-linux.md) и [Node](iot-suite-connecting-devices-node.md). 

## <a name="view-device-telemetry-and-sim-properties"></a>Просмотр данных телеметрии устройства и свойств SIM-карты
После соответствующей настройки учетной записи Telefonica и подключения устройства можно просмотреть данные устройства и SIM-карты.
Можно опубликовать следующие параметры подключения:
* ICCID;
* IP-адрес
* наличие сети;
* состояние SIM-карты;
* сетевое расположение;
* потребляемый трафик данных.
