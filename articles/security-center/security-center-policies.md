# <a name="set-security-policies-in-azure-security-center"></a>Настройка политик безопасности в центре безопасности Azure
В этом пошаговом руководстве описывается, как настроить политики безопасности в центре безопасности.


## <a name="how-security-policies-work"></a>Принцип работы политик безопасности
Центр безопасности автоматически создает политику безопасности по умолчанию для каждой из ваших подписок Azure. Вы можете изменить политику в центре безопасности или использовать [политику Azure](http://docs.microsoft.com/azure/azure-policy/azure-policy-introduction) для создания определений, определения дополнительных политик и назначения политик в группах управления (которые могут представлять всю организацию, подразделение и т. д.), а также контролировать соответствие этим политикам в этих областях.

> [!NOTE]
> Сейчас функция "Политика Azure" находится в ограниченной предварительной версии. Щелкните [здесь](https://aka.ms/getpolicy), чтобы присоединиться. Дополнительные сведения о политиках Azure см. в статье [Create and manage policies to enforce compliance](http://docs.microsoft.com/en-us/azure/azure-policy/create-manage-policy) (Создание политик и управление ими для обеспечения соответствия).

## <a name="how-to-change-security-policies-in-security-center"></a>Как изменить политики безопасности в центре безопасности
Вы можете изменять политику безопасности по умолчанию для каждой из ваших подписок Azure в центре безопасности. Изменить политику безопасности может пользователь с правами владельца, участника или администратора безопасности этой подписки или содержащей ее группы управления. Чтобы просмотреть политики безопасности в центре безопасности, войдите на портал Azure и сделайте следующее:

1. На панели мониторинга **Центр безопасности** в разделе **Общие** выберите **Политика безопасности**.
2. Выберите подписку, для которой вы хотите включить политику безопасности.

    ![Управление политикой](./media/security-center-policies/security-center-policies-fig10.png)

3. В разделе **Компоненты политики** щелкните **Политика безопасности**.

    ![Компоненты политики](./media/security-center-policies/security-center-policies-fig12.png)

4. Это политика по умолчанию, назначенная центру безопасности через функцию "Политика Azure". Вы можете удалить элементы, находящиеся в разделе **Policies and parameters** (Политики и параметры), или добавить другие определения политики, которые находятся в разделе **Available options** (Доступные параметры). Чтобы сделать это, просто щелкните знак "плюс" рядом с именем определения.

    ![Определения политик](./media/security-center-policies/security-center-policies-fig11.png)

5. Если требуются более подробные сведения о политике, щелкните ее, и откроется страница с подробными сведениями и код JSON, содержащий структуру [определения политики (https://docs.microsoft.com/azure/azure-resource-manager/resource-manager-policy#policy-definition-structure):

    ![JSON](./media/security-center-policies/security-center-policies-fig13.png)

6. После внесения изменений нажмите кнопку **Сохранить**.
