# <a name="introduction-to-azure-security-center"></a>Введение в Центр безопасности Azure
Узнайте о Центре безопасности Azure, его основных возможностях и о том, как он работает.

## <a name="what-is-azure-security-center"></a>Что такое Центр безопасности Azure?
Центр безопасности Azure предоставляет возможности унифицированного управления безопасностью и расширенной защиты от угроз для рабочих нагрузок в Azure, в локальной среде и в других облаках.  Он повышает вашу информированность о гибридных облачных рабочих нагрузках, обеспечивает управление ими, а также предоставляет активные средства защиты, снижающие уровень риска угроз, и интеллектуальные средства обнаружения, которые позволяют быть в курсе быстро развивающихся кибератак.

В колонке "Обзор" центра безопасности можно просмотреть состояние безопасности рабочих нагрузок Azure и тех, которые не относятся к Azure. Вы также можете обнаружить и оценить уровень безопасности рабочих нагрузок, а также определять и снижать риски.

## <a name="why-use-security-center"></a>Для чего используется центр безопасности?

**Унифицированное представление и управление**

- **Сведения о состоянии безопасности гибридных рабочих нагрузок**. Управляйте безопасностью всех гибридных облачных рабочих нагрузок, включая локальную среду, Azure и другие облачные платформы, на одной консоли. Благодаря встроенным панелям мониторинга вы можете мгновенно получить сведения о проблемах безопасности, требующих внимания.
- **Просмотр облачных рабочих нагрузок**. Не отставайте от быстро меняющихся облачных рабочих нагрузок. Автоматически обнаруживайте и адаптируйте новые ресурсы, создаваемые в подписках Azure.
- **Централизованное управление политиками.** Обеспечьте соответствие стандартам компании или нормативным требованиям к безопасности за счет централизованного управления политиками во всех гибридных облачных нагрузках.
- **Данные безопасности из множества источников.** Выполняйте сбор, поиск и анализ данных о безопасности из разных источников, включая подключенные партнерские решения, например сетевые брандмауэры и другие службы Майкрософт. 
- **Интеграция с имеющимися рабочими процессами безопасности.** Получайте доступ к сведениям о безопасности, интегрируйте и анализируйте их с помощью API REST для подключения имеющихся инструментов и процессов.
- **Отчеты о соответствии.** Используйте данные безопасности и аналитически данные, чтобы обеспечить соответствие требованиям и легко создавать подтверждающие данные для аудиторов.

**Адаптивное предотвращение угроз**

- **Непрерывная оценка безопасности.** Используйте сотни встроенных элементов проверки безопасности или создайте собственный для мониторинга безопасности компьютеров, сетей и служб Azure. Определите программное обеспечение и конфигурации, уязвимые к атакам.
- **Практические рекомендации.** Устраняйте уязвимости системы безопасности до того, как ими могут воспользоваться злоумышленники, используя приоритетные практические рекомендации по безопасности и встроенные сборники тренировочных заданий автоматизации.
- **Адаптивные элементы управления приложениями.** Блокируйте вредоносные и другие нежелательные программы, используя рекомендации списка разрешений с поддержкой машинного обучения, адаптированных к определенным рабочим нагрузкам Azure. 
- **Защита доступа к сети.** Уменьшите количество сетевых атак с помощью управляемого JIT-доступа к портам управления на виртуальных машинах Azure, чтобы значительно сократить вероятность атак методом подбора и других сетевых атак.

**Интеллектуальная система обнаружения угроз и ответные меры**

- **Обширная отраслевая аналитика угроз.** Обнаруживайте новые и развивающиеся угрозы безопасности с помощью службы Microsoft Intelligent Security Graph, которая использует миллиарды сигналов служб и систем Майкрософт по всему миру.
- **Расширенное обнаружение угроз.** Определяйте атаки и уязвимости нулевого дня с помощью встроенного анализа поведения и машинного обучения. Обнаруживайте входящие атаки и отслеживайте дальнейшую работу систем за счет мониторинга сетей, компьютеров и облачных служб.
- **Приоритетные оповещения и инциденты.** Сосредоточьтесь на важнейших угрозах, используя приоритетные средства оповещения и инциденты системы безопасности, которые сопоставляют разные типы предупреждений и позволяют увидеть общую картину атаки. Создавайте также собственные пользовательские оповещения системы безопасности.
- **Упрощенный анализ.** Быстро оценивайте область и влияние атак с помощью визуальных интерактивных средств. Используйте предопределенные или нерегламентированные запросы для более глубокого исследования данных безопасности. 
- **Контекстная аналитика угроз.** Визуализируйте источник атаки на интерактивной карте мира. Используйте встроенные отчеты об исследовании угроз, чтобы получить ценные сведения о методах и целях известных злоумышленников.

## <a name="get-started"></a>Начало работы
Для начала работы с Центром безопасности необходима подписка Microsoft Azure. Центр безопасности включается с помощью подписки Azure. Если у вас нет подписки, вы можете зарегистрироваться для получения [бесплатной пробной версии](https://azure.microsoft.com/pricing/free-trial/). 

[Краткое руководство по центру безопасности Azure](https://docs.microsoft.com/azure/security-center/security-center-get-started) поможет вам быстро ознакомиться с компонентами мониторинга безопасности и управления политиками в центре безопасности.
