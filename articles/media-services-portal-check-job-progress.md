#Практическое руководство. проверка хода выполнения задания

Это одна из статей серии [Рабочий процесс для видео по запросу в службах мультимедиа](../media-services-video-on-demand-workflow) . 

##Проверка состояния задания с помощью портала

1. На [портале управления](http://go.microsoft.com/fwlink/?LinkID=256666&clcid=0x409) щелкните **службы мультимедиа**, а затем щелкните имя учетной записи служб мультимедиа.
2. Выберите страницу **ЗАДАНИЯ**. 

	На этой странице можно увидеть состояние всех заданий. Можно также использовать фильтр по состоянию, времени или идентификатору задания.
