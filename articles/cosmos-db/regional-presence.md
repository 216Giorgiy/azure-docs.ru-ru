# <a name="regional-presence-of-azure-cosmos-db"></a>Географическое присутствие службы Azure Cosmos DB

Сейчас Azure доступна в [54 регионах](https://azure.microsoft.com/global-infrastructure/regions/) по всему миру. Azure Cosmos DB — это базовая служба в Azure, которая доступна во всех регионах, где предоставляется платформа Azure.

![Региональная доступность Azure Cosmos DB](./media/regional-presence/regional-presence.png)

Cosmos DB доступна для клиентов в пяти различных облачных средах Azure:

* **общедоступное облако Azure** — доступно по всему миру;

* **Azure для Китая** — доступна благодаря партнерским отношениям между корпорацией Майкрософт и компанией 21Vianet (крупнейшим интернет-провайдером в стране).

* **Azure для Германии** — предоставляет службы по модели доверенного лица по вопросам данных. Это означает, что данные клиента хранятся в Германии в T-Systems International GmbH, дочернем подразделении компании Deutsche Telecom, выступающем в роли доверенного лица по вопросам данных в Германии.

* **Azure для государственных организаций** — доступна в четырех регионах в Соединенных Штатах для государственных организаций США и их партнеров. 

* **Azure для государственных организаций для министерства обороны (DoD)** — доступна в двух регионах Соединенных Штатов для Министерства обороны США.

## <a name="regional-presence-with-global-distribution"></a>Географическое присутствие с глобальным распределением

Различные API-интерфейсы, предоставляемые Azure Cosmos DB (включая SQL, MongoDB, Cassandra, Gremlin и табличное хранилище Azure), доступны во всех регионах Azure. Например, у вас есть API-интерфейсы MongoDB и Cassandra, предоставляемые Azure Cosmos DB не только во всех глобальных регионах Azure, но также и в независимых регионах Azure, таких как Azure для Китая, Германии, для государственных организаций и Министерства обороны.

Azure Cosmos DB — это [глобально распределенная](distribute-data-globally.md) база данных. Вы можете связать любое количество регионов Azure с учетной записью Azure Cosmos, и ваши данные будут автоматически и прозрачно реплицированы. Добавить или удалить регион в учетной записи Azure Cosmos можно в любое время. Благодаря комплексной возможности глобального распределения и протоколу репликации с несколькими источниками Azure Cosmos DB обеспечивает задержку операций чтения и записи менее 10 мс на уровне 99-го процентиля, возможность чтения и записи на уровне 99,999, а также возможность гибко масштабировать подготовленную пропускную способность для операций чтения и записи во всех регионах, связанных с учетной записью Azure Cosmos. Azure Cosmos DB также предлагает пять четко определенных моделей согласованности, и вы можете применить к своим данным конкретную модель. Наконец, Azure Cosmos DB — единственная служба баз данных в индустрии, которая предоставляет полное Соглашение об уровне обслуживания, охватывающее подготовленную пропускную способность, задержку на уровне 99-го процентиля, высокий уровень доступности и согласованность.
