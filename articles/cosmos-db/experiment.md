# <a name="azure-cosmos-db-documentation-experiment"></a>Эксперимент с документацией Azure Cosmos DB

Azure Cosmos DB — это многомодельная глобально распределенная база данных. Так как с Azure Cosmos DB можно выполнять множество задач (например, создавать графы, создавать базы данных документов или переносить данные из хранилища MongoDB, Cassandra или табличного хранилища Azure), организация документация может оказаться сложной задачей. Поэтому мы предлагаем вам помощь.

Мы тестируем организацию оглавления и [стартовой страницы](https://docs.microsoft.com/en-us/azure/cosmos-db/) Azure Cosmos DB (см. рисунок ниже). И мы будем рады получить отзыв о предлагаемых изменениях.

## <a name="to-load-both-views"></a>Загрузка обоих представлений

1. Чтобы открыть оба представления параллельно, перейдите на [стартовую страницу](https://docs.microsoft.com/en-us/azure/cosmos-db/) из избранного веб-браузера. 
2. Ориентируйтесь на изображения выше, чтобы определить, какая версия отображается. 
3. Откройте браузер в режиме InPrivate или анонимном режиме и загрузите [стартовую страницу](https://docs.microsoft.com/en-us/azure/cosmos-db/) в новом окне браузера. Снова ориентируйтесь на изображения выше, чтобы определить, какая версия отображается. 

   Если не отображаются оба представления, закройте и снова откройте окно браузера в режиме InPrivate или анонимном режиме и загрузите [стартовую страницу](https://docs.microsoft.com/en-us/azure/cosmos-db/) еще раз. Для загрузки обеих версий в отдельных окнах может потребоваться несколько попыток. Если обновить окно в анонимном режиме, другая версия страницы не загрузится, поэтому нужно открыть новое окно. 

4. Затем вы сможете изменять размер окон, чтобы зафиксировать их параллельно, или переключаться между представлениями для сравнения. 

5. После того как вы ознакомились с организацией и макетом экспериментального интерфейса, нажмите кнопку **Опрос эксперимента** ниже, чтобы отправить отзыв. Отправьте форму не позднее полуночи 4 апреля 2018, чтобы отзыв был проверен и учтен.
