# <a name="azure-cosmos-db-whitepapers"></a>Техническая документация по Azure Cosmos DB 

Техническая документация позволяет более подробно изучить основные понятия Azure Cosmos DB. В этой статье представлен список доступных технических документов по Azure Cosmos DB.

| **Технический документ** | **Описание** |
| --- | --- |
| [Total Cost of Ownership of a NoSQL Database Cloud Service](https://aka.ms/cosmosdb-tco-paper) (Совокупная стоимость владения облачных служб для нереляционных баз данных) |В этом документе сравнивается общая стоимость владения и использования программного обеспечения в разных конфигурациях развертывания, таких как локальная база данных и база данных с облачной инфраструктурой. В нем также представлена оценка совокупной стоимости владения при высоком, среднем и низком уровнях потребления. |
|[Schema-Agnostic Indexing with Azure Cosmos DB](http://www.vldb.org/pvldb/vol8/p1668-shukla.pdf) (Схемонезависимое индексирование в Azure Cosmos DB) | В этом документе описывается подсистема индексирования Azure Cosmos DB. В нем также описываются такие возможности Azure Cosmos DB, как представление документов, язык запросов, метод индексирования документов, поддержка основного индекса и взаимодействие на ранних этапах разработки.|
| [Azure Cosmos DB and GDPR](https://servicetrust.microsoft.com/ViewPage/TrustDocuments?command=Download&downloadType=Document&downloadId=87cc6456-4b23-473c-94d3-6c713b8b8956&docTab=6d000410-c9e9-11e7-9a91-892aae8839ad_FAQ_and_White_Papers) (Azure Cosmos DB и GDPR)| В этом документе представлены рекомендации для клиентов Azure Cosmos DB, которые управляют облачными и (или) локальными базами данных и должны гарантировать, что персональные данные в их системах баз данных обрабатываются и защищены в соответствии с этими правилами. |
