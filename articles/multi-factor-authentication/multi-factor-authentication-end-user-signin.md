# Варианты входа при помощи Azure Multi-Factor Authentication
Предоставляемая информация описывает варианты входа в систему.


## Каким будет ваш вариант входа
Процесс входа зависит от избранного способа и использования многофакторной проверки подлинности. В этом разделе предоставляются сведения о том, что следует ожидать при входе. Ниже выберите вариант, который соответствует вашим действиям.


Ваш способ входа|Описание
:------------- | :------------- | 
[Вход с помощью мобильного или рабочего телефона](multi-factor-authentication-end-user-signin-phone.md) | Таким будет вход с мобильного или рабочего телефона.
[Вход с помощью уведомлений мобильного приложения](multi-factor-authentication-end-user-signin-app-notify.md) | Таким будет вход с мобильного приложения с использованием уведомлений.
[Вход с мобильного приложения с помощью кода подтверждения](multi-factor-authentication-end-user-signin-app-verify.md)|Таким будет вход с мобильного приложения с использованием кода подтверждения.
[Вход с помощью альтернативного метода](multi-factor-authentication-end-user-signin-alt.md)|Такая информация отобразится, если вы захотите воспользоваться альтернативным методом.
