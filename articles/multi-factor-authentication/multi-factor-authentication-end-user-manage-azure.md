# Управление параметрами Azure Multi-Factor Authentication на портале Azure

При использовании многофакторной проверки подлинности в службе Azure вам потребуется управлять параметрами дополнительной проверки безопасности через портал Azure.

## Получение доступа к параметрам дополнительной проверки безопасности на портале Azure

<ol>
<li>Войдите на портал Azure. <li>В верхней части портала Azure щелкните свое имя пользователя. Появится окно раскрывающегося списка. <li>В раскрывающемся списке выберите «Дополнительная проверка безопасности». Откроется раздел дополнительной проверки безопасности.

<center>![Setup](./media/multi-factor-authentication-end-user-manage-azure/changesettings.png)</center>

<li>Вы переместитесь на страницу корректировки, где можно изменить параметры.</li>

<center>![Setup](./media/multi-factor-authentication-end-user-manage-azure/proofup.png)</center>
