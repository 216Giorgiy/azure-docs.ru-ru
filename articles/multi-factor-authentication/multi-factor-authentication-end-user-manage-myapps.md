# Управление параметрами Azure Multi-Factor Authentication на портале Myapps


При использовании многофакторной проверки подлинности в службе Office 365 вам потребуется управлять дополнительными параметрами проверки безопасности через портал Office 365.

## Получение доступа к дополнительным параметрам проверки безопасности на портале Myapps
<ol>
<li>Войдите на сайт [https://myapps.microsoft.com](https://myapps.microsoft.com).</li>
<li>Вверху выберите профиль.</li>
<li>Щелкните правой кнопкой мыши свое имя пользователя и выберите элемент «Дополнительная проверка безопасности».</li>

<center>![Cloud](./media/multi-factor-authentication-end-user-manage-myapps/myapp.png)</center>

<li>Вы переместитесь на страницу корректировки, где можно изменить параметры.</li>

<center>![Setup](./media/multi-factor-authentication-end-user-manage-myapps/proofup.png)</center>
