# <a name="zoom-levels-and-tile-grid"></a>Уровни масштаба и параметры сетки
В службах Azure Location Based Services используется сферическая система координат проекции Меркатора (EPSG: 3857).

Карта мира разбита на квадратные плитки. Для растровой отрисовки предусмотрено 19 уровней масштаба (от 0 до 18). Для векторной отрисовки предусмотрен 21 уровень масштаба (от 0 до 20). На нулевом уровне масштаба вся карта мира помещается на одной плитке:

![Плитка карты мира](./media/zoom-levels-and-tile-grid/world0.png)

На первом уровне масштаба карта мира отображается на 4 плитках: квадрат 2 x 2.

![Плитка карты мира вверху слева](./media/zoom-levels-and-tile-grid/world1a.png)     ![Плитка карты мира вверху справа](./media/zoom-levels-and-tile-grid/world1c.png) 

![Плитка карты мира внизу слева](./media/zoom-levels-and-tile-grid/world1b.png)     ![Плитка карты мира внизу справа](./media/zoom-levels-and-tile-grid/world1d.png) 


На каждом последующем уровне масштаба плитки предыдущего разделяются на четыре плитки, создавая сетку 2<sup>масштаб</sup> x 2<sup>масштаб</sup>. 20-й уровень масштаба — это сетка 2<sup>20</sup> x 2<sup>20</sup>, или 1 048 576 x 1 048 576 плиток (всего 1 099 511 627 776).

Плитки формируются на основе уровня масштаба и координат x и y, которые соответствуют позиции плитки на сетке для этого уровня.

При выборе уровня масштаба помните, что каждое расположение — это фиксированная позиция на соответствующей плитке. Это означает, что количество плиток, необходимое для отображения заданной территории, зависит от конкретного размещения сетки масштабирования на карте мира. Например, есть две точки, расстояние между которыми составляет 900 метров. Для отображения этого маршрута *может* понадобиться только три плитки на 17-ом уровне масштабирования. Но если западная точка находится а правой части своей плитки, а восточная — в левой части своей, может понадобиться четыре плитки:

![Демонстрация увеличения масштаба](./media/zoom-levels-and-tile-grid/zoomdemo_scaled.png) 

После выбора уровня масштаба вычисляются значения x и y. Верхняя плитка слева на каждой сетке масштаба имеет такие координаты: x = 0, y = 0. Нижняя плитка справа имеет такие координаты: x = 2<sup>масштаб -1</sup>, y = 2<sup>масштаб - 1</sup>.
