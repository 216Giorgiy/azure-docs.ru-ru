# Управление рабочей областью машинного обучения Azure 
В рабочих областях машинного обучения Microsoft Azure (Azure ML) вы можете управлять доступом к своим веб-службам Azure ML, администрировать их и развертывать в производственной среде. Кроме того, вкладка панели мониторинга отображает сведения об использовании рабочей области и позволяет быстро получить информацию о ней.  

На вкладке **ВЕБ-СЛУЖБЫ** можно добавлять, обновлять и удалять веб-службы Azure ML.  

## Переход на страницу управления рабочей областью ##
1.	Войдите в систему с использованием учетной записи Azure.
2.	На панели служб Microsoft Azure выберите **МАШИННОЕ ОБУЧЕНИЕ**.
3.	В списке **машинное обучение** щелкните рабочую область.
