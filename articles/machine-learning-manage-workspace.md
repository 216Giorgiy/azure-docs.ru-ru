# Управление рабочей областью машинного обучения Azure

Вы можете работать с рабочими областями машинного обучения Microsoft Azure (Azure ML), чтобы управлять доступом, развертывать производственные веб-службы и управлять своими веб-службами Azure ML. Кроме того, вкладка панели мониторинга отображает сведения об использовании рабочей области и позволяет быстро получить информацию о самой рабочей области.

На вкладке **ВЕБ-СЛУЖБЫ** можно добавлять, обновлять или удалять веб-службу Azure ML.

## Переход на страницу управления рабочей областью

1.  Войдите в систему с использованием учетной записи Azure.
2.  На панели служб Microsoft Azure выберите **МАШИННОЕ ОБУЧЕНИЕ**.
3.  В списке **машинное обучение** щелкните рабочую область.
