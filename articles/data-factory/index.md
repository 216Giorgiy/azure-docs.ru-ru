# <a name="azure-data-factory-documentation"></a>Документация по фабрике данных Azure

Узнайте, как использовать фабрику данных, облачную службу интеграции данных, для объединения служб хранения, перемещения и обработки данных в автоматизированные конвейеры данных. Руководства и другая документация, в которой показано, как настраивать и администрировать конвейеры данных, а также перемещать и преобразовывать данные для анализа.
