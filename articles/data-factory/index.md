# <a name="azure-data-factory-documentation"></a>Документация по фабрике данных Azure

Фабрика данных Azure представляет собой облачную службу интеграции информации, которая автоматизирует перемещение и преобразование данных. Фабрика данных Azure создает решения по интеграции данных, которые получают, преобразуют и обрабатывают данные из различных хранилищ, а также публикуют обработанные данные в хранилищах данных.
