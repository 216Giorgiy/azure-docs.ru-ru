# Фабрика данных Azure - примеры

## Образцы на портале Azure
Быстро развертывать, просмотреть и проверить образец фабрика данных Azure с помощью **Образец конвейеры** выноску на портале.

1. Создайте новую фабрику данных или откройте фабрику существующих данных. В разделе [Приступая к работе с фабрикой данных Azure][data-factory-get-started] действий для создания фабрики данных.
2. В **ФАБРИКА данных** щелкните выноску фабрики данных **Образец конвейеры** плитки.

	![Пример мозаичного конвейеры](./media/data-factory-samples/SamplePipelinesTile.png)

2. В **Образец конвейеры** стоечный модуль, щелкните **Пример** которую требуется развернуть.
	
	![Образец конвейеры выноска](./media/data-factory-samples/SampleTile.png)

3. Задайте параметры конфигурации для данного образца. Например ваш ключ учетной записи и имени учетной записи хранилища Azure, имя сервера Azure SQL, база данных, идентификатор пользователя и пароль, и т. д...

	![Образец выноска](./media/data-factory-samples/SampleBlade.png)

4. После указания параметров конфигурации, нажмите кнопку **Создать** чтобы создать и развернуть образец конвейеры и связанных служб и таблицы, используемые в конвейеры.
5. Вы увидите состояние развертывания на плитке образец щелчка ранее в **Образец конвейеры** выноски.

	![Состояние развертывания](./media/data-factory-samples/DeploymentStatus.png)

6. При появлении **успешно развернуто** сообщения на плитке примера, закройте **Образец конвейеры** выноски.
5. На **ФАБРИКА данных** выноску, вы увидите добавить связанные службы, наборы данных и конвейеры для фабрики данных.  

	![Выноска фабрика данных](./media/data-factory-samples/DataFactoryBladeAfter.png)
   

В следующей таблице приведены описания образцов, доступных в **Образец конвейеры** плитки.

Пример имени | description
----------- | -----------
Профилирование клиентов игры | Contoso представляет игры компании, которая создает игр для нескольких платформ: игровых консолей, карманный устройств и персональных компьютеров (ПК). Каждая из таких игр создает множество журналов. Цель компании Contoso — собрать и проанализировать журналы, созданные в этих игр для получения сведений об использовании, определите возможности увеличения продаж и перекрестные продажи, разрабатывать новые привлекательные функции и т. д... для улучшения бизнес и предоставлять клиентам удобство работы. В этом примере собирает образцы журналов, процессы и дополняет их ссылочными данными и преобразования данных для оценки эффективности маркетинговой кампании, открыт Contoso.
