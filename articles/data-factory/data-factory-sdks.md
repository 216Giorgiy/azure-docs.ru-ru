# Пакеты SDK для фабрики данных Azure

Можно создавать и отслеживать фабрики, а также управлять ими с помощью классического портала Azure, Azure PowerShell, библиотеки классов .NET или интерфейса API REST.

Метод | Расположение ресурса | Материалы для разработчиков
---------------------------------------------------- | ------------------------------ | -----------
Классический портал Azure | [https://portal.azure.com/](https://portal.azure.com) | 
Azure PowerShell | Загрузите последнюю версию [Azure PowerShell](http://go.microsoft.com/?linkid=9811175&clcid=0x409). | [Справочник по командлетам](https://msdn.microsoft.com/library/dn820234.aspx)<p>[Справочник по скриптам JSON](https://msdn.microsoft.com/library/azure/dn835050.aspx) </p> 
Библиотека классов .NET | Пакет .NET SDK для фабрики данных Azure позволяет создавать и расширять фабрики данных Azure, наблюдать за их состоянием и управлять ими с помощью действий .NET. Чтобы приступить к работе, ознакомьтесь со статьями [Использование настраиваемых действий в конвейере фабрики данных Azure](data-factory-use-custom-activities.md) и [Создание, отслеживание фабрик данных Azure и управление ими с помощью пакета Data Factory .NET SDK](data-factory-create-data-factories-programmatically.md).<p><b>Загрузка последнего пакета NuGet</b><br/>Вы можете загрузить последний пакет NuGet библиотеки управления фабриками данных Azure с этого сайта: [https://www.nuget.org/packages/Microsoft.Azure.Management.DataFactories/](https://www.nuget.org/packages/Microsoft.Azure.Management.DataFactories/)</p><p>**Использование консоли диспетчера пакетов в Visual Studio**<br/>Чтобы получить последнюю библиотеку управления фабриками данных Azure, можно выполнить следующую команду в консоли диспетчера пакетов в Visual Studio:</p>Install-Package Microsoft.Azure.Management.DataFactories</p> | [Справочник по .NET SDK](https://msdn.microsoft.com/library/dn883654.aspx)
Интерфейс REST API | Вы можете использовать интерфейс REST API для создания, отслеживания фабрик данных Azure и управления ими. | [справочник по API REST](https://msdn.microsoft.com/library/dn906738.aspx)
