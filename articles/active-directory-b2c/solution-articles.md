# <a name="solutions-and-training-for-azure-active-directory-b2c"></a>Решения и обучение для Azure Active Directory B2C

Azure Active Directory (Azure AD) B2C позволяет организациям реализовать бизнес-решения, обеспечивающие их связь с клиентами. Следующие руководства по решениям и обучению являются загружаемыми документами, которые помогут вам реализовать эти решения.

| Название | ОПИСАНИЕ |
| ----- | ----------- |
| [Управление удостоверениями клиентов с использованием Azure AD B2C](https://channel9.msdn.com/Shows/On-NET/Customer-Identity-Management-with-Azure-AD-B2C) | В этом обзоре службы Парах Джайн ([@jainparakh](https://twitter.com/jainparakh)) из команды Azure AD B2C сообщает общие сведения о работе службы, а также показывает, как можно быстро подключить возможности B2C к приложению ASP.NET Core. |
| [Преимущества использования Azure Active Directory B2C](https://aka.ms/b2coverview) | Узнайте о преимуществах и распространенных сценариях Azure AD B2C, и как ваши приложения могут использовать эту службу CIAM. |
| [Получение практических знаний об Azure AD B2C. Курс для разработчиков](https://aka.ms/learnAADB2C) | Этот комплексный курс проводит разработчиков через все этапы разработки приложений с Azure AD B2C в качестве механизма проверки подлинности. Десять подробных модулей с лабораторными работами охватывают все вопросы: от настройки подписки Azure до создания пользовательских политик, которые определяют пути взаимодействия для привлечения клиентов. |
| [Предоставление партнерам, поставщикам и клиентам доступа к приложениям с помощью Azure Active Directory](https://aka.ms/aadexternalidentities) | Успешность каждой организации, независимо от ее размера, отрасли, уровня защищенности и соблюдения стандартов, зависит от способности организации сотрудничать с другими организациями и налаживать контакты с клиентами.<br><br>Освещая Azure AD, Azure AD B2C и Azure AD B2B Collaboration, в этом руководстве подробно рассматривается ценность для бизнеса и механизм создания приложения или веб-интерфейса, которые предоставляют возможности консолидированной проверки подлинности, адаптированные к контексту ваших сотрудников, партнеров по бизнесу, поставщиков и клиентов. |
| [Миграция проверки подлинности приложения в Azure AD B2C в гибридной среде](https://aka.ms/MigratetoAADB2C) | В современных организациях цифровое преобразование и переход в облако происходит поэтапно, требуя от большинства организаций, по крайней мере временно, работать в гибридной среде удостоверений.<br><br>В этом руководстве рассматривается создание плана миграции для перемещения первого приложения Azure AD B2C, а также выполнение этой процедуры в гибридной среде удостоверений. |
| [Применение общего регламента по защите данных (GDPR) для клиентских приложений](https://aka.ms/AADB2CandGDPR) | Все организации, участвующие в проектах, в которых обрабатываются персональные данные и которые обслуживают граждан стран ЕС, должны учитывать требования GDPR во всех своих приложениях для клиентов.<br><br>В данном руководстве рассматривается использование Azure AD B2C как гибкого компонента в общем подходе по соответствию GDPR, включая то, как компоненты Azure AD B2C соблюдают ключевые права GDPR частных лиц. |
