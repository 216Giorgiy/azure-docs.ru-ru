# Создание клиента Azure Active Directory (Azure AD) или клиента Azure Active Directory B2C: проблемы и решения

[AZURE.INCLUDE [active-directory-b2c-preview-note](../../includes/active-directory-b2c-preview-note.md)]

## Создание клиента Azure AD

Если вам не удается создать клиент Azure AD с первого раза, попробуйте сделать это еще раз. Если проблема не исчезнет, обратитесь в службу поддержки.

## Создание клиента Azure AD B2C (предварительная версия)

При возникновении проблем во время [создания клиента Azure AD B2C](active-directory-b2c-get-started.md) попробуйте сделать следующее:

- Если клиент Azure AD B2C не отображается в списке клиентов, повторите попытку.
- Если клиент Azure AD B2C отображается в списке клиентов, но вы получили сообщение об ошибке, удалите клиент, который вы только что создали, и повторите попытку. Например, может появиться следующее сообщение об ошибке: "Не удалось создать клиент B2C "contosob2c". Посетите эту [ссылку](http://go.microsoft.com/fwlink/?LinkID=624192&clcid=0x409) для получения дополнительных инструкций."
- Если ни одно из приведенных решений не помогло, обратитесь в службу поддержки. Узнайте больше о том, как [регистрировать запросы в службу поддержки для Azure AD B2C](active-directory-b2c-support.md).
