# Предварительная версия Azure Active Directory B2C: работоспособность службы

Эта страница содержит уведомления о незначительных проблемах, состоянии службы и устраненных рисках. Можно создать закладку для этой страницы, чтобы использовать ее в будущем. Также продолжайте отслеживать [панель мониторинга состояния Azure](https://azure.microsoft.com/status/).

### 22\.03.2016: ошибка единого входа в клиентах B2C

Ошибка единого входа была выявлена в 13:00 (тихоокеанское время) 17.03.2016 и устранена в 10:00 (тихоокеанское время) 18.03.2016. В этот период времени пострадало менее 100 потребителей. Мы продолжаем внимательно следить за этой проблемой. Условия, при которых потребитель может с ней столкнуться:

1. Вы установили политику входа в систему с "локальными учетными записями" в качестве единственного поставщика удостоверений.
2. В первый раз потребитель успешно входит в Azure AD B2C.
3. Затем он пытается войти снова, но вместо страницы единого входа на экране отображается сообщение об ошибке.

Единственным способом обойти проблему для потребителя (после шага 3) было закрыть, а затем открыть браузер и повторно пройти аутентификацию.
