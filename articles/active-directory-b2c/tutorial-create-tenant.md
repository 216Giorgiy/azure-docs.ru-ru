# <a name="tutorial-create-an-azure-active-directory-b2c-tenant"></a>Руководство по созданию клиента Azure Active Directory B2C

Чтобы приложения могли взаимодействовать с Azure Active Directory (Azure AD) B2C, их нужно зарегистрировать в клиенте, которым вы управляете.

В этой статье раскрываются следующие темы:

> [!div class="checklist"]
> * Создание клиента Azure AD B2C
> * Привязка клиента к подписке

Сведения о регистрации приложения приводятся в следующем руководстве.

Если у вас еще нет подписки Azure, [создайте бесплатную учетную запись Azure](https://azure.microsoft.com/free/?WT.mc_id=A261C142F), прежде чем начинать работу.

Войдите на [портал Azure](https://portal.azure.com/).

## <a name="create-an-azure-ad-b2c-tenant"></a>Создание клиента Azure AD B2C

1. Выберите **Создать ресурс** в левом верхнем углу окна портала Azure.
2. В поле поиска над списком ресурсов Azure Marketplace найдите и выберите **Active Directory B2C**, а затем щелкните **Создать**.
3. Выберите **Создать B2C-клиент Azure AD**, введите название организации и исходное доменное имя, которое будет использоваться в имени клиента, а затем выберите страну и нажмите кнопку **Создать**. Обязательно проверьте, правильно ли указана страна клиента, так как вы не сможете изменить ее позже.

    ![Создание клиента](./media/tutorial-create-tenant/create-tenant.png)

    В этом примере имя клиента — contoso0522Tenant.onmicrosoft.com.

Чтобы приступить к управлению новым клиентом, щелкните слово **здесь** в фразе **Щелкните здесь, чтобы управлять новым каталогом**. Появится сообщение **об устранении неполадок**, в котором указано, что вам необходимо связать подписку с новым клиентом. 

## <a name="link-your-tenant-to-your-subscription"></a>Привязка клиента к подписке

Клиент Azure AD B2C нужно связать с подпиской Azure, чтобы включить все возможности и вносить плату за использование. Если не установить связь между клиентом и подпиской, приложения не будут работать правильно.

1. Убедитесь, что используется каталог с подпиской, с которой необходимо связать новый клиент. Для этого перейдите к этому каталогу в правом верхнем углу окна портала Azure.

    ![Переключение каталогов](./media/tutorial-create-tenant/switch-directories.png)

    Затем выберите каталог, который содержит вашу подписку.

    ![Выбор каталога](./media/tutorial-create-tenant/select-directory.png)

2. Выберите **Создать ресурс** в левом верхнем углу окна портала Azure.
3. В поле поиска над списком ресурсов Azure Marketplace найдите и выберите **Active Directory B2C**, а затем щелкните **Создать**.
4. Щелкните **Связать существующий клиент B2C Azure AD с вашей подпиской Azure**, выберите созданный вами клиент и подписку, введите имя группы ресурсов *myContosoTenantRG*, примите расположение и нажмите кнопку **Создать**.
