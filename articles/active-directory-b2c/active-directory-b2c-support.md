# Предварительная версия Azure Active Directory B2C: регистрация запросов в службу поддержки

[AZURE.INCLUDE [active-directory-b2c-preview-note](../../includes/active-directory-b2c-preview-note.md)]

Вы можете регистрировать запросы в службу поддержки для Azure Active Directory (AD) B2C на портале Azure следующим образом.

1. [Выполните следующие действия, чтобы перейти к колонке функций B2C на портале Azure](active-directory-b2c-app-registration.md#navigate-to-the-b2c-features-blade).
2. Переключитесь с вашего клиента B2C на другой клиент, с которым сопоставлена подписка Azure. Это может быть клиент вашего сотрудника или клиент по умолчанию, который был создан, когда вы зарегистрировались для получения подписки Azure. Чтобы узнать больше, см. раздел [Как подписка Azure связана с Azure AD](active-directory-how-subscriptions-associated-directory.md#how-an-azure-subscription-is-related-to-azure-ad).

    ![Поддержка. Переключение клиентов](./media/active-directory-b2c-support/support-switch-dir.png)

3. После переключения клиентов нажмите кнопку **Справка и поддержка**.

    ![Поддержка. Справка и поддержка](./media/active-directory-b2c-support/support-support.png)

4. Щелкните **Новый запрос в службу поддержки**.

    ![Поддержка. Новый запрос](./media/active-directory-b2c-support/support-new.png)

5. В колонке **Основы** укажите эти сведения и нажмите кнопку **Далее**.

    - **Тип запроса** — **Технический**.
	- Выберите соответствующую **Подписку**.
    - **Служба** — **Active Directory**.
    - Выберите соответствующий **План поддержки**. Если у вас его нет, вы можете зарегистрировать его [здесь](https://azure.microsoft.com/ru-RU/support/plans/).

    ![Поддержка. Основная информация](./media/active-directory-b2c-support/support-basics.png)

6. В колонке **Проблемы** укажите эти сведения и нажмите кнопку **Далее**.

    - Выберите соответствующий уровень **Серьезности**.
    - **Тип проблемы** — **Предварительная версия B2C**.
    - Выберите соответствующую **Категорию**.
	- Опишите свою проблему в поле **Сведения**. Укажите подробные сведения о проблеме, такие как имя клиента B2C, описание проблемы, сообщения об ошибках, идентификаторы корреляции (при наличии) и т. д.
    - В поле **Интервал времени** укажите дату и время (включая часовой пояс) возникновения проблемы.
    - Отправьте все снимки экрана и файлы, которые, по вашему мнению, помогут в устранении этой проблемы, в разделе **Отправка файлов**.

    ![Поддержка. Проблема](./media/active-directory-b2c-support/support-problem.png)

7. В колонке **Контактные данные** добавьте контактные данные. Щелкните **Создать**. *Примечание. В предварительной версии поддержка Azure AD B2C доступна только на английском языке.*

    ![Поддержка. Контактная информация](./media/active-directory-b2c-support/support-contact.png)

8. После отправки запроса в службу поддержки вы можете отслеживать его, щелкнув **Справка и поддержка** на начальной панели, а затем — **Управление запросами в службу поддержки**.

## Известная проблема: регистрация запроса в службу поддержки в контексте клиента B2C

Если вы пропустили описанный выше шаг 2 и пытаетесь создать запрос на поддержку в контексте вашего клиента B2C, появится указанная ниже ошибка.

> [AZURE.IMPORTANT]
Не пытайтесь зарегистрироваться для получения новой подписки Azure в вашем клиенте B2C.

![Поддержка — нет подписки](./media/active-directory-b2c-support/support-no-sub.png)
