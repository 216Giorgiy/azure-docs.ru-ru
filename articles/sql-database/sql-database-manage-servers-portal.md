# <a name="create-and-manage-azure-sql-database-servers-with-the-azure-portal"></a>Создание и администрирование серверов баз данных SQL Azure с помощью портала Azure

Вы можете создавать и администрировать серверы баз данных SQL Azure с помощью [портала Azure](https://portal.azure.com/), PowerShell, REST API или языка C#. Этот раздел посвящен использованию портала Azure. PowerShell: см. дополнительные сведения о [создании и администрировании серверов баз данных с помощью PowerShell](sql-database-manage-servers-powershell.md).

## <a name="create-an-azure-sql-database-server-using-the-azure-portal"></a>Создание сервера базы данных SQL Azure с помощью портала Azure

1. На [портале Azure](https://portal.azure.com/) откройте вкладку **Серверы SQL Server**. 

    ![Серверы SQL Server](./media/sql-database-get-started/new-sql-server.png)

2. Щелкните **Добавить**, чтобы создать сервер SQL Server.

    ![Добавление нового сервера SQL Server](./media/sql-database-get-started/new-sql-server-add.png)

> [!TIP]
> Чтобы приступить к работе с порталом Azure и SQL Server Management Studio, ознакомьтесь со статьей [Начало работы с серверами баз данных SQL Azure, базами данных и правилами брандмауэра с использованием портала Azure и SQL Server Management Studio](sql-database-get-started.md).
>

## <a name="view-and-update-azure-sql-server-settings-using-the-azure-portal"></a>Просмотр и обновление параметров сервера SQL Azure с помощью портала Azure
1. На [портале Azure](https://portal.azure.com/) откройте вкладку **Серверы SQL Server**. 

    ![Создание нового сервера SQL Server](./media/sql-database-get-started/new-sql-server.png)

2. Щелкните сервер, с которым собираетесь работать, а затем выберите нужный параметр в колонке серверов SQL. 

    ![Колонка сервера SQL Server](./media/sql-database-get-started/sql-server-blade.png)

> [!TIP]
> Портал Azure: см. дополнительные сведения о [создании баз данных с помощью портала Azure](sql-database-get-started.md).
>
