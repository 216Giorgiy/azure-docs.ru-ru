# <a name="view-and-update-an-azure-sql-database-server-and-its-settings"></a>Просмотр и обновление сервера базы данных SQL Azure и его параметров

Просматривать и обновлять сервер базы данных SQL Azure, а также его параметры, можно с помощью [портала Azure](sql-database-manage-portal.md), [PowerShell](sql-database-manage-powershell.md), REST API или C#. 

## <a name="view-and-update-an-azure-sql-server-settings-using-the-azure-portal"></a>Просмотр и обновление параметров сервера SQL Azure с помощью портала Azure
1. На [портале Azure](https://portal.azure.com/) откройте вкладку **Серверы SQL Server**. 

    ![Создание нового сервера SQL Server](./media/sql-database-get-started/new-sql-server.png)

2. Щелкните сервер, с которым собираетесь работать, а затем выберите нужный параметр в колонке серверов SQL. 

    ![Колонка сервера SQL Server](./media/sql-database-get-started/sql-server-blade.png)

> [!TIP]
> Чтобы приступить к работе с порталом Azure, ознакомьтесь со статьей [Начало работы с серверами баз данных SQL Azure, базами данных и правилами брандмауэра с использованием портала Azure и SQL Server Management Studio](sql-database-get-started.md).
>
