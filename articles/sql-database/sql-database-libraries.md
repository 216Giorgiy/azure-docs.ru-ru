### Поддержка ODBC

При использовании мастера источника данных (DSN) для определения источника данных для базы данных SQL Azure выберите вариант **Проверка подлинности учетной записи SQL Server**, а затем выберите **Получить параметры конфигурации, используемые по умолчанию, от SQL**. В поля **идентификатор входа** и **пароль** введите имя пользователя и пароль для подключения к серверу базы данных SQL Azure. Снимите флажок **Получить параметры конфигурации, используемые по умолчанию**. Щелкните **Использовать по умолчанию базу данных** и введите имя базы данных SQL Azure, даже если оно не отображается в списке. Обратите внимание, что в списке **Изменить язык системных сообщений SQL Server** имеется несколько языков.

В этом выпуске база данных SQL Microsoft Azure поддерживает только английский язык, который и нужно выбрать. База данных SQL Microsoft Azure не поддерживает **зеркальный сервер** или **присоединение базы данных**, поэтому не заполняйте эти поля. Нажмите кнопку **Проверить подключение**.

При использовании драйвера ODBC собственного клиента SQL Server 2008 при нажатии на кнопку **Проверить подключение** может появиться сообщение об ошибке, сообщающее, что **master.dbo.syscharsets** не поддерживается. Пропустите эту ошибку, сохраните источник данных и пользуйтесь им.

### OLEDB для DB2 и SQL Server для разработки DRDA

Поставщик данных Microsoft OLE DB для DB2 версии 5.0 (поставщик данных) позволяет создавать распределенные приложения, предназначенные для баз данных IBM DB2. Поставщик данных использует архитектуру доступа к данным Microsoft SQL Server вместе с сетевым клиентом Microsoft для DB2, который выполняет роль приложения, выполняющего запрос для распределенной реляционной базы данных DRDA. Поставщик данных преобразует команды COM OLE DB и типы данных в кодовых точках протокола и форматы данных DRDA.

Дополнительные сведения см. в следующих статьях:

- [Поставщик Microsoft OLE DB для DB2 версии 5.0](http://msdn.microsoft.com/library/dn745875.aspx)
- [Поставщик Microsoft OLE DB для DB2 версии 4.0 для Microsoft SQL Server 2012](http://www.microsoft.com/download/details.aspx?id=29100)
