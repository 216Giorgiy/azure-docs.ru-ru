# Обзор: средства управления для Базы данных SQL

В этом разделе рассматриваются и сравниваются средства и параметры для управления Базами данных SQL, чтобы вы могли выбрать для себя подходящий инструмент для работы и бизнеса. Выбор правильного средства зависит от количества баз данных, которыми вы управляете, выполняемых задач, а также то того, как часто вы их выполняете.



## Портал Azure


 [Портала Azure](http://portal.azure.com) — это веб-портал управления, где можно создавать, обновлять и удалять Базы данных SQL Azure и логические серверы, а также отслеживать ресурсы базы данных. Он превосходно подойдет для тех, кто только знакомится с Azure или управляет небольшим количеством баз данных, а также в случае, если требуется оперативно выполнить какую-либо задачу по управлению.

Более подробные сведения об использовании портала см. в статье [Управление Базами данных SQL с помощью портала Azure](sql-database-manage-portal.md).

## SQL Server Management Studio и SQL Server Data Tools в Visual Studio


SQL Server Management Studio (SSMS) и SQL Server Data Tools (SSDT) в Visual Studio представляют собой клиентские средства, которые выполняются на компьютере и позволяют подключаться к базам данных в облаке, разрабатывать их, а также управлять ими. Если вы не новичок в разработке приложений и уже знакомы с Visual Studio или другими интегрированными средами разработки (IDE), предлагаем обратить внимание на SSDT в составе Visual Studio. Если требуются дополнительные возможности SQL, которые недоступны в SSDT, такие как управление Базами данных SQL Server в гибридных средах, воспользуйтесь средой SSMS.

Дополнительные сведения об управлении Базами данных SQL Azure с помощью SSMS и SSDT см. в статье [Управление Базами данных SQL с помощью SSMS](sql-database-manage-azure-ssms.md)


## Программы командной строки

Для управления Базами данных SQL Azure и автоматизации развертывания ресурсов Azure можно использовать средства командной строки, такие как PowerShell. Мы рекомендуем использовать это средство для управления большим количеством Баз данных SQL Azure и развертывания изменений ресурсов в рабочей среде.

Дополнительные сведения об управлении Базами данных SQL Azure с помощью средств командной строки см. [здесь](sql-database-command-line-tools.md).
