# Обзор Базы данных SQL

База данных SQL — это реляционная облачная служба баз данных на основе ядра Microsoft SQL Server со встроенными критически важными функциями. База данных SQL обеспечивает разработчикам облачных служб и архитекторам решений прогнозируемую производительность, масштабируемость, непрерывность бизнес-процессов, защиту данных и почти полностью автоматизированное администрирование. Важнее всего то, что База данных SQL поддерживает существующие инструменты, библиотеки и API SQL Server, упрощая переход в облачную среду. Базу данных SQL можно [настроить и запустить в считанные секунды](sql-database-get-started.md) без необходимости управлять виртуальными машинами или инфраструктурой. Программные функции, схожие с возможностями администратора базы данных, обеспечивают эффективность разработки, а средства самостоятельного управления позволяют сократить необходимость обслуживания баз данных до минимума. Это позволяет сосредоточиться на разработке приложения и управлении им.

База данных SQL позволяет [увеличивать масштаб](sql-database-service-tiers.md) приложения или [разворачивать его](sql-database-elastic-scale-get-started.md) до тысяч баз данных с прогнозируемой производительностью, которую при необходимости можно увеличить или уменьшить. Условия ведущего в отрасли [SLA](http://azure.microsoft.com/support/legal/sla/) гарантируют доступность в течение 99,99 % времени, что означает практически полное отсутствие простоя приложений. Встроенные средства защиты данных, такие как аудит, восстановление и георепликация, обеспечивают полную безопасность ваших данных. База данных SQL позволяет восстанавливать данные на любой момент времени и транзакцию за последние 35 дней. Вы также можете реплицировать данные в любом регионе Azure и беспрепятственно внедрить геополитику аварийного восстановления, настроенную согласно потребностям вашего предприятия. [Последняя версия Базы данных SQL](sql-database-preview-whats-new.md), V12, обеспечивает практически полную совместимость с ядром SQL Server.

База данных SQL доступна на [уровнях служб](sql-database-service-tiers.md) Basic, Standard и Premium с поддержкой рабочих нагрузок различной интенсивности, что позволяет совмещать и смешивать уровни для создания инновационных архитектур приложений. Благодаря возможностям Azure вы сможете подбирать сочетания служб Azure с Базой данных SQL, удовлетворяющие задачам ваших уникальных и современных архитектур приложений, повышающие эффективность использования ресурсов и снижающие расходы, а также открывающие новые возможности развития бизнеса.

Начните с [бесплатной пробной версии](http://azure.microsoft.com/pricing/free-trial/) и узнайте о том, [как создать свою первую базу данных SQL](sql-database-get-started.md)!
