# Средства работы с эластичными базами данных SQL Azure

Функция эластичных баз данных SQL Azure предназначена для упрощения разработки уровня данных и управления, в частности для разработчиков программного обеспечения как услуги (SaaS), которые используют большое количество баз данных для поддержки клиентов с динамичной инфраструктурой.

**Средства работы с эластичными базами данных** помогают создавать и управлять приложениями, использующими шаблоны сегментирования для масштабирования многих баз данных. Сюда относятся клиентская библиотека эластичных баз данных, поддерживающая управление сегментами, зависящая от данных маршрутизация и возможности запросов между сегментами для ADO.Net. Также включено средство разбиения и объединения, которое упрощает управление сегментами, позволяя перемещать данные (разбиение) из одного сегмента в другой (объединение). Эти средства дополняют пулы эластичных баз данных, которые поддерживают прогнозируемую стоимость и гарантированные характеристики производительности для любого количества баз данных. Связанная служба заданий является простым и надежным средством для обновления и управления любыми базами данных в пуле с помощью сценариев T-SQL.
