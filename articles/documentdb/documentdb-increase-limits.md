# Запрос увеличения ограничения учетной записи DocumentDB

[Microsoft Azure DocumentDB](https://azure.microsoft.com/services/documentdb/) применяет ряд ограничений и квот по умолчанию. Несколько квот могут быть изменены при обращении в службу поддержки Azure. В этой статье показано, как запросить увеличение ограничения учетной записи.

Ознакомившись с данной статьей, вы сможете ответить на следующие вопросы.

-	Какие квоты DocumentDB могут быть изменены при обращении в службу поддержки Azure?
-	Как запросить изменение квоты учетной записи DocumentDB?

##<a id="AdjustableQuotas"></a> Изменяемые квоты учетной записи DocumentDB

В следующей таблице описаны квоты DocumentDB, которые могут быть изменены при обращении в службу поддержки Azure:

|Сущность |Квота (план Standard)|
|-------|--------|
|Учетные записи базы данных |5
|Количество хранимых процедур, триггеров и пользовательских функций для одной коллекции |25 на каждую
|Максимальное количество коллекций в учетной записи базы данных |100
|Максимальный объем хранилища документов базы данных (100 коллекций) |1 TБ
|Максимальное количество пользовательских функций в запросе |2
|Максимальное количество соединений в запросе |5
|Максимальное количество предложений AND в запросе |20
|Максимальное количество предложений OR в запросе |10
|Максимальное число значений в выражении IN |100
|Максимальное количество точек в аргументе многоугольника в запросе ST\_WITHIN |16
|Максимальное количество созданных коллекций в минуту |5
|Максимальное количество операций масштабирования в минуту |5

##<a id="RequestQuotaIncrease"></a> Запрос изменения квоты
Ниже показано, как запросить изменение квоты.

1. На [портале Azure](https://portal.azure.com) нажмите **Обзор**, а затем щелкните **Справка и поддержка**.

	![Снимок экрана запуска справки и поддержки](media/documentdb-increase-limits/helpsupport.png)

2. В колонке **Справки и поддержка** щелкните **Получить поддержку**.

	![Снимок экрана создания запроса в службу поддержки](media/documentdb-increase-limits/getsupport.png)

3. В колонке **Создать запрос поддержки** щелкните **Основные**. Затем задайте для параметра **Тип запроса** значение **Квота**, для параметра **Подписка** укажите вашу подписку, которая включает учетную запись DocumentDB, задайте для параметра **Служба** значение **DocumentDB** и для параметра **План поддержки** значение **Поддержка квоты включена**. Наконец, нажмите кнопку **Далее**.

	![Снимок экрана: тип запроса поддержки](media/documentdb-increase-limits/supportrequest1.png)

4. В колонке **Задача** выберите уровень серьезности. Задайте для параметра **Тип задачи** значение **DocumentDB** и добавьте сведения об увеличении вашей квоты в области **Сведения**. Нажмите кнопку **Далее**.

	![Снимок экрана: средство выбора запроса поддержки](media/documentdb-increase-limits/supportrequest2.png)

5. Наконец, заполните колонку **Контактные данные**.

	![Снимок экрана: средство выбора ресурса запроса поддержки](media/documentdb-increase-limits/supportrequest3.png)

После создания запроса в службу поддержки вы получите номер запроса по электронной почте. Чтобы просмотреть запрос, нажмите кнопку **Запросы технической поддержки** в колонке **Справка и поддержка**.

![Снимок экрана: колонка запросов поддержки](media/documentdb-increase-limits/supportrequest4.png)
