# Поддержка протокола DocumentDB для MongoDB

Теперь базы данных Microsoft Azure DocumentDB можно использовать как хранилище данных для приложений, написанных для MongoDB. Приложения, написанные для MongoDB, могут легко взаимодействовать с DocumentDB, используя существующие драйверы для MongoDB и функции этой предварительной версии. Разработчики могут использовать свои текущие средства и навыки работы с MongoDB, чтобы создавать приложения, использующие полностью управляемые и масштабируемые базы данных NoSQL DocumentDB.

В настоящее время мы предлагаем пользователям протестировать новые функции предварительной версии. Если вы хотите принять участие в тестировании предварительной версии, следуйте указаниям, приведенным ниже, чтобы запросить доступ к предварительной версии программы. Мы свяжемся с вами, когда ваша кандидатура будет утверждена для участия в тестировании.

## Предварительные требования

Прежде чем запрашивать доступ к предварительной версии программы, необходимо иметь следующее.

- **Подписка Azure.**. См. страницу [Создайте бесплатную учетную запись Azure уже сегодня](https://azure.microsoft.com/free/).

## Запрос доступа к поддержке протокола DocumentDB для предварительной версии MongoDB  

Чтобы получить доступ к предварительной версии, выполните следующие действия.

1. Войдите на [портал Azure](https://portal.azure.com).
2. Последовательно щелкните **Создать**, **Данные+хранилище**, **Просмотреть все**, **Дополнительно**, затем выберите **DocumentDB — поддержка протокола для MongoDB**.

	![Снимок экрана: колонки "Marketplace" и "Данные+хранилище" с выделенным элементом "DocumentDB — поддержка протокола для MongoDB" \(база данных Mongo\)](./media/documentdb-protocol-mongodb/marketplacegallery1.png)

3. Или выполните поиск по запросу "Протокол DocumentDB" в категории **Данные+хранилище**.

	![Снимок экрана: колонки поиска "Marketplace" и "Данные+хранилище" с выделенным элементом "DocumentDB — поддержка протокола для MongoDB" \(база данных Mongo\)](./media/documentdb-protocol-mongodb/marketplacegallery2.png)

4. Щелкните **Создать**, чтобы начать регистрацию в предварительной версии.

	![Колонка "DocumentDB — поддержка протокола для MongoDB" на портале Azure](./media/documentdb-protocol-mongodb/marketplacegallery3.png)

5. В колонке **Новая учетная запись DocumentDB** щелкните **Зарегистрироваться в предварительной версии**. Проверьте данные и нажмите кнопку **ОК**. После включения подписки для общедоступной предварительной версии вы получите электронное письмо.

	![Колонка "Зарегистрироваться сегодня" для "DocumentDB — поддержка протокола для MongoDB" на портале Azure](./media/documentdb-protocol-mongodb/registerforpreview.png)


## Дальнейшие действия
- После того как ваша кандидатура будет утверждена для тестирования предварительной версии, вы получите по электронной почте приветственное сообщение с инструкциями по созданию учетной записи и началу работы.
