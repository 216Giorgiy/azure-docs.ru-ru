#Ограничения и квоты DocumentDB

В таблице ниже перечислены ограничения и квоты, действующие для DocumentDB. Квоты, перечисленные со звездочкой (*), [могут быть изменены при обращении в службу поддержки Azure](documentdb-increase-limits.md).

|Сущность |Квота (план Standard)|
|-------|--------|
|Учетные записи базы данных* |5
|Количество баз данных в учетной записи базы данных |100
|Количество пользователей в учетной записи базы данных (итоговое для всех баз данных) |500 000
|Количество разрешений в учетной записи базы данных (итоговое для всех баз данных) |2 000 000
|Объем хранилища вложений в учетной записи базы данных (предварительная версия) |2 ГБ
|Максимальное количество единиц запросов в секунду для одной коллекции |2500
|Количество хранимых процедур, триггеров и определяемых пользователем функций для одной коллекции* |25 на каждую
|Максимальное время выполнения хранимой процедуры или триггера |5 с
|Подготовленный размер хранилища или коллекции |10 ГБ
|Максимальное количество коллекций в учетной записи базы данных* |100
|Максимальный объем хранилища документов базы данных (100 коллекций)* |1 TБ
|Максимальная длина свойства идентификатора |255 символов
|Максимальное количество элементов на одной странице |1000
