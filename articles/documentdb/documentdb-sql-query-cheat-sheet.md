# <a name="documentdb-sql-query-cheat-sheet-pdf"></a>Памятка по SQL-запросам DocumentDB в формате PDF
**Памятка по SQL-запросам DocumentDB** помогает быстро создавать запросы для данных DocumentDB, так как в удобном для печати справочном листе в формате PDF отображаются общие запросы, ключевые слова, встроенные функции и операторы базы данных. 

DocumentDB поддерживает реляционные, иерархические и пространственные запросы документов JSON с использованием [SQL](documentdb-sql-query.md) без указания схемы или вторичных индексов. Кроме стандартных ключевых слов и операторов ANSI-SQL, DocumentDB поддерживает определяемые пользователем функции (UDF) JavaScript, операторы JavaScript и множество встроенных функций.

## <a name="download-the-documentdb-sql-query-cheat-sheet-pdf"></a>Скачивание PDF-файла памятки по SQL-запросам DocumentDB
Скачайте памятку по SQL-запросам и используйте ее в качестве краткого справочника, чтобы быстрее создавать запросы. В PDF-файле памятки по SQL-запросам приводятся общие запросы, используемые для получения данных из двух примеров документов JSON. Чтобы держать памятку по SQL-запросам под рукой, напечатайте ее на одной стороне листа в формате Letter (8,5 x 11 дюймов, или 21,59 x 27,94 см).

**Памятку можно скачать по следующей ссылке: [Памятка по Microsoft Azure DocumentDB SQL](http://go.microsoft.com/fwlink/?LinkId=623215)**.
