# Загружаемая памятка по запросам SQL DocumentDB

**Памятка по запросам DocumentDB** помогает быстро создавать запросы для данных DocumentDB, так как в удобном для печати справочном листе отображаются общие запросы, ключевые слова, встроенные функции и операторы.

DocumentDB поддерживает реляционные, иерархические и пространственные запросы документов JSON с использованием [SQL](documentdb-sql-query.md) без указания схемы или вторичных индексов. Кроме стандартных ключевых слов и операторов ANSI-SQL, DocumentDB поддерживает определяемые пользователем функции (UDF) JavaScript, операторы JavaScript и множество встроенных функций.

## Загрузка памятки по запросам DocumentDB

Скачайте памятку по запросам и используйте ее в качестве краткого справочника, чтобы быстрее создавать запросы. В памятке приводятся общие запросы SQL, используемые для получения данных из двух примеров документов JSON. Чтобы держать памятку под рукой, напечатайте ее на одной стороне листа в формате Letter (8,5 x 11 дюймов, или 21,59 x 27,94 см).

**Памятку можно скачать по следующей ссылке: [Памятка по запросам Microsoft DocumentDB](http://go.microsoft.com/fwlink/?LinkId=623215)**.
