# <a name="custom-connectors-in-logic-apps"></a>Пользовательские соединители в Logic Apps

Рабочие процессы и приложения можно создавать с помощью [Azure Logic Apps](https://azure.microsoft.com/services/logic-apps), [Microsoft Flow](https://flow.microsoft.com) или [PowerApps](https://powerapps.microsoft.com) без необходимости писать код. Чтобы помочь вам интегрировать приложения, данные и бизнес-процессы, эти службы предлагают [более 200 соединителей](https://docs.microsoft.com/connectors/). Они подходят для служб и продуктов Майкрософт, таких как Azure и SQL Server, а также сторонних служб, таких как GitHub, Salesforce, Twitter и т. д.

Тем не менее, иногда требуется вызвать API, службы или системы, которые не предоставляются в качестве готовых соединителей. Для поддержки более специализированных сценариев можно создать *пользовательские соединители*, которые будут содержать собственные триггеры и действия. На сайте с документацией о соединителях имеются базовые и расширенные руководства по пользовательским соединителям. Начните с [обзора пользовательских соединителей](https://docs.microsoft.com/connectors/custom-connectors/). Вы можете перейти непосредственно к этим статьям, чтобы получить сведения в определенной области:

* [Создание соединителя Logic Apps](https://docs.microsoft.com/connectors/custom-connectors/create-logic-apps-connector)

* [Создание пользовательского соединителя с помощью определения OpenAPI](https://docs.microsoft.com/connectors/custom-connectors/define-openapi-definition)

* [Create a custom connector from a Postman collection](https://docs.microsoft.com/connectors/custom-connectors/define-postman-collection) (Создание пользовательского соединителя из коллекции Postman)

* [Use a custom connector from a logic app](https://docs.microsoft.com/connectors/custom-connectors/use-custom-connector-logic-apps)(Применение пользовательского соединителя из приложения логики)

* [Совместное использование пользовательского соединителя в организации](https://docs.microsoft.com/connectors/custom-connectors/share)

* [Отправка пользовательских соединителей на сертификацию Майкрософт](https://docs.microsoft.com/connectors/custom-connectors/submit-certification)

* [Часто задаваемые вопросы о настраиваемых соединителях](https://docs.microsoft.com/connectors/custom-connectors/faq)
