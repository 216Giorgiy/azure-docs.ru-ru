# <a name="what-is-an-integration-account"></a>Что такое учетная запись интеграции?

Учетная запись интеграции предоставляет корпоративным приложениям интеграции, в частности приложениям логики, возможность доступа к артефактам B2B, таким как торговые партнеры, соглашения, карты, схемы, сертификаты и т. д., а также управления ими. Чтобы предоставить такой доступ, свяжите учетную запись интеграции с приложением логики, убедившись в том, что они находятся в *одном и том же расположении Azure*.

## <a name="create-an-integration-account"></a>Создание учетной записи интеграции

1. Войдите на [портал Azure](http://portal.azure.com "Портал Azure"). 

2. В главном меню Azure выберите **Все службы**. Введите в поле поиска "интеграция" и выберите **Учетные записи интеграции**.

   ![Создание учетной записи интеграции](./media/logic-apps-enterprise-integration-accounts/account-1.png)

3. В верхней части страницы нажмите кнопку **Добавить**.

   ![Кнопка "Добавить"](./media/logic-apps-enterprise-integration-accounts/account-3.png)

4. Введите имя для учетной записи интеграции и выберите подписку Azure, которую вы хотите использовать. Вы можете создать **группу ресурсов** или выбрать имеющуюся. Выберите **Расположение** для размещения учетной записи интеграции и **ценовую категорию**. Когда вы будете готовы, нажмите кнопку **Создать**.

   ![Ввод сведений для учетной записи интеграции](./media/logic-apps-enterprise-integration-accounts/account-4.png)

   Azure создает учетную запись интеграции в выбранном расположении. Это займет не больше минуты.

5. Обновите страницу. Вы увидите новую учетную запись интеграции в списке.

   ![Отображение новой учетной записи интеграции](./media/logic-apps-enterprise-integration-accounts/account-5.png) 

Теперь свяжите созданную учетную запись интеграции с приложением логики. 

## <a name="link-an-integration-account-to-a-logic-app"></a>Привязка учетной записи интеграции к приложению логики

Чтобы предоставить приложениям логики доступ к артефактам B2B, таким как торговые партнеры, соглашения, карты и схемы, в учетной записи интеграции, свяжите ее с приложением логики. 

1. На портале Azure выберите приложение логики и проверьте его расположение.

   ![Выбор приложения логики, проверка расположения](./media/logic-apps-enterprise-integration-accounts/linkaccount-1.png)

2. В разделе **Параметры** выберите **Учетная запись интеграции**.

   ![Выбор "Учетная запись интеграции"](./media/logic-apps-enterprise-integration-accounts/linkaccount-2.png)

3. Выберите учетную запись интеграции, которую необходимо связать с приложением логики, из списка **Выберите учетную запись интеграции**. Чтобы завершить связывание, нажмите кнопку **Сохранить**.

   ![Выбор учетной записи интеграции](./media/logic-apps-enterprise-integration-accounts/linkaccount-3.png)

   Отобразится уведомление о том, что учетная запись интеграции связана с вашим приложением логики, а все артефакты в вашей учетной записи интеграции доступны для этого приложения логики.

   ![Приложение логики связано с учетной записью интеграции](./media/logic-apps-enterprise-integration-accounts/linkaccount-5.png)

Теперь, когда ваша учетная запись интеграции связана с приложением логики, вы можете использовать в этом приложении соединители "бизнес-бизнес" (B2B). Некоторые распространенные соединители B2B включают в себя проверку XML, кодирование и декодирование неструктурированных файлов.  

## <a name="delete-your-integration-account"></a>Удаление учетной записи интеграции

1. В главном меню Azure выберите **Все службы**. Введите в поле поиска "интеграция" и выберите **Учетные записи интеграции**.

   ![Поиск учетной записи интеграции](./media/logic-apps-enterprise-integration-accounts/account-1.png)

2. Выберите учетную запись интеграции, которую нужно удалить.

    ![Выбор учетной записи интеграции для удаления](./media/logic-apps-enterprise-integration-accounts/account-5.png)

3. В меню выберите **Удалить**.

    ![Выбор команды "Удалить"](./media/logic-apps-enterprise-integration-accounts/delete.png)

4. Подтвердите свой выбор, чтобы удалить учетную запись интеграции.

## <a name="move-your-integration-account"></a>Перемещение учетной записи интеграции

Чтобы переместить учетную запись интеграции в другую подписку Azure или группу ресурсов, выполните описанные ниже действия. После перемещения учетной записи интеграции необходимо обновить все скрипты, чтобы в них использовались новые идентификаторы ресурсов.

1. В главном меню Azure выберите **Все службы**. Введите в поле поиска "интеграция" и выберите **Учетные записи интеграции**.

   ![Поиск учетной записи интеграции](./media/logic-apps-enterprise-integration-accounts/account-1.png)

2. Выберите учетную запись интеграции, которую нужно переместить. В разделе **Параметры** выберите **Свойства**.

   ![Выбор учетной записи интеграции для перемещения Раздел "Параметры", выбор "Свойства"](./media/logic-apps-enterprise-integration-accounts/move.png)

3. Измените группу ресурсов или подписку Azure, связанную с вашей учетной записью интеграции.

   ![Выбор "Изменить группу ресурсов" или "Изменить подписку"](./media/logic-apps-enterprise-integration-accounts/move-2.png)
