    <h1>Документация по API текстовой аналитики</h1>
    <div class="intro" style="min-width: 200px">
        <p>API текстовой аналитики — это облачная служба, которая предоставляет расширенную обработку необработанного текста на естественных языках и включает в себя три основные функции: анализ тональности, извлечение ключевых фраз и определение языка. Узнайте, как анализировать содержимое разными способами, ознакомившись с нашими шаблонами быстрого запуска, примерами и руководствами.</p>
    </div>
