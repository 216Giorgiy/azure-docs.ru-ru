Перед созданием примера необходимо подписаться на [API средства для поиска аномалий](https://labs.cognitive.microsoft.com/en-us/project-anomaly-finder), который является частью службы Microsoft Cognitive Services. Нажмите кнопку "Подписаться" и войдите в систему с помощью своей учетной записи Майкрософт (Hotmail и т. д), чтобы подписаться на API, после чего воспользуйтесь ключами бесплатной подписки (см. [здесь](https://labs.cognitive.microsoft.com/en-us/subscriptions?productId=/products/AnomalyDetection.Preview&source=labs)). Вы можете использовать как Ключ 1, так и Ключ 2.
