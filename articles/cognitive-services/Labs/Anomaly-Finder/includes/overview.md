## <a name="overview"></a>Обзор
API средства для поиска аномалий позволяет отслеживать данные с привязкой ко времени и обнаруживать в них аномалии благодаря методам машинного обучения, которые адаптируются к уникальным данным и автоматически применяют правильную статистическую модель. Это позволяет организациям поддерживать качество данных, предоставлять надежные услуги, выявлять бизнес-инциденты и улучшать свои подходы к бизнесу, не выполняя трудноразрешимые задачи идентификации нестандартных данных вручную.
