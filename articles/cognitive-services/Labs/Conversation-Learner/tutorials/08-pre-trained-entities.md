# <a name="how-to-add-pre-trained-entities"></a>Добавление предварительно обученных сущностей
В этом руководстве описано, как добавить предварительно обученные сущности в модель Conversation Learner.

## <a name="video"></a>Видео

[![Предварительная версия учебника по предварительно обученным сущностям](https://aka.ms/cl_Tutorial_v3_PreTrainedEntities_Preview)](https://aka.ms/cl_Tutorial_v3_PreTrainedEntities)

## <a name="requirements"></a>Требования
Для работы с этим руководством требуется запущенный бот обучения общего назначения.

    npm run tutorial-general

## <a name="details"></a>Сведения

Предварительно обученные сущности распознают общие типы сущностей, например числа, даты, денежные суммы и другие.  Они работают сразу же, не требуют никакого обучения, и их поведение не может быть изменено в отличие от пользовательских сущностей.  По умолчанию предварительно обученные сущности многозначны и содержат все обнаруженные экземпляры сущности.

## <a name="steps"></a>Действия

### <a name="create-the-model"></a>Создание модели

1. В пользовательском веб-интерфейсе щелкните "Создать модель".
2. В поле "Имя" введите PretrainedEntities и нажмите клавишу ВВОД.
3. Нажмите кнопку "Создать".

### <a name="entity-creation"></a>Создание сущности

1. На левой панели щелкните "Сущности", а затем нажмите кнопку "Создать сущность".
2. В поле "Тип сущности" укажите Pre-Trained/datetimeV2.
3. Отметьте флажком пункт "Многозначная".
    - Многозначные сущности накапливают одно значение или несколько.
    - Предварительно обученные сущности не могут иметь отрицаемых свойств.
4. Нажмите кнопку "Создать".

![](../media/tutorial7_entities_a.PNG)

### <a name="create-the-first-action"></a>Создание первого действия

1. На левой панели щелкните "Действия", а затем нажмите кнопку "Создать действие".
2. В поле "Ответ бота..." введите The date is $builtin-datetimev2 (Дата — $builtin-datetimev2).
3. Нажмите кнопку "Создать".

![](../media/tutorial7_actions_a.PNG)

### <a name="create-the-second-action"></a>Создание второго действия

1. На левой панели щелкните "Действия", а затем нажмите кнопку "Создать действие".
2. В поле "Ответ бота..." введите What's the date? (Какой сегодня день?).
    - Предварительно обученные сущности не могут использоваться в качестве обязательных, так как они распознаются по умолчанию для всех высказываний пользователей.
3. В поле "Запрещенные сущности" введите builtin-datetimev2.
4. Нажмите кнопку "Создать".

![](../media/tutorial7_actions2_a.PNG)

### <a name="train-the-model"></a>Обучение модели

1. На левой панели щелкните "Обучающие диалоги" и нажмите кнопку "Создать обучающий диалог".
2. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите текст hello (привет).
3. Нажмите кнопку "Оценить действия".
4. Выберите ответ What's the date? (Какой сегодня день?).
5. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите текст today (сегодня).
    - Высказывание today (сегодня) автоматически распознается в предварительно обученных моделях в LUIS.
    - При наведении указателя мыши на значения предварительно обученных сущностей отображаются дополнительные данные, предоставляемые LUIS.
