# <a name="how-to-use-negatable-entities-with-a-conversation-learner-model"></a>Использование отрицаемых сущностей в модели Conversation Learner

В этом руководстве описано свойство отрицаемости для сущностей.

## <a name="video"></a>Видео

[![Предварительная версия учебника по отрицаемым сущностям](https://aka.ms/cl_Tutorial_v3_NegatableEntities_Preview)](https://aka.ms/cl_Tutorial_v3_NegatableEntities)

## <a name="requirements"></a>Требования
Для работы с этим руководством требуется запущенный бот обучения общего назначения.

    npm run tutorial-general

## <a name="details"></a>Сведения
С помощью свойства отрицаемости сущности можно пометить как обычные (положительные) и отрицательные экземпляры сущности, выполнять обучение на основе положительных и отрицательных моделей и очищать значение существующей сущности. Сущности с установленным свойством отрицаемости называются отрицаемыми сущностями в Conversation Leaner.

## <a name="steps"></a>Действия

### <a name="create-the-model"></a>Создание модели

1. В пользовательском веб-интерфейсе щелкните "Создать модель".
2. В поле "Имя" введите NegatableEntity и нажмите клавишу ВВОД.
3. Нажмите кнопку "Создать".

### <a name="entity-creation"></a>Создание сущности

1. На левой панели щелкните "Сущности", а затем нажмите кнопку "Создать сущность".
2. Выберите "Пользовательская" в поле "Тип сущности".
3. В поле "Имя сущности" укажите name (имя).
4. Отметьте флажком пункт "Отрицаемая".
    - При выборе этого флажка пользователь может указать значение сущности или указать, что что-то является *отрицанием* значения сущности. В последнем случае будет удалено соответствующее значение сущности.
5. Нажмите кнопку "Создать".

![](../media/tutorial5_entities.PNG)

### <a name="create-the-first-action"></a>Создание первого действия

1. На левой панели щелкните "Действия", а затем нажмите кнопку "Создать действие".
2. В поле "Ответ бота..." введите I don't know your name (Я не знаю, как Вас зовут).
3. В поле "Запрещенные сущности" введите name (имя).
4. Нажмите кнопку "Создать".

### <a name="create-the-second-action"></a>Создание второго действия

1. На левой панели щелкните "Действия", а затем нажмите кнопку "Создать действие".
2. В поле "Ответ бота..." введите I know your name. It is $name (Я знаю, как Вас зовут. Вас зовут $имя).
3. Нажмите кнопку "Создать".

> [!NOTE]
> Сущность name (имя) была автоматически добавлена как "Обязательная сущность" с помощью ссылки в ответе.

Теперь у вас есть два действия.

![](../media/tutorial5_actions.PNG)

### <a name="train-the-model"></a>Обучение модели

1. На левой панели щелкните "Обучающие диалоги" и нажмите кнопку "Создать обучающий диалог".
2. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите текст hello (привет).
3. Нажмите кнопку "Оценить действия".
4. Выберите ответ I don't know your name (Я не знаю, как Вас зовут).
    - Процентиль равен 100%, так как это единственное допустимое действие на основе ограничений.
5. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите текст My name is Frank (Меня зовут Фрэнк).
6. Выберите Frank (Фрэнк) и щелкните метку +name (+ имя).
    - Сущность name (имя) представлена в двух вариантах: +name и -name.  (+) Плюс добавляет или перезаписывает значение. (-) Минус удаляет значение.
7. Нажмите кнопку "Оценить действия".
    - Сущность name (имя) теперь определена как "Фрэнк" в памяти модели, поэтому теперь доступно действие I know your name. It is $name (Я знаю, как Вас зовут. Вас зовут $имя).
8. Выберите ответ I know your name. It is $name (Я знаю, как Вас зовут. Вас зовут $имя).
9. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите текст My name is not Frank (Меня зовут не Фрэнк).
10. Выберите Frank (Фрэнк) и щелкните метку -name (– имя).
    - Мы выбираем метку -name (– имя), чтобы очистить текущее значение сущности.
11. Нажмите кнопку "Оценить действия".
12. Выберите ответ I don't know your name (Я не знаю, как Вас зовут).
13. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите текст My name is Susan (Меня зовут Сьюзан).
14. Выберите Susan (Сьюзан) и щелкните метку +name (+ имя).
