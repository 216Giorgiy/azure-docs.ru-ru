# <a name="how-to-use-multi-value-entities-with-a-conversation-learner-model"></a>Использование многозначных сущностей в модели Conversation Learner
В этом руководстве показано, как использовать многозначные свойства для сущностей.

## <a name="video"></a>Видео

[![Предварительная версия учебника по многозначным сущностям](https://aka.ms/cl_Tutorial_v3_MultiValued_Preview)](https://aka.ms/cl_Tutorial_v3_MultiValued)

## <a name="requirements"></a>Требования
Для работы с этим руководством требуется запущенный бот обучения общего назначения.

    npm run tutorial-general

## <a name="details"></a>Сведения
Многозначные сущности накапливают значения в список, а не хранят только одно значение.  Эти сущности полезны в тех случаях, когда пользователи могут указать более одного значения. Начинки для пиццы, например.

Сущности, помеченные как многозначные, будут содержать каждый распознанный экземпляр сущности в списке в памяти бота. Последующие распознавания обновляют значение сущности, а не перезаписывают его.

## <a name="steps"></a>Действия

### <a name="create-the-model"></a>Создание модели

1. В пользовательском веб-интерфейсе щелкните "Создать модель".
2. В поле "Имя" введите MultiValueEntities и нажмите клавишу ВВОД.
3. Нажмите кнопку "Создать".

### <a name="entity-creation"></a>Создание сущности

1. На левой панели щелкните "Сущности", а затем нажмите кнопку "Создать сущность".
2. Выберите "Пользовательская обучаемая" в поле "Тип сущности".
3. Введите toppings (начинки) в поле "Имя сущности".
4. Отметьте флажком пункт "Многозначная".
    - Многозначные сущности накапливают одно значение или несколько.
5. Отметьте флажком пункт "Подлежит отрицанию".
    - Свойство "Подлежит отрицанию" описано в другом учебнике.
6. Нажмите кнопку "Создать".

![](../media/tutorial6_entities.PNG)

### <a name="create-the-first-action"></a>Создание первого действия

1. На левой панели щелкните "Действия", а затем нажмите кнопку "Создать действие".
2. В поле "Ответ бота..." введите Here are your toppings: $Toppings (Вы выбрали такие начинки: $Toppings).
    - Знак доллара в начале указывает ссылку на сущность.
3. Нажмите кнопку "Создать".

### <a name="create-the-second-action"></a>Создание второго действия

1. На левой панели щелкните "Действия", а затем нажмите кнопку "Создать действие".
2. В поле "Ответ бота..." введите What toppings would you like? (Какие начинки вы предпочитаете?).
3. В поле "Запрещенные сущности" введите toppings (начинки).
4. Нажмите кнопку "Создать".

Теперь у вас есть два действия.

![](../media/tutorial6_actions.PNG)

### <a name="train-the-model"></a>Обучение модели

1. На левой панели щелкните "Обучающие диалоги" и нажмите кнопку "Создать обучающий диалог".
2. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите hi (привет).
3. Нажмите кнопку "Оценить действия".
4. Выберите ответ What toppings would you like? (Какие начинки вы предпочитаете?).
    - Процентиль — 100 %, так как это единственное допустимое действие на основе ограничений.
5. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите текст cheese and mushrooms (сыр и грибы).
6. Нажмите cheese (сыр) и выберите метку +toppings (+ начинки).
7. Нажмите mushrooms (грибы) и выберите метку +toppings (+ начинки).
8. Нажмите кнопку "Оценить действия".
9. Выберите ответ Here are your toppings: $toppings (Вы выбрали такие начинки: $toppings).
10. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите текст add pepper (добавить перец).
11. Нажмите pepper (перец) и выберите метку +toppings (+ начинки).
12. Нажмите кнопку "Оценить действия".
13. Выберите ответ Here are your toppings: $toppings (Вы выбрали такие начинки: $toppings).
14. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите текст remove cheese (убрать сыр).
15. Нажмите cheese (сыр) и выберите метку -toppings (– начинки).
16. Нажмите кнопку "Оценить действия".
17. Выберите ответ Here are your toppings: $toppings (Вы выбрали такие начинки: $toppings).
