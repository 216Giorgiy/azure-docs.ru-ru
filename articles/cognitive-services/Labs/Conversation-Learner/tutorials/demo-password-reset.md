# <a name="demo-password-reset"></a>Демонстрация: сброс пароля
В этой демонстрации показано, как создать простой бот технической поддержки, который поможет сбросить пароль. 

Вы узнаете, как средство для обучения ведению диалога может изучать нестандартные диалоговые потоки информации, сложными последовательности, включая классы за пределами домена. В демонстрации код или сущности не используются.

## <a name="video"></a>Видео

[![Предварительная версия демонстрации сброса пароля](http://aka.ms/cl-demo-password-preview)](http://aka.ms/blis-demo-password)

## <a name="requirements"></a>Требования
Для работы с этим руководством требуется запущенный бот сброса пароля.

    npm run demo-password

### <a name="open-the-demo"></a>Запуск демонстрационного примера

В списке моделей пользовательского веб-интерфейса выберите "Tutorial Demo Password Reset" (Демонстрация сброса пароля для работы с руководством). 

### <a name="actions"></a>Действия

Мы создали набор действий для работы с решениями, в которых используется пароль.

![](../media/tutorial_pw_reset_actions.PNG)

### <a name="training-dialogs"></a>Обучение диалогам

Вы можете обучить бот разным диалогам. Также можно использовать демонстрации классов за пределами домена. Например, запросы пользователя, связанные с направлениями движения, выходят за пределы домена. Боту предоставляются примеры нескольких запросов за пределами домена, и он может отвечать "I can't help with that" (Я не могу вам помочь).

![](../media/tutorial_pw_reset_entities.PNG)

В качестве примера давайте проведем сеанс обучения.

1. Щелкните "Train Dialogs" (Обучение диалогам), а затем — "New Train Dialog" (Обучение новому диалогу).
1. Введите "I lost my password" (Я не помню пароль).
2. Щелкните "Score Action" (Оценить действие).
3. Щелкните "Is that for your local account or Microsoft account?" (Это пароль для локальной учетной записи или учетной записи Майкрософт?).
4. Введите "Local account" (Локальная учетная запись).
5. Щелкните "Score Actions" (Оценить действия).
3. Щелкните, чтобы выбрать "Which version of Windows do you have?" (Какая у вас версия Windows?).
4. Введите "Windows 8".
5. Щелкните "Score Actions" (Оценить действия).
6. Выберите "SOLUTION: how to reset password on Windows 8." (РЕШЕНИЕ. Как сбросить пароль в Windows 8.).
4. Щелкните "Done Teaching" (Завершить обучение).

Давайте проверим, как бот может изучить класс за пределами домена.

1. Щелкните "Train Dialogs" (Обучение диалогам), а затем — "New Train Dialog" (Обучение новому диалогу).
1. Введите "web search" (веб-поиск).
    - Это пример класса за пределами домена. 
2. Щелкните "Score Action" (Оценить действие).
3. Щелкните, чтобы выбрать "Sorry, I can't help with that." (К сожалению, я не могу вам помочь.).
    - Обратите внимание, что оценка этого варианта сейчас низкая. После дополнительного обучения она повысится.
4. Щелкните "Done Teaching" (Завершить обучение).

Вы узнали, как создать демонстрацию базовой технической поддержки и как этот пример может обучаться для предоставления решений и обработки простых запросов.
