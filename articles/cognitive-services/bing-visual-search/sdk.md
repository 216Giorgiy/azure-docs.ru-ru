# <a name="bing-visual-search-sdk"></a>Пакет SDK для API Bing для наглядного поиска

Наглядный поиск Bing предоставляет следующие клиентские пакеты SDK, которые можно использовать для запуска API наглядного поиска:

* [Примеры для .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
    * [Пакет NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.VisualSearch/1.2.0).
    * [библиотеки .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingVisualSearch) с определениями и зависимостями.
* [Примеры для Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * [библиотеки Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/imageSearch) с определениями и зависимостями.
* [Примеры для Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples): 
    * [библиотеки Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples/tree/master/Search) с определениями и зависимостями.
* [Примеры для Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * [библиотеки Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-imagesearch) с определениями и зависимостями.

Примеры пакетов SDK для каждого языка содержат файл сведений, в котором указаны предварительные требования к использованию пакета SDK и инструкции по установке и запуску этих примеров.

### <a name="sdk-tools"></a>Средства пакета SDK
Средства пакета SDK для изучения возможностей визуального поиска Bing можно найти в репозитории Git [Bing Visual Search Tools](https://github.com/Azure-Samples/bing-visualsearch-tools)
