# <a name="bing-spell-check-sdk-samples"></a>Примеры пакета SDK для API проверки орфографии Bing
Примеры использования API проверки орфографии Bing:
1. Поиск запроса с ошибками и анализ ответа.
2. Активация сообщения об ошибке из API.

Пакеты SDK для API проверки орфографии Bing предоставляют доступ к контекстной проверке грамматики и орфографии на следующих языках программирования:
* Начало работы с [примерами для .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7). 
    * [Пакет NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.SpellCheck/1.2.0).
    * См. также определения и зависимости в [библиотеках .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Language/SpellCheck).
* Начало работы с [примерами для Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples). 
    * См. также определения и зависимости в [библиотеках Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/spellCheck).
* Начало работы с [примерами для Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples). 
    * См. также определения и зависимости в [библиотеках Java](https://github.com/Azure/azure-sdk-for-java/tree/master/cognitiveservices/azure-spellcheck).
* Начало работы с [примерами для Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples). 
    * См. также определения и зависимости в [библиотеках Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-language-spellcheck).

Примеры пакетов SDK для каждого языка содержат файл сведений, в котором указаны предварительные требования и инструкции по установке и запуску этих примеров.
