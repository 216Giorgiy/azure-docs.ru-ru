# <a name="emotion-api-frequently-asked-questions"></a>Часто задаваемые вопросы об API распознавания эмоций

> 15 февраля 2019 г. API распознавания эмоций будет отмечен как нерекомендуемый. Сейчас функция распознавания эмоций является общедоступной в составе [API распознавания лиц](https://docs.microsoft.com/azure/cognitive-services/face/).

-----

**Вопрос**. *Какие типы изображений позволяют получить лучшие результаты из API распознавания эмоций?*

**Ответ**. Для получения наилучших результатов используйте полные изображения лиц в анфас, которые ничем не заслонены. Надежность уменьшается из-за наличия неполных изображений лиц в анфас. API распознавания эмоций может не распознать эмоции на изображениях с лицами, повернутыми на более чем 45 градусов.

-----

**Вопрос**. *Сколько эмоций может определить API распознавания эмоций?*

**Ответ**. API распознавания эмоций определяет восемь универсальных эмоций:
* счастье;
* грусть;
* удивление;
* гнев;
* страх;
* презрение;
* отвращение;
* нейтральное выражение.

-----

**Вопрос**. *Есть ли возможность передавать видеопоток в API в реальном времени и одновременно получать результаты?*

**Ответ**. Используйте API распознавания эмоций на основе изображений и вызывайте его для каждого кадра или пропускайте некоторые кадры, чтобы повысить производительность.  Доступны примеры покадрового анализа видео.

-----

**Вопрос**. *Я передаю двоичные данные изображения, но получаю сообщение: "Invalid face image" (Недопустимое изображение лица)**.

**Ответ**. Получение такого сообщения означает, что при анализе изображения с помощью алгоритма возникла проблема.  
* Поддерживаемые форматы входных изображений включают в себя JPEG, PNG, GIF (первый кадр) и BMP
* Размер изображения должен быть не больше 4 МБ
* Диапазон определяемого размера лица составляет от 36 х 36 до 4096 х 4096 пикселей. Лица вне этого диапазона не обнаруживаются
* Некоторые лица могут не обнаруживаться из-за технических проблем, таких как большой поворот головы и существенное перекрытие. Лучшие результаты определяются для лиц в анфас или практически в анфас.
