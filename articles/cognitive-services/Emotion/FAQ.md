**Вопрос**. *Какие типы изображений позволяют получить лучшие результаты из API распознавания эмоций?*

**Ответ**. Для получения наилучших результатов используйте полные изображения лиц в анфас, которые ничем не заслонены. Надежность уменьшается из-за наличия неполных изображений лиц в анфас. API распознавания эмоций может не распознать эмоции на изображениях с лицами, повернутыми на 45 градусов или больше.

-----

**Вопрос**. *Сколько эмоций может определить API распознавания эмоций?*

**Ответ**. API распознавания эмоций определяет восемь универсальных эмоций: 
* счастье;
* грусть;
* удивление;
* гнев;
* страх;
* презрение;
* отвращение; 
* нейтральное выражение. 

-----

**Вопрос**. *Есть ли возможность передавать видеопоток в API в реальном времени и одновременно получать результаты?*

**Ответ**. Используйте API распознавания эмоций на основе изображений и вызывайте его для каждого кадра или пропускайте некоторые кадры, чтобы повысить производительность.  Доступны примеры покадрового анализа видео.

-----

**Вопрос**. *Я передаю двоичные данные изображения, но получаю сообщение "Invalid face image" (Недопустимое изображение лица).**

**Ответ**. Это означает, что при анализе изображения с помощью алгоритма возникла проблема.  
* Поддерживаемые форматы входных изображений включает в себя JPEG, PNG, GIF (первый кадр) и BMP. 
* Размер изображения должен быть не больше 4 МБ.
* Диапазон размеров обнаруживаемых лиц — от 36 x 36 до 4096 x 4096 пикселей. Лица вне этого диапазона не обнаруживаются.
* Некоторые лица могут не обнаруживаться из-за технических проблем, например слишком большого лицевого угла (поворота головы) и существенного перекрытия. Лучшие результаты определяются для лиц в анфас или практически в анфас.
