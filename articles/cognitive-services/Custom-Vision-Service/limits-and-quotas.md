# <a name="limits-and-quotas"></a>Ограничения и квоты

Существуют три категории ключей для Пользовательской службы визуального распознавания. Ресурсы категорий F0 и S0 предоставляются через портал Azure. Подробные сведения о ценах и определениях транзакций можно получить на [странице цен](https://azure.microsoft.com/pricing/details/cognitive-services/custom-vision-service/).  Проекты категории F0 можно обновить до проектов категории S0.

Ресурсы проекта ограниченной пробной версии подключаются к вашему имени для входа в Пользовательскую службу визуального распознавания (иными словами, к учетной записи AAD или учетной записи MSA). Они предназначены для краткосрочного пробного использования службы.  Учетные записи, созданные на этапе предварительной бесплатной пробной версии, то есть до выпуска предварительных версий Azure (1 марта 2018 года), сохранят действовавшие ранее квоты для ограниченных пробных версий.
