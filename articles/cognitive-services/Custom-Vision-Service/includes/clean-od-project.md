## <a name="clean-up-resources"></a>Очистка ресурсов
Если нужно реализовать собственный проект распознавания объектов (или использовать вместо него проект [классификации изображений](../csharp-tutorial.md)), вы можете удалить проект распознавания вилок и ножниц из этого примера. При работе в бесплатной пробной версии Пользовательской службы визуального распознавания можно использовать два проекта.

На веб-сайте [Пользовательской службы визуального распознавания](https://customvision.ai) перейдите в раздел **проектов** и щелкните значок корзины под заголовком "My New Project" (Мой новый проект).
