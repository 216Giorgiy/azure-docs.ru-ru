# <a name="how-to-move-your-limited-trial-project-to-azure-using-the-customvisionai-site"></a>Как переместить проект ограниченной пробной версии в Azure с помощью сайта CustomVision.ai


Так как пользовательская служба визуального распознавания теперь находится в [предварительной версии Azure](https://azure.microsoft.com/services/preview/), заканчивается поддержка для проектов ограниченной пробной версии за пределами Azure. В этом документе описывается использование [концепции пользовательских веб-сайт](https://customvision.ai) перемещение проекта ограниченной пробной версии нужно связать с ресурсом Azure. 

> [!NOTE]
> При перемещении проектов Custom Vision к ресурсу Azure они наследовать базовый [разрешения]( https://docs.microsoft.com/azure/role-based-access-control/role-assignments-portal) этого ресурса Azure. Если владельцы ресурса Azure, проект другим пользователям в вашей организации, они смогут для доступа к проекту на [концепции пользовательских веб-сайт](https://customvision.ai). Аналогично при удалении ресурсов будут удалены проектов.  


Введение в основные понятия Azure подписок и ресурсов, см. в разделе [руководством для разработчиков Azure.](https://docs.microsoft.com/azure/guides/developer/azure-developer-guide#manage-your-subscriptions)


## <a name="prerequisites"></a>Технические условия

Вам потребуется действительная подписка Azure, связанные с одной учетной записи Майкрософт или учетной записи Azure Active Directory (AAD), используйте для входа в [концепции пользовательских веб-сайт](https://customvision.ai). 

Если у вас нет учетной записи Azure [создать учетную запись](https://azure.microsoft.com/free/) бесплатно.


## <a name="create-custom-vision-resources-in-the-azure-portal"></a>Создание пользовательских концепции ресурсов на портале Azure
Использование пользовательской службы визуального распознавания в Azure, необходимо создать ресурсы Custom Vision обучения и прогнозирования в [портала Azure](https://portal.azure.com/?microsoft_azure_marketplace_ItemHideKey=microsoft_azure_cognitiveservices_customvision#create/Microsoft.CognitiveServicesCustomVision). 

 Для перемещения с помощью этого проекта [концепции пользовательских веб-сайт](https://customvision.ai) качества, необходимо создать ресурсы в южно-центральном регионе США, так как все проекты ограниченной пробной версии, размещенные в южно-Центральная часть США. 

Несколько проектов может быть связан один ресурс. Более подробные сведения о [цены и ограничения](https://docs.microsoft.com/azure/cognitive-services/custom-vision-service/limits-and-quotas) доступен. Чтобы продолжить использование пользовательской службы визуального распознавания, бесплатно, можно выбрать уровень F0 на портале Azure. 


## <a name="move-your-limited-trial-project-to-an-azure-resource"></a>Переместить ваш проект ограниченной пробной версии для ресурса Azure

1.  В веб-браузере перейдите к [концепции пользовательских веб-сайт](https://customvision.ai) и выберите __вход__. Откройте проект, который вы хотите перенести учетную запись Azure. 
2.  Откройте страницу параметров для проекта, щелкнув значок шестеренки в верхнем правом углу экрана. 

    ![Параметры проекта расположен значок шестеренки в правой верхней части страницы проекта.](./media/move-your-project-to-azure/settings-icon.png)


3. Щелкните __переместить в Azure__.

    ![Переход на Azure кнопка находится в нижнем левом углу на странице параметров проекта.](./media/move-your-project-to-azure/move-to-azure.jpg)


4. Из раскрывающегося списка на __переместить в Azure__ "Кнопка", выберите ресурс Azure, вы хотите переместить проект. Нажмите кнопку __переместить__. 

5. Если вы не видите ресурс Azure, созданную ранее для пользовательской службы визуального распознавания, возможно, в другом каталоге. Чтобы переместить проект к ресурсу в другом каталоге, выполните приведенные ниже инструкции. 

    ![Окно миграции проекта.](./media/move-your-project-to-azure/Project_Migration_Window.jpg)


## <a name="move-project-to-another-azure-directory"></a>Перемещение проекта в другой каталог Azure 

> [!NOTE]
> В портале Azure и CustomVision.ai можно выбрать каталог из раскрывающегося меню пользователя в правом верхнем углу экрана.   


1. Определите каталог, в котором выполняется к ресурсам Azure. Вы найдете каталог, в списке под именем пользователя в правой верхней строке меню портала Azure. 

    ![Каталог отображается под именем пользователя в правой верхней строке меню портала Azure. .](./media/move-your-project-to-azure/identify_directory.jpg)

2. Найти идентификатор ресурса Custom Vision обучения. Это можно найти на портале Azure, открыв ресурс Custom Vision обучения и выбрав «Свойства» в разделе «Управление ресурсами». Идентификатор вашего ресурса будет здесь. 

    ![Идентификатор ресурса можно узнать на портале Azure, открыв ресурс Custom Vision обучения и выбрав «Свойства» в разделе «Управление ресурсами».](./media/move-your-project-to-azure/resource_ID_azure_portal.jpg)


3. Кроме того, можно найти идентификатор ресурса для ресурса Custom Vision непосредственно в веб-сайт Custom Vision [странице "Параметры"]( https://www.customvision.ai/projects#/settings). Необходимо будет переключиться к ресурсам Azure, состоит в том же каталоге.

    ![Для каждого ресурса на вашей странице "Параметры" на веб-сайте Custom Vision указан идентификатор вашего ресурса.](./media/move-your-project-to-azure/resource_ID_CVS_portal.jpg)

4. Теперь, когда у вас есть идентификатор вашего ресурса, вернитесь в проект Custom Vision, которому вы пытаетесь переместить из ограниченной пробной версии в ресурс Azure. Напоминание, может потребоваться переключиться обратно на ваш исходный каталог, чтобы найти его. Следуйте инструкциям, приведенным [выше](#move-your-limited-trial-project-to-an-azure-resource) откройте страницу параметров проекта и выберите __переместить в Azure__. 


5. При переходе к окну Azure установите флажок для «Переместить в другой каталог Azure?». Выберите каталог, который вы хотите переместить проект и введите идентификатор ресурса для ресурса, к которому вы перемещаете проект. Нажмите кнопку __переместить__. 



5. Помните, что проект сейчас находится в другом каталоге. Для поиска проекта, необходимо будет переключиться на том же каталоге на веб-портале Custom Vision, находящееся в проекте. Оба на портале Azure и [концепции пользовательских веб-сайт](https://customvision.ai), можно выбрать каталог в учетной записи выберите в раскрывающемся меню в правом верхнем углу экрана. 

## <a name="next-steps"></a>Дальнейшие действия

Теперь проект перемещен к ресурсу Azure. Необходимо будет обновить ключи во всех приложениях, которые вы написали обучения и прогнозирования.
