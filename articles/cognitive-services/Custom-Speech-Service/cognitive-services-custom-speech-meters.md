# <a name="custom-speech-service-meters-and-quotas"></a>Единицы измерения и квоты Пользовательской службы распознавания речи

С помощью облачной Пользовательской службы распознавания речи можно настраивать модели речи для преобразования речи в текст.

Чтобы приступить к работе с Пользовательской службой распознавания речи, перейдите на [портал Пользовательской службы распознавания речи](https://cris.ai).

Сведения о текущих единицах измерения и ценах можно получить на странице [Цены на Cognitive Services. Пользовательская служба распознавания речи](https://azure.microsoft.com/pricing/details/cognitive-services/custom-speech-service/).

## <a name="tiers-explained"></a>Описание категорий
Для тестирования и создания прототипов рекомендуется ценовая категория "Бесплатный" (F0). Для рабочих систем рекомендуется использовать категорию S2. С помощью категории S2 можно масштабировать развертывание до числа единиц масштабирования (SU), необходимых для вашего сценария.

> [!NOTE]
> *Невозможно* переходить из категории F0 в S2 и наоборот.
>

## <a name="meters-explained"></a>Описание единиц измерения

### <a name="scale-out"></a>Scale Out
Scale Out — это новая функция, которая представлена в новой модели ценообразования. С помощью Scale Out можно выбрать число параллельных запросов, которые может обрабатывать модель.

Можно задать число параллельных запросов с помощью меры единиц масштабирования в представлении **Create Model Deployment** (Создание развертывания модели). Дополнительные сведения см. в статье [Создание пользовательской конечной точки преобразования речи в текст](CustomSpeech-How-to-Topics/cognitive-services-custom-speech-create-endpoint.md). В зависимости от объема трафика, использование которого предусмотрено в модели, можно выбрать соответствующее количество единиц масштабирования. 

> [!NOTE]
> Каждая единица масштабирования гарантирует обработку 5 параллельных запросов. Вы можете приобрести 1 или несколько единиц масштабирования в соответствии с потребностями. Так как число единиц масштабирования увеличивается с шагом, равным 1, то число гарантируемых параллельных запросов увеличивается с шагом, равным 5.
>

### <a name="log-management"></a>Управление журналами
Вы можете отключить отслеживание звуковых данных для новой развернутой модели за дополнительную плату. Пользовательская служба распознавания речи не будет вести журнал запросов звуковых данных или расшифровок из этой модели.
