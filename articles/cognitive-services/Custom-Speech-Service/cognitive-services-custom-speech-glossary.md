# <a name="glossary"></a>Глоссарий

## <a name="a"></a>A

### <a name="acoustic-model"></a>Акустическая модель

Акустическая модель — это классификатор, сопоставляющий краткие звуковые фрагменты с одной из нескольких фонем или звуковых единиц конкретного языка. Например, слово "речь" состоит из трех фонем: [р’эч’]. Эти классификации выполняются приблизительно 100 раз в секунду.

## <a name="b"></a>b

## <a name="c"></a>C

### <a name="conversational-model"></a>Разговорная модель

Модель, подходящая для распознавания речи при общении. Разговорная акустическая модель Майкрософт адаптирована для речи, как правило, направленной к другому человеку.

## <a name="d"></a>D

### <a name="deployment"></a>Развертывание

Процесс превращения адаптированной пользовательской модели в службу с общедоступным универсальным кодом ресурса (URI).

## <a name="i"></a>I

### <a name="inverse-text-normalization"></a>Обратная нормализация текста

Процесс преобразования необработанного (неформатированного) текста обратно в форматированный текст, т. е. в текст с учетом регистра и знаков препинания, называется обратной нормализацией текста (ITN).

## <a name="l"></a>L

### <a name="language-model"></a>Языковая модель

Языковая модель — это распределение вероятностей для последовательностей слов. Она помогает системе выбрать подходящую из нескольких созвучных последовательностей слов, оценивая вероятность возникновения таких последовательностей.

## <a name="n"></a>Нет

### <a name="normalization"></a>Нормализация

Нормализация (текста): преобразование полученного текста (т. е. расшифровки) в стандартную однозначную форму, распознаваемую системой.

## <a name="s"></a>S

### <a name="search-and-dictate-model"></a>Модель поиска и диктовки

Акустическая модель, подходящая для обработки команд. Акустическая модель поиска и диктовки Майкрософт подходит для речи, направленной к приложению или устройству, например команд.

### <a name="subscription-key"></a>Ключ подписки

Ключ подписки — это строка, которую необходимо указать в качестве параметра строки запроса для вызова любой модели Пользовательской службы распознавания речи. Ключ подписки можно получить на [портале Azure](https://portal.azure.com/#create/Microsoft.CognitiveServices/apitype/CustomSpeech), после чего его можно будет найти в разделе "My Subscriptions" (Мои подписки) на портале Пользовательской службы распознавания речи.

## <a name="t"></a>T

### <a name="transcription"></a>Расшифровка

Расшифровка: фрагмент текста, полученный в результате обработки части звукового WAV-файла.
