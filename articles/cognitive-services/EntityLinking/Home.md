# <a name="what-is-the-entity-linking-intelligence-service-api"></a>Что такое API аналитической службы связывания сущностей?

Вас приветствует Аналитическая служба связывания сущностей — веб-служба, призванная помочь разработчикам выполнять задачи, относящиеся к связыванию сущностей.

## <a name="entity-linking"></a>Связывание сущностей

Иногда в разных контекстах слово может использоваться в качестве именованной сущности, глагола или другой словоформы в данном предложении. Например, в случае, когда слово "таймс" является именованной сущностью, оно все равно может относиться к двум отдельно различимым сущностям, таким как "Нью-Йорк Таймс" или "Таймс-сквер". При рассматривании конкретного абзаца в документе Аналитическая служба связывания сущностей распознает и идентифицирует каждую отдельную сущность на основе контекста.  

На рисунке ниже показан пример связывания сущностей. В частности, при использовании Википедии Аналитическая служба связывания сущностей распознает все сущности, упомянутые во входном тексте, и связывает их с соответствующими опорными точками согласно идентификаторам страниц.
