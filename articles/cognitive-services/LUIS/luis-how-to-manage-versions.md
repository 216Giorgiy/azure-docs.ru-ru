# <a name="manage-versions"></a>Управление версиями

Каждый раз, работая с этой моделью, создавайте другую [версию](luis-concept-version.md) приложения. 

## <a name="set-active-version"></a>Настройте активную версию
Для работы с версиями откройте приложение, выбрав его имя на странице **Мои приложения**, затем выберите на верхней панели **Параметры**.

![Страница "Версии"](./media/luis-how-to-manage-versions/settings.png)

На странице**Параметры** содержатся сведения, которые позволяют настроить параметры для всего приложения, включая версии, а также участников совместной работы. 

## <a name="clone-a-version"></a>Клонирование версии
1. На странице **Параметры** после разделов приложения "Параметры" и "Участники совместной работы" найдите строку версии, которую необходимо клонировать. Выберите многоточие (...) в крайнем правом углу. 

    ![Свойства записи версии](./media/luis-how-to-manage-versions/version-section.png)

2. Выберите **Клонировать** в списке.

    ![Выбор свойств записи версии](./media/luis-how-to-manage-versions/version-three-dots-modal.png)

3. В диалоговом окне**Версия клона** введите имя для новой версии, например "0,2".

   ![Диалоговое окно "Версия клона"](./media/luis-how-to-manage-versions/version-clone-version-dialog.png)
 
 > [!NOTE]
 > Идентификатор версии может состоять только из символов, цифр или точки ".". Его длина не может превышать 10 символов.
 
 Новая версия с указанным именем создается и настраивается как активная версия.
 
  ![Версия создана и добавлена в список](./media/luis-how-to-manage-versions/new-version.png)

 > [!NOTE]
 > Как показано на предыдущем изображении, опубликованная версия связывается с цветной отметкой, указывающей тип слота, где она опубликована: рабочая среда (зеленый), промежуточное хранение (красный) и оба значения (черный). Обучение и публикация данных отображаются для каждой опубликованной версии.

## <a name="set-active-version"></a>Настройте активную версию
1. На странице**Параметры** в списке **Версии** выберите многоточие (...) в крайнем правом углу.

2. Во всплывающем списке выберите **Установить как активный**.

    ![Настройте активную версию](./media/luis-how-to-manage-versions/set-active-version.png)

    Активная версия выделена голубым цветом, как показано на следующем снимке экрана.

    ![Активная версия](./media/luis-how-to-manage-versions/set-active-version-done.png) 


## <a name="import-version"></a>Импорт версии
Версию можно импортировать из JSON-файла. После импорта версии она становится активной.

**Чтобы импортировать версию**, выполните следующие действия.

1. На странице **Параметры** выберите кнопку **Импорт новой версии**.

    ![Кнопка "Импорт"](./media/luis-how-to-manage-versions/import-version.png)

2. Нажмите **Обзор** и выберите JSON-файл.

    ![Диалоговое окно "Импорт версии"](./media/luis-how-to-manage-versions/import-version-dialog.png)

Необходимо задать идентификатор версии, если версия уже существует в JSON-файле приложения.

## <a name="export-version"></a>Экспорт версии
Версию можно экспортировать в JSON-файл.

**Чтобы экспортировать версию**, выполните следующие действия.

1. На странице**Параметры** в списке **Версии** выберите многоточие (...) в крайнем правом углу.

2. Во всплывающем списке действий выберите **Экспорт** и место для сохранения файла.

## <a name="delete-a-version"></a>Удаление версии
Версии можно удалять, но по крайней мере одна версия приложения должна остаться. Можно удалить все версии, кроме активной. 

1. На странице**Параметры** в списке **Версии** выберите многоточие (...) в крайнем правом углу.

2. Во всплывающем списке действий выберите **Удалить** и место для сохранения файла.

    ![Подтверждение удаления версии](./media/luis-how-to-manage-versions/delete-menu.png) 


## <a name="rename-a-version"></a>Переименование версии
Версии можно переименовать при условии, что такое имя версии не используется.  

1. На странице**Параметры** в списке **Версии** выберите многоточие (...) в крайнем правом углу.

2. Во всплывающем списке действий выберите **Переименовать**.

3. Введите имя новой версии и нажмите кнопку **Готово**.

    ![Подтверждение переименования версии](./media/luis-how-to-manage-versions/rename-popup.png)
