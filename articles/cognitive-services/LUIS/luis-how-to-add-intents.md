# <a name="manage-intents"></a>Управление намерениями 
Добавьте [намерения](luis-concept-intent.md) в приложение LUIS для определения групп вопросов или команд, которые имеют одинаковые намерения. 

Добавление и управление намерениями осуществляется на странице **Intents** (Намерения), доступной по ссылке **Intents** на левой панели LUIS. 

Ниже приведена процедура добавления намерения Bookflight в приложении TravelAgent.

## <a name="add-intent"></a>Добавление намерения

1. Откройте приложение (например, TravelAgent), щелкнув его имя на странице **Мои приложения**, а затем щелкните **Intents** (Намерения) на левой панели. 
2. На странице **намерений** щелкните **Create new intent** (Создать намерение).

    ![Список намерений](./media/luis-how-to-add-intents/IntentsList.png)
3. В диалоговом окне **создания намерения** введите имя намерения "BookFlight" и нажмите кнопку **Готово**.

    ![Добавление намерения](./media/luis-how-to-add-intents/Addintent-dialogbox.png)

    На странице сведений о добавленном намерении [добавьте фрагменты речи](#add-an-utterance-on-intent-page).

## <a name="rename-intent"></a>Переименование намерения

1. На странице **намерения** щелкните значок переименования ![Rename Intent](./media/luis-how-to-add-intents/Rename-Intent-btn.png) (Переименовать намерение) рядом с именем намерения. 

2. На странице **намерения** в диалоговом окне отображается текущее имя намерения. Измените имя намерения и нажмите клавишу ВВОД. Новое имя сохраняется и отображается на странице намерения.

    ![Изменение намерения](./media/luis-how-to-add-intents/EditIntent-dialogbox.png)

## <a name="delete-intent"></a>Удаление намерения
При удалении намерения, отличного от намерения None, можно добавить все фрагменты речи в намерение None. Это удобно, если требуется перенести фрагменты речи вместо их удаления.   

1. На странице **намерения** нажмите кнопку **Delete Intent** (Удалить намерение) справа от имени намерения. 

    ![Кнопка "Удалить намерение"](./media/luis-how-to-add-intents/DeleteIntent.png)

2. В диалоговом окне подтверждения нажмите кнопку "ОК".

## <a name="add-an-utterance-on-intent-page"></a>Добавление фрагмента речи на странице намерения

На странице намерения введите соответствующий фрагмент речи, который ожидается от пользователей (например, `book 2 adult business tickets to Paris tomorrow on Air France`), в текстовом поле под именем намерения и нажмите клавишу ВВОД. 
 
>[!NOTE]
>LUIS преобразует все фрагменты речи в нижний регистр.

![Снимок экрана страницы сведений о намерении с выделенным фрагментом речи](./media/luis-how-to-add-intents/add-new-utterance-to-intent.png) 

Фрагменты речи добавляются в список фрагментов для текущего намерения. После добавления фрагмента речи [пометьте все сущности](luis-how-to-add-example-utterances.md) во фрагментах и выполните [обучение](luis-how-to-train.md) приложения. 

## <a name="create-a-pattern-from-an-utterance"></a>Создание шаблона на основе фрагмента речи
См. раздел [Добавление шаблона на основе существующего фрагмента речи на странице сущности или намерения](luis-how-to-model-intent-pattern.md#add-pattern-from-existing-utterance-on-intent-or-entity-page).

## <a name="edit-an-utterance-on-intent-page"></a>Изменение фрагмента речи на странице намерения

Чтобы изменить фрагмент речи, выберите значок с многоточием (...) в правом конце строки этого фрагмента, а затем пункт **Изменить**. Измените текст, затем нажмите клавишу ВВОД на клавиатуре.

![Снимок экрана страницы сведений о намерении с выделенным значком с многоточием](./media/luis-how-to-add-intents/edit-utterance.png) 

## <a name="reassign-utterances-on-intent-page"></a>Переназначение фрагментов речи на странице намерения
Вы можете изменить намерение одного или нескольких фрагментов речи путем переназначения их другому намерению. 

Чтобы переназначить один фрагмент речи, в правом конце строки фрагмента речи выберите нужное имя намерения в столбце **Labeled intent** (Намерение с меткой). Фрагмент речи удаляется из списка фрагментов текущего намерения. 

![Снимок экрана страницы намерения BookFlight: намерение фрагмента речи выбрано в столбце Labeled intent](./media/luis-how-to-add-intents/reassign-1-utterance.png)

Чтобы изменить назначение нескольких фрагментов речи, установите флажки слева от фрагментов и выберите **Reassign intent** (Переназначить намерение). Выберите нужное намерение в списке.

![Снимок экрана страницы намерения BookFlight: выбран фрагмент речи и выделена кнопка переназначения намерения](./media/luis-how-to-add-intents/delete-several-utterances.png) 

## <a name="delete-utterances-on-intent-page"></a>Удаление фрагментов речи на странице намерения

Чтобы удалить фрагмент речи, выберите значок с многоточием (...) в правом конце строки этого фрагмента, а затем пункт **Удалить**. Фрагмент речи удаляется из списка и приложения LUIS.

![Снимок экрана страницы сведений о намерении с выделенным пунктом "Удалить"](./media/luis-how-to-add-intents/delete-utterance-ddl.png)

Удаление нескольких фрагментов речи

1. Установите флажки слева от фрагментов речи, а затем выберите **Delete utterances(s)** (Удалить фрагменты речи). 

    ![Снимок экрана страницы сведений о намерении: выбраны фрагменты речи и выделена кнопка удаления фрагментов речи](./media/luis-how-to-add-intents/delete-several-utterances.png)

2. Выберите **Готово** во всплывающем диалоговом окне **Delete utterances?** (Удалить фрагменты речи?).

## <a name="search-in-utterances-on-intent-page"></a>Поиск фрагментов речи на странице намерения
На странице намерения можно искать фрагменты речи, которые содержат текст (слова или фразы). Например, вы могли заметить ошибку, которая относится к конкретному слову, и хотите найти все примеры, включающие это слово. 

1. Щелкните значок лупы на панели инструментов.

    ![Снимок экрана страницы намерений с выделенным значком поиска (лупы)](./media/luis-how-to-add-intents/magnifying-glass.png)

2. Появится текстовое поле поиска. Введите слово или фразу в поле поиска в правом верхнем углу списка фрагментов речи. Список фрагментов речи обновится, и будут показаны только те фрагменты, которые включают искомый текст. 

    ![Снимок экрана страницы намерений с выделенным текстовым полем поиска](./media/luis-how-to-add-intents/search-textbox.png)

    Чтобы отменить поиск и восстановить полный список фрагментов речи, удалите введенный в поле поиска текст. Чтобы закрыть текстовое поле поиска, снова щелкните значок лупы на панели инструментов.

## <a name="prediction-discrepancy-errors-on-intent-page"></a>Ошибки несоответствия прогнозирования на странице намерения
Для фрагмента речи в намерении может возникать расхождение между выбранным намерением и оценкой прогнозирования. Служба LUIS указывает это несоответствие с помощью красной рамки вокруг оценки. 

![Снимок экрана страницы намерения BookFlight с выделенной несоответствующей оценкой прогнозирования](./media/luis-how-to-add-intents/score-discrepancy.png) 

## <a name="filter-by-intent-prediction-discrepancy-errors-on-intent-page"></a>Фильтрация по ошибкам несоответствия прогнозирования намерения на странице намерения
Чтобы отфильтровать список фрагментов речи для вывода только фрагментов с несоответствием прогнозирования намерения, на панели инструментов переключите представление **Показать все** на **Только ошибки**. 

## <a name="filter-by-entity-type-on-intent-page"></a>Фильтрация по типу сущности на странице намерения
Используйте раскрывающееся меню **Entity filters** (Фильтры сущностей) на панели инструментов для фильтрации фрагментов речи по сущности. 

![Снимок экрана страницы намерений с выделенным фильтром по типу сущности](./media/luis-how-to-add-intents/filter-by-entities.png) 

Чтобы удалить фильтр, выберите синее поле фильтра с нужным словом или фразой под панелью инструментов.

## <a name="switch-to-token-view-on-intent-page"></a>Переключение в представление маркеров на странице намерения
Переключите выключатель **Tokens View** (Представление маркеров), чтобы вывести на экран маркеры вместо имен типов сущностей. Кроме того, для переключения представления можно использовать клавиши **CTRL + E** на клавиатуре. 

![Снимок экрана намерения BookFlight с выделенной кнопкой Token View (Представление маркеров)](./media/luis-how-to-add-intents/toggle-tokens-view.png)

## <a name="train-your-app-after-changing-model-with-intents"></a>Обучение приложения после изменения модели с намерениями
После добавления, изменения и удаления намерений выполните [обучение](luis-how-to-train.md) и [публикацию](luis-how-to-publish-app.md) приложения, чтобы применить изменения к запросам конечной точки.
