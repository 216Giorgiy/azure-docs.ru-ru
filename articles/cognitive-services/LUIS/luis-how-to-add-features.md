# <a name="use-features-to-improve-your-luis-apps-performance"></a>Использование функций для повышения производительности приложения LUIS  

В приложение LUIS можно добавить функции для повышения его точности. Функции помогают приложению предоставлять указания, в которых некоторые слова и фразы являются частью категории. Если приложение LUIS обучить распознавать один элемент категории, то оно распознает другие подобным образом.

## <a name="add-phrase-list"></a>Добавление списка фраз

1. Откройте приложение, щелкнув его имя на странице **Мои приложения**, щелкните **Сборка**, а затем на левой панели приложения щелкните **Phrase lists** (Списки фраз). 

    ![Навигация по списку фраз](./media/luis-add-features/phrase-list-nav.png)

2. На странице **Списки фраз** щелкните **Create new phrase list** (Создать список фраз). 
 
    ![Создание списка фраз](./media/luis-add-features/create-new-phrase-list.png)
    
3. В диалоговом окне **Add Phrase List** (Добавление списка фраз) введите имя списка фраз как"Города". В поле **Значение** введите значения списка фраз. Можно ввести только одно значение или набор значений, разделенные запятыми и нажать **ВВОД**.

    ![Добавление списка фраз "Города"](./media/luis-add-features/add-phrase-list-cities.png)

4. Интеллектуальная служба распознавания речи может предложить связанные значения для добавления в список фраз. Щелкните **Рекомендовать**, чтобы получить группу предложенных значений, которые семантически связаны с добавленными значениями. Щелкните любое предложенное значение или нажмите **Add All** (Добавить все), чтобы добавить все.

    ![Предложенные значения списка фраз](./media/luis-add-features/related-values.png)

5. Щелкните **These values are interchangeable** (Эти значения являются взаимозаменяемыми), если добавленные значения списка фраз являются альтернативными, которые можно использовать как взаимозаменяемые.

    ![Предложенные значения списка фраз](./media/luis-add-features/interchangeable.png)

6. Выберите команду **Сохранить**. Список фраз "Города" добавляется на страницу **Список фраз**.

    ![Список фраз добавлен](./media/luis-add-features/phrase-list-cities.png)

## <a name="edit-phrase-list"></a>Изменение списка фраз

Щелкните имя списка фраз на странице **Список фраз**. Откроется диалоговое окно **Edit Phrase List** (Изменение списка фраз). Внесите любые необходимые правки и щелкните **Сохранить**.

 ![Список фраз добавлен](./media/luis-add-features/edit-phrase-list.png)

## <a name="delete-phrase-list"></a>Удаление списка фраз 

Нажмите кнопку с многоточием (***...***) в конце строки и выберите **Удалить**.

 ![Удаление добавленного списка](./media/luis-add-features/delete-phrase-list.png)

## <a name="deactivate-phrase-list"></a>Отключение списка фраз 

Нажмите кнопку с многоточием (***...***) в конце строки и выберите **Отключить**.

 ![Отключение добавленного списка](./media/luis-add-features/deactivate-phrase-list.png)

## <a name="pattern-regular-expression-feature"></a>Функция шаблона (регулярное выражение) 
**Эту функцию не рекомендуется использовать**. Интеллектуальная служба распознавания речи не может добавить новые функции шаблона. Любые существующие функции шаблона поддерживаются до мая 2018 г. Внесите вклад в стандартное регулярное выражение LUIS, соответствующее PR в хранилище [Recognizers-Text репозитория GitHub](https://github.com/Microsoft/Recognizers-Text). 

## <a name="next-steps"></a>Дополнительная информация

После добавления, редактирования, удаления или отключения списка фраз обучите и протестируйте приложение повторно, чтобы увидеть, улучшилась ли производительность. Дополнительные сведения см. в разделе [Тестирование приложения LUIS](luis-interactive-test.md).
