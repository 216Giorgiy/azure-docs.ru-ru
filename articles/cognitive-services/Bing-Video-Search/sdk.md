# <a name="bing-search-sdk"></a>Пакет SDK для Поиска Bing
Примеры использования API Bing для поиска видео включают в себя следующие сценарии:
1.  Запрос простого видео.
2.  Ограничение области поиска видео с помощью фильтрации (например, для поиска бесплатных или коротких видео, или видео с определенным форматным соотношением).
3.  Поиск видео, набирающих сейчас популярность в различных социальных сетях.
4.  Запрос с использованием более подробных сведений о видео, таких как источник видео, название и т. д.

Пакеты SDK для службы "Поиск Bing" предоставляют доступ к функции поиска в Интернете на перечисленных ниже языках программирования:
* Начало работы с [примерами для .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
    * [Пакет NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.VideoSearch/1.2.0)
    * Ознакомьтесь также с определениями и зависимостями в [библиотеках .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingVideoSearch).
* Начало работы с [примерами для Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * Ознакомьтесь также с определениями и зависимостями в [библиотеках Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/videoSearch).
* Начало работы с [примерами для Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * Ознакомьтесь также с определениями и зависимостями в [библиотеках Java](hhttps://github.com/Azure-Samples/cognitive-services-java-sdk-samples/tree/master/Search/BingNewsSearch).
* Начало работы с [примерами для Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * Ознакомьтесь также с определениями и зависимостями в [библиотеках Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-videosearch).

Примеры пакетов SDK для каждого языка содержат файл сведений, в котором указаны предварительные требования и инструкции по установке и запуску этих примеров.
