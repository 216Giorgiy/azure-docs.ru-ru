# <a name="what-is-computer-vision"></a>Что собой представляет компьютерное зрение

Облачная служба API компьютерного зрения предоставляет разработчикам доступ к расширенным алгоритмам обработки изображений и возврата данных. API компьютерного зрения работает с популярными форматами изображений, такими как JPEG и PNG. Чтобы проанализировать изображение, нужно передать его или указать его URL-адрес. Алгоритмы API компьютерного зрения могут анализировать содержимое изображения по-разному, в зависимости от интересующих вас визуальных компонентов. Например, API компьютерного зрения может определить, содержит ли изображение содержимое для взрослых или непристойного характера, или найти все лица на изображении.

API компьютерного зрения можно вызвать в приложении с помощью [клиентских библиотек](quickstarts-sdk/csharp-analyze-sdk.md) или использовать, напрямую вызвав [REST API](vision-api-how-to-topics/howtocallvisionapi.md), для следующих сценариев:

- [Анализ изображений для получения полезных сведений](#analyzing-images-for-insight).
- [Извлечение текста из изображений](#extracting-text-from-images).
- [Модерация содержимого на изображениях](#moderating-content-in-images).

## <a name="analyzing-images-for-insight"></a>Анализ изображений для получения полезных сведений

С помощью API компьютерного зрения можно проанализировать изображения, чтобы обнаружить и получить сведения о визуальных компонентах и характеристиках изображений. Вы можете передать содержимое изображения для анализа локальных изображений или указать URL-адрес изображения для анализа удаленных изображений.

При анализе изображения API компьютерного зрения может выполнять следующие действия:

| Действие | ОПИСАНИЕ |
| ------ | ----------- |
|**[Добавление тегов к визуальным компонентам](quickstarts/csharp-analyze.md)**|Определение и добавление тегов к визуальным компонентам на изображении на основе более 2000 распознаваемых объектов, живых существ, пейзажей и действий. В случаях, когда теги могут быть неоднозначными или необщеизвестными, в ответ входят "подсказки", помогающие разъяснить смысл тега на основе известного параметра. Добавление тегов не ограничивается основным предметом, например человеком на переднем плане, но также включает параметр (внутри помещения или снаружи), мебель, инструменты, растения, животных, аксессуары, гаджеты и т. д.|
|**[Классификация изображения](quickstarts/csharp-analyze.md)**|Определение и классификация всего изображения с помощью [классификации категорий](Category-Taxonomy.md) с родительскими или дочерними наследственными иерархиями. Категории могут использоваться отдельно или с новыми моделями добавления тегов.<br/>В настоящее время единственным поддерживаемым языком для добавления тегов и классификации изображений является английский.|
|**[Описание изображения](quickstarts/csharp-analyze.md)**|Создание описания всего изображения на удобочитаемом языке с использованием полных предложений. Алгоритмы API компьютерного зрения создают различные описания в зависимости от объектов, определенных на изображении. Каждое описание оценивается и получает оценку достоверности. Затем возвращается список, упорядоченный от наибольшей оценки достоверности к наименьшей.<br/>Пример бота, использующего эту технологию для создания подписей к изображениям, можно найти [на сайте GitHub](https://github.com/Microsoft/BotBuilder-Samples/tree/master/CSharp/intelligence-ImageCaption).|
|**[Распознавание лиц](quickstarts/csharp-analyze.md)** |Обнаружение лиц на изображении, а также предоставление сведений о каждом обнаруженном лице. API компьютерного зрения возвращает координаты, прямоугольник, пол и возраст для каждого обнаруженного лица.<br/>API компьютерного зрения предоставляет подмножество функций, которые можно найти в [API распознавания лиц](/azure/cognitive-services/face/), и вы можете использовать службу "Распознавание лиц" для более детального анализа, такого как идентификация по лицу и определение позы.|
|**[Обнаружение типов изображения](quickstarts/csharp-analyze.md)**|Выявление характеристик изображения, например, является ли изображение графическим или же это иллюстрация.|
|**[Обнаружение содержимого, связанного с определенными сферами](quickstarts/python-domain.md)**|Использование модели предметной области для обнаружения и идентификации отдельного предметного содержимого в изображении, например знаменитостей и достопримечательностей. К примеру, если на изображении люди, API компьютерного зрения может использовать модель предметной области для знаменитостей, включенных в службу, чтобы определить, соответствуют ли люди, обнаруженные на изображении, знаменитостям.|
|**[Обнаружение цветовой схемы](quickstarts/csharp-analyze.md)**|Анализ использования цвета в изображении. API компьютерного зрения может определить, является ли изображение черно-белым или цветным, а для цветных изображений может определить основные цвета и акцентные.|
|**[Создание эскиза](quickstarts/csharp-thumb.md)**|Анализ содержимого изображения, чтобы создать для него соответствующий эскиз. API компьютерного зрения сначала генерирует высококачественный эскиз, а затем анализирует объекты внутри изображения, чтобы определить *интересующую область*. Затем он обрезает изображение для соответствия требованиям интересующей области. Для удовлетворения потребностей пользователя созданный эскиз можно пропорционально изменять в размерах.|

## <a name="extracting-text-from-images"></a>Извлечение текста из изображений

API компьютерного зрения можно использовать для [извлечения текста с помощью функции распознавания текста OCR](quickstarts/csharp-print-text.md) и его последующей вставки в поток символов, обрабатываемых компьютером. При необходимости OCR исправляет поворот в градусах распознанного текста относительно горизонтальной оси изображения и предоставляет координаты кадра каждого слова. OCR поддерживает 25 языков и автоматически определяет язык извлеченного текста.

Вы также можете [распознать печатный и рукописный текст](quickstarts/csharp-hand-text.md) на изображении. API компьютерного зрения может обнаруживать и извлекать печатный и рукописный текст из изображений различных объектов с различными поверхностями и фонами, например из квитанций, плакатов, визитных карточек, писем и настенных досок. В настоящее время распознавание печатного и рукописного текста доступно в режиме предварительной версии, а английский язык является единственным поддерживаемым языком.  

## <a name="moderating-content-in-images"></a>Модерация контента на изображениях

API компьютерного зрения можно использовать для [обнаружения на изображениях содержимого для взрослых и непристойного характера](quickstarts/csharp-analyze.md). Он оценивает вероятность того, что изображение содержит соответствующее содержимое, и генерирует оценку достоверности для такого содержимого. Фильтр, определяющий содержимое для взрослых и носящее непристойный характер, можно настроить в соответствии с предпочтением пользователя по степени принятия материалов данного характера.

## <a name="image-requirements"></a>Требования к образам

API компьютерного зрения может анализировать изображения, отвечающие следующим требованиям:

- Изображение должно быть представлено в формате JPEG, PNG, GIF или BMP.
- Размер файла изображения должен быть меньше 4 МБ.
- Размеры изображения должны быть больше 50 x 50 пикселей.  
  Для распознавания текста размеры изображения должны быть от 40 x 40 до 3200 x 3200 пикселей и изображение не может превышать 10 мегапикселей.
