# <a name="extract-text-with-optical-character-recognition"></a>Извлечение текста с использованием технологии оптического распознавания символов

Функция оптического распознавания символов (OCR) API компьютерного зрения позволяет выявлять текстовое содержимое на изображении и извлекать обнаруженный текст в поток символов, пригодный для машинного чтения. Результат вы можете использовать для поиска и множества других целей, например, для медицинских записей, безопасности и банковских операций. 

OCR поддерживает 25 языков: арабский, китайский (упрощенное письмо), китайский (традиционное письмо), чешский, датский, голландский, английский, финский, французский, немецкий, греческий, венгерский, итальянский, японский, корейский, норвежский, польский, португальский, румынский, русский, сербский (кириллица и латиница), словацкий, испанский, шведский и турецкий. OCR автоматически распознает язык в обнаруженном тексте.

При необходимости OCR исправляет поворот в градусах распознанного текста с помощью смещения относительно горизонтальной оси изображения. OCR также предоставляет координаты рамки для каждого слова, как показано на иллюстрации ниже.

## <a name="image-requirements"></a>Требования к образам

API компьютерного зрения может извлечь текст с помощью OCR из изображений, которые отвечают следующим требованиям:

* Изображение должно быть представлено в формате JPEG, PNG, GIF или BMP.
* Размер входного изображения должен быть от 50 x 50 и до 4200 x 4200 пикселей.
* Текст на изображении можно многократно поворачивать на 90 градусов, а также на небольшой угол в пределах 40 градусов.

## <a name="improving-ocr-accuracy"></a>Повышение точности распознавания текста

Точность распознавания текста зависит от качества изображения. Неточное прочтение может быть вызвано следующими ситуациями:

* Размытость изображения.
* Рукописный текст или курсив.
* Художественные стили шрифтов.
* Мелкий размер текста.
* Сложный фон, тени или отблеск поверх текста, или искажение перспективы.
* Слишком большой размер или отсутствие больших букв в начале слов
* Подстрочный, надстрочный или зачеркнутый текст.

### <a name="ocr-limitations"></a>Ограничения OCR

На изображениях, где преобладает текст, может быть ложное срабатывание частично распознанных слов. На некоторых изображениях, особенно без текста, точность может во многом зависеть от типа изображения.
