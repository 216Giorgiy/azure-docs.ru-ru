# <a name="extracting-text-with-ocr"></a>Извлечение текста с использованием OCR

Технология оптического распознавания символов (OCR) в API компьютерного зрения позволяет выявлять текстовое содержимое на изображении и извлекать обнаруженный текст в поток символов, пригодный для машинного чтения. Результат можно использовать для поиска и множества других целей, таких как медицинские записи, безопасность и банковские операции. Язык определяется автоматически. OCR экономит время и обеспечивает удобство для пользователей, позволяя им фотографировать текст, а не записывать его.

OCR поддерживает 25 языков. Вот они: арабский, китайский (упрощенное письмо), китайский (традиционное письмо), чешский, датский, голландский, английский, финский, французский, немецкий, греческий, венгерский, итальянский, японский, корейский, норвежский, польский, португальский, румынский, русский, сербский (кириллица и латиница), словацкий, испанский, шведский и турецкий.

При необходимости OCR исправляет поворот в градусах распознанного текста относительно горизонтальной оси изображения. OCR предоставляет координаты рамки для каждого слова, как показано на иллюстрации ниже.

## <a name="ocr-requirements"></a>Требования OCR

API компьютерного зрения может извлечь текст с помощью OCR из изображений, которые отвечают следующим требованиям:

* Изображение должно быть представлено в формате JPEG, PNG, GIF или BMP.
* Размер входного изображения должен быть от 50 x 50 и до 4200 x 4200 пикселей.
* Размер изображения не должен превышать 10 мегапикселей.

Входное изображения можно многократно поворачивать на 90 градусов, а также на небольшой угол в пределах 40 градусов.

## <a name="improving-ocr-accuracy"></a>Повышение точности распознавания текста

Точность распознавания текста зависит от качества изображения. Неточное прочтение может быть вызвано следующими ситуациями:

* Размытость изображения.
* Рукописный текст или курсив.
* Художественные стили шрифтов.
* Мелкий размер текста.
* Сложный фон, тени или отблеск поверх текста, или искажение перспективы.
* Слишком большой размер или отсутствие больших букв в начале слов
* Подстрочный, надстрочный или зачеркнутый текст.

### <a name="ocr-limitations"></a>Ограничения OCR

На фотографиях, где преобладает текст, может быть ложное срабатывание частично распознанных слов. На некоторых фотографиях, особенно без текста, точность может во многом зависеть от типа изображения.
