# <a name="detecting-domain-specific-content"></a>Обнаружение содержимого, связанного с определенными предметными областями

Кроме добавления тегов и классификации верхнего уровня, компьютерное зрение также поддерживает специализированные (связанные с определенными предметными областями) сведения. Специализированные сведения могут быть реализованы изолированным методом либо с общей классификацией. Это работает как средство для дальнейшего уточнения 86 категорий таксономии путем добавления моделей, предназначенных для определенных сфер.

Есть два варианта использования моделей, предназначенных для определенных сфер:

* Ограниченный анализ  
  Анализ только выбранной модели с помощью вызова HTTP POST. Если вы знаете, какую модель хотите использовать, укажите ее имя. Будут получены только сведения, относящиеся к этой модели. Например, вы можете использовать этот вариант только для поиска, ориентированного на распознавания знаменитостей. Ответ содержит список потенциально совпадающих знаменитостей вместе с их оценками достоверности.
* Расширенный анализ  
  Анализ дает дополнительные сведения, относящиеся к категориям из таксономии 86 категорий. Этот вариант доступен для использования в приложениях, где пользователи хотят получить анализ обычного изображения с дополнительными сведениями из одной или нескольких моделей, предназначенных для определенных сфер. При вызове этого метода сначала вызывается классификатор таксономии 86 категорий. Если какие-либо категории соответствуют известным или сопоставленным моделям, следует второй этап вызовов классификатора. Например, если параметр `details` в вызове HTTP POST имеет значение "all" или включает "celebrities", метод вызывает классификатор знаменитостей после вызова классификатора 86 категорий. Если изображение классифицируется как относящееся к категории `people_` или ее подкатегории, вызывается классификатор знаменитостей.

## <a name="listing-domain-specific-models"></a>Получение списка моделей предметных областей

Вы можете получить список моделей предметных областей, поддерживаемых компьютерным зрением. В настоящее время при обнаружении содержимого, связанного с определенными предметными областями, компьютерное зрение поддерживает следующие модели предметных областей:

| ИМЯ | ОПИСАНИЕ |
|------|-------------|
| celebrities | Распознавание знаменитостей; поддерживается для изображений, которые были классифицированы как относящиеся к категории `people_` |
| landmarks | Распознавание ориентиров; поддерживается для изображений, которые были классифицированы как относящиеся к категории `outdoor_` или `building_` |

### <a name="domain-model-list-example"></a>Пример списка моделей предметных областей

Приведенный ниже ответ в формате JSON содержит список моделей предметных областей, поддерживаемых компьютерным зрением.

```json
{
    "models": [
        {
            "name": "celebrities",
            "categories": ["people_", "人_", "pessoas_", "gente_"]
        },
        {
            "name": "landmarks",
            "categories": ["outdoor_", "户外_", "屋外_", "aoarlivre_", "alairelibre_",
                "building_", "建筑_", "建物_", "edifício_"]
        }
    ]
}
```
