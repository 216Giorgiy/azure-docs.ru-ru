# <a name="detecting-color-schemes"></a>Определение цветовых схем

Компьютерное зрение извлекает цвета с изображения. Затем цвета подвергаются анализу в трех различных контекстах: преобладающий цвет переднего плана, преобладающий цвет фона и преобладающие цвета изображения в целом. Цвета объединяются в 12 групп, которые соответствуют преобладающим акцентным цветам. Акцентные цвета — черный, синий, коричневый, серый, зеленый, оранжевый, розовый, сиреневый, красный, сине-зеленый, белый и желтый. Компьютерное зрение анализирует цвета, извлеченные с изображения, и возвращает акцентный цвет, который представляет собой самый впечатляющий для зрителей цвет, через сочетание преобладающих цветов и насыщенности. В зависимости от цвета на изображении, просто черно-белый или акцентные, цвета могут возвращаться в шестнадцатеричном цветовом коде. Компьютерное зрение также возвращает логическое значение, которое указывает, является ли изображение черно-белым.

## <a name="color-scheme-detection-examples"></a>Примеры определения цветовой схемы

В приведенном ниже примере показан ответ JSON, возвращаемый компьютерным зрением при определении цветовой схемы на образце изображения. В этом случае изображение не является черно-белым, но преобладающий цвет переднего плана и фона — черный, а преобладающие цвета изображения в целом — черный и белый.

![Гора](./Images/mountain_vista.png)

```json
{
    "color": {
        "dominantColorForeground": "Black",
        "dominantColorBackground": "Black",
        "dominantColors": ["Black", "White"],
        "accentColor": "BB6D10",
        "isBwImg": false
    },
    "requestId": "0dc394bf-db50-4871-bdcc-13707d9405ea",
    "metadata": {
        "height": 202,
        "width": 300,
        "format": "Jpeg"
    }
}
```
