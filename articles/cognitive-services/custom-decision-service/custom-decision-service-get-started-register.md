# <a name="register-your-application"></a>Регистрация приложения

Чтобы использовать Пользовательскую службу принятия решений для приложения, зарегистрируйте его на портале. В этой статье описано, как это сделать.

1. Перейдите на [основную страницу](https://ds.microsoft.com/) Пользовательской службы принятия решений. На ленте щелкните **Мой портал**, как показано на рисунке:

    ![Мой портал](./media/portal.png)

    Если вы еще не выполнили вход на портал, вам будет предложено войти с использованием [учетной записи Майкрософт](https://account.microsoft.com/account). Войдя на портал, в правом верхнем углу страницы вы увидите учетную запись Майкрософт.

2. Чтобы зарегистрировать приложение, нажмите кнопку **Новое приложение**.

3. В диалоговом окне выберите идентификатор приложения. Пользовательской службе принятия решений требуется уникальный идентификатор для каждого приложения. Если такой идентификатор уже используется, система попросит выбрать другой.

4. Укажите API набора действий. Этот параметр представляет собой веб-канал RSS или Atom, который передает доступное для вашего приложения содержимое в Пользовательскую службу принятия решений. Введите имя для веб-канала и URL-адрес, из которого он будет обслуживаться. Чтобы выполнить этот шаг позже, нажмите кнопку **Веб-каналы**, а затем кнопку **Новый веб-канал**. Ниже описан пример, в котором создается RSS-канал.

5. Чтобы зарегистрировать свое приложение, установите флажок **Пользовательское приложение** в левом нижнем углу. Введите [строку подключения](../../storage/common/storage-configure-connection-string.md) для учетной записи хранения Azure, где регистрируются данные вашего приложения. Дополнительные сведения о том, как создать учетную запись хранения, см. в руководстве по [созданию, администрированию и удалению учетных записей](../../storage/common/storage-create-storage-account.md).
