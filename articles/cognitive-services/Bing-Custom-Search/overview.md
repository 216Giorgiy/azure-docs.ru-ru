# <a name="what-is-bing-custom-search"></a>Что такое "Пользовательский поиск Bing"?

Служба "Пользовательский поиск Bing" позволяет создавать специально адаптированные интерфейсы для поиска по темам, которые действительно важны для вас. Например, если ваш веб-сайт предоставляет функцию поиска, то вы можете указать домены, веб-сайты и веб-страницы, по которым Bing будет выполнять поиск. Ваши пользователи увидят результаты поиска, адаптированные к интересующей их информации, вместо того, чтобы пролистывать результаты поиска с несоответствующим содержимым.

Чтобы создать пользовательское представление веб-страницы, перейдите на [портал](https://customsearch.ai) службы "Пользовательский поиск Bing". Этот портал позволяет создать экземпляр пользовательского поиска с указанием доменов, веб-сайтов и веб-страниц, по которым будет выполняться поиск Bing, а также веб-сайтов, по которым поиск выполнять не требуется. Помимо указания известных вам URL-адресов вы также можете воспользоваться порталом для поиска соответствующего содержимого, которое также требуется добавить.

С помощью портала также можно закрепить определенную веб-страницу в верхней части результатов поиска, если пользователь введет какое-то конкретное условие поиска. 

После определения параметров вы можете интегрировать полученный экземпляр пользовательского поиска в свой веб-сайт, в классическое или мобильное приложении, вызывая API пользовательского поиска. При использовании веб-сайта или веб-приложения вы можете разрешить размещенному пользовательскому интерфейсу выполнять функции интерфейса поиска.

## <a name="adding-custom-search-box-suggestions"></a>Добавление предложений пользовательского поиска

Возможности пользовательского поиска можно расширить за счет предложений пользовательского поиска. Эта функция предоставляет наиболее релевантные предложения поиска. Когда пользователь вводит текст в поле поиска, в раскрывающемся списке содержатся предлагаемые строки запросов на основе частично введенной пользователем строки запроса. Вы можете указать, возвращать ли только пользовательские предложения или также включать предложения Bing. [Дополнительные сведения](define-custom-suggestions.md).

## <a name="adding-custom-image-search-experience"></a>Добавление возможностей пользовательского поиска изображений

Возможности пользовательского поиска можно расширить за счет изображений. Аналогично веб-результатам, пользовательский поиск поддерживает поиск изображений в списке веб-сайтов вашего экземпляра. [Дополнительные сведения](get-images-from-instance.md).

## <a name="adding-custom-video-search-experience"></a>Добавление возможностей пользовательского поиска видео

Возможности пользовательского поиска можно расширить за счет видео. Аналогично веб-результатам, пользовательский поиск поддерживает поиск видео в списке веб-сайтов вашего экземпляра. [Дополнительные сведения](get-videos-from-instance.md).

## <a name="sharing-your-custom-search-instance-with-others"></a>Использование экземпляра пользовательского поиска совместно с другими пользователями

Вы можете разрешить совместное редактирование и тестирование своего экземпляра, предоставив к нему доступ участникам вашей команды. [Дополнительные сведения](share-your-custom-search.md).
