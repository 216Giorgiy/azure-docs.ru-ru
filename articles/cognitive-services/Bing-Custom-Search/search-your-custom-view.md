# <a name="call-your-custom-search"></a>Обращение к пользовательскому поиску
Прежде чем создать первый вызов к API пользовательского поиска, который вернет результаты поиска для созданного экземпляра, вам необходимо получить ключ подписки Cognitive Services. Чтобы получить ключ для API пользовательского поиска, перейдите на страницу [Пробная версия Cognitive Services](https://azure.microsoft.com/try/cognitive-services/?api=bing-custom-search).

> [!NOTE]
> Существующие клиенты службы "Пользовательский поиск Bing", у которых есть ключи предварительной версии, подготовленные не позднее 15 октября 2017 г., смогут использовать эти ключи до 30 ноября 2017 г. или до исчерпания максимально допустимого числа запросов. После этого им необходимо перейти на общедоступную версию Azure.

## <a name="try-it-out"></a>Попробуйте сейчас
После настройки интерфейса для пользовательского поиска вы можете проверить его конфигурацию через портал пользовательского поиска. Войдите на портал [Пользовательский поиск](https://customsearch.ai), выберите нужный экземпляр пользовательского поиска и щелкните вкладку **Production** (Рабочая среда). Откроется вкладка **Endpoints** (Конечные точки). Доступные для использования конечные точки будут разными в зависимости от типа используемой подписки, как описано на [странице цен](https://azure.microsoft.com/pricing/details/cognitive-services/bing-custom-search/). Чтобы протестировать конечную точку, выберите ее из раскрывающегося списка и задайте соответствующие параметры конфигурации. 

Ниже приведены доступные параметры.

- **Query** (Запрос) — условие, по которому выполняется поиск. Оно доступно только для конечных точек веб-поиска, изображений и автозаполнения.
- **Custom Configuration ID** (Идентификатор пользовательской конфигурации) — идентификатор конфигурации для выбранного экземпляра пользовательского поиска. Это поле доступно только для чтения.
- **Market** (Рынок) — обозначает рынок, по которому возвращаются результаты. Он доступен только для конечных точек веб-поиска, изображений и размещенного пользовательского интерфейса.
- **Subscription Key** (Ключ подписки) — ключ подписки для тестирования. Вы можете выбрать ключ из раскрывающегося списка или ввести его вручную.
- **Safe Search** (Безопасный поиск) — фильтр, который позволяет убрать из результатов веб-страницы для взрослых. Он доступен только для конечных точек веб-поиска, изображений и размещенного пользовательского интерфейса.
- **Count** (Число) — количество возвращаемых в ответе результатов поиска. Доступно только для конечных точек веб-поиска и изображений.
- **Offset** (Сдвиг) — количество возвращаемых в ответе результатов поиска. Доступно только для конечных точек веб-поиска и изображений.

Завершив выбор всех необходимых параметров для веб-поиска, поиска изображений или автозаполнения, щелкните **Call** (Вызов), чтобы просмотреть ответ в формате JSON в области справа. 

Если вы выбрали конечную точку размещенного пользовательского интерфейса, в панели справа можно проверить взаимодействие с ее интерфейсом.
