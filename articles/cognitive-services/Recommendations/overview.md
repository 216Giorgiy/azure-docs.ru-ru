# <a name="recommendations-api-discontinued"></a>API рекомендаций (не поддерживается)

Предварительная версия API рекомендаций больше не находится в активной разработке и официально исключена из использования с 15 февраля. 

В качестве альтернативы рекомендуем использовать новый [шаблон Recommendations Solution](http://aka.ms/recopcs) с открытым исходным кодом, который позволяет размещать механизм рекомендаций в своей собственной подписке Azure. Он поддерживает основные сценарии API рекомендаций, а также предоставляет свободу развертывания в любом регионе Azure и более точно контролирует вычислительные ресурсы, используемые для обучения моделей и подготовки рекомендаций. 

Дополнительные сведения о шаблоне Recommendations Solution см. в [документации](https://github.com/Microsoft/Product-Recommendations).
