# <a name="microsoft-speech-client-samples"></a>Образцы клиента речи Microsoft

Служба распознавания речи Microsoft содержит полноценные примеры, с помощью которых показано, как использовать API распознавания речи Microsoft в разных вариантах использования, например для распознавания команд, непрерывного распознавания и определения намерений. Все образцы доступны в GitHub и могут быть загружены по следующим ссылкам: README.md в каждом репозитории, а также на странице [клиентских библиотек](GetStarted/GetStartedClientLibraries.md), которая содержит подробную информацию о том, как компилировать и запускать образцы.

- [Примеры REST API](https://github.com/Azure-Samples/SpeechToText-REST)

- [Примеры JavaScript](https://github.com/Azure-Samples/SpeechToText-WebSockets-Javascript)

- [Примеры клиента C# Desktop](https://github.com/Azure-Samples/Cognitive-Speech-STT-Windows)

- [Примеры службы C#](https://github.com/Azure-Samples/Cognitive-Speech-STT-ServiceLibrary)

- [Примеры Java для Android](https://github.com/Azure-Samples/Cognitive-Speech-STT-Android)

- [Примеры ObjectiveC для iOS](https://github.com/Azure-Samples/Cognitive-Speech-STT-iOS)

Все пакеты SDK и образцы Microsoft Cognitive Services лицензированы лицензией MIT. Дополнительные сведения можно найти в разделе [ЛИЦЕНЗИЯ](https://github.com/Microsoft/Cognitive-Speech-STT-JavaScript/blob/master/LICENSE.md).
