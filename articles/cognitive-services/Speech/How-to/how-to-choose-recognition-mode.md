# <a name="speech-recognition-modes"></a>Режимы распознавания речи

API *преобразования речи в текст* Microsoft поддерживает несколько режимов распознавания речи. Выберите для вашего приложения режим, который дает наилучшие результаты распознавания.

| Mode | ОПИСАНИЕ |
|---|---|
| *интерактивный* | Распознавание "Команды и элемента управления" для интерактивных сценариев пользовательских приложений. Пользователи произносят короткие фразы, которые подразумеваются как команды приложению. |
| *диктовка* | Непрерывное распознавание для сценариев диктовки. Пользователи произносят более длинные предложения, которые отображаются в виде текста. Пользователи используют более формальный стиль речи. |
| *беседа* | Непрерывное распознавание для воспроизводимых бесед между людьми. Пользователи используют менее формальный стиль речи и могут чередовать более длинные предложения с короткими фразами.

> [!NOTE]
> Эти режимы можно применять при использовании [REST API](../GetStarted/GetStartedREST.md). [Клиентские библиотеки](../GetStarted/GetStartedClientLibraries.md) используют разные параметры для указания режима распознавания. Дополнительные сведения см. в документации к нужной клиентской библиотеке.

Для получения дополнительных сведений см. страницу [Recognition Modes](../concepts.md#recognition-modes) (Режимы распознавания).
