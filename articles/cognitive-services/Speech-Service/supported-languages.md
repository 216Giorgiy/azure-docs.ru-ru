# <a name="supported-languages-for-speech-service"></a>Поддерживаемые языки для службы "Речь"

| de-DE | Немецкий (Германия) | ja-JP | Японский (Япония) |

### <a name="customization"></a>Настройка

Настраиваемые голоса **преобразования текста в речь** поддерживают английский язык (en-US) и китайский язык (zh-CN).

## <a name="speech-translation"></a>Перевод речи

**API перевода речи** поддерживает перевод речи в речь и речи в текст для нескольких языков. Язык оригинала должен обязательно быть указан в следующей таблице. Доступность языка перевода зависит от того, поддерживается ли для языка оригинала речь или только текст.

Этот список языков можно получить программным образом или через конечную точку API-интерфейса REST Languages. В списке содержатся все коды языков, а также их названия на английском (или другом поддерживаемом языке). Этот список обновляется службой Microsoft Translator по мере добавления поддержки новых языков.
