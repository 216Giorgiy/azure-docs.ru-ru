# <a name="about-the-speech-translation-api"></a>Сведения об API перевода речи

API перевода речи от корпорации Майкрософт позволяет применить в средствах, приложениях и устройствах полнофункциональное решение для перевода устной речи на многих языках в режиме реального времени. Один API можно использовать как для преобразования речи в речь, так и для преобразования речи в текст.

Через API перевода речи от Майкрософт клиентские приложения могут передавать в службу потоковые аудиоданные и получать в ответ поток результатов. Эти результаты содержат распознанный текст на исходном языке и его перевод на целевой язык. Могут предоставляться промежуточные результаты перевода еще до завершения высказывания, а также окончательный перевод после его завершения.

При необходимости можно синтезировать звуковую версию окончательного перевода, что позволяет реализовать полностью голосовую среду перевода речи.

API перевода речи использует протокол WebSocket для создания дуплексного коммуникационного канала между клиентом и сервером. Но вам не нужно самостоятельно организовывать работу с WebSockets, пакет SDK для распознавания речи сделает все за вас.

API перевода речи использует те же технологии, что и в различных продуктах и службах Майкрософт. Эта служба уже используется тысячами компаний по всему миру в разных приложениях и рабочих процессах.

## <a name="about-the-technology"></a>Сведения о технологии

Базовая система перевода от корпорации Майкрософт использует два разных подхода: статистический машинный перевод (SMT) и нейронный машинный перевод (NMT). Второй из них, который основывается на искусственном интеллекте и нейронных сетях, представляет собой современный подход к машинному переводу. NMT дает более качественные результаты — повышается не только точность, но и плавность и естественность речи. Секрет такого улучшения кроется в том, что технология NMT использует для перевода слов контекст всего предложения.

В настоящее время корпорация Майкрософт перенесла почти все популярные языки на платформу NMT, а SMT применяется только для менее используемых языков. Все [языки, доступные для преобразования речи в речь](supported-languages.md#speech-translation), обрабатываются на платформе NMT. Преобразование речи в текст может выполняться на SMT или NMT в зависимости от конкретной пары языков. Если NMT поддерживается для целевого языка, весь перевод осуществляется на платформе NMT. Если для целевого языка NMT не поддерживается, применяется "гибридный" подход к переводу с применением NMT, SMT и промежуточного перевода на английский язык.

Все различия между этими моделями полностью обрабатываются системой перевода. Пользователи заметят только улучшение качества перевода, особенно для китайского, японского и арабского языков.
