# <a name="about-the-speech-devices-sdk-preview"></a>Сведения о пакете SDK для речевых устройств (предварительная версия)

Служба речи (Майкрософт) поддерживает широкий ряд устройств и источников звука. Теперь ваши речевые приложения могут стать еще эффективнее благодаря соответствующему оборудованию и программному обеспечению. Пакет SDK для речевых устройств — это предварительно настроенная библиотека, связанная с определенным оборудованием с поддержкой микрофона. Пакет SDK упрощает интеграцию устройств с облачной службой речи (Майкрософт) и создание удобного и привлекательного пользовательского интерфейса для клиентов.

Пакет SDK позволяет создавать устройства, реагирующие на окружающую среду, и настроить уникальное для вашей торговой марки слово для активации взаимодействия с пользователем. Он использует многоканальный звук для обеспечения более точного распознавания речи благодаря шумоподавлению, подавлению эха, обнаружению голоса в дальней зоне и формированию луча.

Пакет SDK упрощает реализацию разнообразных сценариев речевого взаимодействия, таких как система заказов в автокафе, помощники в автомобиле или в доме и смарт-динамики. На реплики пользователей можно отвечать текстом, общаться с ними с помощью настраиваемых голосов или голосов по умолчанию, предоставлять результаты поиска, переводить на другие языки и т. д.
