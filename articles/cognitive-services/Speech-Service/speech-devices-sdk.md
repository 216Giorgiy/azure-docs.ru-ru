# <a name="about-the-speech-devices-sdk-preview"></a>Сведения о пакете SDK для речевых устройств (предварительная версия)

[Служба распознавания речи](overview.md) поддерживает широкий ряд устройств и источников звука. Теперь ваши речевые приложения могут стать еще эффективнее благодаря соответствующему оборудованию и программному обеспечению. Пакет SDK для речевых устройств — это предварительно настроенная библиотека, связанная со специально созданными пакетами разработки для микрофонной решетки.

Пакет SDK для речевых устройств помогает решать следующие задачи:
* быстрое тестирование новых сценариев голосовой связи;
* более простая интеграция облачной службы речи в устройства;
* создание удобного интерфейса пользователя для ваших клиентов.

Пакет SDK для речевых устройств использует [пакет SDK для распознавания речи](speech-sdk.md). Пакет SDK для распознавания речи служит для отправки аудиоданных, обрабатываемых расширенным алгоритмом обработки звука, из микрофонной решетки устройства в [службу распознавания речи](overview.md). Он использует многоканальный звук для обеспечения более точного [распознавания речи](speech-to-text.md) благодаря шумоподавлению, подавлению эха, обнаружению голоса в дальней зоне, формированию луча и подавлению реверберации.

С помощью пакета SDK для речевых устройств также можно создавать устройства, реагирующие на окружающую среду, и настроить [уникальное для вашей торговой марки слово](speech-devices-sdk-create-kws.md) для активации взаимодействия с пользователем.

Пакет SDK для речевых устройств упрощает реализацию разнообразных сценариев речевого взаимодействия, таких как система заказов в автокафе, помощники в магазине или в доме и смарт-динамики. На реплики пользователей можно отвечать текстом, общаться с ними с помощью [настраиваемых голосов](how-to-customize-voice-font.md) или голосов по умолчанию, предоставлять результаты поиска, [переводить](speech-translation.md) на другие языки и т. д. Мы будем рады видеть вашу сборку!
