### <a name="customization"></a>Настройка

Настройка голоса недоступна для английского языка (en US), континентального китайского (zh-CN) и итальянского (it-IT).

> [!NOTE]
> Обучение итальянского голоса начинается с набора данных, состоящего из более чем 2000 высказываний. Китайско-английские двуязычные модели также поддерживаются с помощью начального набора данных, состоящего из более чем 2000 высказываний.

## <a name="speech-translation"></a>Перевод речи

**API перевода речи** поддерживает перевод речи в речь и речи в текст для нескольких языков. Язык оригинала должен обязательно быть указан в приведенной ниже таблице. Доступность языка перевода зависит от того, поддерживается ли для языка оригинала речь или только текст.

Этот список языков можно получить с помощью программных средств через конечную точку API-интерфейса REST Languages. В списке содержатся все коды языков, а также их названия на английском (или другом поддерживаемом языке). Этот список обновляется службой Microsoft Translator по мере добавления поддержки новых языков.
