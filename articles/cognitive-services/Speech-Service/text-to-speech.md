# <a name="about-the-text-to-speech-api"></a>Сведения об API преобразования текста в речь

API **преобразования текста в речь** (TTS) преобразует входной текст в естественно звучащую речь (также называется *синтезом речи*).

Для синтезирования речи приложение отправляет HTTP-запросы POST в API преобразования текста в речь. Далее текст синтезируется в человеческую речь и возвращается в виде звукового файла. Поддерживаются различные голоса и языки.

Синтез речи задействуется в следующих сценариях:

* *Повышение доступности.* Технология **преобразования текста в речь** позволяет владельцам содержимого и издателям реагировать на различные способы взаимодействия пользователей с содержимым. Люди с нарушениями зрения или трудностями чтения оценят возможность использовать содержимое в звуковой форме. Голосовой вывод также облегчает восприятие текстового содержимого, например газет или блогов, на мобильных устройствах в поездках или во время занятий спортом.

* *Реагирование на потребности в многозадачности.* **Преобразование текста в речь** позволяет быстро и c комфортом получать важную информацию за рулем автомобиля или в других ситуациях, когда читать текст неудобно. Распространенная область применения в этой связи — автомобильная навигация.

* *Улучшение обучения.* Разные люди предпочитают учиться по-разному. Эксперты по онлайн-обучению доказали, что предоставление речевого материала вместе с текстом помогает легче усваивать и запоминать информацию.

* *Создание интуитивно понятных ботов и помощников.* Голосовое общение может быть неотъемлемой частью интеллектуального чат-бота или виртуального помощника. Все больше компаний разрабатывают чат-боты для обеспечения взаимодействия клиентов со службами. Речь добавляет новое измерение, позволяя воспринимать ответы ботов на слух (например, по телефону).

## <a name="voice-support"></a>Поддержка речи

Служба **преобразования текста в речь** Майкрософт предлагает свыше 75 голосов для более чем 45 языков и языковых стандартов. Чтобы использовать эти стандартные голоса, достаточно указать название голоса и еще несколько параметров при вызове REST API службы. Дополнительные сведения о поддерживаемых языках, языковых стандартах и голосах см. [здесь](language-support.md#text-to-speech).

### <a name="neural-voices"></a>Синтезирование голоса с помощью нейронных сетей

Преобразование текста в речь на основе нейронных сетей может сделать взаимодействие с чат-ботами и виртуальными помощниками более естественным и приятным. Кроме того, эта функция позволяет преобразовывать цифровой текст из электронных книг в аудиокниги, а также помогает совершенствовать автомобильные системы навигации. Преобразование текста в речь на основе нейронных сетей отличается человекоподобной естественной просодией и четким произношение слов, что значительно уменьшает усталость от прослушивания при взаимодействии с системами искусственного интеллекта. Дополнительные сведения о синтезировании голоса с помощью нейронных сетей см. [здесь](language-support.md#text-to-speech).

### <a name="custom-voices"></a>Настраиваемое синтезирование голоса

Настройка голоса для преобразования текста в речь позволяет создать узнаваемый уникальный *голос*. Чтобы создать голос, сделайте студийную запись и отправьте связанные сценарии в качестве данных обучения. Затем служба создает модель уникального голоса, настроенную для записи. Затем этот голос можно использовать для синтезирования речи. Дополнительные сведения см. [здесь](how-to-customize-voice-font.md).

| Отправка наборов данных для адаптации голоса | Yes | Нет  |
| Создание моделей голосов и управление ими | Yes | Нет  |

| Создание тестов голосов и управление ими| Yes | Нет  |
