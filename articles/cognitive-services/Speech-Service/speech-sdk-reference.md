# <a name="speech-sdk-reference"></a>Справочник по пакету SDK для службы "Речь"

Пакет средств разработки программного обеспечения (SDK) для службы "Речь" предоставляет приложениям доступ к функциям службы "Речь", что облегчает разработку программного обеспечения с поддержкой речи. Сейчас пакеты SDK предоставляют доступ к **преобразованию речи в текст** и **переводу речи**. При преобразовании текста в речь используются вызовы [REST] (https://docs.microsoft.com/azure/cognitive-services/speech-service/rest-apis) POST по протоколу HTTP.

В таблице ниже перечислены поддерживаемые языки программирования и операционные системы.

|Язык программирования|платформа|Справочник по API|
|-|-|-|
|C#|Windows|[Обзор](https://aka.ms/csspeech/csharpref)
|C/C++|Windows, Linux|[Обзор](https://aka.ms/csspeech/cppref)
|Java\*|Устройства|[Обзор](https://aka.ms/csspeech/javaref)

\* *Пакет SDK для Java входит в [пакет SDK для речевых устройств](speech-devices-sdk.md).*
