# <a name="speech-sdk-api-reference"></a>Справочник по API пакета SDK для службы "Речь"

Пакет средств разработки программного обеспечения (SDK) для службы "Речь" предоставляет приложениям доступ к функциям службы "Речь", что облегчает разработку программного обеспечения с поддержкой речи. Сейчас пакеты SDK предоставляют доступ к **преобразованию речи в текст**, **переводу речи** и **распознаванию намерений**. Общие сведения о возможностях и поддерживаемых платформах можно найти на [начальной странице](https://aka.ms/csspeech) документации.
