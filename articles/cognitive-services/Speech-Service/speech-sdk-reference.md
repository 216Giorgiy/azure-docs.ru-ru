# <a name="speech-sdk-reference"></a>Справочник по пакету SDK для службы "Речь"

Пакет средств разработки программного обеспечения (SDK) для службы "Речь" предоставляет приложениям доступ к функциям службы "Речь", что облегчает разработку программного обеспечения с поддержкой речи. Сейчас пакеты SDK предоставляют доступ к **преобразованию речи в текст**, **переводу речи** и **распознаванию намерений**.
