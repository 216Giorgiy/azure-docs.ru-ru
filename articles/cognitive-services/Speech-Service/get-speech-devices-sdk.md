# <a name="get-the-cognitive-services-speech-devices-sdk"></a>Получение пакета SDK для речевых устройств в Cognitive Services

## <a name="requesting-access"></a>Запрос доступа

Пакет SDK для речевых устройств предоставляется в рамках ограниченной предварительной версии и требует регистрации в программе. На данный момент предпочтение для предоставления доступа к этому продукту отдается крупным компаниям.

Чтобы получить доступ к пакету SDK для речевых устройств, выполните следующие действия.

1. Перейдите к [бланку регистрации](https://aka.ms/sdsdk-signup) для получения пакета SDK для речевых устройств (Майкрософт).
1. Прочитайте [лицензионное соглашение](speech-devices-sdk-license.md).
1. Если вы согласны с условиями лицензионного соглашения, установите флажок "Принимаю".
1. Ответьте на вопросы в форме.
1. Отправьте форму. 
1. Если ваш адрес электронной почты еще не входит в Azure Active Directory, после утверждения вашей заявки вы получите по электронной почте приглашение, аналогичное приведенному ниже. Если ваш адрес электронной почты уже существует в Azure Active Directory, после утверждения вашей заявки вы получите сообщение электронной почты от команды разработчиков службы "Речь" и сможете сразу перейти к [загрузке пакета SDK для речевых устройств](#download-the-speech-devices-sdk).

## <a name="approval-e-mail"></a>Электронное письмо с утверждением

```
From: Microsoft Speech Team from Microsoft (via Microsoft) <invites@microsoft.com> 
Subject: You're invited to the Microsoft organization 
```

![сообщение электронной почты](media/speech-devices-sdk/get-sdk-1.png)

## <a name="accept-access"></a>Принятие доступа
Выполните следующие действия, чтобы присоединиться к Azure Active Directory с адресом электронной почты, указанным во время регистрации. Этот процесс предоставляет вам доступ к [сайту загрузки](https://shares.datatransfer.microsoft.com/) пакета SDK для речевых устройств.

1. Нажмите кнопку **Начало работы** в полученном сообщении электронной почты. Если ваша организация уже является клиентом Office 365, вам будет предложено выполнить вход, и вы можете сразу перейти к шагу 8.

2. Нажмите кнопку **Далее** в окне запущенного браузера.

    ![окно проверки подлинности](media/speech-devices-sdk/get-sdk-2.png)

3. Создайте учетную запись Майкрософт, если у вас ее еще нет. Введите тот же адрес электронной почты, на который вы получили приглашение на шаге 6 выше.

    ![создание учетной записи Майкрософт](media/speech-devices-sdk/get-sdk-3.png)

4. Нажмите кнопку **Далее**, чтобы создать пароль.

5. При появлении запроса для проверки электронной почты вернитесь в папку входящих сообщений, чтобы получить отправленный вам код проверки.
 
7. Вставьте или введите код безопасности из сообщения электронной почты в диалоговом окне. В нашем примере это "8406". Нажмите кнопку **Далее**.

    ![проверка электронной почты](media/speech-devices-sdk/get-sdk-6.png)
 
8. Появление приложения панели доступа в окне браузера означает, что ваш адрес электронной почты (из шага 6) теперь входит в Azure Active Directory. Теперь вы получаете доступ к сайту загрузки пакета SDK для речевых устройств.

## <a name="download-the-speech-devices-sdk"></a>Загрузка пакета SDK для речевых устройств

Перейдите на [сайт загрузки пакета SDK для речевых устройств](https://shares.datatransfer.microsoft.com/) и выполните вход с учетной записью Майкрософт, созданной ранее. Теперь вы можете скачать пакет SDK для речевых устройств, связанные с ним примеры кода и справочные материалы, выполнив следующие действия.

![сайт загрузки пакета SDK](media/speech-devices-sdk/get-sdk-7.png)

1. Загрузите и установите средства Aspera Connect при получении соответствующего запроса.

    ![загрузка Aspera Connect](media/speech-devices-sdk/get-sdk-8.png)
 
1. Нажмите кнопку **Да**, чтобы переключиться на Aspera Connect.

    ![переключение на Aspera Connect](media/speech-devices-sdk/get-sdk-9.png)
 
1. Нажмите кнопку **Разрешить** для подтверждения загрузки файлов с помощью Aspera Connect.

    ![загрузка с помощью Aspera Connect](media/speech-devices-sdk/get-sdk-10.png)
 
1. Закройте окно передачи Aspera Connect после загрузки файлов.

    ![Окно передачи Aspera Connect](media/speech-devices-sdk/get-sdk-11.png)
 
По умолчанию файлы загружаются в папку **Загрузки**. Теперь вы можете выйти из системы этого сайта.
