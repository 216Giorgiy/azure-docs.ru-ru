# <a name="about-the-speech-to-text-api"></a>Сведения об API преобразования речи в текст

API **преобразования речи в текст** *расшифровывает* потоки аудиоданных в текст, который приложение может отобразить для пользователя или использовать в качестве входных данных для команды. Эти API-интерфейсы можно использовать с клиентской библиотекой пакетов SDK (на поддерживаемых платформах и языках) или REST API.

API **преобразования речи в текст** предоставляет следующие возможности:

- Современные технологии распознавания речи, разработанные корпорацией Майкрософт — точно такие же, как и в Cortana, Office и других продуктах от Майкрософт.

- Непрерывное распознавание в реальном времени. С помощью **преобразования речи в текст** пользователи могут преобразовывать звук в текст в режиме реального времени. Также эта возможность поддерживает получение промежуточных результатов, то есть уже распознанных слов на текущий момент. Служба автоматически распознает конец речи. Пользователи также могут выбирать дополнительные параметры форматирования, например регистр символов и правила пунктуации, маскировку нецензурных выражений и нормализацию текста.

- Оптимизированные результаты **преобразования речи в текст** для интерактивных сценариев, диалогов и диктовки. 

- Поддержка речи на множестве языков с различными диалектами. Полный список поддерживаемых языков в каждом режиме распознавания см. в статье о [поддерживаемых языках](supported-languages.md#speech-to-text).

- Настраиваемые языковые и акустические модели, которые позволяют адаптировать приложение к особенностям речи пользователей, среде общения и специализированному словарю.

- Распознавание естественной речи. Благодаря интеграции с [Интеллектуальной службой распознавания речи](https://docs.microsoft.com/azure/cognitive-services/luis/) (LUIS) вы можете извлекать из речи сведения о намерениях и сущностях. Пользователям не обязательно знать словарь, используемый приложением, а достаточно лишь своими словами описать свои требования.

## <a name="api-capabilities"></a>Возможности API

Некоторые возможности API **преобразования речи в текст** недоступны через интерфейс REST. В следующей таблице перечислены возможности каждого метода доступа к API.

| Вариант использования  | REST | Пакеты SDK |
|-----|-----|-----|----|
| Преобразование коротких высказываний, например команд (длительностью до 15 с); без промежуточных результатов | Yes | Yes |
| Преобразование длительных высказываний (более 15 с) | Нет  | Yes |
| Преобразование потокового аудио с выдачей промежуточных результатов при необходимости | Нет  | Yes |
| Распознавание намерений говорящего с помощью API распознавания речи | Нет\* | Yes |

\* *Для получения сведений о намерениях и сущностях через LUIS потребуется отдельная подписка LUIS. При ее наличии пакет SDK сможет обращаться к LUIS и возвращать сведения о намерениях и сущностях вместе с транскрибированием речи. Вы можете самостоятельно вызывать LUIS с помощью REST API для получения сведений о намерениях и сущностях, используя подписку LUIS.*
