> [!NOTE]
> Но сначала получите ключ подписки.
> Вы можете получить ключи бесплатной пробной версии подписки на странице [подписки на Cognitive Services](https://azure.microsoft.com/try/cognitive-services/). После выбора SAPI щелкните **Get API Key** (Получить ключ API), чтобы получить два ключа — основной и дополнительный. Можно использовать любой из ключей. Оба ключа привязаны к одной квоте.
>
> Платный ключ подписки можно получить через панель мониторинга в Azure.
