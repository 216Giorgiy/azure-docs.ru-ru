# <a name="create-a-custom-speech-to-text-endpoint"></a>Создание пользовательской конечной точки преобразования речи в текст

После создания пользовательской акустической модели или языковой модели их можно развернуть в пользовательской конечной точке преобразования речи в текст. 

## <a name="create-an-endpoint"></a>Создание конечной точки
Чтобы создать пользовательскую конечную точку, в меню **Настраиваемое распознавание речи** в верхней части страницы выберите **Пользовательское распознавание речи**. Откроется страница **Конечные точки**, которая содержит таблицу текущих пользовательских конечных точек. Если вы еще не создали конечные точки, таблица будет пустой. Текущий языковой стандарт отражается в заголовке таблицы. 

Чтобы создать развертывание для другого языка, выберите **Change Locale** (Изменить языковой стандарт). См. дополнительные сведения о поддерживаемых языках.

Чтобы создать конечную точку, выберите **Создать**. В области **Создание конечной точки** введите данные пользовательского развертывания в поля **Имя** и **Описание**.

В поле со списком **Подписка** выберите подписку, которую вы хотите использовать. Если это подписка S0 (платная версия), можно выбрать параллельные запросы.

Можно также включить или отключить ведение журнала содержимого. Таким образом можно выбрать, будет ли сохраняться трафик конечной точки. Если эта функция не включена, трафик не сохраняется. Ссылки для скачивания всего регистрируемого содержимого можно найти, щелкнув "Конечная точка" >"Подробности".

Из списка **Acoustic Model** (Акустическая модель) выберите соответствующую акустическую модель, а из списка **Языковая модель** — соответствующую языковую модель. При выборе акустической и языковой моделей всегда доступны базовые модели Майкрософт. Выбор базовой модели ограничивает возможные сочетания. Невозможно смешивать базовые разговорные модели с базовыми моделями поиска и диктовки.

> [!NOTE]
> Обязательно примите условия использования и сведения о ценах, установив соответствующий флажок.
>

После выбора акустической и языковой моделей щелкните **Создать**. Вы вернетесь на страницу **Развертывания**. Теперь таблица содержит запись, которая соответствует новой конечной точке. Состояние конечной точки отражает ее текущее состояние во время создания. Создание экземпляра новой конечной точки с пользовательскими моделями может занять до 30 минут. Если состояние развертывания — *Выполнено*, конечная точка готова к использованию.

Когда развертывание будет готово, имя конечной точки станет ссылкой. При щелчке этой ссылки отображается страница **Сведения о конечной точке**, которая содержит URL-адреса, используемые вашей пользовательской конечной точкой для HTTP-запросов, или клиентскую библиотеку службы "Речь" в Microsoft Cognitive Services, которая использует веб-сокеты.
