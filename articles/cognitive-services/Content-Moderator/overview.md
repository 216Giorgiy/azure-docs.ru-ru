# <a name="what-is-content-moderator"></a>Что такое Content Moderator?

Модерация контента — это процесс отслеживания возможного оскорбительного, нежелательного и представляющего риск содержимого. Модерировать можно такой контент, как изображения. текст и видео.

## <a name="where-it-is-used"></a>Сценарии использования

В следующем списке приведены несколько примеров сценариев использования Content Moderator:

- Интернет-магазины с модерацией каталогов продукции и создаваемого пользователями содержимого.
- Компании-разработчики игр, модерирующие создаваемые пользователями игровые артефакты и чаты.
- Социальная платформа обмена сообщениями с модерацией изображений, текста и видео, добавляемых пользователями.
- Коммерческие медиакомпании, внедряющие централизованную модерацию своего контента.
- Поставщики образовательных решений K-12, фильтрующие неуместное и оскорбительное содержимое для учащихся и преподавателей.

## <a name="what-it-includes"></a>Компоненты

Content Moderator включает несколько API веб-служб и встроенное средство пользовательской проверки, которое помогает модерировать изображения, текст и видео.

![Блок-схема Content Moderator](images/content-moderator-block-diagram.png)

## <a name="apis"></a>Интерфейсы API

Content Moderator включает следующие API:
  - [**API модерации текста**](text-moderation-api.md) позволяет проверять текст на наличие ругательств, откровенных, непристойных, оскорбительных выражений и персональных данных (PII).
  - [**API настраиваемых списков терминов**](try-terms-list-api.md) служит для сопоставления с настраиваемыми списками терминов в дополнение ко встроенным. Эти списки позволяют блокировать или разрешать определенное содержимое в соответствии с политиками управления содержимым.  
  - [**API модерации изображений**](image-moderation-api.md) позволяет проверять изображения на наличие предназначенного для взрослых или специфического содержимого, выявлять в изображениях текст с помощью функции оптического распознавания символов (OCR), а также распознавать лица.
  - [**API настраиваемых списков изображений**](try-image-list-api.md) служит для сопоставления с настраиваемыми списками изображений — предварительно идентифицированным содержимым, который не нужно заново классифицировать.
  - [**API модерации видео**](video-moderation-api.md): позволяет проверять видео на наличие потенциального предназначенного для взрослых или специфического содержимого.
  - [**API-интерфейсы просмотра**](try-review-api-job.md) — операции [Задания](try-review-api-job.md), [Просмотры](try-review-api-review.md) и [Рабочий процесс](try-review-api-workflow.md) — позволяют создавать и автоматизировать рабочие процессы с участием человека с использованием средства проверки.

## <a name="human-review-tool"></a>Средство просмотра человеком

В подписку Content Moderator включено встроенное [средство пользовательской проверки](Review-Tool-User-Guide/human-in-the-loop.md). С помощью упомянутого ранее API просмотра можно создавать просмотры текста, изображений и видео для проверки модераторами вручную и принятия окончательных решений.
