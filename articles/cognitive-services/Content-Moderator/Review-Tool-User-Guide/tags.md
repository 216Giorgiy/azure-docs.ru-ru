# <a name="about-tags"></a>О тегах #

Помимо двух тегов по умолчанию **isadult** (**a**) и **isracy** (**r**) можно создавать настраиваемые теги для более узконаправленной проверки. Затем эти настраиваемые теги используются при пользовательской проверке и могут назначаться изображениям или тексту.

## <a name="create-tags"></a>Создание тегов ##

1.  Выберите "Tags" (Теги) на вкладке "Settings" (Параметры).

  ![Теги модерации контента](images/tags-1.png)

2.  Введите двухбуквенный короткий код для тега.
3.  Введите имя тега. Оно должно быть кратким и понятным. Например, **isbullying**.
4.  Введите описание.
5.  Нажмите Добавить.
6.  Завершив создание тегов, нажмите кнопку "Save" (Сохранить).

![Определение тегов модерации контента](images/tags-2-define.png)

## <a name="using-custom-tags"></a>Использование настраиваемых тегов ##

Настраиваемые теги используются во время пользовательской проверки. Они отображаются в области предварительного просмотра, и рецензент выбирает их щелчком мыши.

![Использование тегов модерации контента](images/tags-3-use.png)

Вы можете отключать различные теги для разных проверок, устанавливая или снимая соответствующие флажки в столбце "Is Visible" (Видимый).
 
![Отключение тегов модерации контента](images/tags-4-disable.png)

Хотя невозможно удалить два тега по умолчанию, **isadult** и **isracy**, можно удалить любые настраиваемые теги, которые вы определили. Щелкните значок корзины рядом с тегом, который требуется удалить.

![Удаление тегов модерации контента](images/tags-5-delete.png)
