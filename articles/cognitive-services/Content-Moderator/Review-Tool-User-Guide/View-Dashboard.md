# <a name="view-dashboard"></a>Просмотр панели мониторинга

Щелкните вкладку **Dashboard** (Панель мониторинга), чтобы просмотреть основные метрики по всем проверкам содержимого, выполненным в инструменте. Можно увидеть общее число, число выполненных и число текущих проверок изображений и текста. Можно просмотреть метрики для команды по умолчанию, а также для ваших подкоманд.
