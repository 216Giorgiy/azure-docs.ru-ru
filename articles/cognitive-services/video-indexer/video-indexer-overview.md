# <a name="what-is-video-indexer-preview"></a>Что такое Индексатор видео? (предварительная версия)

Служба "Индексатор видео" — это облачное приложение, созданное с помощью функции "Аналитика мультимедиа Azure", Cognitive Services (например, API распознавания лиц, Microsoft Translator, API компьютерного зрения и Пользовательская служба распознавания речи) и службы "Поиск Azure". В службе "Индексатор видео" используются технологии искусственного интеллекта, что обеспечивает следующие возможности для получения аналитических сведений:

- **Автоматическое распознавание языка.** Служба "Индексатор видео" может автоматически определять язык видео. Автоматическое распознавание сейчас поддерживается для следующих языков: английский, испанский, французский, немецкий, итальянский, китайский (упрощенное письмо), японский, русский. Если язык не удается распознать, будет использоваться базовый язык (английский).
- **Расшифровка аудио.** В службе "Индексатор видео" есть функция преобразования текста в речь, которая позволяет расшифровывать произнесенные слова. В число поддерживаемых языков входят английский, испанский, французский, немецкий, итальянский, китайский (упрощенное письмо), португальский (Бразилия), японский и русский (в будущем этот список значительно пополнится). 
- **Отслеживание и определение лиц.** Технологии распознавания лиц позволяют обнаруживать лица в видео. Обнаруженные лица сопоставляются с изображениями в базе данных знаменитостей, чтобы определить, какие знаменитости присутствуют в видео. Кроме того, можно отметить лица, которые не совпадают с изображениями в базе данных знаменитостей. В службе "Индексатор видео" можно создать модель распознавания лиц на основе таких меток, чтобы распознавать эти лица в видео, которые будут отправляться в дальнейшем.
- **Индексирование речи говорящего.** В службе "Индексатор видео" можно сопоставить фрагменты речи и определить, кто именно и когда произнес определенные слова.
- **Распознавание визуального текста.** С помощью этой технологии в службе "Индексатор видео" извлекается текст, отображаемый в видео.  
- **Распознавание голоса.** Эта функция службы "Индексатор видео" позволяет отделять фоновый шум от голоса. 
- **Обнаружение сцены.** В службе "Индексатор видео" можно выполнять визуальный анализ видео, чтобы определить, когда в нем сменяется сцена.
- **Извлечение опорного кадра.** В службе "Индексатор видео" автоматически определяются опорные кадры видео. 
- **Анализ тональности.** В службе "Индексатор видео" выполняется анализ тональности в тексте, извлеченном с помощью преобразования речи в текст и распознавания текста. Для такой информации указываются тональность (положительная, отрицательная или нейтральная) и коды времени.
- **Перевод.** Служба "Индексатор видео" позволяет переводить расшифровку аудио с одного языка на другой. Поддерживаются следующие языки: английский, испанский, французский, немецкий, итальянский, китайский (упрощенное письмо), португальский (Бразилия), японский и русский. После перевода можно даже создать субтитры в проигрывателе видео на других языках.
- **Модерация визуального контента.** Эта технология позволяет обнаружить в видео материалы для взрослых или материалы непристойного характера. Ее можно использовать для фильтрации содержимого. 
- **Извлечение ключевых слов.** Служба "Индексатор видео" извлекает ключевые слова на основе расшифровки произнесенных слов и текста, полученного с помощью средства распознавания визуального текста.
- **Метки.** В службе "Индексатор видео" можно добавлять метки для визуальных объектов (например, кошка, собака, стол, автомобиль) и для действий (например, стоит, бежит или летит).
- **Торговые марки.** Служба "Индексатор видео" извлекает торговые марки компаний на основе расшифровки произнесенных слов и текста, полученного с помощью средства распознавания визуального текста.

По завершении обработки и анализа в службе "Индексатор видео" вы сможете просматривать, курировать и публиковать аналитические сведения, а также выполнять по ним поиск.

Служба "Индексатор видео" подходит как для администраторов содержимого, так и для разработчиков. Администраторы содержимого могут использовать веб-портал службы "Индексатор видео". При этом им не нужно писать код (см. руководство по [началу работы с порталом службы "Индексатор видео"](video-indexer-get-started.md)). Разработчики могут использовать API для обработки содержимого в масштабе (см. руководство по [использованию REST API службы "Индексатор видео"](video-indexer-use-apis.md)). Кроме того, служба позволяет публиковать потоковое видео и извлеченные аналитические сведения в собственных приложениях с помощью мини-приложений (см. руководство по [внедрению визуальных мини-приложений в собственные приложения](video-indexer-embed-widgets.md)).

Вы можете зарегистрироваться для использования службы с помощью существующей учетной записи AAD, LinkedIn, Facebook, Google или MSA. Дополнительные сведения см. в руководстве по [началу работы](video-indexer-get-started.md).

## <a name="scenarios"></a>Сценарии

Ниже описано несколько сценариев, в которых можно использовать службу "Индексатор видео":

- Поиск. С помощью аналитических сведений, извлеченных из видео, можно усовершенствовать систему поиска по библиотеке видео. Например, индексирование лиц и произнесенных слов позволяет выполнять поиск моментов в видео, когда определенный человек произносит определенные слова или когда два определенных человека находятся в кадре вместе. Поиск на основе таких аналитических сведений, извлеченных из видео, — полезная функция для информационных агентств, образовательных учреждений, вещательных компаний, владельцев развлекательного содержимого, корпоративных бизнес-приложений и в целом для любой отрасли, где применяются библиотеки видео, по которым нужно выполнять поиск.

- Монетизация. Служба "Индексатор видео" может повысить рентабельность использования видео. Например, отрасли, в которых как источник дохода используется реклама (например, СМИ, социальные сети, и т. д.), могут предоставлять более актуальные объявления, используя извлеченные аналитические сведения в качестве дополнительных сигналов для сервера рекламы (например, реклама спортивной обуви более актуальна при трансляции футбольного матча, чем при трансляции соревнований по плаванию).

- Вовлеченность пользователей. С помощью аналитических сведений, полученных из видео, можно повысить уровень вовлеченности пользователей, размещая для них релевантные фрагменты видео. Предположим, у нас есть учебное видео, в котором на протяжении 30 минут рассматриваются сферы, а на протяжении следующих 30 минут — пирамиды. Учащемуся, который знакомится с пирамидами, будет удобнее, если после 30-й минуты установить метку, чтобы видео начало воспроизводиться с этого момента.

См. [дополнительные сведения](http://aka.ms/videoindexerblog).
