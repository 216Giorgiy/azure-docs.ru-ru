# <a name="bing-search-sdk"></a>Пакет SDK для Поиска Bing
Примеры использования API Bing для поиска изображений включают в себя следующие сценарии:
1.  Запрос простого изображения.
2.  Ограничение области поиска с помощью фильтрации изображений по определенным типам и форматным соотношениям.
3.  Поиск изображений, набирающих сейчас популярность в различных социальных сетях.
4.  Запрос с указанием более подробных сведений об изображении, таких как источник изображения, подпись к изображению и т. д.

Пакеты SDK для службы "Поиск Bing" предоставляют доступ к функции поиска в Интернете на перечисленных ниже языках программирования:
* Начало работы с [примерами для .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
    * [Пакет NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.ImageSearch/1.2.0)
    * Ознакомьтесь также с определениями и зависимостями в [библиотеках .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingImageSearch).
* Начало работы с [примерами для Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * Ознакомьтесь также с определениями и зависимостями в [библиотеках Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/imageSearch).
* Начало работы с [примерами для Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * Ознакомьтесь также с определениями и зависимостями в [библиотеках Java](https://github.com/Azure/azure-sdk-for-java/tree/master/azure-cognitiveservices/search/bingimagesearch).
* Начало работы с [примерами для Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * Ознакомьтесь также с определениями и зависимостями в [библиотеках Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-imagesearch).

Примеры пакетов SDK для каждого языка содержат файл сведений, в котором указаны предварительные требования и инструкции по установке и запуску этих примеров.
