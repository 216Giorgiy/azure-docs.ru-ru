# <a name="what-products-can-be-published-via-the-cloud-partner-portal"></a>Продукты, которые могут быть опубликованы через облачный портал для партнеров

Облачный портал для партнеров выступает в качестве центра для публикации образа виртуальной машины с сертификацией Azure в Microsoft Azure Marketplace или в приложении Microsoft Cortana или Microsoft R Solutions в Microsoft AppSource.  

Образы виртуальных машин, предварительно настроенные с полностью установленной операционной системой и одним или несколькими приложениями. Чтобы опубликовать образ виртуальной машины в Azure Marketplace, необходимо соблюдать [предварительные технические требования](https://docs.microsoft.com/azure/marketplace-publishing/marketplace-publishing-vm-image-creation-prerequisites) для сертификации образа.
