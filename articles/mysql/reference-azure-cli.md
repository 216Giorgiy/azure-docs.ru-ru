# <a name="commands"></a>Команды
|||
|---|---|
|az mysql server|Ниже приведены команды управления серверами. На серверах содержатся базы данных, хранилища данных и эластичные пулы.|
|az mysql server create|Создает сервер MySQL.|
|az mysql server delete|Удаляет сервер MySQL.|
|az mysql server list|Возвращает список всех серверов MySQL в группе ресурсов.|
|az mysql server show|Отображает сведения о сервере MySQL.|
|az mysql server update|Обновляет сервер.|
|az mysql server restore|Восстанавливает сервер до предыдущей точки во времени.|
|az mysql server firewall-rule|Управляет правилами брандмауэра сервера.|
|az mysql server firewall-rule create|Создает правило брандмауэра.|
|az mysql server firewall-rule delete|Удаляет правило брандмауэра.|
|az mysql server firewall-rule list|Выводит список правил брандмауэра.|
|az mysql server firewall-rule show|Отображает сведения о правиле брандмауэра.|
|az mysql server firewall-rule update|Обновляет правило брандмауэра.|
|az mysql db|Управляет базами данных.|
|az mysql db list|Выводит список всех баз данных MySQL на сервере.|
|az mysql server-logs list|Выводит список файлов журнала для сервера.|
|az mysql server-logs download|Скачивает журналы с сервера в локальную среду.|
|az mysql server configuration list|Выводит список параметров конфигурации сервера.|
|az mysql server configuration show|Отображает сведения о параметре конфигурации сервера.|
|az mysql server configuration update|Изменяет значение параметра конфигурации сервера.|
|||
