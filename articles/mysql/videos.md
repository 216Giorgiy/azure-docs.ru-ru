# <a name="azure-database-for-mysql-videos"></a>Видеоролики, посвященные базе данных Azure для MySQL

[Как начать работу с новой службой "База данных Azure для MySQL"](https://channel9.msdn.com/events/Build/2017/B8045)

<iframe src="https://channel9.msdn.com/Events/Build/2017/B8045/player" width="800" height="450" allowFullScreen frameBorder="0"></iframe>

В этом видеоролике, представленном корпорацией Майкрософт на пресс-конференции в 2017 г., описывается управляемая служба MySQL в Azure, предлагаемая корпорацией Майкрософт. Рассматривается стратегия корпорации Майкрософт по поддержке СУБД с открытым кодом в Azure. Обсуждается, что для вас как разработчика означает разработка и развертывание приложений, использующих MySQL в Azure. Представлен обзор архитектуры службы, а также демонстрируется, как база данных Azure для MySQL интегрируется с другими службами Azure, например с веб-приложениями.
