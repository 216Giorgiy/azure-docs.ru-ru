#Масштабирование службы мультимедиа  

[WACOM.INCLUDE [disclaimer](../includes/disclaimer.md)]


Службы мультимедиа можно масштабировать, указав число **зарезервированных единиц потоковой передачи по требованию** и **зарезервированных единиц кодирования**, которые требуется подготовить для вашей учетной записи. 


<h2>Зарезервированные модули потоковой передачи по требованию</h2>

Зарезервированные единицы потоковой передачи по требованию предоставляют как выделенные мощности исходящего трафика, которые могут быть приобретены с шагом 200 Мбит/с, так и дополнительную функциональность, которая в настоящее время включает [возможности динамической упаковки](http://go.microsoft.com/fwlink/?LinkId=276874). По умолчанию потоковая передача по требованию настроена в модели общего экземпляра, для которой серверные ресурсы (например, вычислительные ресурсы, мощности исходящего трафика и т. д.) совместно используются со всеми другими пользователями. Чтобы повысить производительность потоковой передачи по требованию, рекомендуется приобретать зарезервированные единицы потоковой передачи по требованию. 

Чтобы изменить число зарезервированных единиц потоковой передачи по требованию, выполните следующие действия:

1. На [портале управления](https://manage.windowsazure.com/) щелкните **Службы мультимедиа**. Затем щелкните имя службы мультимедиа.

2. Выберите страницу ИСТОЧНИКИ. Затем щелкните исходный элемент, который требуется изменить.

	![Origin page](./media/media-services-how-to-scale/media-services-origin-page.png)

3. Чтобы указать число зарезервированных единиц, перейдите на вкладку МАСШТАБ и переместите ползунок **зарезервированной мощности**.

	![Страница «Масштаб»](./media/media-services-how-to-scale/media-services-origin-scale.png)

4. Нажмите кнопку СОХРАНИТЬ, чтобы сохранить изменения.

	Распределение всех новых единиц потоковой передачи по требованию занимает около 20 минут. 

	 
	**Примечание.** В настоящее переход от любого положительного значения единиц потоковой передачи по требованию к нулевому, может привести к отключению потоковой передачи по требованию на период до одного часа.


	**Примечание.** Для расчета затрат используется наибольшее число единиц, указанных для 24-часового периода. Дополнительные сведения о ценах см. в разделе [Сведения о ценах для служб мультимедиа](http://go.microsoft.com/fwlink/?LinkId=275107).

<h2>Зарезервированные модули кодирования</h2>

Количество подготовленных зарезервированных единиц кодирования равняется числу задач мультимедиа, которое может быть одновременно обработано в данной учетной записи. Например, если в учетной записи имеется 5 зарезервированных единиц, то будет запущено 5 одновременно выполняемых задач мультимедиа, если имеются задачи, которые требуется обработать. Остальные задачи будут ожидать в очереди, они будут последовательно отбираться для обработки сразу же после завершения выполнения текущей задачи. Если учетная запись не имеет подготовленных зарезервированных единиц, задачи будут отбираться последовательно. В этом случае время ожидания между завершением одной задачи и началом выполнения следующей будет зависеть от доступности ресурсов в системе.

Чтобы изменить число зарезервированных единиц кодирования, выполните следующие действия:

1. На [портале управления](https://manage.windowsazure.com/) щелкните **Службы мультимедиа**. Затем щелкните имя службы мультимедиа.

2. Выберите страницу ОБРАБОТЧИКИ. 

	![Страница "Процессоры"](./media/media-services-how-to-scale/media-services-encoding-scale.png) 

3. Нажмите кнопку СОХРАНИТЬ, чтобы сохранить изменения.

	Почти сразу же распределяются новые зарезервированные единицы кодирования.

	**Примечание.** Для расчета затрат используется наибольшее число единиц, указанных для 24-часового периода.

<h2>Открытие запроса в службу поддержки</h2>


По умолчанию каждая учетная запись служб мультимедиа может масштабироваться до 25 зарезервированных единиц кодирования и 5 зарезервированных единиц потоковой передачи по требованию. Вы можете запросить более высокий предел, открыв запрос в службу поддержки.

Чтобы открыть запрос в службу поддержки, выполните следующее: 

1. Войдите в свою учетную запись Azure на [портале управления](http://manage.windowsazure.com).
2. Перейти в [службу поддержки](http://www.windowsazure.com/ru-ru/support/contact/).
3. Выберите "Получить поддержку".
4. Выберите свою подписку.
5. В группе типа поддержки выберите "Техническая".
6. Щелкните "Создать запрос в службу поддержки".
7. Выберите "Службы мультимедиа Azure" в списке продуктов, представленных на следующей странице.
8. Выберите "Обработка мультимедиа" как "Тип проблемы", а затем выберите "Резервирование единицы" в разделе категорий.
9. Нажмите кнопку "Продолжить".
10. Следуйте инструкциям на следующей странице, а затем введите сведения о том, сколько требуется зарезервированных единиц кодирования или потоковой передачи по требованию.
11. Нажмите кнопку "Отправить" для создания запроса в службу поддержки.
