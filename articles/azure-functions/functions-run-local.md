# <a name="how-to-code-and-test-azure-functions-locally"></a>Как программировать и тестировать функции Azure в локальной среде
> [!NOTE]
> Сейчас доступна только предварительная версия возможностей для разработки и инструментов Функций Azure. В окончательном выпуске все возможности и инструменты будут значительно улучшены. Самый простой способ запустить узел Функций Azure локально — использовать [интерфейс командной строки Функций Azure](https://go.microsoft.com/fwlink/?linkid=832752) и установить его из npm. В настоящее время поддерживается только Windows. Полная документация будет опубликована в ближайшее время. Чтобы отправить отзыв об интерфейсе командной строки Функций Azure или сообщить об ошибках в его работе, зарегистрируйте проблему в репозитории GitHub [azure-webjobs-sdk-script](https://github.com/Azure/azure-webjobs-sdk-script).
