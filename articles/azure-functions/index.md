#<a name="azure-functions-documentation"></a>Документация по функциям Azure

Функции Azure — это решение для быстрого запуска фрагментов кода (функций) в облаке. Вы можете быстро писать код для решения своих задач, не беспокоясь о наличии приложения или инфраструктуры для его выполнения. Также вы можете разрабатывать решения на удобном для вас языке, включая C#, F #, Node.js, Python или PHP.
