## <a name="clean-up-resources"></a>Очистка ресурсов

Если вы не собираетесь использовать это приложение дальше, удалите все ресурсы, созданные в ходе работы с этим кратким руководством, чтобы не оплачивать их.

1. На портале Azure выберите **Группа ресурсов** слева, а затем выберите созданную группу ресурсов. Также можно использовать поле поиска для поиска ресурса по его имени.

1. В открывшемся окне выберите группу ресурсов и щелкните **Удалить группу ресурсов**.

1. В новом окне введите имя группы ресурсов, которую требуется удалить, и щелкните **Удалить**.
