# <a name="view-all-your-billing-accounts-in-azure-portal"></a>Просмотр всех учетных записей выставления счетов на портале Azure  

Учетные записи выставления счетов определяют отношения выставления счетов Azure. Вы можете использовать несколько учетных записей выставления счетов. Например, у вас может быть подписка Azure, которую вы используете для личных проектов. Вы также можете получить доступ к Azure через соглашение Enterprise вашей организации. На портале Azure выберите [Cost Management + Billing](https://portal.azure.com/#blade/Microsoft_Azure_Billing/BillingMenuBlade/Overview) (Управление затратами + выставление счетов), чтобы просмотреть все учетные записи выставления счетов.

В настоящее время вы можете просматривать только личные учетные записи выставления счетов и учетные записи выставления счетов соглашения Enterprise. По умолчанию на странице [Cost Management + Billing](https://portal.azure.com/#blade/Microsoft_Azure_Billing/BillingMenuBlade/Overview) (Управление затратами + выставление счетов) отображается ваша личная учетная запись выставления счетов. Однако вы можете перейти к **учетным записям выставления счетов** и изменить учетную запись.

Все свои расходы личных учетных записей выставления счетов вы можете просматривать на [портале Azure](https://portal.azure.com). Но для учетных записей с соглашением Enterprise вы можете просматривать только покупки зарезервированного экземпляра, баланс и расходы за текущий месяц. Эти функции соглашения Enterprise находятся в режиме предварительной версии и доступны только в том случае, если вы находитесь в роли администратора Enterprise.

## <a name="change-billing-account-view"></a>Просмотр изменения учетной записи выставления счетов 

1.  Войдите на [портал Azure](https://portal.azure.com).

2.  В левой части портала выберите **Cost Management + Billing** (Управление затратами + выставление счетов). 

3.  По умолчанию, вы увидите свою личную учетную запись (если она у вас есть). Если у вас нет личной учетной записи выставления счетов или вы хотите перейти на учетную запись соглашения Enterprise, выберите **Учетные записи выставления счетов**.

4.  Выберите учетную запись для просмотра сведений. Для соглашения Enterprise вы увидите только те учетные записи выставления счетов, в которых вы являетесь администратором Enterprise. Другие роли соглашения Enterprise будут поддерживаться в будущем.
