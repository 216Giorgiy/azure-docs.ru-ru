# <a name="create-an-additional-subscription-in-the-azure-portal"></a>Создание дополнительной подписки на портале Azure

Вы можете создавать дополнительные подписки для своей учетной записи в Azure. Дополнительную подписку можно использовать, чтобы избежать превышения ограничений на подписки и создать отдельные среды для выставления счетов и обеспечения безопасности, а также чтобы изолировать данные для обеспечения соответствия.

Чтобы создавать подписки Azure в рамках корпоративного Соглашения Enterprise, требуется роль владельца учетной записи для вашей организации.

Если у вас еще нет подписки Azure, создайте [бесплатную учетную запись](https://azure.microsoft.com/free).

## <a name="create-an-additional-azure-subscription"></a>Создание дополнительной подписки Azure

1. Войдите на [портале Azure](https://portal.azure.com).
1. Выполните поиск по запросу **Подписки**.

   ![Снимок экрана: поиск подписок на портале](./media/billing-create-subscription/billing-search-subscription-portal.png)

1. Выберите **Добавить**.

   ![Снимок экрана: кнопка "Добавить" в представлении "Подписки"](./media/billing-create-subscription/subscription-add.png)

Дополнительные подписки можно также создать программными средствами. См. дополнительные сведения о [программными средствами](../azure-resource-manager/programmatically-create-subscription.md).
