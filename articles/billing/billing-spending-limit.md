# <a name="understand-azure-spending-limit-and-how-to-remove-it"></a>Общие сведения о предельной сумме расходов Azure и ее отключении

Предельная сумма расходов Azure — это предел затрат по подписке Azure. Для всех новых клиентов, которые зарегистрировали бесплатную пробную версию или приобрели на несколько месяцев предложение с деньгами на счете, по умолчанию установлена предельная сумма расходов. Она составляет 0 долл. США. Ее невозможно изменить. Предельная сумма расходов не используется для подписок с оплатой по мере использования и тарифных планов с фиксированной ежемесячной платой. Список всех доступных предложений Azure и сведения о доступности предельной суммы расходов см. [здесь](https://azure.microsoft.com/support/legal/offer-details/).

## <a name="what-happens-when-i-reach-the-spending-limit"></a>Что происходит при достижении предельной суммы расходов

Если вы превысили ежемесячную сумму, предусмотренную для вашего предложения, ваши развернутые службы отключаются до конца расчетного месяца для предотвращения дополнительных расходов. Например, ваши развернутые облачные службы удаляются из рабочей среды, виртуальные машины Azure останавливаются и выделенные для них ресурсы освобождаются. Если вы не хотите, чтобы ваши службы отключались, снимите ограничение на предельную сумму расходов. Когда службы отключены, данные в учетных записях хранения и базах данных доступны только для чтения и только администраторам. В начале следующего расчетного месяца, если для вашего предложения предусмотрены деньги на счете на несколько месяцев, ваша подписка будет снова активирована. После этого вы сможете повторно развернуть свои облачные службы и получить полный доступ к своим учетным записям хранения и базам данных.

Если для бесплатной пробной подписки достигнута предельная сумма расходов, вы можете в течение 90 дней снова включить подписку и задать ее автоматическое [обновление до нашего стандартного предложения с оплатой по мере использования](billing-upgrade-azure-subscription.md).

При достижении предельной суммы расходов для предложения вы получаете уведомления. Войдите в [Центр учетных записей Azure](https://account.windowsazure.com), щелкните **Учетная запись** и выберите **Подписки**. Для подписок, в которых достигнута предельная сумма расходов, отображается соответствующее уведомление.

## <a name="things-you-are-charged-for-even-if-you-have-a-spending-limit-enabled"></a>За что взимается плата, даже если установлена предельная сумма расходов

За использование некоторых служб Azure и [покупки в Marketplace](https://azure.microsoft.com/marketplace/) с вашего счета будут списываться средства, даже если установлена предельная сумма расходов. Примером таких служб и покупок могут быть лицензии на Visual Studio, служба Azure Active Directory Premium, планы поддержки и большинство сторонних служб, продаваемых в Marketplace.


## <a name="when-not-to-use-the-spending-limit"></a>Когда не следует использовать ограничение на предельную сумму расходов

Из-за установленной предельной суммы расходов у вас могут возникнуть проблемы с развертыванием или использованием некоторых служб Майкрософт, а также служб, продаваемых в Marketplace. Ниже описаны ситуации, в которых в подписке следует снять ограничение на предельную сумму расходов.

- Вы планируете развернуть образы от таких поставщиков, как Oracle, или таких служб, как Visual Studio Team Services. В этом случае предельная сумма расходов будет превышена почти мгновенно, и ваша подписка будет отключена.

- У вас есть службы, работу которых нельзя прерывать.

- У вас есть службы или ресурсы с определенными параметрами (например, виртуальные IP-адреса), которые вы не хотите терять. При освобождении ресурсов и служб такие параметры сбрасываются.


## <a name="remove-the-spending-limit"></a>Снятие ограничения на предельную сумму расходов

Вы можете снять ограничение предельной суммы расходов в любой момент при условии, что в вашей подписке указан действительный метод оплаты. Для предложений с деньгами на счете на несколько месяцев это ограничение можно активировать повторно. Ограничение будет установлено в начале следующего цикла выставления счетов.

Чтобы снять ограничение, выполните следующие действия:

1. Войдите в [Центр учетных записей Azure](https://account.windowsazure.com).

2. Выберите подписку.

3. Если подписка отключена из-за превышения установленной суммы расходов, щелкните уведомление "Для подписки достигнута предельная сумма расходов, подписка отключена для предотвращения дополнительных затрат". В противном случае в области **Состояние подписки** щелкните **Снять предельную сумму расходов**.

4. Выберите нужный вам вариант.

|Параметр|Результат|
|-------|-----|
|Удалить предельную сумму расходов на неопределенный срок|Удаление предельной суммы расходов без ее автоматического включения в начале следующего расчетного периода.|
|Удалить предельную сумму расходов для текущего расчетного периода|Удаление предельной суммы расходов с ее автоматическим включением в начале следующего расчетного периода.|
