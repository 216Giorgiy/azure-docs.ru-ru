# <a name="avoid-getting-charged-for-your-azure-free-account"></a>Как не платить за бесплатную учетную запись Azure

Бесплатная учетная запись Azure предоставляет деньги на счете в Azure на сумму 200 долларов США, которые можно потратить в течение первых 30 дней, и 12 месяцев использования бесплатных служб. Дополнительные сведения см. в статьях [Create your Azure free account today](https://azure.microsoft.com/free/) (Создание бесплатной учетной записи Azure) и [Создание служб, доступных в бесплатной учетной записи Azure](billing-create-free-services-included-free-account.md). В зависимости от состояния кредита вы можете использовать свой кредит или получить счет за использование, который не покрывается преимуществами бесплатной учетной записи Azure.

## <a name="you-still-have-active-azure-free-account-credit"></a>У вас все еще есть деньги на счете в бесплатной учетной записи Azure 
Если у вас все еще есть деньги на счете в Azure, срок действия которых не истек, их можно использовать для оплаты любых счетов, выставляемых в бесплатной учетной записи Azure.

## <a name="your-credit-ran-out-or-has-expired"></a>Деньги на счете в Azure закончились, или их срок действия истек
Если через 30 дней деньги на счете в Azure закончились или истек срок их действия, Azure отключит вашу подписку. Плата взимается, только если вы обновили подписку до подписки с оплатой по мере использования. Дополнительные сведения см. в статье [Обновление с бесплатной пробной версии или подписки Azure Microsoft Imagine до подписки с оплатой по мере использования](billing-upgrade-azure-subscription.md). После обновления подписки будет взиматься плата по тарифам оплаты по мере использования для всех затрат в бесплатной учетной записи Azure. 


Рассмотрим некоторые из причин, из-за которых может взиматься плата в бесплатной учетной записи Azure.


### <a name="your-usage-exceeds-the-limits-of-free-services-included-with-azure-free-account"></a>Потребление превышает ограничения бесплатных служб, включаемых в бесплатную учетную запись Azure

Вы получаете ограниченное количество бесплатных служб каждый месяц в бесплатной учетной записи Azure. Количество бесплатных служб истекает в конце месяца и не переносится на следующий месяц. Например, вы получаете 5 ГБ хранилища файлов каждый месяц. Если в месяц использовалось 2 ГБ, оставшиеся 3 ГБ не переносятся на следующий месяц. Чтобы избежать выставления счетов, потребляйте ресурсы в допустимых пределах. Дополнительные сведения об ограничениях на бесплатные службы см. в статье [Вопросы и ответы о бесплатной учетной записи Azure](https://azure.microsoft.com/free/free-account-faq/). Чтобы проверить потребление бесплатных служб, ознакомьтесь со статьей [Проверка использования бесплатных служб, включаемых в бесплатную учетную запись Azure](billing-check-free-service-usage.md). 

### <a name="you-are-using-services-not-included-with-azure-free-account"></a>Используются службы, не входящие в бесплатную учетную запись Azure

Выставление счетов за использование служб, которые не включены в бесплатную учетную запись Azure, будет осуществляться согласно тарифам оплаты по мере использования. Дополнительные сведения о службах, входящих в бесплатную учетную запись, см. в статье [Вопросы и ответы о бесплатной учетной записи Azure](https://azure.microsoft.com/free/free-account-faq/). Потребление можно проверить на портале Azure или через файл потребления Azure. Дополнительные сведения см. в разделе [Регулярно просматривайте на портале детализацию затрат и график расходов](billing-getting-started.md#costs) и статье [Скачивание или просмотр счета на оплату и данных о ежедневном использовании в Azure](billing-download-azure-invoice-daily-usage-date.md). 

### <a name="you-have-reached-the-end-of-your-azure-free-account-benefits"></a>Использованы все преимущества бесплатной учетной записи Azure

Срок действия преимуществ бесплатной учетной записи Azure истекает через 12 месяцев. После этого Azure выставляет счета за использование служб согласно тарифам оплаты по мере использования. Портал Azure можно использовать для удаления ресурсов неиспользуемых служб. Если вы не собираетесь использовать какие-либо службы Azure, можно [отказаться от подписки](billing-how-to-cancel-azure-subscription.md).
