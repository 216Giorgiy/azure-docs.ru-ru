# <a name="device-models"></a>Модель устройства

При настройке симуляции вы можете использовать существующую модель устройства с предопределенным набором датчиков или создать пользовательскую модель устройства и настроить для нее нужный набор датчиков. Пользовательский набор датчиков позволяет точнее моделировать реальные устройства.

## <a name="pre-configured-device-models"></a>Предварительно настроенные модели устройств

Акселератор решений "Симулятор устройств" предлагает три предварительно настроенные модели устройств: Chillers (холодильники), Elevators (лифты) и Trucks (грузовики).

Предварительно настроенные модели устройств содержат несколько датчиков и расширенных функций, которые определены в файле JavaScript. Вы не можете изменять эти функции через веб-интерфейс.

В следующей таблице показан список конфигураций для каждой предварительно настроенной модели устройства.

| Модель устройства  | Датчик           | Единица измерения  |
| ------------- | ---------------- | ----- |
| Холодильник       | Влажность         | %     |
|               | pressure         | фунтов на кв. дюйм  |
|               | Температура      | F     |
| Лифт      | Этаж            |       |
|               | Вибрация        | ММ    |
|               | температура;      | F     |
| Грузовик         | Широта         |       |
|               | Долгота        |       |
|               | Скорость            | миль в час   |
|               | Температура груза | F     |

## <a name="custom-device-models"></a>Пользовательские модели устройств

Пользовательские модели устройств позволяют определить собственный набор датчиков, который лучше соответствует реальным устройствам. В пользовательском устройстве может быть до 10 датчиков.

Выбрав пользовательский тип для модели устройств, вы сможете использовать действие **+Add sensor** (+Добавить датчик).

[![Настройка датчиков](./media/iot-accelerators-device-simulation-custom-model/configuresensors-inline.png)](./media/iot-accelerators-device-simulation-custom-model/configuresensors-expanded.png#lightbox)

Пользовательские датчики обладают перечисленными ниже свойствами:

| Поле     | ОПИСАНИЕ |
| --------- | ----------- |
| Имя датчика | Понятное имя датчика, такое как **температура** или **скорость**.  |
| Поведение  | Реакции на событие позволяют передавать разные данные телеметрии между сообщениями, симулируя реальные данные. **Приращение** увеличивает значение на один в каждом отправленном сообщении, начиная с минимального значения. После достижения максимального значения оно вернется к минимальному значению и снова начнет увеличиваться. **Уменьшение** ведет себя подобно **приращению**, но в обратном порядке. Реакция на событие **Случайно** создает случайное значение в промежутке между максимальным и минимальным значением. |
| Минимальное значение | Нижняя граница допустимого диапазона. |
| Максимальное значение | Верхняя граница допустимого диапазона. |
| Единица измерения      | Единица измерения для датчика, например "°F" или "миль в час". |
