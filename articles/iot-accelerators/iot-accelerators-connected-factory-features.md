# <a name="what-is-azure-iot-suite-connected-factory"></a>Что такое подключенная фабрика Azure IoT Suite?

Подключенная фабрика реализует в одном решении с открытым исходным кодом эталонную архитектуру Azure для промышленного Интернета вещей, разработанную корпорацией Майкрософт. Вы можете использовать ее в качестве отправной точки для коммерческих продуктов. Готовые версии решения на основе подключенной фабрики вы можете развернуть в подписке Azure из [Azure IoT Suite](https://www.azureiotsuite.com/#solutions/types/CF).

![Панель мониторинга для решения подключенной фабрики](./media/iot-accelerators-connected-factory-features/dashboard.png)

Подключенная фабрика реализует следующие возможности.

## <a name="industrial-device-interoperability"></a>Взаимодействие между промышленными устройствами

- Подключение к промышленным ресурсам, поддерживающим интерфейс OPC UA.
- Использование имитированных производственных линий (под управлением серверов OPC UA в контейнерах Docker) для получения от них данных телеметрии в режиме реального времени.
- Просмотр информационной модели OPC UA для серверов OPC UA через панель мониторинга для облака.

## <a name="remote-management"></a>Удаленное управление

- Настройка активов OPC UA с помощью облачной панели мониторинга (вызов методов, чтение и запись данных).
- Публикация и отмена публикации данных телеметрии, получаемых от активов OPC UA через облачную панель мониторинга.

## <a name="cloud-dashboard"></a>Облачная панель мониторинга

- Предварительный просмотр данных телеметрии прямо на облачной панели мониторинга.
- Просмотр тенденций для данных телеметрии и создание корреляций с помощью панели мониторинга обозревателя службы "Аналитика временных рядов".
- Просмотр вычисляемой общей эффективности работы оборудования (OEE) и ключевых показателей эффективности (KPI) на облачной панели мониторинга.
- Просмотр иерархии промышленных активов в топологии дерева и на интерактивной карте.
- Просмотр, подтверждение и закрытие оповещений через облачную панель мониторинга.

## <a name="azure-time-series-insights"></a>Azure Time Series Insights

- [Служба "Аналитика временных рядов"](../time-series-insights/time-series-insights-overview.md) разработана для того, чтобы сохранять, визуализировать и запрашивать большие объемы данных временных рядов. Подключенная фабрика использует эту службу.
- Подключенная фабрика интегрируется с этой службой, позволяя выполнять глубокий анализ данных устройства в режиме реального времени.

## <a name="rules-and-alerts"></a>Правила и оповещения

[Настройка правил для предупреждений на основе пороговых значений](iot-accelerators-connected-factory-configure.md).

## <a name="end-to-end-security"></a>Комплексная безопасность

- Настройка разрешений безопасности для пользователей с помощью управления доступом на основе ролей (RBAC).
- Сквозное шифрование реализуется с помощью аутентификации OPC UA (с использованием сертификатов X.509) и маркеров безопасности.

## <a name="customizability"></a>Возможности настройки

- Вы можете [настроить](../iot-suite/iot-suite-v1-guidance-on-customizing-preconfigured-solutions.md) решение в соответствии с потребностями конкретной организации.
- Полный исходный код решения вы найдете на сайте GitHub. См. также репозиторий [предварительно настроенного решения подключенной фабрики](https://github.com/Azure/azure-iot-connected-factory).
