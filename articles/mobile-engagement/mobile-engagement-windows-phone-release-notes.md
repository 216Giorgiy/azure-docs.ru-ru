#Заметки о выпуске пакета SDK для Windows Phone Silverlight

##3\.1.0 (11/20/2015)
Входит в пакет NuGet *MicrosoftAzure.MobileEngagement* **версии 3.2.0**

-   Исправлена проблема сбоев при запросе идентификатора устройства сразу после инициализации.
-   Улучшены журналы консоли.

##3\.0.0 (03.04.2015)

-   Обновлен стандартный значок уведомления.
-   При щелчке уведомления отправляется отклик на действие с системным уведомлением.
-   Исправлено состояние гонки для системных уведомлений.

##Версия 2.0.0 (17.02.2015)

-   Первый выпуск Azure Mobile Engagement.
-   Конфигурация appId/sdkKey заменена конфигурацией строки подключения.
-   Улучшения безопасности.
