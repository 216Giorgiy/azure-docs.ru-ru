#Заметки о выпуске веб-пакета SDK для Azure Mobile Engagement

##2\.0.1 (10.06.2016)

-   Пакет SDK отключен в IE8 и IE9.
-   Исправлено обнаружение веб-браузера Opera.

##2\.0.0 (07.06.2016)

-   Первый выпуск Azure Mobile Engagement.
-   Конфигурация appId заменена на конфигурацию строки подключения.
-   Добавлены API для включения и отключения агента.
-   Улучшения безопасности.
-   Использование собственных API JSON из браузеров.
-   Удален API для отправки и получения сообщений между устройствами.
-   Удалены обратные вызовы, связанные с нерекомендованным соединением XMPP. 
-   Эта версия не поддерживает функцию Reach.
