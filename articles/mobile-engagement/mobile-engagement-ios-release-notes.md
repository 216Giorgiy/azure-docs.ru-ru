#Заметки о выпуске

##3\.0.0 (06/19/2015)

-   Mobile Engagement использует автоматические push-уведомления.

##2\.2.0 (05/21/2015)

-   В основе идентификатора устройства Mobile Engagement для устройств c версией ОС ниже iOS 6 теперь используется GUID, генерируемый во время установки.

##2\.1.0 (24.04.2015)

-   Добавлена поддержка Swift.
-   При щелчке уведомления теперь после открытия приложения вызывается URL-адрес действия.
-   В пакет SDK добавлен отсутствующий файл заголовка.
-   Исправлена проблема, при которой отключалось средство создания отчетов об аварийном завершении Mobile Engagement.

##Версия 2.0.0 (17.02.2015)

-   Первый выпуск Azure Mobile Engagement
-   Конфигурация appId/sdkKey заменена конфигурацией строки подключения.
-   Удален API для отправки и получения произвольных сообщений XMPP из произвольных сущностей XMPP.
-   Удален API для отправки и получения сообщений между устройствами.
-   Улучшения безопасности.
-   Удалено средство отслеживания SmartAd.
