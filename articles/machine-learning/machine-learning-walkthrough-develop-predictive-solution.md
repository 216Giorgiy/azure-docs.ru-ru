# <a name="walkthrough-develop-a-predictive-analytics-solution-for-credit-risk-assessment-in-azure-machine-learning"></a>Пошаговое руководство по разработке решения для прогнозной аналитики в службе машинного обучения Azure для оценки кредитных рисков

В этом пошаговом руководстве вы узнаете, как разработать решение в студии машинного обучения. Мы разработаем модель прогнозной аналитики в студии машинного обучения, а затем развернем эту модель в веб-службе машинного обучения Azure для создания прогнозов на основе новых данных. 

> [!TIP]
> В этом руководстве предполагается, что вы уже работали со студией машинного обучения и имеете некоторое представление о машинном обучении.
> 
>Если вы не использовали **студию машинного обучения**, прежде чем приступать к работе с этим руководством, [создайте свой первый эксперимент по обработке и анализу данных в студии машинного обучения Azure](machine-learning-create-experiment.md). В процессе ознакомления со студией машинного обучения вы научитесь выполнять базовые операции: переносить модули в эксперимент, соединять их друг с другом, запускать эксперимент и просматривать результаты.
>
>Если вы еще плохо знакомы с машинным обучением, рекомендуем начать изучение с серии видеоматериалов [Обработка и анализ данных для начинающих](machine-learning-data-science-for-beginners-the-5-questions-data-science-answers.md). В этих видео простым языком объясняются базовые принципы машинного обучения.
> 

## <a name="the-problem"></a>Задача

Предположим, требуется спрогнозировать кредитный риск претендентов на кредит на основе сведений, которые они предоставляют в заявке на кредит.  

Оценка кредитных рисков — несомненно, сложная задача, но мы немного упростим ее условия. Затем мы сможем использовать ее как пример создания решения для прогнозной аналитики с помощью машинного обучения Microsoft Azure, студии машинного обучения и веб-службы машинного обучения.  

## <a name="the-solution"></a>Решение

В рамках этого руководства мы сначала обратимся к общедоступным данным по кредитным рискам, разработаем и обучим на основе этих данных прогнозную модель, а затем развернем ее как веб-службу, которую смогут использовать для оценки рисков по кредитам и другие пользователи.

Создание нашего решения для оценки кредитных рисков предполагает такие действия:

2. [Отправка существующих данных](machine-learning-walkthrough-2-upload-data.md)
3. [Создание нового эксперимента](machine-learning-walkthrough-3-create-new-experiment.md)

6. [Доступ к веб-службе](machine-learning-walkthrough-6-access-web-service.md)
