# <a name="discover-custom-machine-learning-modules-in-the-cortana-intelligence-gallery"></a>Изучите пользовательские модули машинного обучения в коллекции Cortana Intelligence
[!INCLUDE [machine-learning-gallery-item-selector](../../includes/machine-learning-gallery-item-selector.md)]

## <a name="custom-modules-for-machine-learning-studio"></a>Пользовательские модули для Студии машинного обучения
В коллекции Cortana Intelligence есть несколько **[пользовательских модулей](https://gallery.cortanaintelligence.com/customModules)**, которые расширяют возможности Студии машинного обучения Azure. Вы можете загрузить эти модули и использовать их в экспериментах, чтобы создать еще более сложные решения для прогнозной аналитики.

## <a name="discover"></a>Обнаружение
Для просмотра пользовательских модулей в коллекции откройте элемент [Коллекция](http://gallery.cortanaintelligence.com), наведите указатель мыши на элемент **Дополнительно** в верхней части домашней страницы "Коллекция" и выберите пункт **Пользовательские модули**.

![Выбор пользовательских модулей на домашней странице "Коллекция"](media/machine-learning-gallery-custom-modules/select-custom-modules-in-gallery.png)

 На странице **[Пользовательские модули](https://gallery.cortanaintelligence.com/customModules)**
 отображается список популярных модулей.
Щелкните **Показать все**, чтобы просмотреть все пользовательские модули.
На открывшейся странице можно просмотреть все пользовательские модули в коллекции. Здесь также можно воспользоваться фильтрами в левой части страницы или строкой поиска вверху, чтобы найти определенные модули.

### <a name="comment-and-share"></a>Комментарии и общий доступ
 Щелкните любой пользовательский модуль, чтобы открыть страницу сведений и узнать о назначении и использовании модуля. На этой странице в разделе комментариев можно оставлять замечания, отзывы и задавать вопросы. Ссылку на страницу можно опубликовать в социальных сетях LinkedIn и Twitter для друзей и коллег. Кроме того, ссылку можно отправить по электронной почте, чтобы другие пользователи также могли просмотреть найденный вами пользовательский модуль.

![Расскажите друзьям о том, что вы нашли](media/machine-learning-gallery-how-to-use-contribute-publish/share-links.png)

![Добавление комментариев](media/machine-learning-gallery-how-to-use-contribute-publish/comments.png)

## <a name="download"></a>Загрузить
Любой пользовательский модуль из коллекции можно использовать в своих экспериментах.
Получить копию модуля можно двумя способами.

* **Из коллекции** — загружаемый из коллекции модуль добавляется на палитру модулей в рабочей области. Можно также получить пример эксперимента, который поможет разобраться, как можно использовать этот модуль.
* **Из в Студии машинного обучения** — работая в Студии, можно импортировать любой пользовательский модуль, который будет добавлен на палитру модулей в рабочей области.

Когда пользовательский модуль окажется на палитре модулей, вы сможете использовать его в любых экспериментах в рабочей области, как и любой другой модуль.

1. Создайте новый эксперимент или откройте существующий
2. На палитре модулей слева от холста эксперимента щелкните элемент **Пользовательские**, чтобы открыть список пользовательских модулей, доступных в рабочей области.
   
    ![Список пользовательских модулей на палитре Студии](media/machine-learning-gallery-custom-modules/custom-module-in-studio-palette.png)
3. Выберите импортированный модуль и перетащите его в эксперимент.

### <a name="from-the-gallery"></a>Из коллекции
Чтобы скачать копию пользовательского модуля из коллекции, выполните следующее.

1. Откройте страницу сведений о модуле в коллекции.
2. Щелкните **Открыть в Studio**.
   
    ![Откройте пользовательский модуль из коллекции](media/machine-learning-gallery-custom-modules/open-custom-module-from-gallery.png)
   
    Каждый пользовательский модуль содержит пример эксперимента, в котором демонстрируется использование этого модуля. После нажатия кнопки **Открыть в Studio** пример эксперимента загружается в рабочую область Студии машинного обучения и открывается там. (Если вы еще не вошли в Студию, прежде чем эксперимент будет скопирован в рабочую область, вам будет предложено войти, используя учетную запись Майкрософт.)

Пользовательский модуль копируется в рабочую область и размещается на палитре модулей рядом с остальными встроенными и пользовательскими модулями Студии. Теперь вы можете использовать его в рабочей области так же, как любой другой модуль.

### <a name="from-within-machine-learning-studio"></a>Из Студии машинного обучения Azure
Вы можете импортировать те же пользовательские модули из коллекции, работая в Студии машинного обучения.

1. В студии машинного обучения щелкните **+New** (Создать).
2. Выберите **Модуль**, и вы увидите список модулей коллекции. Вы можете выбрать любой модуль из этого списка или найти конкретный модуль с помощью поля поиска.
3. Наведите указатель на нужный модуль и выберите **Импорт модуля** (Чтобы просмотреть сведения о модуле, выберите **Просмотреть в коллекции**, чтобы перейти на страницу коллекции со сведениями об этом модуле.)
   
    ![Импорт пользовательского модуля в Студию машинного обучения](media/machine-learning-gallery-custom-modules/add-custom-module-in-studio.png)

Пользовательский модуль копируется в рабочую область и размещается на палитре модулей рядом с остальными встроенными и пользовательскими модулями Студии. Теперь вы можете использовать его в рабочей области так же, как любой другой модуль.
