#<a name="heading"></a>Выборка данных в процессе аналитики Кортаны

Это **меню** содержит ссылки на разделы, описывающие выборку данных из различных сред хранения. Эта задача является одним из этапов процесса аналитики Кортаны (CAP).

[AZURE.INCLUDE [cap-sample-data-selector](../../includes/cap-explore-data-selector.md)]

## Введение

Этот документ описывает выборку данных, содержащихся в хранилище больших двоичных объектов Azure, путем их программной загрузки и последующей выборки с помощью примера кода на языке Python. Для этого необходимо выполнить следующие шаги:

В этом документе описано, как осуществлять выборку данных в трех расположениях, которые обычно применяются в процессе аналитики Кортаны.

- Выборка **данных контейнера больших двоичных объектов Azure** осуществляется путем их программного скачивания и последующей выборки с помощью примера кода на языке Python.
- Выборка **данных SQL Server** осуществляется с помощью как SQL, так и языка программирования Python. 
- Выборка **данных таблицы Hive** осуществляется с помощью запросов Hive.
