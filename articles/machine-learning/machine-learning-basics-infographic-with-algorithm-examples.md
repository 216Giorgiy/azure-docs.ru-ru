# Загружаемая инфографика по основам машинного обучения с примерами алгоритмов

Скачайте доходчивую инфографику по основам машинного обучения и ознакомьтесь с популярными алгоритмами, которые используются для ответа на распространенные вопросы о машинном обучении. Ссылки на примеры алгоритмов помогут начинающему пользователю машинного обучения понять, какие алгоритмы использовать и для чего они применяются.

## Популярные алгоритмы, доступные в Студии машинного обучения

[Студия машинного обучения Azure](https://studio.azureml.net/) поставляется с большим числом алгоритмов машинного обучения, которые можно использовать для решения проблем прогнозной аналитики. Указанная ниже загружаемая инфографика демонстрирует, как использовать четыре типа алгоритмов машинного обучения — **регрессию**, **обнаружение аномалий**, **кластеризацию** и **классификацию** — для ответа на вопросы о машинном обучении.

[Коллекция Cortana Intelligence](https://gallery.cortanaintelligence.com/) содержит множество примеров, показывающих, как применять эти алгоритмы на практике. Ссылки в этой инфографике ведут на рабочие примеры использования каждого алгоритма.
