# Загружаемая инфографика по основам машинного обучения с примерами алгоритмов
Скачайте понятную обзорную инфографику с основами машинного обучения. Узнайте о часто используемых алгоритмах и получите ответы на распространенные вопросы по машинному обучению. Примеры алгоритмов помогут начинающему пользователю машинного обучения понять, какие алгоритмы используются и с какой целью.

## Популярные алгоритмы, доступные в Студии машинного обучения
[Студия машинного обучения Azure](https://studio.azureml.net/) включает большую библиотеку алгоритмов для прогнозной аналитики. На инфографике показаны популярные семейства алгоритмов: **регрессия**, **обнаружение аномалий**, **кластеризация** и **классификация**.

[Коллекция Cortana Intelligence](https://gallery.cortanaintelligence.com/) демонстрирует, как применять эти алгоритмы на практике. Чтобы ознакомиться с рабочими примерами, переходите по ссылкам в инфографике.

* Полный список всех алгоритмов и модулей машинного обучения, доступных в Студии машинного обучения, см. в разделе [Полный список модулей Студии машинного обучения][a-z-list] в справке по алгоритмам и модулям Студии машинного обучения.
