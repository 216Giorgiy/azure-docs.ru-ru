# <a name="get-support-and-training-for-azure-machine-learning-services"></a>Службы машинного обучения Azure: поддержка и обучение

В этой статье объясняется, как получить дополнительные сведения о службах машинного обучения Azure и справку при возникновении проблем и вопросов. 

## <a name="learn-more-about-azure-machine-learning"></a>Дополнительные сведения о службе "Машинное обучение Azure"

В [центре документации по Машинному обучению Azure](../desktop-workbench/overview-what-is-azure-ml.md) доступны обучающие видео и пошаговые руководства. В этих пошаговых руководствах представлены общие сведения о службах и описание жизненного цикла обработки и анализа данных. Мы регулярно добавляем новые материалы в учебный центр машинного обучения. 

Кроме того, есть [видео, которые можно просмотреть, чтобы получить дополнительные сведения](https://azure.microsoft.com/resources/videos/index/?services=machine-learning).

Вы можете **отправлять запросы** на получение дополнительных учебных материалов, используя ссылку отзыва в нижней части статьи.

## <a name="get-support-for-azure-machine-learning-services"></a>Получение поддержки для служб машинного обучения Azure

Поддержку для служб машинного обучения Azure можно получить следующими способами:

+ **Техническая поддержка.** Чтобы обратиться в службу технической поддержки, откройте страницу [поддержки Azure](https://azure.microsoft.com/support/options/) и выберите машинное обучение. 

+ **Форум пользователей MSDN.** Задавайте вопросы, отвечайте на вопросы и обращайтесь к другим пользователям на [форуме службы поддержки служб машинного обучения Azure на MSDN](https://social.msdn.microsoft.com/Forums/windowsdesktop/home?forum=MachineLearning).

+ **Stack Overflow.** Посетите на сайте [StackOverflow](https://stackoverflow.com/questions/tagged/azure-machine-learning) сообщество службы "Машинное обучение Azure", отмеченное тегом Azure-Machine-Learning.

+ **Делитесь предложениями по продукту** и запросами на новые компоненты в нашем [канале обратной связи службы "Машинное обучение Azure](https://feedback.azure.com/forums/257792-machine-learning). Канал можно найти по ссылке в нижней части каждой статьи.
