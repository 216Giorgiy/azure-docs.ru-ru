Вы можете **отправлять запросы** на получение дополнительных учебных материалов, используя кнопку **Содержимое отзыва** в конце каждой статьи.

## <a name="get-service-support"></a>Получение поддержки службы 

Ознакомьтесь со следующими ресурсами поддержки:

+ **Техническая поддержка для клиентов Azure**. [Создание запроса на поддержку Azure](https://docs.microsoft.com/azure/azure-supportability/how-to-create-azure-support-request).

+ **Форум пользователей**. Задавайте свои вопросы, отвечайте на вопросы других пользователей и общайтесь на [форуме поддержки Службы машинного обучения Azure на MSDN](https://aka.ms/aml-forum-service).

+ **Stack Overflow**. Посетите сообщество Службы машинного обучения Azure на сайте [StackOverflow](https://stackoverflow.com/questions/tagged/azure-machine-learning), отмеченное тегом Azure-Machine-Learning.

+ **Поделитесь предложениями о продуктах** и запросами на функции на нашем [канале обратной связи службы "Машинное обучение Azure"](https://feedback.azure.com/forums/257792-machine-learning). Чтобы поделиться своим отзывом, выберите **Отзыв о продукте** в конце каждой статьи.
