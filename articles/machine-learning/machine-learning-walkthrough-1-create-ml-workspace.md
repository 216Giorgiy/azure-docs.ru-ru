Для использования студии машинного обучения требуется рабочая область машинного обучения Microsoft Azure. Такая рабочая область содержит инструменты, необходимые для создания, публикации экспериментов и управления ими.

##Создание рабочей области  

1.	Войдите в систему с помощью учетной записи Microsoft Azure.
2.	На панели служб Microsoft Azure выберите **МАШИННОЕ ОБУЧЕНИЕ**. ![Создание рабочей области][1]

3.	Щелкните **СОЗДАТЬ РАБОЧУЮ ОБЛАСТЬ ML**.
4.	На странице **БЫСТРОЕ СОЗДАНИЕ** введите информацию о рабочей области и выберите **СОЗДАТЬ РАБОЧУЮ ОБЛАСТЬ ML**.

	> [AZURE.NOTE]В поле **ВЛАДЕЛЕЦ РАБОЧЕЙ ОБЛАСТИ** введите свою учетную запись Майкрософт (например, name@outlook.com) или учетную запись в организации.

После создания рабочей области машинного обучения она появится в списке на странице **Машинное обучение**.

> [AZURE.TIP]Чтобы сделать общими эксперименты, над которыми вы работаете, пригласите других пользователей в свою рабочую область. Это можно сделать в Студии машинного обучения на странице **ПАРАМЕТРЫ**. Для каждого пользователя необходима учетная запись Майкрософт или учетная запись организации.

Подробнее: [Управление рабочей областью машинного обучения Azure](machine-learning-manage-workspace.md).
