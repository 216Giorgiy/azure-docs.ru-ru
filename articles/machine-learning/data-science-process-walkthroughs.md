# Пошаговые руководства по процессу обработки и анализа данных

Перечисленные здесь пошаговые руководства содержат указания по процессу обработки и анализа данных группы для **конкретных сценариев** и показывают, как объединять облачные и локальные инструменты и службы в рабочий процесс или конвейер, чтобы создать интеллектуальное приложение.

## Использование SQL Server
В пошаговом руководстве[Процесс обработки и анализа данных группы на практике: использование SQL Server](machine-learning-data-science-process-sql-walkthrough.md) показано, как создать и развернуть модель машинного обучения с помощью SQL Server и общедоступного набора данных [NYC Taxi Trips](http://www.andresmh.com/nyctaxitrips/) (Поездки такси Нью-Йорка).


## Использование кластеров HDInsight Hadoop
В пошаговом руководстве [Процесс обработки и анализа данных группы на практике: использование кластеров HDInsight Hadoop](machine-learning-data-science-process-hive-walkthrough.md) мы используем [кластер Azure HDInsight Hadoop](https://azure.microsoft.com/services/hdinsight/) для хранения, изучения и определения характеристик данных из общедоступного набора данных [NYC Taxi Trips](http://www.andresmh.com/nyctaxitrips/) (Поездки такси Нью-Йорка).


## Процесс обработки и анализа данных группы на практике: использование кластеров Azure HDInsight Hadoop с набором данных объемом 1 ТБ
В пошаговом руководстве [Процесс обработки и анализа данных группы на практике: использование кластеров Azure HDInsight Hadoop с набором данных объемом 1 ТБ](machine-learning-data-science-process-hive-criteo-walkthrough.md) мы представляем комплексный сценарий, в котором [кластер Azure HDInsight Hadoop](https://azure.microsoft.com/services/hdinsight/) используется для хранения, изучения, реконструирования характеристик и сокращения выборки данных из одного из общедоступных наборов данных [Criteo](http://labs.criteo.com/downloads/download-terabyte-click-logs/).

## Обработка и анализ данных с помощью Spark в Azure HDInsight
В пошаговом руководстве [Обработка и анализ данных с помощью Spark в Azure HDInsight](machine-learning-data-science-spark-overview.md) показано комплексное использование процесса обработки и анализа данных группы в сценарии, где [кластер Azure HDInsight Hadoop](https://azure.microsoft.com/services/hdinsight/) используется для хранения, изучения, реконструирования характеристик и сокращения выборки данных из одного из общедоступных наборов данных [NYC Taxi Trips](http://www.andresmh.com/nyctaxitrips/) (Поездки такси Нью-Йорка).


## Масштабируемая обработка и анализ данных в озере данных Azure
В пошаговом руководстве [Масштабируемая обработка и анализ данных в озере данных Azure](machine-learning-data-science-process-data-lake-walkthrough.md) на примере набора данных о поездках и тарифах такси в Нью-Йорке показано, как использовать озеро данных Azure для выполнения задач по исследованию и двоичной классификации данных, чтобы спрогнозировать вероятность получения чаевых за поездку.

## Комплексное пошаговое руководство по использованию служб SQL Server R для обработки и анализа данных
В [комплексном пошаговом руководстве по использованию служб SQL Server R для обработки и анализа данных](https://msdn.microsoft.com/library/mt612857.aspx) используется известный общедоступный набор данных "NYC Taxi Trips" (Поездки такси Нью-Йорка). Сочетание кода R, данных SQL Server и пользовательских функций SQL используется для создания модели классификации, которая показывает вероятность получения водителем чаевых за поездку. Кроме того, модель R развертывается в SQL Server и использует данные сервера для формирования оценок на основе модели.

## Внутренняя расширенная аналитика базы данных для разработчиков для SQL

Цель пошагового руководства [Внутренняя расширенная аналитика базы данных для разработчиков для SQL](https://msdn.microsoft.com/library/mt683480.aspx) — предоставить программистам на SQL практический опыт создания решений расширенной аналитики с помощью служб R SQL Server. Вы узнаете, как внедрить R в приложение или решение для бизнес-аналитики, поместив код R в хранимые процедуры.
