# Приложения машинного обучения в Azure Marketplace

С помощью средств Машинного обучения Azure специалисты по обработке и анализу данных могут монетизировать свои технологии и творческие способности, как это делают разработчики приложений для смартфонов. Публикуя свои службы машинного обучения в каталоге Azure Marketplace, вы позволяете людям со всего мира за определенную плату подписаться на опубликованные вами облачные API-интерфейсы и использовать их в своих приложениях.

[AZURE.INCLUDE [machine-learning-free-trial](../../includes/machine-learning-free-trial.md)]

Однако каталог Azure Marketplace может предложить намного больше, чем простейшие аналитические API-интерфейсы. Например, мы создали несколько завершенных комплексных приложений машинного обучения, которые решают конкретные производственные задачи. Чтобы использовать эти приложения машинного обучения, не нужно быть специалистом по обработке данных или экспертом в машинном обучении. Наука уже встроена в наши решения. Пользователи могут просто подставить свои данные и начать использовать приложения.

Примеры таких приложений.

##Рекомендации##
 Самостоятельный и очень мощный обработчик рекомендаций, который позволяет легко добавлять персонализированные рекомендации продуктов для вашего веб-сайта или рассылки электронных сообщений.

[Ссылка на магазин данных](http://datamarket.azure.com/dataset/amla/recommendations)

[Краткое руководство](machine-learning-recommendation-api-quick-start-guide.md)
  

##Обнаружение аномалий##
API, который может обнаруживает аномальные события в данных временных рядов (например, телеметрия сервера или данные, использующие датчики).

[Ссылка на магазин данных](https://datamarket.azure.com/dataset/aml_labs/anomalydetection)

[Документация](machine-learning-apps-anomaly-detection.md)

Эти API позволяют вам использовать мощные службы на основе машинного обучения, построенные с помощью машинного обучения Azure. Для этого вам не нужно быть специалистом в обработке данных или тратить время на самостоятельную разработку приложений. Просто внесите свои данные и используйте службы.

##Текстовая аналитика##
Этот API можно использовать, чтобы анализировать неструктурированный текст для таких задач, как анализ настроений и извлечение ключевых фраз. Для использования этого API данные для обучения не нужны, поэтому можно просто использовать свои текстовые данные. Этот программный интерфейс использует передовые методы глубокого анализа естественного языка.

[Ссылка на магазин данных](https://datamarket.azure.com/dataset/aml_labs/anomalydetection)

[Документация](machine-learning-apps-text-analytics.md)
