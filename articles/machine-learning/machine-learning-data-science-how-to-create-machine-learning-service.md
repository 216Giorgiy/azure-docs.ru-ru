# Процесс обработки и анализа данных в облаке для машинного обучения

В этом руководстве по процессу обработки и анализа данных в облаке показано, как использовать платформу Azure для обработки и анализа данных, а также создания рабочих моделей с помощью Машинного обучения Azure. На схеме показаны типичные задачи обработки и анализа данных, такие как прием данных из различных источников, подготовка этих данных к использованию в Машинном обучении Azure и применение Машинного обучения Azure для создания рабочих моделей, которые могут использовать приложения конечных пользователей (программно или другим способом). Так как схема включает в себя базовые шаги типичной комплексной задачи обработки и анализа данных, не все шаги являются обязательными, ровно как и последовательность их выполнения, которая может меняться в зависимости от расположения и размера данных, а также сложности их структуры.

Шаг, который называется **Расширенная обработка данных в Azure**, связан со второй схемой. Если вы вернетесь к выполнению первой схемы, продолжайте работу с этого шага, чтобы принять обработанные данные в Студии машинного обучения Azure, а затем создать и развернуть модели.
