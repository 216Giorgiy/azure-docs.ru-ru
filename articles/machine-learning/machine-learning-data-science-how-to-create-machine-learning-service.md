# Разработка расширенных аналитических решений в Azure

В руководстве по обработке аналитических данных и технологии расширенного анализа (ADAPT) рассматриваются задачи обработки и анализа данных, которые обычно используются для создания и развертывания прогнозных моделей в службе машинного обучения Azure. В нем демонстрируется выполнение на платформе Azure таких разнообразных задач, как прием данных из различных источников, их подготовка к использованию в машинном обучении Azure и создание на базе эксперимента машинного обучения рабочих моделей, которые могут использовать приложения конечных пользователей (программно или другим способом). Схема включает базовую последовательность действий типичной комплексной задачи обработки и анализа данных, однако не все эти шаги являются обязательными, а их последовательность может зависеть от расположения и размера данных, а также сложности их структуры.

[AZURE.INCLUDE [machine-learning-free-trial](../../includes/machine-learning-free-trial.md)]

Этап **Расширенная обработка данных в Azure** связан со вторым руководством, в котором описан пошаговый процесс подготовки данных, используемых в эксперименте машинного обучения Azure. Если вы уже ознакомились с ним, переходите к следующему шагу в этом руководстве, в котором подготовленные данные направляются в систему машинного обучения Azure, а затем — к последнему этапу, связанному с использованием модели.
