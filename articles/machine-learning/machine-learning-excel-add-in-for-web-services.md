# Надстройка Excel для веб-служб машинного обучения Azure

Excel упрощает непосредственный вызов веб-служб без необходимости написания какого-либо кода.

## Действия для использования существующей веб-службы в книге

1. Откройте [пример файла Excel](http://aka.ms/amlexcel-sample-2), который содержит надстройки Excel и данные о пассажирах "Титаника".
2. Выберите веб-службу, щелкнув по ней. В данном примере это "Прогноз выживших на "Титанике" (пример надстройки Excel)".

    ![Выбор веб-службы][01]

3. Вы перейдете в раздел **Прогноз**. Эта книга уже содержит образцы данных. В пустой книге можно выбрать ячейку в Excel и щелкнуть **Использовать образцы данных**.
4. Выделите данные с заголовками и щелкните значок диапазона входных данных. Убедитесь, что установлен флажок "Мои данные содержат заголовки".
5. В разделе **Вывод** введите номер ячейки, в которую следует поместить выходные данные. В данном случае это H1.
6. Нажмите **Выполнить прогноз**.

	![Раздел «Прогноз»][02]

## Действия для добавления новой веб-службы

1. Опубликуйте веб-службу (как это сделать, описано на [этой странице](machine-learning-walkthrough-5-publish-web-service.md)) или воспользуйтесь существующей веб-службой.
2. В Excel перейдите в раздел **Веб-службы** (если вы находитесь в разделе **Прогноз**, щелкните стрелку "Назад", чтобы перейти к списку веб-служб).

	![Переход к выбору веб-службы][03]

3. Щелкните **Добавить веб-службу**.
4. Щелкните **ВЕБ-СЛУЖБЫ** на левой панели Студии машинного обучения, а затем выберите веб-службу.

	![Выбор веб-службы в Студии машинного обучения][04]

5. Скопируйте ключ API для веб-службы.

	![Ключ API в Студии машинного обучения][05]

6. Вставьте ключ API в текстовое поле с меткой **Ключ API** в надстройке Excel.
7. На вкладке **ПАНЕЛЬ МОНИТОРИНГА** веб-службы щелкните ссылку **ЗАПРОС/ОТВЕТ**.
8. Найдите раздел **Адрес конечной точки OData**. Скопируйте URL-адрес запроса и вставьте его в текстовое поле с меткой **URL-адрес**.
9. Щелкните **Добавить**.

	![URL-адрес и ключ API][06]

10.	Чтобы использовать веб-службу, следуйте инструкциям, указанным выше в разделе «Действия для использования существующей веб-службы».

## Предоставление общего доступа к книге

Во время сохранения книги также сохраняются ключи API для веб-служб, которые вы добавили. Это означает, что доступ к книге следует предоставлять только тем лицам, которым вы доверяете.
