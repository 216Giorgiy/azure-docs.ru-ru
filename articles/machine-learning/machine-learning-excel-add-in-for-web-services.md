# Надстройка Excel для веб-служб машинного обучения Azure

Excel упрощает непосредственный вызов веб-служб без необходимости написания какого-либо кода.

## Действия для использования существующей веб-службы

1. Откройте образец файла Excel здесь или на вкладке **ПАНЕЛЬ МОНИТОРИНГА** веб-служб.
2. Выберите веб-службу, щелкнув ее. В данном примере это «Прогноз выживших на "Титанике" (пример надстройки Excel)».

    ![Выбор веб-службы][01]

3. Вы перейдете в раздел **Прогноз**. Выберите ячейку в Excel и щелкните **Использовать образцы данных**. При необходимости отредактируйте данные в Excel.

	![Раздел «Прогноз»][02]

4. Выделите данные и щелкните значок диапазона входных данных.
5. В разделе **Вывод** введите номер ячейки, в которую следует поместить выходные данные.
6. Нажмите **Прогноз**.

## Действия для добавления новой веб-службы

1. Опубликуйте веб-службу (на [этой странице](machine-learning-walkthrough-5-publish-web-service.md) описано, как это сделать) или найдите существующую веб-службу.
2. Щелкните раздел **ВЕБ-СЛУЖБЫ** в левой части Студии машинного обучения, а затем выберите веб-службу, которую нужно использовать.
3. На вкладке **ПАНЕЛЬ МОНИТОРИНГА** веб-службы скопируйте ключ API для этой службы.
4. В Excel перейдите в раздел **Веб-службы** (если вы находитесь в разделе **Прогноз**, щелкните стрелку «Назад», чтобы перейти к списку веб-служб).

	![Раздел «Веб-службы»][03]

5. Щелкните **Добавить веб-службу**.
6. Вставьте ключ в текстовое поле с меткой **Ключ API**.
7. На вкладке **ПАНЕЛЬ МОНИТОРИНГА** веб-службы щелкните ссылку **Страница справки API**.
8. Скопируйте URI запроса из поля **Запрос** и вставьте его в текстовое поле с меткой **URL-адрес**.
9. Щелкните **Добавить**.

	![URL-адрес и ключ API][04]

10.	Чтобы использовать веб-службу, следуйте инструкциям, указанным выше в разделе «Действия для использования существующей веб-службы».

## Предоставление общего доступа к книге

Во время сохранения книги также сохраняются ключи API для веб-служб, которые вы добавили. Это означает, что доступ к книге следует предоставлять только тем лицам, которым вы доверяете.
