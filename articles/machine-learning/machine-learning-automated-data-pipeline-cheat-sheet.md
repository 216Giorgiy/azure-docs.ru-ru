# Памятка по работе с автоматическим конвейером данных для создания прогнозов машинного обучения Azure

**Памятка по работе с автоматическим конвейером данных машинного обучения Microsoft Azure** поможет вам разобраться в технологии, которую вы можете использовать для передачи данных в веб-службу машинного обучения и их последующей оценки с помощью модели прогнозной аналитики.

Выбор способа перемещения данных в конечную точку веб-службы для последующей оценки зависит от того, хранятся ли ваши данные на локальном компьютере, находятся в облаке или транслируются в режиме реального времени. Это памятка рассказывает, какие решения необходимо принять, и содержит ссылки на статьи, которые помогут вам в разработке решения.
