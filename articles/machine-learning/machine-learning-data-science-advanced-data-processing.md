# Руководство по обучению. Расширенная обработка данных в Azure 

На этой схеме представлен пошаговый процесс подготовки данных, используемых в эксперименте Машинного обучения Azure. Она показывает, как выполнить перемещение, обработку и выборку данных для подготовки к машинному обучению. Представленный здесь процесс подготовки данных является одним из этапов, описанных в статье [Процесс обработки и анализа данных в облаке для машинного обучения](machine-learning-data-science-how-to-create-machine-learning-service.md). В руководстве по обработке аналитических данных и технологии расширенного анализа (ADAPT) рассматриваются задачи обработка и анализ данных, которые обычно используются для создания и развертывания прогнозных моделей в службе машинного обучения Azure.
