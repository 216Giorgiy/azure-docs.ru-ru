# <a name="azure-machine-learning-studio-documentation"></a>Документация по студии машинного обучения Azure

Студия машинного обучения Azure — это инструмент для совместной работы, поддерживающий функцию перетаскивания объектов и предназначенный для создания, тестирования и развертывания решений для прогнозного анализа данных.
В руководствах, видеороликах и примерах моделей объясняется, как с помощью студии создавать и развертывать модели машинного обучения.
