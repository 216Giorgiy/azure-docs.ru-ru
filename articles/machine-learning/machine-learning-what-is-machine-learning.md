# Что такое машинное обучение?
Машинное обучение используется повсюду. При совершении покупки через Интернет машинное обучение помогает рекомендовать другие продукты на основании уже приобретенных вами товаров. При проведении кредитной карты через терминал машинное обучение позволяет обнаружить мошенничество и уведомит вас о подозрительных транзакциях. Машинное обучение — это процесс создания моделей, которые могут обучаться на существующих данных и прогностически анализировать данные, которые поступят в будущем.

## Что такое машинное обучение Azure?
Машинное обучение Azure представляет собой мощную облачную службу прогностической аналитики, при помощи которой можно без лишних временных затрат создавать аналитические решения. Мы создали машинное обучение Azure, чтобы проявить более демократичный подход к этому подходу, исключив наиболее сложные аспекты, связанные с созданием и развертыванием технологии машинного обучения, чтобы сделать ее доступной для всех. Она представляет собой полностью управляемую службу, а значит, не потребуется приобретать аппаратное обеспечение и вручную управлять виртуальными машинами.

Можно использовать браузерный инструмент, который называется [Студия машинного обучения](machine-learning-what-is-ml-studio.md) для быстрого создания и автоматизации рабочих процессов машинного обучения. Используя перетаскивание, вы можете поместить сотни существующих [библиотек машинного обучения](https://msdn.microsoft.com/library/azure/f5c746fd-dcea-4929-ba50-2a79c4c067d7), которые помогут вам быстро запустить собственные решения для прогностического анализа, после чего добавить собственные скрипты на языках [R](machine-learning-r-quickstart.md) и [Python](machine-learning-execute-python-scripts.md), чтобы расширить их. Студию можно использовать в любом браузере. Она позволяет без лишних временных затрат разрабатывать и повторно использовать решения. ![Эксперименты прогностической аналитики в облаке при помощи студии машинного обучения Azure](./media/machine-learning-what-is-machine-learning/AzureMLStudio.png)

Вы можете без труда обнаруживать и создавать [веб-службы](machine-learning-publish-a-machine-learning-web-service.md), [обучать и переобучать свои модели через интерфейсы API](machine-learning-retrain-models-programmatically.md), [управлять конечными точками](machine-learning-create-endpoint.md) и [масштабировать веб-службы](machine-learning-scaling-endpoints.md) в зависимости от клиентов, а также настраивать диагностику для отслеживания и отладки служб. Новейшие функции включают:

- Возможность создания пользовательских модулей при помощи языка R, использования собственных скриптов обучения и прогнозирования на языке R и добавления скриптов Python с использованием большой экосистемы библиотек, таких как numpy, scipy, pandas или scikit-learn. Теперь для обучения можно использовать терабайты данных при помощи [обучения со счетчиком][learning-with-counts], использовать PCA или одноклассовый SVM для обнаружения аномалий и без труда изменять, фильтровать и очищать данные при помощи знакомого SQLite. 
- [Коллекция сообщества машинного обучения](machine-learning-gallery-how-to-use-contribute-publish.md) позволяет обнаруживать и использовать интересные эксперименты, авторство которых принадлежит другим участникам. Теперь можно задавать вопросы или размещать комментарии об экспериментах в [коллекции](http://gallery.azureml.net) или опубликовывать свои собственные. Вы можете делиться ссылками на интересные эксперименты через социальные каналы, такие как LinkedIn и Twitter. Коллекция является привлекательным способом освоения машинного обучения Azure и обучения у других членов сообщества. ![Попробуйте образцы прогностических экспериментов или представьте свои в коллекции машинного обучения Azure](./media/machine-learning-what-is-machine-learning/AzureMLGallery.png)
- Вы можете приобретать [приложения Marketplace](https://datamarket.azure.com/browse?query=machine+learning) через свою подписку Azure и использовать законченные веб-службы для рекомендаций, анализа текста и обнаружения аномалий сразу после приобретения необходимых компонентов в Azure Marketplace. 
- Пошаговое руководство в мире науки о данных предоставляет инструкции, от необработанных данных до создания готовой веб-службы, облегчая таким образом путь к облачным методам использования данных. Мы добавили в машинное обучение Azure возможность использования популярных инструментов, таких как iPython Notebook и Python Tools для Visual Studio.

## Начните работу уже сейчас
Вы можете изучить базовые принципы прогностической аналитики и машинного обучения, используя [пошаговый учебник](machine-learning-create-experiment.md) и [создание сборок из образцов](machine-learning-sample-experiments.md). Чтобы начать работать с машинным обучением, подписка Azure или кредитная карта не требуется при условии, что используется студия для работы с экспериментами.
