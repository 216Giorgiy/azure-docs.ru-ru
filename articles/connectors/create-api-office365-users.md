# Начало работы с API-интерфейсом Office 365 Пользователи

Поставщик подключения Office 365 Пользователи позволяет получать доступ к профилям пользователей в вашей организации с помощью учетной записи Office 365. Вы можете выполнять различные действия, такие как получение своего профиля и профиля пользователя, получение руководителя пользователя и непосредственных подчиненных, обновление профиля пользователя.

>[AZURE.NOTE] Эта версия статьи предназначена для приложений логики со схемой версии 2015-08-01-preview. Чтобы получить версию схемы 2014-12-01-preview, щелкните [API Office 365](../app-service-logic/app-service-logic-connector-office365.md).


С помощью Office 365 Пользователи вы можете:

* создавать приложения логики;
* создавать высокопроизводительные приложения.

Сведения о добавлении API в PowerApps Enterprise см. в статье [Регистрация API в PowerApps](../power-apps/powerapps-register-from-available-apis.md).

Сведения о добавлении операции в приложения логики см. в статье [Создание приложения логики](../app-service-logic/app-service-logic-create-a-logic-app.md).

## Поговорим о триггерах и действиях

API Office 365 Пользователи можно использовать в качестве действия. Триггеры отсутствуют. Все API-интерфейсы поддерживают данные в форматах JSON и XML.

 API Office 365 Пользователи предоставляет следующие триггеры и действия.

### Действия Office 365 Пользователи
Вы можете выполнять перечисленные ниже действия:

|Действие|Описание|
|--- | ---|
|MyProfile|Извлекает профиль для текущего пользователя|
|Профиль пользователя|Извлекает профиль конкретного пользователя|
|Manager|Извлекает профиль руководителя указанного пользователя|
|DirectReports|Получение непосредственных подчиненных|
|SearchUser|Извлекает результаты поиска профилей пользователей|
## Создание подключения к Office 365 Пользователи
Чтобы использовать API Office 365 Пользователи, сначала нужно создать **подключение**, а затем указать сведения для приведенных ниже свойств.

|Свойство| Обязательно|Описание|
| ---|---|---|
|токен|Да|Указание учетных данных Office 365|


>[AZURE.TIP] Это подключение можно использовать в других приложениях логики.
