# <a name="get-started-with-the-excel-connector"></a>Начало работы с соединителем Excel
В Logic Apps нет соединителя Excel. 

## <a name="to-use-excel-data"></a>Использование данных Excel
Вы можете сохранить данные Excel как файл с разделителями-запятыми (CSV) в папку хранилища, например [OneDrive](connectors-create-api-onedrive.md). Этот CSV-файл можно использовать с [соединителем неструктурированного файла](../logic-apps/logic-apps-enterprise-integration-flatfile.md).

## <a name="connector-specific-details"></a>Сведения о соединителях

Информацию о существующих ограничениях, а также о триггерах и действиях, определенных в Swagger, см. в статье со [сведениями о соединителях](/connectors/excel/).

## <a name="more-connectors"></a>Дополнительные сведения о соединителях
Вы можете вернуться к [списку интерфейсов API](apis-list.md).
