# Приступая к работе с API FTP

Подключение к Excel для вставки строки, удаления строки и многого другого. API Excel можно использовать из:

- PowerApps.

С помощью Excel можно:

- Добавить API Excel в PowerApps Enterprise. После этого пользователи смогут использовать этот API в своих приложениях. 

Сведения о добавлении API в PowerApps Enterprise см. в статье [Регистрация API в PowerApps](../power-apps/powerapps-register-from-available-apis.md).

## Триггеры и действия
Excel содержит следующее действие. Триггеры отсутствуют.

|Триггер|Действия|
|--- | ---|
|None | <ul><li>Получение строк</li><li>Вставка строки</li><li>Удаление строки</li><li>Получение строки</li><li>Получение таблиц</li><li>Обновление строки</li></ul>

Все API поддерживают данные в форматах JSON и XML.

## Справочник по REST API Swagger
Относится к версии 1.0.

### Вставка новой строки в таблицу Excel
```POST: /datasets/{dataset}/tables/{table}/items```



| Имя| Тип данных|Обязательно|Местонахождение|Значение по умолчанию|Описание|
| ---|---|---|---|---|---|
|dataset|string|Да|path|Нет|Имя файла Excel|
|таблица|строка|Да|path|Нет|Имя таблицы Excel|
|item| |Да|текст|Нет|Строка, вставляемая в указанную таблицу Excel|


### Ответ

|Имя|Описание|
|---|---|
|200|ОК|
|по умолчанию|Операция завершилась ошибкой.|
