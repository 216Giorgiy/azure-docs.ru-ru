# Начало работы с API-интерфейсом SQL Azure
Подключитесь к SQL Azure, чтобы управлять таблицами и строками, например: вставлять строки, получать таблицы и выполнять многие другие действия.

API SQL Azure можно использовать из:

- приложений логики. 

>[AZURE.NOTE] Эта версия статьи предназначена для приложений логики со схемой версии 2015-08-01-preview. Чтобы получить версию схемы 2014-12-01-preview, щелкните [соединитель SQL](../app-service-logic/app-service-logic-connector-sql.md).

С помощью SQL Azure можно:

- формировать бизнес-процессы на основе данных, получаемых из SQL Azure; 
- использовать действия для получения строки, вставки строки и т. д. Эти действия получают ответ и делают выходные данные доступными для использования другими действиями. Например, можно получить строку данных из SQL Azure и затем добавить эти данные в Excel. 

Сведения о добавлении операции в приложения логики см. в статье [Создание приложения логики](../app-service-logic/app-service-logic-create-a-logic-app.md).


## Триггеры и действия
SQL позволяет выполнять следующие действия. Триггеры отсутствуют.

триггеры; | Действия
--- | ---
None | <ul><li>Получение строки</li><li>Получение строк</li><li>Вставка строки</li><li>Удаление строки</li><li>Получение таблиц</li><li>Обновление строки</li></ul>

Все API-интерфейсы поддерживают данные в форматах JSON и XML.

## Создание подключения к SQL
При добавлении этого API в приложения логики введите указанные ниже значения:

|Свойство| Обязательно|Описание|
| ---|---|---|
|Строка подключения SQL|Да|Введите строку подключения SQL Azure|

После создания подключения укажите свойства SQL, например имя таблицы. Эти свойства описаны далее в **справочнике по REST API**.

>[AZURE.TIP] Это подключение можно использовать в других приложениях логики.

## Справочник по REST API Swagger
Относится к версии 1.0.

### Получение строки 
Извлекает одну строку из таблицы SQL. ```GET: /datasets/default/tables/{table}/items/{id}```

| Имя| Тип данных|Обязательно|Местонахождение|Значение по умолчанию|Описание|
| ---|---|---|---|---|---|
|таблица|строка|Да|path|Нет|Имя таблицы SQL|
|id|строка|Да|path|Нет|Уникальный идентификатор извлекаемой строки|

#### Ответ
|Имя|Описание|
|---|---|
|200|ОК|
|по умолчанию|Операция завершилась ошибкой.|


### Получение строк 
Извлекает строки из таблицы SQL. ```GET: /datasets/default/tables/{table}/items```

| Имя| Тип данных|Обязательно|Местонахождение|Значение по умолчанию|Описание|
| ---|---|---|---|---|---|
|таблица|строка|Да|path|Нет|Имя таблицы SQL|
|$skip|целое число|Нет|запрос|Нет|Количество пропускаемых записей (значение по умолчанию — 0)|
|$top|целое число|Нет|запрос|Нет|Максимальное количество извлекаемых записей (значение по умолчанию — 256)|
|$filter|string|Нет|запрос|Нет|Запрос фильтра ODATA для ограничения количества записей|
|$orderby|строка|Нет|запрос|Нет|Запрос orderBy ODATA для указания порядка записей|

#### Ответ
|Имя|Описание|
|---|---|
|200|ОК|
|по умолчанию|Операция завершилась ошибкой.|

#### Ответ

|---|---|
