# Начало работы с API-интерфейсом Office 365 Видео

API-интерфейс Office 365 Видео предназначен для работы с каналами и видеофайлами Office 365.

>[AZURE.NOTE] Эта версия статьи предназначена для приложений логики со схемой версии 2015-08-01-preview. Для схемы версии 2014-12-01-preview щелкните [соединитель Office 365 Видео] (../app-service-logic/app-service-logic-connector-Office365 Video Connector.md).

С помощью Office 365 Видео вы можете:

* создавать приложения логики.

Сведения о добавлении операции в приложения логики см. в статье [Создание приложения логики](../app-service-logic/app-service-logic-create-a-logic-app.md).

## Поговорим о триггерах и действиях

API Office 365 Видео можно использовать в качестве действия. Триггеры отсутствуют. Все API-интерфейсы поддерживают данные в форматах JSON и XML.

 API Office 365 Видео предоставляет следующие триггеры и действия.

### Действия API Office 365 Видео
Вы можете выполнять перечисленные ниже действия:

|Действие|Описание|
|--- | ---|
|IsVideoPortalEnabled|Проверяет состояние видеопортала и определяет, включены ли видеослужбы|
|ListViewableChannels|Получает все каналы, права на просмотр которых имеет пользователь|
|ListVideos|Перечисляет все видео Office 365, имеющиеся в канале|
|GetVideo|Получает сведения о конкретном видео Office 365|
|GetPlaybackUrl|Получает URL-адрес воспроизведения манифеста служб мультимедиа Azure для видео|
|GetStreamingKeyAccessToken|Получает токен носителя для получения прав расшифровки видео|
