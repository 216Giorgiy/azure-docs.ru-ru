# Создание рабочей области машинного обучения Azure 
Рабочая область машинного обучения Azure (рабочая область ML) содержит средства, которые требуются для управления и публикации экспериментов. Дополнительные сведения о рабочей области машинного обучения см. по ссылке.
## Создание рабочей области ##
1. Войдите в систему с использованием учетной записи Azure.
2. На панели служб Microsoft Azure выберите **МАШИННОЕ ОБУЧЕНИЕ**.

![][1]
 

1. Щелкните **СОЗДАТЬ РАБОЧУЮ ОБЛАСТЬ ML**.
2. На странице **БЫСТРО СОЗДАТЬ** выполните указанные ниже действия.
	1. Введите информацию о рабочей области.    
	ПРИМЕЧАНИЕ. В поле **ВЛАДЕЛЕЦ РАБОЧЕЙ ОБЛАСТИ** введите свою учетную запись Майкрософт (name@outlook.com) или учетную запись в организации.
	2. Щелкните **СОЗДАТЬ РАБОЧУЮ ОБЛАСТЬ ML**.
3. После создания рабочей области машинного обучения она появится в списке на странице **Машинное обучение**.
