# Создание рабочей области машинного обучения Azure 
Рабочая область Машинного обучения Azure (рабочая область машинного обучения) содержит инструменты, которые требуются для публикации экспериментов и управления ими. Дополнительную информацию о рабочей области машинного обучения см. по ссылке.
## Создание рабочей области ##
1. Войдите в систему с помощью учетной записи Microsoft Azure.
2. На панели служб Microsoft Azure выберите **МАШИННОЕ ОБУЧЕНИЕ**.

![][1]
 

1. Выберите **СОЗДАТЬ РАБОЧУЮ ОБЛАСТЬ МАШИННОГО ОБУЧЕНИЯ**.
2. На странице **БЫСТРОЕ СОЗДАНИЕ**:
	1. Введите информацию о рабочей области.    
	> [AZURE.NOTE] В поле **ВЛАДЕЛЕЦ РАБОЧЕЙ ОБЛАСТИ** введите свою учетную запись Майкрософт (name@outlook.com) или учетную запись в организации.
	2. Выберите **СОЗДАТЬ РАБОЧУЮ ОБЛАСТЬ МАШИННОГО ОБУЧЕНИЯ**.
3. После создания рабочей области машинного обучения она появится в списке на странице **Машинное обучение**.
