# <a name="tutorial-set-up-a-classroom-lab"></a>Руководство. Настройка аудиторной лаборатории 
В этом руководстве вы настроите аудиторную лабораторию с виртуальными машинами, используемыми учащимися в аудитории.  

Вот какие действия выполняются в этом руководстве:

> [!div class="checklist"]
> * Создание аудиторной лаборатории.
> * Настройка аудиторной лаборатории.
> * Отправка ссылки для регистрации учащимся

## <a name="prerequisites"></a>предварительным требованиям
Чтобы настроить лабораторию аудитории в учетной записи лаборатории, вы должны быть членом роли **Создатель лаборатории** в учетной записи лаборатории. Владелец лаборатории может [назначить пользователю роль создателя лаборатории](tutorial-setup-lab-account.md#add-a-user-to-the-lab-creator-role).


## <a name="create-a-classroom-lab"></a>Создание лаборатории для аудитории

1. Перейдите на [веб-сайт Служб лабораторий Azure](https://labs.azure.com).
2. Выберите **Вход** и введите свои учетные данные. 
3. В окне **New Lab** (Создание лаборатории) выполните следующие действия. 
    1. Укажите **имя** аудиторной лаборатории. 
    2. Выберите **размер** виртуальной машины, которую вы планируете использовать в аудитории.
    3. Выберите **образ** для создания виртуальной машины.
    4. Укажите **учетные данные по умолчанию** для входа в виртуальные машины в лаборатории. 
    7. Щелкните **Сохранить**.

        ![Создание лаборатории для аудитории](../media/tutorial-setup-classroom-lab/new-lab-window.png)
1. Откроется **панель мониторинга** лаборатории. 
    
    ![Панель мониторинга аудиторной лаборатории](../media/tutorial-setup-classroom-lab/classroom-lab-home-page.png)

## <a name="configure-usage-policy"></a>Настройка политики использования

1. Выберите **Usage policy** (Политика использования). 
2. В разделе **Usage policy** (Политика использования) введите **количество пользователей**, которым разрешено использовать лабораторию.
3. Щелкните **Сохранить**. 

    ![Политика использования](../media/tutorial-setup-classroom-lab/usage-policy-settings.png)


## <a name="set-up-the-template"></a>Настройка шаблона 
На основе шаблона, в роли которого выступает базовый образ виртуальной машины, в лаборатории создаются все виртуальные машины пользователей. Настройте шаблон согласно параметрам, которые вы хотите предоставить пользователям лаборатории. Вы можете указать имя и описание шаблона, которое будет отображаться для пользователей лаборатории. Опубликуйте шаблон, сделав его общедоступным, чтобы открыть экземпляры шаблона виртуальной машины для пользователей лаборатории. 

### <a name="set-title-and-description"></a>Настройка заголовка и описания
1. В разделе **Шаблон** выберите **Изменить** (значок карандаша) для шаблона. 
2. В окне **User view** (Пользовательское представление) введите **заголовок** шаблона.
3. Введите **описание** шаблона.
4. Щелкните **Сохранить**.

    ![Описание аудиторной лаборатории](../media/tutorial-setup-classroom-lab/lab-description.png)

### <a name="set-up-the-template-vm"></a>Настройка шаблона виртуальной машины
 Вы можете подключиться к шаблону виртуальной машины и установить на нем все необходимое программное обеспечение, прежде чем сделать его доступным для учащихся. 

1. Подождите, пока шаблон виртуальной машины будет готов. После этого должна стать активной кнопка **Запустить**. Чтобы запустить виртуальную машину, щелкните **Запустить**.

    ![Запуск шаблона виртуальной машины](../media/tutorial-setup-classroom-lab/start-template-vm.png)
1. Чтобы подключиться к виртуальной машине, выберите **Подключить** и следуйте инструкциям. 

    ![Подключение к шаблону виртуальной машины](../media/tutorial-setup-classroom-lab/connect-template-vm.png)
1. Установите программное обеспечение, необходимое для работы учащихся с лабораторией (например, Visual Studio, Обозреватель службы хранилища Azure и т. д.). 
2. Отключитесь (закройте сеанс удаленного рабочего стола) от шаблона виртуальной машины. 
3. **Остановите** шаблон виртуальной машины, щелкнув **Остановить**. 

    ![Остановка шаблона виртуальной машины](../media/tutorial-setup-classroom-lab/stop-template-vm.png)

### <a name="publish-the-template"></a>Публикация шаблона 
Когда вы опубликуете шаблон, Службы лабораторий Azure создадут виртуальные машины в лаборатории с помощью этого шаблона. Число созданных при этом процессе виртуальных машин соответствует максимальному числу разрешенных в лаборатории пользователей, которое можно настроить в политике использования лаборатории. Все виртуальные машины имеют ту же конфигурацию, что и шаблон. 

1. В разделе **Шаблон** выберите **Опубликовать**. 

    ![Публикация шаблона виртуальной машины](../media/tutorial-setup-classroom-lab/public-access.png)
1. В окне **Публикация** выберите параметр **Опубликовано**. 
2. Нажмите кнопку **Опубликовать**. Этот процесс может занять некоторое время в зависимости от числа виртуальных машин, которое соответствует числу разрешенных пользователей лаборатории.
    
    > [!IMPORTANT]
    > После того как шаблон станет общедоступным, его доступ нельзя будет изменить на закрытый. 
4. Перейдите на страницу **Виртуальные машины** и убедитесь, что на ней отображаются пять виртуальных машин в состоянии **Не назначено**. Эти виртуальные машины еще не назначены учащимся. 

    ![Виртуальные машины](../media/tutorial-setup-classroom-lab/virtual-machines.png)
5. Подождите, пока виртуальные машины будут созданы. Они должны быть в состоянии **Остановлено**. На этой странице вы можете запустить виртуальную машину для учащегося, подключиться к ней, а также остановить или удалить ее. На этой странице виртуальные машины можете запускать вы или учащиеся. 

    ![Виртуальные машины в остановленном состоянии](../media/tutorial-setup-classroom-lab/virtual-machines-stopped.png)

## <a name="send-registration-link-to-students"></a>Отправка ссылки для регистрации учащимся

1. Откройте представление **Панель мониторинга**. 
2. Щелкните плитку **User registration** (Регистрация пользователей).

    ![Ссылка для регистрации учащихся](../media/tutorial-setup-classroom-lab/dashboard-user-registration-link.png)
1. В диалоговом окне **User registration** (Регистрация пользователей) щелкните **Копировать**. Ссылка скопируется в буфер обмена. Вставьте ее в редактор электронной почты и отправьте электронное сообщение учащемуся. 

    ![Ссылка для регистрации учащихся](../media/tutorial-setup-classroom-lab/registration-link.png)
2. В диалоговом окне **Регистрация пользователей** щелкните **Закрыть**.
