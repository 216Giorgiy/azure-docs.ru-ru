# <a name="tutorial-set-up-a-classroom-lab"></a>Руководство. Настройка аудиторной лаборатории 
В этом руководстве вы настроите аудиторную лабораторию с набором виртуальных машин, используемых учащимися в аудитории.  

Вот какие действия выполняются в этом руководстве:

> [!div class="checklist"]
> * Создание аудиторной лаборатории.
> * Настройка аудиторной лаборатории.
> * Отправка ссылки для регистрации учащимся.

## <a name="create-a-classroom-lab"></a>Создание аудиторной лаборатории

1. Перейдите на [веб-сайт Служб лабораторий Azure](https://labs.azure.com).
2. В окне **New Lab** (Создание лаборатории) выполните следующие действия. 
    1. Укажите **имя** аудиторной лаборатории. 
    2. Выберите **размер** виртуальной машины, которую вы планируете использовать в аудитории.
    3. Выберите **образ** для создания виртуальной машины.
    4. Укажите **учетные данные по умолчанию** для входа в виртуальные машины в лаборатории. 
    7. Щелкните **Сохранить**.

        ![Создание лаборатории для аудитории](../media/tutorial-setup-classroom-lab/new-lab-window.png)
1. После этого откроется **домашняя страница** лаборатории. 
    
    ![Домашняя страница аудиторной лаборатории](../media/tutorial-setup-classroom-lab/classroom-lab-home-page.png)

## <a name="configure-usage-policy"></a>Настройка политики использования

1. Выберите **Usage policy** (Политика использования). 
2. В разделе **Usage policy** (Политика использования) введите **количество пользователей**, которым разрешено использовать лабораторию.
3. Щелкните **Сохранить**. 

    ![Политика использования](../media/tutorial-setup-classroom-lab/usage-policy-settings.png)

## <a name="set-up-the-template"></a>Настройка шаблона 
На основе шаблона, в роли которого выступает базовый образ виртуальной машины, в лаборатории создаются все виртуальные машины пользователей. Настройте шаблон согласно параметрам, которые вы хотите предоставить пользователям лаборатории. Вы можете указать имя и описание шаблона, которое будет отображаться для пользователей лаборатории. Сделайте шаблон общедоступным, чтобы открыть экземпляры шаблона виртуальной машины для пользователей лаборатории. 

### <a name="set-title-and-description"></a>Настройка заголовка и описания
1. В разделе **Шаблон** выберите **Изменить** (значок карандаша) для шаблона. 
2. В окне **User view** (Пользовательское представление) введите **заголовок** шаблона.
3. Введите **описание** шаблона.
4. Щелкните **Сохранить**.

    ![Описание аудиторной лаборатории](../media/tutorial-setup-classroom-lab/lab-description.png)

### <a name="make-instances-of-the-template-public"></a>Создание общедоступных экземпляров шаблона
Когда вы зададите значение видимости шаблона **Public** (Общедоступный), Службы лабораторий Azure будут создавать виртуальные машины в лаборатории с помощью этого шаблона. Число созданных при этом процессе виртуальных машин соответствует максимальному числу разрешенных в лаборатории пользователей, которое можно настроить в политике использования лаборатории. Все виртуальные машины имеют ту же конфигурацию, что и шаблон. 

1. В разделе **Шаблон** выберите **Видимость**. 
2. На странице **Доступность** выберите **Public** (Общедоступный).
    
    > [!IMPORTANT]
    > После того как шаблон станет общедоступным, его доступ нельзя будет изменить на закрытый. 
3. Щелкните **Сохранить**.

    ![Доступность](../media/tutorial-setup-classroom-lab/public-access.png)

## <a name="send-registration-link-to-students"></a>Отправка ссылки для регистрации учащимся

1. Щелкните плитку **User registration** (Регистрация пользователей).
2. В диалоговом окне **User registration** (Регистрация пользователей) щелкните **Копировать**. Ссылка скопируется в буфер обмена. Вставьте ее в редактор электронной почты и отправьте электронное сообщение учащемуся. 

    ![Ссылка для регистрации учащихся](../media/tutorial-setup-classroom-lab/registration-link.png)
