#Заметки о выпуске

##3.0.0 (17.02.2015)

-   Первый выпуск Azure Mobile Engagement
-   Конфигурация appId заменена конфигурацией строки подключения.
-   Удален API для отправки и получения произвольных сообщений XMPP от произвольных объектов XMPP.
-   Удален API для отправки и получения между устройствами.
-   Усовершенствования безопасности.
-   Удалено отслеживание Google Play и SmartAd.
