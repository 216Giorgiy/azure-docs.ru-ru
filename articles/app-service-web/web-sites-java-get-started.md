# Создание веб-приложения Java в службе приложений Azure

# Создание веб-приложения Java с использованием Azure Marketplace

Здесь описано, как использовать Azure Marketplace для выбора контейнера приложения Java — Apache Tomcat или Jetty — для вашего веб-приложения.

Ниже показано, как будет выглядеть веб-приложение, созданное с помощью Tomcat из Azure Marketplace.

<!--todo:![Web app using Apache Tomcat](./media/web-sites-java-get-started/tomcat.png)-->

Ниже показано, как будет выглядеть веб-приложение, созданное с помощью Jetty из Azure Marketplace.

<!--todo:![Web app using Jetty](./media/web-sites-java-get-started/jetty.png)-->

1. Войдите на [портал Azure](http://go.microsoft.com/fwlink/?LinkId=529715).
2. Щелкните **Создать** в нижнем левом углу страницы.
3. Выберите колонку **Веб + мобильный**.
4. Щелкните **Azure Marketplace** в нижней части колонки **Веб + мобильный**.
5. Выберите **Веб**.
6. В верхней части страницы **Веб** расположено текстовое поле поиска. В этом текстовом поле введите нужный сервер приложений Java, например **Apache Tomcat** или **Jetty**.
4. Щелкните нужный сервер приложений Java.
5. Щелкните **Создать**.
6. Задайте URL-имя.
6. Выберите регион. Например, **Запад США**.
7. Щелкните **Создать**.

Через несколько секунд веб-приложение будет создано. Чтобы просмотреть веб-приложение, в колонке **Веб-приложения** на портале управления Azure выберите созданное веб-приложение, а затем щелкните его URL-адрес.

Теперь, когда вы создали веб-приложение с использованием контейнера приложения, см. раздел **Дальнейшие действия** для получения сведений об отправке вашего приложения в веб-приложение.

# Создание веб-приложения Java с использованием интерфейса настройки Azure

Здесь описано, как использовать пользовательский интерфейс настройки Azure для выбора контейнера приложения Java — Apache Tomcat или Jetty — для веб-приложения.

1. Войдите на портал управления Microsoft Azure.
2. Щелкните **Создать** в нижнем левом углу страницы.
3. Выберите колонку **Веб + мобильный**.
4. Щелкните **Azure Marketplace** в нижней части колонки **Веб + мобильный**.
5. Выберите **Веб**.
6. Щелкните **Веб-приложение**.
7. Щелкните **Создать**. 
8. Задайте URL-имя.
9. Выберите регион. Например, **Запад США**.
10. Щелкните **Создать**.
11. После создания веб-приложения щелкните **Все параметры**.
12. Щелкните **Параметры приложения**.
13. Выберите нужную версию Java.
14. Отображаются параметры для веб-контейнера, например Tomcat и Jetty. Выберите нужный **веб-контейнер**. 
15. Щелкните **Сохранить**. 

Веб-приложение на базе Java будет доступно через несколько секунд. Чтобы убедиться, что оно создано на основе Java, щелкните URL-адрес веб-приложения. Обратите внимание, что на странице будет присутствовать текст о том, что новое веб-приложение создано на основе Java.

Теперь, когда вы создали веб-приложение с использованием контейнера приложения, см. раздел **Дальнейшие действия** для получения сведений об отправке вашего приложения в веб-приложение.
