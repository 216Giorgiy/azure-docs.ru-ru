<div class="content">
    <h1>Документация по веб-приложениям</h1>
    <p style="padding-bottom: 0px; max-width: 1050px;">Веб-приложения Azure позволяют создавать и размещать веб-сайты на любых языках программирования без необходимости управлять инфраструктурой. Они обеспечивают автоматическое масштабирование и высокий уровень доступности, поддерживают Windows и Linux (предварительная версия) и обеспечивают автоматическое развертывание из GitHub, Team Services или любого репозитория Git. Узнайте, как использовать веб-приложения Azure, с помощью наших шаблонов быстрого запуска, примеров и руководств.</p>
