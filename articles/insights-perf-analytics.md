# Мониторинг производительности приложений

На[Портал Azure](http://portal.azure.com)можно выполнить настройку наблюдения для сбора статистические данные и сведения о зависимостях приложения в веб-приложений или виртуальных машин.

Благодаря использованию *расширений* Azure поддерживает мониторинг производительности приложений.*APM*), используя*расширения.*. Эти расширения устанавливаются в приложение. Они позволяют собирать данные и отправлять отчеты в службы мониторинга.

## Добавление расширения

1. Щелкните **Обзор** и выберите веб-приложение или виртуальную машину для инструментирования.

2. Щелкните ссылку.**Мониторинг приложений**плитки в группе**Мониторинг**.

3. Выберите поставщик расширений, которую вы хотите использовать, такие как**Application Insights**или**New Relic**.

![Веб-приложение](./media/insights-perf-analytics/05-extend.png)

Или, если вы используете виртуальной машины:

![Виртуальная машина.](./media/insights-perf-analytics/10-vm1.png)


## Обзор кода

Используйте приложение для создания некоторых телеметрии некоторое время.

1. Затем в веб-приложении или в колонке виртуальной машины отобразится установленное расширение.
2. Щелкните строку, которая представляет приложение, чтобы перейти к этому поставщику.![Щелкните "Обновить"](./media/insights-perf-analytics/06-overview.png)

Можно также использовать**Обзор**Чтобы перейти прямо к Application Insights компонента или New Relic учетной записью, которая использовалась.

После получения для Application Insights в выноске, например, вы можете:-Откройте производительности:

![В сетке щелкните зависимостей, чтобы просмотреть связанные запросы.](./media/insights-perf-analytics/08-requests.png)

### Дополнительные требования к приложениям определенных типов

New Relic автоматически устанавливается без дополнительных средств, но Application Insights имеет одно дополнительное требование.

![Выберите Application Insights в диалоговом окне создания проекта.](./media/insights-perf-analytics/03-add.png)

В ответ на запрос для входа в систему, используются учетные данные учетной записи Azure.
