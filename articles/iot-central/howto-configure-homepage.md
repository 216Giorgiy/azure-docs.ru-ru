# <a name="configuring-homepage"></a>Настройка домашней страницы

Домашней называется та страница, которая открывается по URL-адресу приложения для пользователя, имеющего необходимый доступ. Если при создании приложения вы выбрали шаблон приложения "Sample Contoso" или "Sample Devkits", для приложения будет использована предопределенная домашняя страница. Если же вы использовали шаблон приложения "Custom Application" (Пользовательское приложение), домашняя страница будет пустой.

Вот пример домашней страницы приложения, основанной на шаблоне "Sample Contoso". Чтобы настроить домашнюю страницу для приложения, сначала нажмите **Изменить** вверху справа. 

![Домашняя страница для приложений, основанных на шаблоне "Sample Contoso"](media/howto-configure-homepage/image1.png)

После нажатия **Изменить** вы увидите библиотеку панелей мониторинга на панели слева. Здесь представлены разные примитивы для создания плиток и панелей мониторинга, которые вы можете добавить на домашнюю страницу.

![Библиотека панелей мониторинга](media/howto-configure-homepage/image2.png)

Например, можно добавить плитку **Settings and Properties** (Параметры и свойства), чтобы отображать текущие значения параметров и свойств из некоторого набора. Для этого выберите **Шаблон устройства**, а затем **Экземпляр устройства**. Затем присвойте плитке заголовок и выберите отображаемый **Параметр** или **Свойство**. В нашем примере выбрано **Значение температуры**. Щелкните **Готово**, и выбранная плитка сразу появится на домашней странице.

![Форма настройки сведений об устройстве со сведениями о параметрах и свойствах](media/howto-configure-homepage/image3.png)

Теперь, когда оператор откроет домашнюю страницу, он увидит эту плитку со значениями свойства и (или) параметров устройства:

![Вкладка "Панель мониторинга", на которой отображаются параметры и свойства плитки](media/howto-configure-homepage/image4.png)

Попробуйте применить другие типы плиток, чтобы изучить доступные возможности персонализации домашней страницы.
