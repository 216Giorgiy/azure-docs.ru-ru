# <a name="prepare-and-upload-images-to-your-azure-iot-central-application"></a>Подготовка и отправка изображений в приложение Azure IoT Central

В этой статье описывается, как конструктор может настроить приложение Microsoft Azure IoT Central путем отправки пользовательских изображений. Например, можно настроить панель мониторинга устройства, добавив изображение устройства.

## <a name="before-you-begin"></a>Перед началом работы

Чтобы выполнить действия, описанные в этой статье, необходимо следующее:

1. Приложение Azure IoT Central. Дополнительные сведения см. в статье [Создание приложения Azure IoT Central](howto-create-application.md).
1. Средство для масштабирования и изменения размеров файлов изображений.

## <a name="choose-where-to-use-custom-images"></a>Выбор места для вставки пользовательских изображений

Пользовательские изображения можно добавить в следующие расположения в приложении Azure IoT Central.

* Страница **диспетчера приложений**.

    ![Изображение страницы диспетчера приложений](media/howto-prepare-images/applicationmanager.png)

* Домашняя страница.

    ![Изображение домашней страницы](media/howto-prepare-images/homepage.png)

* Шаблон устройства.

    ![Изображение шаблона устройства](media/howto-prepare-images/devicetemplate.png)

* Плитка на панели мониторинга устройства.

    ![Изображение плитки устройства](media/howto-prepare-images/devicetile.png)

* Плитка на панели мониторинга набора устройств.

    ![Изображение плитки набора устройств](media/howto-prepare-images/devicesettile.png)

## <a name="prepare-the-images"></a>Подготовка изображений

Во всех четырех расположениях можно использовать изображения в формате PNG, GIF или JPEG.

В следующей таблице приведены размеры изображений, которые можно использовать.

| Расположение | Размеры |
| -------- | ------ |
| **Диспетчер приложений** | 268 x 160 пикселей |
| Шаблон устройства | 64 x 64 пикселей |
| Домашняя страница и плитки панели мониторинга | Наименьшая по размеру плитка — 200 x 200 пикселей. Плитки большего размера имеют квадратную или прямоугольную форму размера, кратного размеру небольших плиток. Например, 200 x 400, 400 x 200 или 400 x 400 пикселей. |

Для наилучшего отображения в приложении следует создать изображения, которые соответствуют размерам, предоставленным в предыдущей таблице.

## <a name="upload-the-images"></a>Отправка изображений

В следующих разделах описан процесс отправки изображений, которые будут использоваться в разных расположениях.

### <a name="application-manager"></a>Диспетчер приложений

Чтобы отправить изображение для использования в **диспетчере приложений**, перейдите на страницу **Параметры приложения** в разделе **Администрирование**. Для выполнения этой задачи вам необходимо иметь права администратора.

![Отправка изображения приложения](media/howto-prepare-images/uploadapplicationmanager.png)

Щелкните отправляемое изображение, а затем выберите на локальном компьютере файл для отправки.

### <a name="home-page"></a>главная страница

Чтобы отправить изображение для использования на домашней странице, перейдите на **домашнюю страницу** приложения и включите режим конструктора. Для выполнения этой задачи вы должны быть конструктором.

![Отправка изображения домашней страницы](media/howto-prepare-images/uploadhomepage.png)

Щелкните отправляемое изображение, а затем выберите на локальном компьютере файл для отправки.

После отправки изображения можно изменить его размер при включенном режиме конструктора.

### <a name="device-template"></a>Шаблон устройства

Чтобы отправить изображение для использования в шаблоне устройства, откройте **Device Explorer**, выберите шаблон устройства и устройство, а затем перейдите в режим конструктора. Для выполнения этой задачи вы должны быть конструктором.

![Отправка изображения шаблона устройства](media/howto-prepare-images/uploaddevicetemplate.png)

Щелкните отправляемое изображение, а затем выберите на локальном компьютере файл для отправки.

### <a name="device-dashboard"></a>Панель мониторинга устройства

Чтобы отправить изображение для использования на панели мониторинга устройства, откройте **Device Explorer** и выберите шаблон устройства и устройство. Откройте страницу **Панель мониторинга** и включите режим конструктора. Для выполнения этой задачи вы должны быть конструктором.

![Отправка изображения панели мониторинга устройства](media/howto-prepare-images/uploaddevicedashboard.png)

Щелкните отправляемое изображение, а затем выберите на локальном компьютере файл для отправки.

После отправки изображения можно изменить его размер и расположение при включенном **режиме конструктора**.

### <a name="device-set-dashboard"></a>Панель мониторинга набора устройств

Чтобы отправить изображение для использования на панели мониторинга устройства, перейдите в раздел **Device Sets** (Наборы устройств) и выберите набор устройств и устройство. Откройте страницу **Панель мониторинга** и включите **режим конструктора**.

![Отправка изображения панели мониторинга набора устройств](media/howto-prepare-images/uploaddevicesetdashboard.png)

Щелкните отправляемое изображение, а затем выберите на локальном компьютере файл для отправки.

После отправки изображения можно изменить его размер и расположение при включенном режиме конструктора.
