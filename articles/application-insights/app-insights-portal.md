# Использование портала Application Insights

После того как вы [настроите Application Insights для проекта](app-insights-get-started.md), данные телеметрии о производительности и использовании вашего приложения будут отображаться в ресурсе Application Insights на [портале Azure](https://portal.azure.com).

## Поиск телеметрии в Azure

Выполните вход на [портал Azure](https://portal.azure.com) и перейдите к ресурсу Application Insights, который вы создали для приложения.
