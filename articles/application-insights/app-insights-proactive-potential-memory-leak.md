# <a name="memory-leak-detection-preview"></a>Обнаружение утечек памяти (Предварительная версия)

Application Insights автоматически анализирует потребление памяти каждого процесса в приложении и предупреждать о потенциальных утечек памяти или увеличения объема используемой памяти.

Этот компонент требует без дополнительной настройки, отличного от [Настройка счетчиков производительности](https://docs.microsoft.com/en-us/azure/application-insights/app-insights-performance-counters) для вашего приложения. Он остается активным, когда приложение создает недостаточно памяти телеметрии счетчиков производительности (например, байты исключительного пользования).


## <a name="when-would-i-get-this-type-of-smart-detection-notification"></a>Когда будет получить уведомления смарт-обнаружение такого типа
Типичные уведомление будет следовать согласованное увеличение объема используемой памяти в течение длительного времени (несколько часов), в один или несколько процессов и один или несколько компьютеров, которые являются частью приложения.
Алгоритмы машинного обучения используются для определения увеличения объема используемой памяти, соответствующую шаблону утечку памяти, в отличие от увеличения объема используемой памяти из-за естественным образом увеличение использования приложения.

## <a name="does-my-app-definitely-have-a-problem"></a>В работе моего приложения обязательно существует проблема?
Уведомление не означает, что в работе приложения возникла проблема. Несмотря на то, что утечке памяти обычно указывают на проблемы приложения, эти шаблоны может быть характерны для конкретного процесса, или может иметь естественный коммерческое обоснование и можно пропустить.

## <a name="how-do-i-fix-it"></a>Как ее исправить?
Уведомления содержат диагностических сведений для поддержки в процессе анализа диагностики:
1. **"Рассмотрение".** Уведомление показывает, можно увеличить объем памяти (в ГБ) и диапазон времени, в котором увеличивается объем памяти. Это поможет назначить приоритет проблемы.
2. **Область действия.** Сколько машин короткая шаблон утечка памяти? Сколько исключения во время потенциальную утечку памяти было запущено? Эти сведения можно получить из уведомления.
3. **Выполните диагностику.** Обнаружение содержит шаблон утечки памяти, показывающая потребление памяти процесса со временем. Можно также использовать связанные элементы и отчеты, связывание с вспомогательные сведения, помогающие дальнейшей диагностики проблемы.
