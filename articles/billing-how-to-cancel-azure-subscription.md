# Отмена подписки Azure

>[AZURE.NOTE] Срок плана поддержки Azure — 6 месяцев, а корпорация Майкрософт требует непрерывного соблюдения финансовых обязательств на его протяжении. При досрочной отмене оставшееся обязательство не аннулируется, а предоплаченные суммы не возмещаются. Если подписка отменена по ошибке и ее нужно активировать повторно, вы можете обратиться за помощью в [службу поддержки Azure](billing-how-to-create-billing-support-ticket.md).

Если отменить подписку, все службы будут немедленно удалены. Обязательно создайте резервную копию данных, которые вы хотите сохранить, до отмены.

1. Выполните вход в [Центр учетных записей Azure](https://account.windowsazure.com/subscriptions) в качестве администратора учетной записи.
2. В разделе **Щелкните подписку для просмотра информации и сведений об использовании** выберите подписку, которую нужно отменить. </br> ![selectsub](.\media\billing-how-to-cancel-azure-subscription\Selectsub.png)
3. В правой части страницы выберите **Отменить подписку**. </br> ![cancelsub](.\media\billing-how-to-cancel-azure-subscription\cancelsub.png)
4. Выберите **Да, отменить мою подписку**. </br> ![cancelbox](.\media\billing-how-to-cancel-azure-subscription\cancelbox.png)
5. Щелкните ![checkbutton](.\media\billing-how-to-cancel-azure-subscription\checkbutton.png), чтобы закрыть диалоговое окно и вернуться к странице подписки.
