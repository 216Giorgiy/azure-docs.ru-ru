# Отслеживание сообщений B2B

## Сведения об отслеживании B2B
Взаимодействие B2B включает обработку сообщений между торговыми партнерами. Отношения задаются как соглашения между двумя торговыми партнерами. Посте установки взаимодействия требуется способ наблюдения, чтобы взаимодействие происходило надлежащим образом. 
Предлагая приложения API B2B в составе служб приложений Azure, мы включили отслеживание данных, а также то, что предоставляется на портале Azure. 

## AS2
После создания экземпляра приложения API AS2 перейдите в этот экземпляр и войдите в часть "Отслеживание". Здесь можно просматривать все сведения отслеживания AS2, а также фильтровать их с помощью предоставленных колонок фильтра.

![][1]  

## EDIFACT
После создания экземпляра приложения API EDIFACT перейдите в этот экземпляр и войдите в часть "Отслеживание". Здесь можно просматривать все сведения отслеживания EDIFACT, а также фильтровать их с помощью предоставленных колонок фильтра.
Кроме того, можно также просматривать данные уровня обмена, данные уровня группы и данные уровня набора транзакций, войдя в соответствующее представление. 

Если пакеты были созданы как часть соглашений EDIFACT в связанном приложении API управления торговыми партнерами, то в части "Пакетная обработка" будут перечислены все эти пакеты. Можно войти в пакет, чтобы увидеть сообщения, составляющие активные сообщения (если таковые имеются), а также сведения для пакетов, которые были выполнены в прошлом.

![][2]

X12 и EDIFACT имеют аналогичные представления отслеживания.
