# Кодировщик неструктурированных файлов BizTalk

Соединитель кодирования и декодирования неструктурированных файлов BizTalk обеспечивает взаимодействие данных неструктурированных файлов (таких как Excel и CSV) и данных XML-файлов в вашем приложении. Он может преобразовывать экземпляр неструктурированного файла в XML-файл и наоборот.

##Использование кодировщика неструктурированных файлов BizTalk
1. Чтобы использовать кодировщик BizTalk, необходимо сначала создать экземпляр приложения API кодировщика неструктурированных файлов BizTalk. Это можно сделать либо в процессе, при создании приложения логики, либо выбрав приложение API кодировщика неструктурированных файлов BizTalk на сайте Azure Marketplace.

###Настройка кодировщика неструктурированных файлов BizTalk
Кодировщик неструктурированных файлов BizTalk принимает схемы как часть своей конфигурации. Пользователи могут запустить колонку настройки приложения API, запустив приложение API непосредственно с портала Azure или дважды щелкнув приложение API в рабочей области конструктора.

![Настройка кодировщика неструктурированных файлов BizTalk][1]

Чтобы настроить схему в колонке приложения API, необходимо щелкнуть часть *Схемы*.

![Часть «Схемы» кодировщика неструктурированных файлов BizTalk][2]

Пользователи могут загрузить схемы с диска или создать их на основе экземпляра неструктурированного файла или экземпляра JSON.

![Часть «Схемы» кодировщика неструктурированных файлов BizTalk][3]


###Использование кодировщика неструктурированных файлов BizTalk в рабочей области конструирования
После настройки пользователи могут щелкнуть значок *->* и выбрать действие в списке действий.

![Список действий кодировщика неструктурированных файлов BizTalk][4]
