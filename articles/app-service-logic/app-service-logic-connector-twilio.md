# Использование соединителя Twilio в приложении логики #

Приложения логики могут запускаться на основе разных источников данных и предлагать соединители для получения и обработки данных в рамках потока. 

Соединитель Twilio позволяет отправлять и получать SMS в учетной записи Twilio. С его помощью вы также можете получать номера телефонов и сведения об использовании.

## Создание соединителя Twilio для приложения логики ##
Чтобы использовать соединитель Twilio, необходимо сначала создать экземпляр приложения API соединителя Twilio. Это можно сделать следующим образом.

1.	Откройте Azure Marketplace, нажав кнопку "+ СОЗДАТЬ" в нижнем левом углу портала Azure.
2.	Перейдите в раздел "Мобильные и веб-приложения > Приложения API" и выполните поиск элемента "Соединитель Twilio".
3.	Настройте соединитель Twilio следующим образом.
 
	![][1]
	- **Расположение** - Выберите географическое расположение, где планируется развернуть соединитель.
	- **Подписки** - Выберите подписку, в которой нужно создать соединитель.
	- **Группа ресурсов** - Выберите или создайте группу ресурсов, в которой следует разместить соединитель.
	- **План веб-хостинга** - Выберите или создайте план веб-хостинга.
	- **Ценовой уровень** - Выберите ценовую категорию для соединителя.
	- **Имя** - Задайте имя для этого соединителя Twilio.
	- **Параметры пакета**
		- **ИД безопасности учетной записи** - Уникальный идентификатор учетной записи. ИД безопасности учетной записи для вашей учетной записи может быть извлечен из https://www.twilio.com/user/account/settings 
		- **Маркер авторизации** - Маркер авторизации, связанный с учетной записью. Маркер авторизации для вашей учетной записи можно получить из https://www.twilio.com/user/account/settings


4.	Нажмите кнопку "Создать". Будет создан новый соединитель Twilio.
5.	После создания экземпляра приложения API можно создать приложение логики в той же группе ресурсов для использования соединителя Twilio. 

## Использование соединителя Twilio в приложении логики ##
После создания приложения API можно использовать соединитель Twilio как действие для приложения логики. Для этого необходимо выполнить следующие действия.

1.	Создайте новое приложение логики и выберите ту же группу ресурсов, которую имеет соединитель Twilio.
 
	![][2]
2.	Откройте раздел "Триггеры и действия", чтобы открыть конструктор приложений логики и настроить поток. 
 
	![][3]
3.	Соединитель Twilio отображается в разделе "Приложения API в этой группе ресурсов" в коллекции с правой стороны.
 
	![][4]
4. Приложение API соединителя Twilio можно удалить в редакторе, щелкнув "Соединитель Twilio".
 
5.	Теперь можно использовать соединитель Twilio в потоке. Действие "Отправить сообщение" в потоке можно использовать для отправки сообщения. Настройте входные свойства для действия "Отправить сообщение" следующим образом.
	- **С номера телефона** - Введите номер телефона, включенный для этого типа отправляемого сообщения. Только телефонные номера или сокращенные коды, приобретенные у Twilio, будут работать с этим соединителем.
	- **На номер телефона** - Конечный номер телефона. Принятый формат: +, код страны, номер телефона. Например, +16175551212. Если опустить знак "+", Twilio будет использовать код страны, введенный в поле  'С номера телефона'.
	- **Текст** - Текст сообщения, которое необходимо отправить.
