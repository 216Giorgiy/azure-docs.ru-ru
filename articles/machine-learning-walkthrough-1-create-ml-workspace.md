# Шаг 1. Создание рабочей области машинного обучения Azure

Для использования ML Studio нужно иметь рабочую область ML. Такая рабочая область содержит инструменты, необходимые для создания, управления и публикации экспериментов.

## Создание рабочей области

1.  Войдите в систему с использованием учетной записи Azure.
2.  На панели служб Microsoft Azure выберите **МАШИННОЕ ОБУЧЕНИЕ**.
    ![Создание рабочей области][Создание рабочей области]

3.  Щелкните **СОЗДАТЬ РАБОЧУЮ ОБЛАСТЬ ML**.
4.  На странице **БЫСТРО СОЗДАТЬ** введите информацию о рабочей области, а затем щелкните **СОЗДАТЬ РАБОЧУЮ ОБЛАСТЬ ML**.

    > **ПРИМЕЧАНИЕ**. В поле **ВЛАДЕЛЕЦ РАБОЧЕЙ ОБЛАСТИ** введите свою учетную запись Майкрософт (например, <name@outlook.com>) или учетную запись организации.

После создания рабочей области машинного обучения она появится в списке на странице **Машинное обучение**.

> **Подсказка**. Чтобы сделать общими эксперименты, над которыми вы работаете, других пользователей можно пригласить в свою рабочую область. Это можно сделать в ML Studio на странице **НАСТРОЙКИ**. Для каждого пользователя будет просто нужна учетная запись Майкрософт или учетная запись организации.
