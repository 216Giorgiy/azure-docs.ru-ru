# <a name="tutorial-connect-set-up-activate-azure-data-box-edge-preview"></a>Руководство. Подключение, настройка, активация Azure Data Box Edge (предварительная версия) 

В этом руководстве описывается, как подключить, настроить и активировать устройство Data Box Edge с помощью локального пользовательского веб-интерфейса. 

Процесс установки и активации может занять около 20 минут. 

Из этого руководства вы узнаете, как выполнять следующие задачи:

> [!div class="checklist"]
> * подключение физического устройства;
> * настройка и активация физического устройства.

> [!IMPORTANT]
> В данный момент решение Data Box Edge находится в состоянии предварительной версии. Изучите [Дополнительные условия использования Предварительных выпусков Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/), прежде чем заказывать и развертывать это решение. 


## <a name="prerequisites"></a>Предварительные требования

Перед установкой и настройкой Data Box Edge проверьте следующее.

* Вы установили физическое устройство, как описано в статье [Руководство по установке Azure Data Box Edge (предварительная версия)](data-box-edge-deploy-install.md).
* У вас есть ключ активации из службы Data Box Edge, который вы создали для управления устройством Data Box Edge. Дополнительные сведения см. в разделе [Подготовка к развертыванию Azure Data Box Edge](data-box-edge-deploy-prep.md).

## <a name="connect-to-the-local-web-ui-setup"></a>Подключение к настройке локального пользовательского веб-интерфейса 

1. Настройте адаптер Ethernet на компьютере, который используется для подключения к устройству Edge, указав статический IP-адрес 192.168.100.5 и подсеть 255.255.255.0.
2. Подключите компьютер к порту 1 на вашем устройстве. 
3. Откройте окно браузера и перейдите к локальному пользовательскому веб-интерфейсу на устройстве по адресу https://192.168.100.10. Это действие может занять несколько минут после включения устройства. 
4. Появится сообщение об ошибке или предупреждение, уведомляющее о проблеме с сертификатом безопасности веб-сайта. Щелкните **Перейти на эту веб-страницу**. (Эти шаги могут отличаться в зависимости от используемого браузера)
   
    ![](./media/data-box-edge-deploy-connect-setup-activate/image2.png)

2. Войдите в пользовательский веб-интерфейс устройства. Пароль по умолчанию — *Password1*. 
   
    ![](./media/data-box-edge-deploy-connect-setup-activate/image3.png)

3. Появится запрос на изменение пароля администратора устройства. Введите новый пароль, содержащий от 8 до 16 символов. Пароль должен содержать три знака из следующих категорий: прописные буквы, строчные буквы, цифры и специальные знаки.

Теперь вы находитесь на **панели мониторинга** устройства.

## <a name="set-up-and-activate-the-physical-device"></a>Настройка и активация физического устройства
 
1. Из панели мониторинга вы можете перейти к различным параметрам, необходимым для настройки и регистрации физического устройства с помощью службы Data Box Edge. Настройки в разделах **Имя устройства**, **Параметры сети**, **Параметры веб-прокси** и **Параметры времени** являются необязательными. Обязательны только настройки в разделе **Параметры облака**.
   
    ![](./media/data-box-edge-deploy-connect-setup-activate/set-up-activate-1.png)

2. На странице **Имя устройства** настройте понятное имя для своего устройства. Понятное имя может содержать от 1 до 15 знаков (буквы, цифры и дефисы).

    ![](./media/data-box-edge-deploy-connect-setup-activate/set-up-activate-2.png)

3. (Необязательно) Настройте **Параметры сети**. В физическом устройстве вы увидите шесть сетевых интерфейсов. Порты 1 и 2 — это сетевые интерфейсы со скоростью 1 Гбит/с. Порты 3–6 являются сетевыми интерфейсами со скоростью 25 Гбит/с. Порт 1 автоматически настроен только для управления, тогда как порты 2–6 являются портами данных. Ниже показана страница **Параметры сети**.
    
    ![](./media/data-box-edge-deploy-connect-setup-activate/set-up-activate-3.png)
   
    При настройке параметров сети учитывайте следующее.

    - Если в среде включен DHCP, сетевые интерфейсы настраиваются автоматически. IP-адрес, подсеть, шлюз и DNS назначаются автоматически.
    - Если протокол DHCP не включен, при необходимости вы можете назначить статические IP-адреса.
    - Сетевой интерфейс можно настроить как IPv4.
   
4. (Необязательно) Настройте прокси-сервер доступа в Интернет. Хотя использовать веб-прокси не обязательно, если в вашей сети имеется веб-прокси, настройку для работы с ним можно выполнить только в этом разделе.
   
   ![](./media/data-box-edge-deploy-connect-setup-activate/set-up-activate-4.png)
   
   На странице **Прокси-сервер доступа в Интернет** сделайте следующее:
   
   1. Укажите **URL-адрес веб-прокси** в следующем формате: `http://host-IP address or FDQN:Port number`. URL-адреса HTTPS не поддерживаются.
   2. Выберите тип **аутентификации**: **Базовая** или **Нет**.
   3. Если аутентификация используется, то необходимо указать также **имя пользователя** и **пароль**.
   4. Щелкните **Применить**. Заданные параметры веб-прокси будут проверены и применены.

5. (Необязательно) Настройте параметры времени для своего устройства, такие как часовой пояс, а также основной и дополнительный NTP-серверы. NTP-серверы необходимы, так как устройство должно синхронизировать время, чтобы проходить проверку подлинности в облачных службах.
    
    ![](./media/data-box-edge-deploy-connect-setup-activate/set-up-activate-5.png)
    
    На странице **Параметры времени** сделайте следующее:
    
    1. В раскрывающемся списке выберите **часовой пояс** , учитывая географическое расположение, в котором будет развернуто устройство. Часовой пояс по умолчанию для устройства — тихоокеанское стандартное время (PST). Устройство будет использовать заданный часовой пояс для всех запланированных операций.
    2. Укажите **основной NTP-сервер** для устройства или примите значение по умолчанию, заданное на сайте time.windows.com. Убедитесь, что ваша сеть позволяет передавать NTP-трафик из вашего центра обработки данных в Интернет.
    3. При необходимости укажите **дополнительный NTP-сервер** для устройства.
    4. Щелкните **Применить**. Заданные параметры времени будут проверены и применены.

6. На странице **Параметры облака** активируйте свое устройство с помощью службы Data Box Edge на портале Azure.
    
    1. Введите **ключ активации** для Data Box Edge, который вы получили при работе с разделом [Получение ключа активации](data-box-edge-deploy-prep.md#get-the-activation-key).

    2. Нажмите кнопку **Применить**. 
       
         ![](./media/data-box-edge-deploy-connect-setup-activate/set-up-activate-6.png)
    
    3. После успешной активации устройства отобразятся параметры режима подключения. Эти параметры настраиваются в том случае, если вам нужно работать с устройством в режиме частичного или полного отключения. 

        ![](./media/data-box-edge-deploy-connect-setup-activate/set-up-activate-7.png)    

Настройка устройства завершена. Теперь в него можно добавить общие папки.
