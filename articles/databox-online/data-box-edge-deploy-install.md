# <a name="tutorial-install-azure-data-box-edge-preview"></a>Руководство по установке Azure Data Box Edge (предварительная версия)

Это руководство содержит инструкции по установке Data Box Edge на физическое устройство. Процедура установки включает распаковку устройства, установку в стойку и подключение к нему кабелей. 

Установка может занимать около 2 часов.

Из этого руководства вы узнаете, как выполнять следующие задачи:

> [!div class="checklist"]
> * Распаковка устройства.
> * Установка устройства в стойку.
> * Подключение кабелей к устройству.

> [!IMPORTANT]
> В данный момент решение Data Box Edge находится в состоянии предварительной версии. Изучите [Дополнительные условия использования Предварительных выпусков Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/), прежде чем заказывать и развертывать это решение.

## <a name="prerequisites"></a>Предварительные требования

Ниже приведены предварительные требования для установки физического устройства.

### <a name="for-the-data-box-edge-resource"></a>Для ресурса Data Box Edge

Перед тем как начать, убедитесь в следующем.

* Выполнены все инструкции из статьи [Руководство. Подготовка к развертыванию Azure Data Box Edge (предварительная версия)](data-box-edge-deploy-prep.md).
    * Создан ресурс Data Box Edge для развертывания устройства.
    * Создан ключ активации, чтобы активировать устройство с помощью ресурса Data Box Edge.

 
### <a name="for-the-data-box-edge-physical-device"></a>Для физического устройства Data Box Edge

Перед развертыванием устройства:

- Убедитесь, что устройство надежно установлено на плоской, стабильной и ровной рабочей поверхности (или в подобном положении).
- Убедитесь, что в расположении, в котором планируется выполнить настройку, есть следующее:
    - стандартное питание переменного тока из независимого источника или
    - стоечный блок распределения питания (БРП) с источником бесперебойного питания (ИБП).
- Убедитесь, что в стойке, куда вы планируете установить корпуса, имеется один разъем высотой 1U.

### <a name="for-the-network-in-the-datacenter"></a>Сеть в центре обработки данных

Перед началом работы

- Изучите сетевые требования к развертыванию Data Box Edge и настройте сеть центра обработки данных в соответствии с требованиями. Дополнительные сведения см. в разделе [Требования к сети](data-box-gateway-system-requirements.md#networking-requirements).
- Убедитесь, что минимальная пропускная способность интернет-канала, необходимая для оптимальной работы устройства, составляет 20 Мбит/с.


## <a name="unpack-the-device"></a>распаковка устройства;

Данное устройство поставляется в одной упаковке. Выполните следующие действия для распаковки устройства. 

1. Расположите коробку на ровной плоской поверхности.
2. Проверьте коробку и упаковочный пеноматериал на предмет наличия пробоин, порезов, повреждений от воды или других видимых повреждений. Если коробка или упаковка серьезно повреждены, не открывайте коробку. Обратитесь в службу поддержки Майкрософт, специалисты которой помогут вам в оценке состояния устройства.
3. Распакуйте коробку. После распаковки коробки, убедитесь, что у вас имеется:
    - одно устройство Edge в отдельном корпусе;
    - два кабеля питания;
    - один комплект для монтажа в стойку без инструментов (с двумя боковыми направляющими и монтажным оборудованием).
4. Если вы не получили какие-либо элементы из перечисленных выше, обратитесь в службу технической поддержки Data Box Edge. Следующим шагом является установка устройства в стойку.


## <a name="rack-the-device"></a>установка устройства в стойку;

Устройства должны устанавливаться в стандартную 19-дюймовую стойку. Используйте следующую процедуру, чтобы установить устройство в стандартную 19-дюймовую стойку с помощью передних и задних опор.

> [!IMPORTANT]
> Для правильной работы устройства Data Box Edge должны быть установлены в стойку.


1. Потяните за передний открывающий механизм, чтобы разблокировать внутреннюю направляющую из скользящего узла. Отпустите фиксатор замка и толкните среднюю направляющую внутрь, чтобы повторно подсоединить направляющую. Внутренние и внешние направляющие должны быть разделены.

    ![Установка направляющих стойки](./media/data-box-edge-deploy-install/rack-mount-rail-1.png)

2. Теперь установите внешние направляющие на вертикальные элементы стойки. Чтобы помочь с ориентацией, на направляющих есть пометка "Перед" и этот конец прикреплен к передней части корпуса. 
    
    1. Найдите штифты направляющих на передней и задней частях направляющих в комплекте. Выдвиньте рельсовую направляющую между опорами стойки. Сначала присоедините внешнюю направляющую в задней части стойки. Откорректируйте расположение заднего монтажного держателя, чтобы поместить его в задние отверстия для крепления в стойке.   
    2. Нажмите и удерживайте спусковой крючок на заднем держателе, чтобы открыть металлические крючки. Выровняйте их и вставьте в монтажные отверстия, а затем отпустите спусковой крючок.
    3. Выровняйте передний держатель с монтажным отверстием.
    4. Передний держатель теперь должен поместиться в стойку. Направляющие и опоры можно закрепить с помощью винтов M5 X 10L, если это необходимо. 

    ![Установка направляющих стойки](./media/data-box-edge-deploy-install/rack-mount-rail-2.png)

3. Чтобы прикрепить внутреннюю направляющую на шасси, убедитесь, что отверстия для замочной скважины на внутренней направляющей выровнены с установочными штифтами на стороне шасси. Убедитесь, что головки опорных штифтов шасси выступают через отверстия в замочной скважине внутренней направляющей. Потяните направляющую к передней части корпуса, пока направляющая со слышимым щелчком не встанет на место. Повторите это с другими внутренними направляющими. Толкните шасси с помощью внутренней направляющей на скользящий узел, чтобы выполнить установку в стойку.

    ![Установка направляющих стойки](./media/data-box-edge-deploy-install/rack-mount-rail-3.png)

## <a name="cable-the-device"></a>Подключение кабелей к устройству

Ниже описано, как подключить кабель питания и сети к устройству Edge.

## <a name="prerequisites"></a>Предварительные требования

Перед началом подключения кабелей к устройству, необходимо иметь:

- физическое устройство Edge, распакованное и установленное в стойку;
- два кабеля питания; 
- как минимум один сетевой кабель 1 GbE RJ-45 для подключения к интерфейсу управления (на устройстве есть два сетевых интерфейса 1 GbE: для управления и для данных);
- один трансивер SFP 25 GbE + медный кабель для настройки каждого сетевого интерфейса данных (хотя бы один сетевой интерфейс данных — PORT 2, PORT 3, PORT 4, PORT 5 или PORT 6 — должен быть подключен к Интернету (подключение к Azure));
- доступ к двум блокам распределения питания (БРП) (рекомендуется).

> [!NOTE]
> - Если вы подключаете только один сетевой интерфейс данных, для отправки данных в Azure рекомендуем использовать сетевой интерфейс 25 GbE, например PORT 3, PORT 4, PORT 5 или PORT 6. 
> - Для достижения оптимальной производительности и обработки больших объемов данных рекомендуем подключить все порты данных.
> - Устройство Edge необходимо подключить к сети центра обработки данных таким образом, чтобы оно могло принимать данные с серверов исходных данных. 

Устройство Edge имеет восемь дисков SSD NVMe. На передней панели также есть светодиоды состояния и кнопки питания. Устройство включает резервные блоки питания (РБП) на задней панели. Устройство имеет шесть сетевых интерфейсов: два интерфейса на 1 Гбит/с и четыре интерфейса на 25 Гбит/с. На вашем устройстве есть контроллер управления основной платой (BMC). Определите разные порты на задней панели устройства.
 
  ![Задняя панель подключенного устройства](./media/data-box-edge-deploy-install/backplane-cabled.png)
 
Выполните следующие действия, чтобы подключить к устройству питание и сеть.

1. Подключите кабели питания ко всем резервным блокам питания в корпусе. Чтобы обеспечить высокую доступность, установите оба резервных блока питания и подключите их к разным источникам питания.
2. Присоедините кабели питания ко всем распределительным блокам стойки. Убедитесь, что для двух резервных блоков питания используются разные источники питания.
3. Подключите сетевой интерфейс 1 GbE PORT 1 к компьютеру, используемому для настройки физического устройства. PORT 1 — это выделенный интерфейс управления.
4. Подключите к сети ЦОД или Интернету порт PORT 2, PORT 3, PORT 4, PORT 5 или PORT 6 (или несколько из них). Для порта PORT 2 используйте сетевой кабель RJ-45. Для сетевых интерфейсов 25 GbE используйте трансиверы SFP + медные кабели.
