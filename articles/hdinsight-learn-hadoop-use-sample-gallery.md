# Изучение Hadoop в HDInsight с помощью коллекции материалов для начала работы с HDInsight

С помощью коллекции материалов для начала работы с HDInsight можно легко и быстро запускать примеры приложений Hadoop в HDInsight. Некоторые примеры поставляются с примером данных. Для остальных примеров можно указать собственные данные. Уже сейчас есть 6 примеров, а вскоре их станет еще больше.

- Решения с данными Azure
	- Анализ журналов веб-сайта Microsoft Azure
	- Аналитика службы хранилища Microsoft Azure
- Решения с примером данных
	- Анализ данных датчика
	- Анализ тенденций Twitter
	- Анализ журнала веб-сайта
	- Рекомендации по фильмам с использованием Mahout

**Запуск примера из коллекции материалов для начала работы**

1.	Войдите на [портал управления Azure][azure.management.portal].
2.	Щелкните **HDInsight** в меню слева. Появится список существующих кластеров HDInsight, в том числе Hadoop, Storm и HBase. 
3.	Щелкните кластер, в котором вы хотите запустить пример.
4.	Щелкните **КОНСОЛЬ ЗАПРОСОВ** снизу.
5.	Введите имя пользователя и пароль Hadoop для кластера.
6.	Щелкните **Коллекция материалов для начала работы** сверху.
7.	Выберите один из примеров. К каждому примеру прилагаются подробные указания по его запуску. На следующем рисунке приведен пример анализа тенденций Twitter:

	![HDInsight Twitter trend analysis sample][hdinsight.twitter.sample]
