#Изменение буквы диска для временного диска Windows

Если вам нужно использовать диск D для хранения данных, следуйте приведенным указаниям, чтобы использовать другой диск в качестве временного. Никогда не используйте временный диск для хранения данных, которые нужно сохранить.

Прежде чем начать, потребуется подключить диск данных к виртуальной машине, чтобы хранить файл подкачки Windows (pagefile.sys) во время этой процедуры. Если у вас его нет, ознакомьтесь со статьей [Подключение диска данных к виртуальной машине Windows][Attach]. Указания о том, как узнать, какие диски подключены, см. в подразделе «Поиск диска» раздела [Отключение диска от виртуальной машины Windows][Detach].

Если вы хотите использовать существующий диск данных на диске D, убедитесь, что вы также передали соответствующий VHD в учетную запись хранения. Указания см. в шагах 3 и 4 в разделе [Создание и отправка VHD Windows Server в Azure][VHD].

> [AZURE.WARNING]Если вы меняете размер виртуальной машины, и при этом она перемещается на другой узел, временным диском снова станет диск D.

##Изменение буквы диска

1. Войдите в виртуальную машину. Дополнительную информацию см. в разделе [Как войти в виртуальную машину под управлением Windows Server][Logon].

2. Переместите файл pagefile.sys с диска D на другой диск.

3. Перезапустите виртуальную машину.

4. Снова войдите и измените букву диска с D на E.

5. На [портале Azure](http://manage.windowsazure.com) подключите существующий диск данных или пустой диск данных.

6.	Снова войдите на виртуальную машину, инициализируйте диск и назначьте только что подключенному диску букву D.

7.	Убедитесь, что буква E назначена временному диску.

8.	Переместите файл pagefile.sys с другого диска на диск E.
