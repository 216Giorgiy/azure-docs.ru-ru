## О CoreMark

Показатели для Linux были вычислены при выполнении [CoreMark](http://www.eembc.org/coremark/faq.php) в Ubuntu. Для CoreMark было настроено количество потоков, равное количеству виртуальных процессоров, а также настроен параметр параллелизма PThreads. Целевое количество итераций было скорректировано в соответствии с ожидаемой производительностью, чтобы обеспечить время выполнения не менее 20 секунд (обычно больше). Окончательный результат был получен делением числа выполненных итераций на число секунд, которое потребовалось для выполнения теста. Каждый тест выполнялся по крайней мере семь раз на каждой виртуальной машине. Тесты были выполнены в октябре 2015 года на нескольких виртуальных машинах в каждом общедоступном регионе Azure, в котором на тот момент поддерживались эти виртуальные машины.
