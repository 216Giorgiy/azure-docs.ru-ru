## Дальнейшие действия

* Попробуйте запустить рабочую нагрузку в кластере. См. пример в [руководстве по началу работы](https://technet.microsoft.com/library/jj884144) с пакетом HPC.

* Поэкспериментируйте со средствами HPC для запуска, остановки, добавления и удаления вычислительных узлов в кластере. См. статью [Управление вычислительными узлами в кластере на основе пакета HPC в Azure](virtual-machines-windows-classic-hpcpack-cluster-node-manage.md).
