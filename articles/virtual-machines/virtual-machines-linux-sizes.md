# Размеры виртуальных машин в Azure

В этой статье описаны доступные размеры и разновидности виртуальных машин Azure, которые можно использовать для запуска приложений и рабочих нагрузок Linux. Здесь также предоставлены рекомендации по развертыванию, которые нужно учитывать при планировании использования этих ресурсов. Подробнее о ценах на различные размеры см. в разделе [Цены на Виртуальные машины](https://azure.microsoft.com/pricing/details/virtual-machines/#Linux). Также доступна версия этой статьи для [виртуальных машин Windows](virtual-machines-windows-sizes.md).
