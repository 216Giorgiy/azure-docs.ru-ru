# Отслеживание использования с помощью пользовательских событий и метрик

### Скоро

Application Insights SDK позволяет вставлять строки кода в ваше приложение для отслеживания операций пользователей, чтобы обеспечить возможность настройки приложения с учетом потребностей пользователей.

В настоящее время мы по-прежнему поддерживаем данную функцию на новом портале Application Insights в Microsoft Azure. (При желании можно [посмотреть, как эта функция работает на старом портале][посмотреть, как эта функция работает на старом портале].)

Однако, прямо сейчас можно [собрать журналы событий диагностики, созданные с помощью Trace, NLog или Log4Net, и проводить по ним поиск][собрать журналы событий диагностики, созданные с помощью Trace, NLog или Log4Net, и проводить по ним поиск].
