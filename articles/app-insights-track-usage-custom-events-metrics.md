# Отслеживание использования с помощью пользовательских событий и метрик

### Скоро

Application Insights SDK позволяет вставлять строки кода в ваше приложение для отслеживания операций пользователей, чтобы обеспечить возможность настройки приложения с учетом потребностей пользователей.

Сейчас мы по-прежнему поддерживаем эту функцию на новом портале Application Insights в Microsoft Azure. (При желании можно [посмотреть, как эта функция работает на старом портале](http://msdn.microsoft.com/library/dn481100.aspx).)

При этом вы можете прямо сейчас [собрать журналы событий ][диагностики], созданные с помощью Trace, NLog или Log4Net, и выполнить поиск по ним.
