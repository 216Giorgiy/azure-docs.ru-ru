# Что такое Microsoft Azure Active Directory


###Другие статьи на эту тему
[Что такое Azure AD?](active-directory-whatis.md)<br> [Как это работает?](active-directory-works.md)<br> [Приступая к работе](active-directory-get-started.md)<br> [Дальнейшие действия](active-directory-next-steps.md)<br> [Подробнее](active-directory-learn-map.md)

Azure Active Directory (Azure AD) предоставляет компаниям простой способ управления удостоверениями и доступом не только в облаке, но и локально. Пользователи смогут входить в локальные и веб-приложения с любого устройства на базе iOS, Mac OS X, Android или Windows, используя единую рабочую или учебную учетную запись, а организация — защитить размещенные локально и в облаке конфиденциальные данные и приложения, а также обеспечить безопасность удаленного и локального доступа с помощью интегрированной многофакторной проверки подлинности. Компания также может расширить функциональность локальных каталогов, предоставив информационным работникам надежный и единообразный доступ к корпоративным ресурсам с помощью единой учетной записи. В Azure AD предусмотрены подробные отчеты и аналитика, а также поддерживаются возможности самообслуживания, что позволяет сократить затраты и повысить уровень безопасности. Соглашение об уровне обслуживания Azure AD для корпоративных клиентов поможет вам вести бизнес без перебоев.

![][1]


> [AZURE.NOTE]Чтобы использовать Azure Active Directory, необходима учетная запись Azure. Если у вас нет учетной записи, вы можете [зарегистрироваться для получения бесплатной учетной записи Azure](http://azure.microsoft.com/pricing/free-trial/).
