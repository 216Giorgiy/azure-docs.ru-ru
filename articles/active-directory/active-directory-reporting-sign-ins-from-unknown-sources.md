# Операции входа из неизвестных источников

| Описание | Расположение отчета |
| :-------------     | :-------        |
| <p>В этом отчете указываются пользователи, которые успешно выполнили вход в ваш каталог, при этом им назначен IP-адрес клиента, который распознается Майкрософт как IP-адрес анонимного прокси-сервера. Пользователи часто используют эти прокси-серверы, чтобы скрыть IP-адреса своих компьютеров. Иногда их используют злоумышленники. </p><p> В результатах этого отчета отображается количество успешных операций входа пользователя в ваш каталог со своего адреса и с IP-адреса прокси-сервера.</p> | Каталог > вкладка «Отчеты» |
