# Управление привилегированными пользователями Azure: выполнение проверки безопасности

## Выполнение проверки безопасности
[После запуска функции проверки безопасности](active-directory-privileged-identity-management-how-to-start-security-review.md) можно очень просто проверить привилегированный доступ.

## Для проверяющих: утверждение или запрещение доступа

### Проверка сведений о себе
1. В главном меню управления привилегированными пользователями выберите пункт **Проверка административного доступа**. Отобразится список проверок безопасности.
2. В этом списке выберите **пользователей**, для которых необходимо изменить права доступа. ПРИМЕЧАНИЕ. Права доступа будут изменены на самом деле. В результате этого процесса будет просто создан контрольный список для тех, кто будет изменять права доступа для роли. Если выбрать хотя бы одного пользователя, будут включены кнопки **Утвердить доступ** и **Запретить доступ**.
3. Нажмите кнопку **Утвердить доступ** или **Запретить доступ** для выбранных вами пользователей. В главном меню портала отобразится соответствующее уведомление, а список проверок исчезнет. Закройте колонку **Проверка ролей Azure AD**.

### Самостоятельная проверка
1. Пользователь получит электронное письмо, указывающее, что он должен самостоятельно проверить свои права доступа. В этом письме будет ссылка для входа на портал.
2. После этого пользователь сможет утвердить или запретить собственные права доступа, нажав кнопку **Утвердить доступ** или **Запретить доступ**. Его имя исчезнет из списка.

## Для руководителей по проверкам: управление проверками безопасности

## Завершение или останов проверки
1. Вернитесь на панель мониторинга управления привилегированными пользователями.
2. В списке **Проверки безопасности** щелкните проверку безопасности, которую необходимо завершить. Отобразится колонка со сведениями о проверке безопасности.
3. Для завершения или останова проверки щелкните **Остановить проверку**. После этого проверка будет помещена в архив, а колонка исчезнет.

## Экспорт проверки
Вы можете экспортировать проверку, чтобы использовать ее в Excel или другой программе, способной работать с CSV-файлами.

1. Вернитесь на панель мониторинга управления привилегированными пользователями.
2. На панели мониторинга щелкните раздел **Проверки безопасности**. Отобразится колонка **Проверки безопасности**.
3. Щелкните проверку безопасности, которую необходимо экспортировать. Отобразится колонка со сведениями о проверке безопасности.
4. Нажмите кнопку **Экспортировать**. Начнется скачивание CSV-файла.

## Удаление проверки

> ПРЕДУПРЕЖДЕНИЕ AZURE. Перед удалением не будет отображено никакого предупреждения, поэтому указывайте проверку, которую на самом деле *необходимо* удалить.

1. Вернитесь на панель мониторинга управления привилегированными пользователями.
2. На панели мониторинга щелкните раздел **Проверки безопасности**. Отобразится колонка **Проверки безопасности**.
3. Щелкните проверку безопасности, которую необходимо удалить. Отобразится колонка со сведениями о проверке безопасности.
4. Нажмите кнопку **Удалить**.
