# Управление владельцами группы

## Назначение владельцев группы

**Добавление владельца в группу на портале Azure**

1. На портале Azure щелкните **Active Directory**, а затем выберите имя каталога своей организации.
2. Щелкните вкладку **Группы**.
2. На странице **Группы** щелкните имя группы, в которую требуется добавить владельцев.
3. На странице группы щелкните **Добавить владельцев**.
4. На странице **Добавление владельцев** щелкните имя пользователя, которого нужно добавить в качестве владельца этой группы, и убедитесь, что это имя появилось в области **Выбранные**.


**Удаление владельца из группы на портале Azure**

1. На портале Azure щелкните **Active Directory**, а затем выберите имя каталога своей организации.
2. Щелкните вкладку **Группы**.
3. На странице **Группы** щелкните имя группы, из которой требуется удалить владельцев.
4. На странице группы щелкните вкладку **Владельцы**.
5. На странице группы щелкните имя владельца, которого нужно удалить из этой группы, и щелкните **Удалить**.
