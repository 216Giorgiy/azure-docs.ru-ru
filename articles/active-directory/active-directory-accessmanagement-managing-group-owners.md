# Управление владельцами группы

##Назначение владельцев группы
**Добавление владельца в группу на портале управления Azure**

1. На портале управления щелкните **Active Directory**, а затем выберите имя каталога вашей организации. 
2. Щелкните вкладку **Группы**.
2. На странице «Группы» щелкните имя группы, для которой требуется добавить владельцев. 
3. На странице группы щелкните **Добавить владельцев**.
4. На странице «Добавление владельцев» щелкните имя пользователя, которого хотите добавить в качестве владельца этой группы, и убедитесь, что это имя появилось в области **Выбранные**.


**Удаление владельца из группы на портале управления Azure**

1. На портале управления щелкните **Active Directory**, а затем выберите имя каталога вашей организации.
2. Щелкните вкладку **Группы**.
3. На странице «Группы» щелкните имя группы, из которой требуется удалить владельцев.
4. На странице группы щелкните вкладку **Владельцы**.
5. На странице группы щелкните имя владельца, которого хотите удалить из этой группы, и щелкните **Удалить**.
