# Управление привилегированными пользователями Azure: добавление и удаление роли пользователя

## Добавление и удаление ролей пользователей
Есть несколько способов перехода к колонке **Добавление управляемых пользователей** в интерфейсе управления привилегированными пользователями (PIM). Все они перечислены ниже.

- Панель мониторинга > Пользователи в роли администратора > Добавить или удалить
- Панель мониторинга > Сводка по ролям > Список всех пользователей > Добавить или удалить
- Панель мониторинга > щелкните роль пользователя в таблице ролей (например, глобальный администратор) > Добавить или удалить

## Добавление пользователя к роли
Перейдите к колонке **Добавление управляемых пользователей**.

1. Щелкните **Выбрать роль**. Если вы перешли сюда, щелкнув роль пользователя в таблице ролей, то роль уже будет выбрана.
2. Выберите роль, например **Администратор паролей**, в списке ролей. Откроется колонка **Выбор пользователей**.
3. Введите имя пользователя в поле поиска. По мере ввода будет отображаться список пользователей в каталоге, имена которых соответствуют критерию поиска.
4. Выберите пользователя в списке и нажмите кнопку **Готово**.
5. Нажмите кнопку **ОК**, чтобы сохранить изменение. Выбранный пользователь будет отображаться в списке, но роль будет временной.
6. Если вы хотите сделать роль постоянной, выберите пользователя в списке. Откроется отдельная колонка со сведениями о пользователе. Выберите в меню пользователя команду **сделать постоянной**.
7. Щелкните **Активировать**, чтобы запросить активацию роли пользователя. В текстовом поле **Причина запроса** введите обоснование активации новой роли. Роль пользователя будет автоматически активирована, и глобальным администраторам будет отправлено соответствующее уведомление.

## Удаление пользователя из роли
1. Найдите пользователя в списке ролей пользователей, используя перечисленные выше пути.
2. Щелкните имя пользователя в списке пользователей.
3. Щелкните **Удалить**. Появится запрос на подтверждение.
4. Щелкните **Да**, чтобы удалить роль пользователя.
