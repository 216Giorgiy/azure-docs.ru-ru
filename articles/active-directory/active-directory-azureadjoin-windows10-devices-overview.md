# Windows 10 для предприятия: использование устройств для работы

Windows 10 предоставляет возможность использовать Azure Active Directory. Устройства Windows 10 можно подключить к Azure AD, и пользователи могут входить в Windows под учетными записями Azure AD или могут добавить свой идентификатор Azure, чтобы получить доступ к ресурсам и бизнес-приложениям.
