# Добавление приложения в коллекцию приложений Azure Active Directory

Чтобы добавить приложение, поддерживающее единый вход с Azure Active Directory, в [коллекцию Azure AD](http://azure.microsoft.com/marketplace/active-directory/all/) и Azure Marketplace, группа Azure AD должна проверить интеграцию вашего приложения. Для этого необходимо отправить следующие сведения по адресу <waadpartners@microsoft.com>:

- имя пользователя и пароль тестовой учетной записи пользователя в тестовом экземпляре Azure Active Directory, которая может войти в приложение;
- URL-адрес и инструкции для входа этого тестового пользователя в приложение;
- краткое описание создания подключения между приложением и Azure AD. Для этого могут использоваться следующие способы:
  - встроенная в приложение функция на основе платформы согласия Azure AD;
  - использование Azure AD как поставщика удостоверений (SAML 2.0, WS-Federation или OpenID Connect) с помощью портала администрирования приложения;
  - создание подключения вручную с помощью коллеги в вашей организации.
- Если ваше приложение поддерживает настройку Azure AD как поставщика удостоверений с помощью портала самообслуживания, также укажите тестовую учетную запись администратора для проверки.
