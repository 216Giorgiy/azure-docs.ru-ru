# Приступая к работе с управлением привилегированными пользователями Azure AD


Управление привилегированными пользователями (PIM) в Azure Active Directory (AD) позволяет контролировать и отслеживать пользователей с привилегированными удостоверениями и доступ к ресурсам в Azure AD и других веб-службах Майкрософт, таких как Office 365 или Microsoft Intune.

В этой статье рассказывается, как добавить приложение Azure AD PIM на панель мониторинга портала Azure.

## Добавление приложения для управления привилегированными пользователями

Прежде чем вы сможете работать с приложением для управления привилегированными пользователями Azure AD, его необходимо добавить на панель мониторинга портала Azure.

1. Войдите на [портал Azure](https://portal.azure.com/) как глобальный администратор нужного каталога.
2. Если в организации имеется несколько каталогов, щелкните свое имя пользователя в правом верхнем углу портала Azure и выберите каталог, в котором вы будете использовать управление привилегированными пользователями.
3. Щелкните значок **Создать** в области навигации слева.
4. Выберите **Безопасность + удостоверения**.
5. Выберите **Управление привилегированными пользователями Azure AD**.
6. Установите флажок **Закрепить на панели мониторинга** и нажмите кнопку **Создать**. Откроется приложение "Управление привилегированными пользователями".


Если вы первый пользователь, использующий управление привилегированными пользователями Azure AD в каталоге, запустится [мастер защиты](active-directory-privileged-identity-management-security-wizard.md), который поможет вам выполнить первое назначение. После этого вы автоматически станете первым **администратором безопасности** каталога. Доступ к этому приложению для управления правами доступа других администраторов есть только у администратора безопасности.

С другой стороны, если вам была назначена одна или несколько ролей другим администратором безопасности, вы можете выбрать нужную роль и активировать ее. Если вы сами являетесь администратором безопасности, вам также будет доступно **управление удостоверениями**.
