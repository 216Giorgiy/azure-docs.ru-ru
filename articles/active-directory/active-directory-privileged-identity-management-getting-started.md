# Начало работы с функцией управления привилегированными пользователями Azure

## Запуск расширения для управления привилегированными пользователями

1.  Создайте каталог в Azure Active Directory.
2.  Добавьте пользователей в ваш Azure Active Directory.
3.  Создайте несколько глобальных администраторов в вашем каталоге.
4.  Войдите на портал <http://portal.azure.com> с помощью учетной записи глобального администратора.
5.  Щелкните **имя пользователя** в верхнем левом углу портала.
6.  Выберите **каталог**, который вы только что создали.
7.  Щелкните значок **Создать** в области навигации слева.
8.  В меню "Создание" выберите пункт **Безопасность и идентификатор**.
9.  В меню **Безопасность и идентификатор** выберите пункт **Управление привилегированными пользователями Azure AD**.
10. Установите флажок **Закрепить на начальной панели**, а затем нажмите кнопку **Создать**. Откроется панель мониторинга управления привилегированными пользователями.
