# Приступая к работе с управлением привилегированными пользователями Azure AD

## Запуск расширения для управления привилегированными пользователями

1. Создайте каталог в Azure Active Directory.
2. Добавьте в него пользователей, включая нескольких глобальных администраторов.

    >[AZURE.NOTE] Настройку пользователей необходимо проводить с использованием их рабочих или учебных учетных записей. Учетные записи Майкрософт сейчас нельзя использовать для управления привилегированными пользователями Azure AD.

3. Войдите на [портал Azure](https://portal.azure.com/) как глобальный администратор этого каталога.
4. В правом верхнем углу портала щелкните свое имя.
5. Выберите только что созданный каталог.
6. Щелкните значок **Создать** в области навигации слева.
7. В меню "Создание" выберите пункт **Безопасность и идентификатор**.
8. Выберите **Управление привилегированными пользователями Azure AD**.
9. Оставьте флажок **Закрепить на панели мониторинга** установленным и нажмите кнопку **Создать**. Откроется панель мониторинга управления привилегированными пользователями.
