# Что делать, если вы изменили регистратор DNS для пользовательского доменного имени

Если вы используете пользовательские доменные имена только в Azure AD, никакие дополнительные настройки конфигурации в Azure AD не потребуются.

## Если используются другие службы

Если вы используете пользовательские доменные имена в Office 365, Intune или других службах, учитывающих пользовательские доменные имена, обратитесь к документации этих служб.

-   [Использование пользовательских доменов с Office 365](https://support.office.com/article/Add-your-users-and-domain-to-Office-365-6383f56d-3d09-4dcb-9b41-b5f5a5efd611?ui=ru-RU&rs=ru-RU&ad=US)

-   [Использование пользовательских доменов с Intune](https://technet.microsoft.com/library/dn646966.aspx#BKMK_DomainNames)
