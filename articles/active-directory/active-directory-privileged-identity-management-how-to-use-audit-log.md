# Управление привилегированными пользователями Azure AD: как использовать журнал аудита

Вы можете использовать журнал аудита управления привилегированными пользователями (PIM) для просмотра всех назначенных пользователям и активированных ролей за определенный период времени.

## Переход к журналу аудита
Вы можете открыть журнал аудита, щелкнув **Журнал аудита** на панели мониторинга PIM.

## Диаграмма журнала аудита
Вы можете использовать диаграмму журнала аудита для просмотра общего количества активаций, а также максимального и среднего количества активаций в день. Если в журнале аудита представлены данные для нескольких ролей, их можно отфильтровать.

Для сортировки по времени, действию или роли используйте кнопки **время**, **действие** и **роль**.

## Список журнала аудита
Список журнала аудита содержит следующие столбцы:

- **Запрашивающая сторона** — пользователь, запросивший активацию роли.
- **Пользователь** — пользователь, для которого активируется роль.
- **Роль** — роль, назначенная пользователю.
- **Действие** — действия, выполненные с пользователем или ролью.
- **Время** — время выполнения действия.
- **Обоснование**. Если при активации в поле «Причина» был введен какой-либо текст, он будет отображаться здесь.
- **Срок действия**. Если отображается 9999 год, то роль пользователя постоянна.

## Фильтрация журнала аудита

Вы можете фильтровать информацию в журнале аудита, используя кнопку **Фильтр**. Откроется колонка **Обновление параметров диаграммы**.

### Изменение диапазона дат
Чтобы изменить временной диапазон журнала аудита, нажмите кнопку **Сегодня**, **Прошлая неделя**, **Прошлый месяц** или **Другое**. При выборе варианта **Другое** появятся поля **начальной** и **конечной** дат, что позволит вам самостоятельно выбрать временной диапазон журнала. Можно ввести дату в формате ММ/ДД/ГГГГ или щелкнуть значок **календаря** и выбрать дату в нем.

### Изменение ролей, включенных в журнал

Установите флажок **Роль** рядом с каждой ролью, которую требуется включить в журнал, и снимите его рядом с остальными.

Настроив все необходимые фильтры для журнала аудита, нажмите кнопку «Обновить», чтобы отфильтровать данные в нем. Если данные не появятся сразу же, обновите страницу.
