# Время обратного заполнения отчетов Azure Active Directory
*Данная документация является частью [руководства по отчетам Azure Active Directory](active-directory-reporting-guide.md).*

После включения отчетов для каталога отчеты заполняются данными за определенное количество дней, указанное здесь.

| Отчет | Описание |
| --- | --- |
| Попытки входа из неизвестных источников |0 дней |
| "Операции входа после нескольких неудачных попыток"; |0 дней |
| "Операции входа из нескольких географических регионов". |0 дней |
| Попытки входа с IP-адресов с подозрительными действиями |0 дней |
| Попытки входа с возможно инфицированных устройств |0 дней |
| Нестандартные действия при входе |0 дней |
| Пользователи с аномальными событиями при входе |0 дней |
| Пользователи с утерянными учетными данными |0 дней |
| Отчет аудита |30 дней |
| Действие сброса пароля (Azure AD) |0 дней |
| Действие сброса пароля (Диспетчер удостоверений) |0 дней |
| Событие регистрации сброса пароля (Azure AD) |0 дней |
| Событие регистрации сброса пароля (Диспетчер удостоверений) |0 дней |
| Действия групп по самообслуживанию (Azure AD) |0 дней |
| Действия групп по самообслуживанию (Диспетчер удостоверений) |0 дней |
| Использование приложений |0 дней |
| Действия по подготовке учетных записей |0 дней |
| Состояние смены пароля |0 дней |
| Ошибки подготовки учетной записи |0 дней |
| Использование RMS |0 дней |
| Самые активные пользователи RMS |0 дней |
| Использование устройства с RMS |0 дней |
| Использование приложений с поддержкой RMS |0 дней |
