# <a name="overview-about-adding-accounts-to-the-microsoft-authenticator-app"></a>Общие сведения о добавлении учетных записей в приложение Microsoft Authenticator

>[!Important]
>Эти материалы предназначены для пользователей. Администраторы могут найти дополнительные сведения о том, как настроить и контролировать среду Azure Active Directory (Azure AD), в [документации по Azure Active Directory](https://docs.microsoft.com/azure/active-directory).

Приложение Microsoft Authenticator позволяет войти в учетные записи при использовании двухфакторной проверки подлинности. Двухфакторная проверка подлинности обеспечивает более безопасный доступ к учетным записям, особенно если просматривается конфиденциальная информация. Так как пользователи могут забыть пароли и их могут украсть или скомпрометировать, двухфакторная проверка подлинности — это дополнительный этап обеспечения безопасности, который усиливает защиту учетной записи от несанкционированного доступа.

Приложение Microsoft Authenticator можно использовать несколькими способами, в том числе:

- Предоставление запроса на второй способ проверки после входа с использованием имени пользователя и пароля.

- Обеспечение входа без пароля — с использованием имени пользователя и данных отпечатка, лица или ПИН-кода на мобильном устройстве.

## <a name="what-account-types-can-i-add"></a>Какие типы учетных записей можно добавить
Вы можете включить двухфакторную проверку подлинности для множества учетных записей, которые вы используете каждый день, в том числе:

|Тип учетной записи|ОПИСАНИЕ|
|------------|-----------|
|Рабочие или учебные учетные записи|Учетная запись, предоставленная вашей компанией или учебным заведением, например alain@contoso.com.|
|Личные учетные записи|Учетные записи Майкрософт или сторонних компаний, которые вы открыли лично, такие как Outlook.com, Xbox LIVE, Google, Facebook и многие другие.|
