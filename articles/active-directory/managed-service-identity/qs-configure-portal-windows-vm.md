# <a name="configure-a-vm-managed-service-identity-using-the-azure-portal"></a>Настройка управляемого удостоверения службы виртуальной машины Azure с помощью портала Azure

[!INCLUDE[preview-notice](../../../includes/active-directory-msi-preview-notice.md)]

Управляемое удостоверение службы предоставляет службы Azure с автоматически управляемыми удостоверениями в Azure Active Directory. Это удостоверение можно использовать для аутентификации в любой службе, которая поддерживает аутентификацию Azure AD, не храня какие-либо учетные данные в коде. 

Из этой статьи вы узнаете, как включить и отключить назначаемое системой и пользователем удостоверение для виртуальной машины Azure с помощью портала Azure. 

## <a name="prerequisites"></a>Предварительные требования

- Если вы не работали с компонентом "Управляемое удостоверение службы", изучите [общие сведения](overview.md).
- Если у вас нет учетной записи Azure, [зарегистрируйтесь для получения бесплатной пробной учетной записи](https://azure.microsoft.com/free/), прежде чем продолжать.
- Для выполнения операций управления, описанных в этой статье, учетной записи требуется следующая роль.
    - [Участник виртуальных машин](/azure/role-based-access-control/built-in-roles#virtual-machine-contributor) для включения и удаления назначаемого системой удостоверения на виртуальной машине Azure.

## <a name="system-assigned-identity"></a>Системное удостоверение

В этом разделе вы узнаете, как включить и отключить назначаемое системой удостоверение для виртуальной машины с помощью портала Azure.

### <a name="enable-system-assigned-identity-during-creation-of-a-vm"></a>Включение назначаемого системой удостоверения во время создания виртуальной машины

Сейчас портал Azure не поддерживает включение назначаемого системой удостоверения во время создания виртуальной машины. Вместо этого ознакомьтесь с одним из следующих кратких руководств по созданию виртуальной машины, а затем перейдите к следующему разделу со сведениями о включении назначаемого системой удостоверения в виртуальной машине:

- [Создание виртуальной машины Windows с помощью портала Azure](../../virtual-machines/windows/quick-create-portal.md#create-virtual-machine)
- [Создание виртуальной машины Linux с помощью портала Azure](../../virtual-machines/linux/quick-create-portal.md#create-virtual-machine)  

### <a name="enable-system-assigned-identity-on-an-existing-vm"></a>Включение назначаемого системой удостоверения в имеющейся виртуальной машине

Чтобы включить системное удостоверения на виртуальной машине, которая изначально была подготовлена без него, сделайте следующее.

1. Войдите на [портал Azure](https://portal.azure.com) с помощью учетной записи, связанной с подпиской Azure, которая содержит виртуальную машину.

2. Перейдите к нужной виртуальной машине и выберите **Удостоверение**.

3. На вкладке **Назначено системой** в области **Состояние** нажмите кнопку **Вкл.**, а затем — **Сохранить**.

   ![Снимок экрана страницы "Конфигурация"](../managed-service-identity/media/msi-qs-configure-portal-windows-vm/create-windows-vm-portal-configuration-blade.png)  

### <a name="remove-system-assigned-identity-from-a-vm"></a>Удаление назначаемого системой удостоверения из виртуальной машины

При наличии виртуальной машины, которой больше не требуется системное удостоверение, сделайте следующее.

1. Войдите на [портал Azure](https://portal.azure.com) с помощью учетной записи, связанной с подпиской Azure, которая содержит виртуальную машину. 

2. Перейдите к нужной виртуальной машине и выберите **Удостоверение**.

3. На вкладке **Назначено системой** в области **Состояние** нажмите кнопку **Выкл.**, а затем — **Сохранить**.

   ![Снимок экрана страницы "Конфигурация"](../managed-service-identity/media/msi-qs-configure-portal-windows-vm/create-windows-vm-portal-configuration-blade-disable.png)

## <a name="user-assigned-identity"></a>Удостоверение, назначенное пользователем

 В этом разделе вы узнаете, как добавить и удалить назначаемое пользователем удостоверение для виртуальной машины с помощью портала Azure.

### <a name="assign-a-user-assigned-identity-during-the-creation-of-a-vm"></a>Добавление назначаемого пользователем удостоверения во время создания виртуальной машины

Сейчас портал Azure не поддерживает присвоение назначаемого пользователем удостоверения во время создания виртуальной машины. Вместо этого ознакомьтесь с одним из следующих кратких руководств по созданию виртуальной машины, а затем перейдите к следующему разделу со сведениями о присвоении виртуальной машине назначаемого пользователем удостоверения:

- [Создание виртуальной машины Windows с помощью портала Azure](../../virtual-machines/windows/quick-create-portal.md#create-virtual-machine)
- [Создание виртуальной машины Linux с помощью портала Azure](../../virtual-machines/linux/quick-create-portal.md#create-virtual-machine)

### <a name="assign-a-user-assigned-identity-to-an-existing-vm"></a>Присвоение назначаемого пользователем удостоверения имеющейся виртуальной машине

1. Войдите на [портал Azure](https://portal.azure.com) с помощью учетной записи, связанной с подпиской Azure, которая содержит виртуальную машину.
2. Перейдите в нужную виртуальную машину и выберите **Удостоверение**, **Назначено пользователем**, а затем — **\+Добавить**.

   ![Добавление назначаемого пользователем удостоверения в виртуальную машину](./media/msi-qs-configure-portal-windows-vm/add-user-assigned-identity-vm-screenshot1.png)

3. Выберите назначаемое пользователем удостоверение, которое нужно добавить в виртуальную машину, а затем нажмите кнопку **Добавить**.

    ![Добавление назначаемого пользователем удостоверения в виртуальную машину](./media/msi-qs-configure-portal-windows-vm/add-user-assigned-identity-vm-screenshot2.png)

### <a name="remove-a-user-assigned-identity-from-a-vm"></a>Удаление назначаемого пользователем удостоверения из виртуальной машины

1. Войдите на [портал Azure](https://portal.azure.com) с помощью учетной записи, связанной с подпиской Azure, которая содержит виртуальную машину.
2. Перейдите в нужную виртуальную машину и выберите **Удостоверение**, **Назначено пользователем**, выберите имя назначенного пользователем удостоверения, которое нужно удалить, а затем нажмите кнопку **Удалить** (выберите **Да** на панели подтверждения).

   ![Удаление назначаемого пользователем удостоверения из виртуальной машины](./media/msi-qs-configure-portal-windows-vm/remove-user-assigned-identity-vm-screenshot.png)
