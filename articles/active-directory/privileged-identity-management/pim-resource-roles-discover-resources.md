# <a name="discover-azure-resources-to-manage-in-pim"></a>Обнаружение ресурсов Azure и управление ими с помощью управления привилегированными пользователями

Узнайте, как обнаруживать ресурсы Azure и управлять ими с помощью управления привилегированными пользователями (PIM) в Azure Active Directory (Azure AD). Эта информация может быть полезна организациям, которые уже используют PIM для защиты ресурсов администратора, и владельцам подписок, которым требуется решение для защиты рабочих ресурсов.

При первой настройке управления привилегированными пользователями для ресурсов Azure следует найти и выбрать ресурсы для защиты. Количество ресурсов, которыми можно управлять с помощью PIM, не ограничено. Однако рекомендуется начать с наиболее важных (рабочих) ресурсов.

> [!NOTE]
> Для управления с помощью PIM можно выполнять поиск и выбирать только группу управления или ресурсы подписки. При управлении группой управления или подпиской с помощью PIM можно также управлять ее дочерними ресурсами.

## <a name="discover-resources"></a>Обнаружение ресурсов

На портале Azure перейдите к панели **Управление привилегированными пользователями**. В меню слева в разделе **Управление** выберите **Ресурсы Azure**.

![Панель "Управление привилегированными пользователями — ресурсы Azure"](media/azure-pim-resource-rbac/aadpim_manage_azure_resources.png)

Если вы впервые используете управление привилегированными пользователями для ресурсов Azure, сначала нужно найти доступные для управления ресурсы. На панели **Обнаружение ресурсов** нажмите кнопку **Обнаружить ресурсы**, чтобы запустить процесс обнаружения.

![Панель "Обнаружение ресурсов"](media/azure-pim-resource-rbac/aadpim_first_run_discovery.png)

Если управление привилегированными пользователями для ресурсов Azure уже используется другим администратором ресурса или каталога в вашей организации или вам назначена допустимая роль для ресурса, отображение списка содержит такое сообщение: **Discover resources or activate an eligible role assignment to continue** (Выполните обнаружение ресурсов или активируйте допустимую роль, чтобы продолжить процесс). 

![Кнопка "Обнаружить ресурсы" на панели "Управление привилегированными пользователями — ресурсы Azure"](media/azure-pim-resource-rbac/aadpim_discover_eligible_not_active.png)

После нажатия кнопки **Обнаружить ресурсы**, находящейся в верхнем меню или в центре панели, появится список подписок, которыми можно управлять. Выделенные подписки уже защищены с помощью PIM.

> [!NOTE]
> Чтобы другой администратор ресурсов не мог удалить настройки управления привилегированными пользователями, для уже заданной для управления подписки не поддерживается отмена управления.

![Панель "Ресурсы Azure — обнаружение"](media/azure-pim-resource-rbac/aadpim_discovery_some_selected.png)

В столбце **Ресурсы** наведите указатель мыши на подписку, которую требуется защитить с помощью PIM. Затем установите флажок слева от имени ресурса. Можно выбрать сразу несколько подписок.

![Список ресурсов на панели "Ресурсы Azure — обнаружение"](media/azure-pim-resource-rbac/aadpim_discovery_all_selected.png)

Чтобы начать процесс подключения, в верхнем меню щелкните **Управление ресурсом**.

![Кнопка "Управление ресурсом" на панели "Ресурсы Azure — обнаружение"](media/azure-pim-resource-rbac/aadpim_discovery_click_manage.png)

Теперь для выбранных ресурсов применяется управление привилегированными пользователями. Чтобы закрыть экран обнаружения, в правом верхнем углу нажмите кнопку **X**. Чтобы начать управление параметрами PIM и назначение членов, в меню в верхней части панели **Управление привилегированными пользователями — ресурсы Azure** нажмите кнопку **Обновить**.

![Кнопка "Обновить" в верхнем меню панели "Управление привилегированными пользователями — ресурсы Azure".](media/azure-pim-resource-rbac/aadpim_discovery_resources_refresh.png)
