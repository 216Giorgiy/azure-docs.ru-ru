# <a name="discover-and-manage-azure-resources"></a>Поиск ресурсов Azure и управление ими

Если ваша организация уже использует управление привилегированными пользователями Azure AD для защиты администраторов в каталоге или вы изучаете способы защиты ресурсов в рабочей среде вашей подписки, эта статья предназначена именно для вас.

При первом включении управления привилегированными пользователями для ресурсов Azure следует обнаружить и выбрать ресурсы для защиты. Не существует ограничений на число ресурсов, которыми можно управлять с помощью управления привилегированными пользователями, но мы рекомендуем сперва применить защиту для наиболее важных (рабочих) ресурсов.

> [!Note]
> Определять и выбирать для управления можно только ресурсы подписки. Если вы выберете подписку для управления в PIM, управление будет применено и ко всем дочерним ресурсам.

## <a name="discover-resources"></a>Обнаружение ресурсов

Перейдите к управлению привилегированными пользователями Azure AD и выберите нужные ресурсы Azure в разделе "Управление" в меню навигации слева.

![](media/azure-pim-resource-rbac/aadpim_manage_azure_resources.png)

Если вы впервые настраиваете управление привилегированными пользователями для ресурсов Azure, нужно найти доступные для управления ресурсы.
Нажмите кнопку "Обнаружение ресурсов" в центре экрана, чтобы запустить процесс обнаружения.

![](media/azure-pim-resource-rbac/aadpim_first_run_discovery.png)

Если управление привилегированными пользователями для ресурсов Azure уже используется другим администратором ресурса или каталога в вашей организации, или вам назначена допустимая роль для ресурса, отображение списка будет содержать такое сообщение: "Discover resources or activate an eligible role assignment to continue" (Выполните обнаружение ресурсов или активируйте допустимую роль, чтобы продолжить процесс). 

![](media/azure-pim-resource-rbac/aadpim_discover_eligible_not_active.png)

Нажав кнопку обнаружения ресурсов на панели действий или в центре экрана, вы увидите список доступных для управления подписок. Если на этом этапе здесь есть выделенные подписки, значит они уже защищены с помощью управления привилегированными пользователями.

> [!Note]
> Чтобы другой администратор ресурсов не мог удалить настройки управления привилегированными пользователями, для подписки не поддерживается отмена управления.

![](media/azure-pim-resource-rbac/aadpim_discovery_some_selected.png)

Наведите указатель мыши на подписку, которую вы хотите защитить с помощью управления привилегированными пользователями, и установите флажок на левом краю строки. Можно выбрать сразу несколько подписок.

![](media/azure-pim-resource-rbac/aadpim_discovery_all_selected.png)

Чтобы запустить процесс подключения, нажмите кнопку "Управление ресурсами" в строке в верхней части экрана.

![](media/azure-pim-resource-rbac/aadpim_discovery_click_manage.png)

Теперь для выбранных ресурсов применяется управление привилегированными пользователями. Закройте экран обнаружения с помощью кнопки "X" в правом верхнем углу страницы, затем щелкните "Обновить" на панели в верхней части экрана "Управление ресурсами Azure", чтобы настроить параметры и назначить члены для управления привилегированными пользователями.

![](media/azure-pim-resource-rbac/aadpim_discovery_resources_refresh.png)
