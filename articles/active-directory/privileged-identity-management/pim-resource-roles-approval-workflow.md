# <a name="approve-or-deny-requests-for-azure-resource-roles-in-pim"></a>Подтверждение или отклонение запросов для ролей ресурсов Azure в PIM

Чтобы утвердить или отклонить запрос, вы должны быть участником в списке утверждающих. 

1. В PIM на вкладке в меню слева выберите **Утвердить запросы** и найдите запрос.

   ![Область "Утверждение запросов"](media/azure-pim-resource-rbac/aadpim_rbac_approve_requests_list.png)

2. Выберите запрос, обоснуйте свой выбор и выберите **Утвердить** или **Отклонить**. Запрос будет обработан.

   ![Выбранный запрос с подробными сведениями](media/azure-pim-resource-rbac/aadpim_rbac_approve_request_approved.png)

## <a name="workflow-notifications"></a>Уведомления в рамках рабочего процесса

Ниже приводятся некоторые факты об уведомлениях в рамках рабочего процесса.

- Все участники в списке утверждающих получают уведомления по электронной почте, когда требуется рассмотреть запрос на утверждение роли. В этих уведомлениях содержится прямая ссылка на запрос, который утверждающий может подтвердить или отклонить.
- Запросы разрешаются первым участником в списке. Он утверждает или отклоняет запрос. 
- Когда утверждающий отвечает на запрос, все участники в списке утверждающих получают уведомление об этом действии. 
- Администраторы ресурсов получают уведомление, когда роль участника, запрос которого утвержден, становится активной. 

>[!Note]
>Если администратор ресурсов считает, что не нужно активировать роль для участника, запрос которого был утвержден, он может удалить назначение активной роли в PIM. Хотя администраторы ресурсов не получают уведомления об ожидающих запросах (если только они не являются участниками в списке утверждающих лиц), они могут просмотреть и отменить запросы в режиме ожидания для всех пользователей, просмотрев эти запросы в PIM.
