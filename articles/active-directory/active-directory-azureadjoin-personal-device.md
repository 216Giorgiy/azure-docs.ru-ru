# Присоединение личного устройства к своей организации

Присоединение личного устройства с Windows 10 к своей организации
--------------------------------------------------------------------------------------------
1.	В меню **Пуск** выберите **Параметры**. 
2.	Выберите пункт **Учетные записи**, а затем нажмите кнопку **Ваша учетная запись**.
3.	Выберите команду **Добавить рабочую или учебную учетную запись** и введите свою корпоративную учетную запись.
4.	Откроется страница входа организации. Введите имя пользователя и пароль и нажмите кнопку **ОК**.
5.	Затем появится окно многофакторной проверки подлинности. Проверка подлинности настраивается ИТ-специалистами компании.
6.	Azure AD проверит, требуется ли дополнительная регистрация пользователя или устройства в рамках политики управления мобильными устройствами (MDM). 
7.	Windows зарегистрирует устройство в каталоге организации в Azure AD, а также в MDM.
8.	Если вы являетесь управляемым пользователем, Windows завершит процесс установки, автоматически выполнит вход и отобразит рабочий стол пользователя.
9.	Если вы являетесь федеративным пользователем, то появится экран входа в Windows, где нужно будет ввести учетные данные.
