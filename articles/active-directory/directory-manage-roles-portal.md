# <a name="view-all-members-of-an-administrator-role-in-azure-active-directory"></a>Просмотр всех членов роли администратора в Azure Active Directory

Теперь вы можете просматривать всех членов роли администратора на портале Azure Active Directory и управлять ими. Если вы часто управляете назначениями ролей, возможно, вам пригодится эта функция. Если вы когда-нибудь задумывались, для чего нужны все эти роли, просмотрите подробный список разрешений для каждой роли администратора в Azure AD.

Вы можете увидеть и свои разрешения. Нажмите **Ваша роль**, чтобы открыть свою страницу пользователя и просмотреть список своих активных назначенных ролей. Нажмите на многоточие справа от каждой строки, чтобы открыть подробное описание роли.

![Список ролей на портале Azure AD](./media/directory-manage-roles-portal/role-list.png)

Выберите всю строку, чтобы просмотреть список назначенных членов. Нажмите **Управление в PIM**, чтобы открыть дополнительные возможности управления. Администраторы привилегированных ролей могут менять "постоянные" назначения (пользователь всегда выступает в этой роли) на "разрешенные" (роль назначается в отдельных случаях). Если у вас нет PIM, вы можете нажать **Управление в PIM**, чтобы зарегистрироваться для получения пробной версии. Для управления привилегированными пользователями (PIM) требуется [план лицензирования Azure AD Premium P2](./privileged-identity-management/subscription-requirements.md).

![Список членов роли администратора](./media/directory-manage-roles-portal/member-list.png)

Если вы являетесь глобальным администратором или администратором привилегированной роли, вы можете добавлять или удалять членов, отфильтровывать список или нажимать на участников, чтобы перейти на их страницу пользователя и просмотреть активные назначенные роли. 

## <a name="role-details-in-the-portal"></a>Сведения о ролях на портале

При просмотре членов роли нажмите **Описание**, чтобы просмотреть полный список разрешений для этой роли. Страница содержит ссылки на соответствующую документацию со сведениями об управлении ролями в каталоге.

![список разрешений для роли администратора](./media/directory-manage-roles-portal/role-description.png)
