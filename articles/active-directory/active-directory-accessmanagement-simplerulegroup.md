# Создание простого правила для настройки динамического членства для группы

**Чтобы включить динамическое членство для определенной группы, выполните следующие действия:**

1. На портале управления Azure на вкладке **Группы** выберите группу, которую необходимо изменить, и затем на ее вкладке **Настройка** для параметра **Включить динамическое членство** установите значение **Да**.


2. Теперь можно настроить простое правило для группы, которое будет управлять динамическим членством для функций этой группы. Убедитесь, что переключатель **Добавить пользователей:** установлен, затем выберите свойство пользователя из раскрывающегося меню (например, department, jobTitle и т. д.).

3. После этого выберите условие («Не равно», «Равно», «Не начинается с», «Начинается с», «Не содержит», «Содержит», «Не соответствует», «Соответствует») и укажите значение для выбранного свойства пользователя. Например, если назначена группа для приложения SaaS и вы включаете динамическое членство для этой группы, задав правило, согласно которому для параметра **Добавить пользователей:** задано свойство jobTitle, которое равно (-eq) Sales Rep, то все пользователи в каталоге Azure AD, название должности которых — Sales Rep, будут иметь доступ к этому приложению SaaS.
