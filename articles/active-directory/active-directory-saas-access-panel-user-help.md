# <a name="do-you-need-help-with-the-my-apps-portal"></a>Нужна помощь по работе с порталом "Мои приложения"?

Скорее всего, вы попали на эту страницу из-за проблем с использованием портала "Мои приложения". Зачастую для устранения проблемы необходимо обратиться в службу технической поддержки или к администратору. Здесь же приведены ответы на часто задаваемые вопросы, которые могут помочь справиться с проблемой самостоятельно.

## <a name="i-am-having-trouble-signing-into-the-my-apps-portal"></a>У меня возникают проблемы при входе на портал "Мои приложения"

Общие проблемы, которые следует проверить:

- Убедитесь в том, что вы выполняете вход с правильным URL-адресом: [https://myapps.microsoft.com](https://myapps.microsoft.com)

- Попробуйте добавить этот URL-адрес в список надежных сайтов в браузере.

- Удостоверьтесь в том, что пароль действителен и вы не забыли его. Дополнительные сведения о смене пароля см. в [этой статье](active-directory-passwords-update-your-own-password.md).

- Убедитесь в том, что ваши контактные данные для проверки подлинности актуальны и доступ не блокируется из-за них. Дополнительные сведения о настройке данных для проверки подлинности см. в [этой статье](https://docs.microsoft.com/azure/multi-factor-authentication/end-user/multi-factor-authentication-end-user).

- Попробуйте очистить файлы cookie в браузере и повторите попытку входа.

Если при попытке входа по-прежнему возникают проблемы, обратитесь к своему администратору за дальнейшей помощью.


## <a name="how-do-i-update-my-password"></a>Как сменить свой пароль?

Если вы забыли пароль или хотите изменить его, ИТ-специалисты не давали его вам или ваша учетная запись заблокирована, см. статью [Я не помню свой пароль Azure AD](active-directory-passwords-update-your-own-password.md).

## <a name="how-do-i-register-for-password-reset"></a>Как зарегистрироваться для сброса пароля?

Как пользователь вы можете самостоятельно сбросить свой пароль или разблокировать учетную запись, не обращаясь к администратору. Прежде чем использовать эту функцию, необходимо зарегистрировать методы аутентификации или подтвердить стандартные методы, которые заполнил администратор. Дополнительные сведения см. в статье [Регистрация для самостоятельного сброса пароля](active-directory-passwords-reset-register.md).


## <a name="i-am-having-trouble-installing-the-my-apps-secure-sign-in-extension"></a>У меня возникли трудности, установив Мои приложения безопасного входа расширение

Проверьте, выполняются ли требования к браузеру.

- Для работы с порталом требуется браузер с включенной поддержкой JavaScript и CSS. Если вы используете приложения единого входа по паролю, необходимо установить соответствующее вспомогательное расширение. Это расширение автоматически скачивается при запуске приложения, настроенного для работы с единым входом по паролю.

- Для использования расширения предъявляются следующие требования к браузеру:
    - Edge в Windows 10 Anniversary Edition или более поздней версии;
    - Chrome в Windows 7 или более поздней версии и в Mac OS X;
    - Firefox 26.0 или более поздней версии в Windows XP с пакетом обновления 2 (SP2) или более поздней версии и в Mac OS X 10.6 или более поздней версии.
    - Internet Explorer 8, 9, 10, 11 в Windows 7 или более поздней версии (ограниченная поддержка)

Чтобы скачать расширение для Chrome и Edge, воспользуйтесь следующими ссылками:

- [Расширение для Chrome](https://chrome.google.com/webstore/detail/access-panel-extension/ggjhpefgjjfobnfoldnjipclpcfbgbhl)

- [Расширение для Edge](https://www.microsoft.com/store/apps/9pc9sckkzk84)

Если после установки возникают проблемы, попробуйте выполнить указанные ниже действия.

- В параметрах расширений браузера проверьте, включено ли расширение.

- Перезапустите браузер и войдите на портал "Мои приложения".

- Очистите файлы cookie в браузере и войдите на портал "Мои приложения".
- Выполните [Устранение расширение панели доступа для Internet Explorer](https://docs.microsoft.com/en-us/azure/active-directory/active-directory-saas-ie-troubleshooting) руководство доступ средства диагностики и пошаговые инструкции по настройке расширения для IE.

## <a name="how-do-i-use-the-my-apps-secure-sign-in-extension"></a>Как использовать мои приложения безопасного входа расширение?
Изменение URL-адрес по умолчанию My Apps для расширения

Если вы используете другой Мои приложения URL-адрес чем https://myapps.microsoft.com необходимо настроить URL-адрес, по умолчанию хотя следующие действия:
1. Пока не выполнен вход в модуль, **щелкните правой кнопкой мыши** значок расширения.
2. Щелкните **выберите Мои URL-адрес приложения** в меню.
3. **Выберите** URL-адрес по умолчанию.
4. Щелкните значок расширения.
5. Войдите на расширении, выбрав **вход приступить к работе**.

Вход непосредственно в приложение из браузера
1. После установки расширения, войдите на расширении, выбрав **вход приступить к работе**.
2. Перейдите к **URL-адрес входа** вы хотите войти в приложения, обычно это URL-адрес приложения, которое отображает форму входа.
3. Расширение следует изменить состояние и позволяют знать пароль, щелкнув **значок расширения** входа

Запустите приложение из расширения
1. После установки расширения, войдите на расширении, выбрав **вход приступить к работе**.
2. Щелкните значок расширения, чтобы открыть соответствующее меню.
3. **Поиск** для приложения доступна на портале Мои приложения.
4. Щелкните приложение из **результаты поиска** для его запуска.
5. Последние три приложений, запущенных также будут отображаться в **недавно использованные** ярлык списка

> [!NOTE]
> Эти параметры доступны только для Edge, Chrome, Firefox.

## <a name="how-do-i-add-a-new-app"></a>Как добавить новое приложение?

1.  На странице **Приложения** щелкните **Добавить приложение**.

2.  Найдите приложение, которое нужно добавить, и нажмите кнопку **Добавить**.

**Примечания:**

- Этот параметр будет доступен только в том случае, если администратор включил его для вашей учетной записи.

- Если приложению требуется разрешение, может понадобиться получить подтверждение от администратора.


## <a name="how-do-i-manage-my-group-memberships"></a>Как управлять членством в группах?

1. Нажмите кнопку **группы** плитки. 
2. Чтобы создать группу, в области группы, имеющееся, щелкните **создать группу**и следуйте инструкциям.
3. Чтобы присоединиться к группе, в области группы в, щелкните **присоединиться к группе**и следуйте инструкциям.

**Примечания:**

- Этот параметр будет доступен только в том случае, если администратор включил его для вашей учетной записи.

- Выбрав группу, в которой вы состоите, вы можете просмотреть подробные сведения и покинуть ее.

- Щелкнув группу, владельцем которой вы являетесь, вы можете просмотреть подробные сведения, добавить или удалить участников или покинуть группу.
