# <a name="azure-ad-self-service-password-reset-for-the-it-professional"></a>Самостоятельный сброс пароля в Azure AD для ИТ-специалистов

Самостоятельный сброс пароля (SSPR) Azure Active Directory (Azure AD) позволяет пользователям сбрасывать пароли самостоятельно, когда это требуется, а также дает администраторам контроль над сбросом паролей. Пользователям больше не требуется обращаться в службу поддержки, чтобы сбросить пароль.

* **Самостоятельная смена пароля**. Пользователь знает пароль, но хочет изменить его на новый.
* **Самостоятельный сброс пароля**. Пользователю не удается выполнить вход, и он хочет сбросить пароль, используя один или несколько из приведенных ниже проверенных методов аутентификации:
   * текстовое сообщение, отправляемое на проверенный мобильный телефон;
   * звонок на проверенный мобильный или рабочий телефон;
   * отправка сообщения в проверенную учетную запись электронной почты;
   * ответы на свои контрольные вопросы.
* **Самостоятельное разблокирование учетной записи**. Пользователю не удалось выполнить вход с паролем, и его учетная запись была заблокирована. Ему требуется разблокировать свою учетную запись без вмешательства администратора с помощью доступных методов аутентификации.

## <a name="why-choose-azure-ad-self-service-password-reset"></a>В чем преимущества самостоятельного сброса пароля в Azure AD

* **Снижение затрат.** На сброс пароля обычно приходится 20 % обращений в службу технической поддержки ИТ-организации. 
* **Повышение удобства работы пользователей** и **сокращение нагрузки на службу технической поддержки** благодаря предоставлению пользователям возможности сразу устранять собственные проблемы с паролями, не обращаясь в службу технической поддержки.
* **Поддержка мобильности.** Пользователи могут сбрасывать пароли независимо от своего местонахождения.
* **Контроль** на политиками безопасности. Администраторы могут по-прежнему применять уже имеющиеся политики.

Если вы готовы, вы можете приступить к работе с Azure AD SSPR, используя [краткое руководство по началу работы](active-directory-passwords-getting-started.md), и быстро реализовать возможность сброса паролей для пользователей.

## <a name="azure-ad-self-service-password-reset-availability"></a>Доступность самостоятельного сброса пароля в Azure AD

В зависимости от подписки самостоятельный сброс пароля в Azure AD доступен на трех уровнях.

* **Бесплатная подписка Azure AD.** Только администраторы облака могут сбрасывать свои пароли.
* **Azure AD Basic** или любая **платная подписка Office 365.** Пользователи облака могут сбрасывать свои пароли.
* **Azure AD Premium.** Любой пользователь или администратор, включая пользователей облака, федеративных пользователей и пользователей с синхронизацией паролей, могут сбрасывать свои пароли. Для локальных паролей требуется включить компонент обратной записи паролей.
