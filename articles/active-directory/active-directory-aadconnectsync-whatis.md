# Azure AD Connect Sync: настройка параметров синхронизации

Службы синхронизации Active Directory Azure (Azure AD Connect Sync) — это основной компонент Azure Connect AD. Этот компонент отвечает за все операции, относящиеся к синхронизации данных удостоверения между локальной средой и Azure AD в облаке. С точки зрения синхронизации Azure AD Connect Sync является преемником таких решений, как DirSync, Azure AD Sync и Forefront Identity Manager с настроенным соединителем Active Directory.
