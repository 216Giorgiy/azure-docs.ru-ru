# Операции входа из нескольких географических регионов

| Описание | Расположение отчета |
| :-------------     | :-------        |
| <p>В этом отчете содержится информация об успешных попытках входа пользователя, от имени которого были совершены две попытки входа из разных регионов, причем времени между этими попытками было недостаточно, чтобы пользователь мог добраться из одного региона в другой. Возможные причины:</p><ul><li>совместное использование пароля несколькими пользователями;</li><li>пользователь использует удаленный рабочий стол для запуска веб-браузера для входа;</li><li>злоумышленник вошел в учетную запись пользователя из другой страны.</li></ul><p>В результатах этого отчета будут показаны успешные события входа, время, разделяющее эти события, регионы, из которых они были выполнены, и рассчитанная продолжительность поездки из одного региона в другой.</p><p>Показанное время поездки — это лишь примерная оценка, и она может отличаться от фактической продолжительности поездки из одного расположения в другое. Кроме того, события не создаются для операций входа, выполненных в соседних регионах.</p> | Каталог > вкладка «Отчеты» |
