# <a name="register-for-self-service-password-reset"></a>Регистрация для самостоятельного сброса пароля

Если ваш администратор включил эту функцию, вы можете самостоятельно сбросить свой пароль (функция SSPR) или разблокировать учетную запись. Прежде чем использовать эту функцию, необходимо зарегистрировать методы аутентификации или подтвердить стандартные методы, которые заполнил администратор.

## <a name="register-or-confirm-authentication-data-with-sspr"></a>Регистрация или подтверждение данных проверки подлинности для SSPR

1. Откройте веб-браузер на устройстве и перейдите на [страницу регистрации для сброса пароля](http://aka.ms/ssprsetup)
2. Введите имя пользователя и пароль, предоставленные администратором
3. В зависимости от параметров, утвержденных вашим администратором, вы увидите один или несколько следующих элементов для настройки или подтверждения, которые используются, если необходимо сбросить пароль.
    * Рабочий телефон — этот параметр может указать только администратор.
    * Телефон для аутентификации — в качестве этого параметра необходимо указать другой номер телефона, к которому у вас есть доступ, например мобильный телефон, который может принимать текстовые сообщения или вызовы. Администратор может заполнить эти данные за вас, если у него есть разрешение использовать эту информацию.
    * Электронная почта для аутентификации — для этого параметра необходимо указать альтернативный электронный адрес с доступом без пароля, который вы хотите сбросить.
    * Секретные вопросы — этот параметр содержит утвержденный администратором список вопросов, на которые вы должны ответить. Вы не можете использовать один и тот же ответ для нескольких вопросов.
4. Укажите и проверьте сведения, необходимые администратору. Если доступно несколько вариантов, рекомендуется зарегистрировать несколько методов, чтобы выполнить аутентификацию, даже если другой метод недоступен (например, во время отъезда вы не можете получить доступ к рабочему телефону).

    ![Зарегистрируйте методы аутентификации и нажмите кнопку "Готово"][Register]

5. Выполнив шаг 4, нажмите кнопку **Готово**, и вы сможете выполнять самостоятельный сброс пароля, когда понадобится.

Если вы вводите номер телефона или адрес электронной почты для аутентификации, эти данные не отображаются в глобальном каталоге. Эти данные видите только вы и администраторы. Ответы на ваши секретные вопросы видите только вы.

Администраторы могут потребовать подтверждения аутентификации через определенное время, чтобы убедиться, что вы все еще используете правильные методы выполнения этой процедуры.
