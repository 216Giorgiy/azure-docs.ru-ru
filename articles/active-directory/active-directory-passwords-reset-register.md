# <a name="register-for-self-service-password-reset"></a>Регистрация для самостоятельного сброса пароля

Как пользователь вы можете самостоятельно сбросить свой пароль или разблокировать учетную запись, не обращаясь к администратору. Прежде чем использовать эту функцию, необходимо зарегистрировать методы аутентификации или подтвердить стандартные методы, которые заполнил администратор.

## <a name="register-or-confirm-authentication-data-with-sspr"></a>Регистрация или подтверждение данных проверки подлинности для SSPR

1. Откройте веб-браузер на устройстве и перейдите на [страницу регистрации для сброса пароля](http://aka.ms/ssprsetup)
2. Введите имя пользователя и пароль, предоставленные администратором
3. В зависимости от установленной вашим IT-специалистом конфигурации, вы сможете использовать для настройки и проверки один или несколько параметров. Администратор может заполнить некоторые данные за вас, если у него есть разрешение на использование этой информации.
    * Рабочий телефон — этот параметр устанавливает только администратор.
    * Телефон для аутентификации — здесь нужно указать другой телефонный номер, к которому у вас есть доступ, например мобильный телефон с возможностью приема текстовых сообщений или вызовов.
    * Электронная почта для аутентификации — здесь нужно указать запасной адрес электронной почты, доступ к которой осуществляется без использования пароля, который вы хотите сбросить.
    * Контрольные вопросы — этот параметр содержит утвержденный администратором список вопросов, на которые вы должны ответить. Вы не можете использовать один и тот же вопрос или ответ несколько раз.
4. Укажите и проверьте сведения, необходимые администратору. Если доступно несколько вариантов, рекомендуется зарегистрировать несколько методов аутентификации. Так вы обеспечите возможность доступа, когда один из методов будет недоступен (например, если вы в дороге и у вас нет доступа к рабочему телефону).

    ![Зарегистрируйте методы аутентификации и нажмите кнопку "Готово"][Register]

5. Выполнив шаг 4, нажмите кнопку **Готово**, и вы сможете выполнять самостоятельный сброс пароля, когда понадобится.

Если вы вводите номер телефона или адрес электронной почты для аутентификации, эти данные не отображаются в глобальном каталоге. Эти данные видите только вы и администраторы. Ответы на ваши секретные вопросы видите только вы.

Администраторы могут потребовать подтверждения аутентификации через определенное время, чтобы убедиться, что вы все еще используете правильные методы выполнения этой процедуры.
