# Пользователи с аномальными событиями при входе

| Описание | Расположение отчета |
| :-------------     | :-------        |
| <p>Используйте этот отчет, чтобы получать информацию обо всех учетных записях пользователей, для которых были обнаружены аномальные действия при входе. В этом отчете содержатся данные из всех остальных отчетов об аномальных действиях. В результатах этого отчета приводится подробная информация о пользователе, причины определения его события входа как аномального, дата, время и другая информация о событии.</p> | Каталог > вкладка «Отчеты» |
