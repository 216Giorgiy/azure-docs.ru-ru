# Пользователи с аномальными событиями при входе
Это совокупный отчет, который объединяет данные о подозрительных операциях входа из следующих отчетов: <ul><li>Операции входа из неизвестных источников</li>; <li>Операции входа после нескольких сбоев</li>; <li>Операции входа из нескольких географических регионов</li>; <li>Входы с IP-адресов с подозрительными действиями</li>; <li>Попытки входа с возможно инфицированных устройств</li>; <li>Нестандартные действия при входе</li>. </ul>
