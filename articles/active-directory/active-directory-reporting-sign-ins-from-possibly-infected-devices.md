# Операции входа с возможно инфицированных устройств

| Описание | Расположение отчета |
| :-------------     | :-------        |
| <p>Используйте этот отчет, чтобы получать информацию об операциях входа с устройств, на которых могут быть запущены вредоносные программы. Мы сопоставляем IP-адреса входа с IP-адресами, с которых была предпринята попытка подключения к серверу с вредоносными программами.</p><p>Рекомендация. Так как этот отчет предполагает, что IP-адрес в обоих случаях связан с одним устройством, мы советуем связаться с пользователем и для уверенности отсканировать его устройство.</p><p>Дополнительную информацию об устранении вредоносных программ см. в разделе [Центр по защите от вредоносных программ](http://go.microsoft.com/fwlink/?linkid=335773). </p> | Каталог > вкладка «Отчеты» |
