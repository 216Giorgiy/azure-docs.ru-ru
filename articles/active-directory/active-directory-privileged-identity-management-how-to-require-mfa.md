# Управление привилегированными пользователями Azure: как сделать использование MFA обязательным

Настоятельно рекомендуем обязательно использовать многофакторную проверку подлинности для всех администраторов.

##Как сделать использование MFA обязательным с помощью управления привилегированными пользователями Azure
Когда вы выполняете вход как администратор управления привилегированными пользователями, отображаются оповещения о необходимости обязательно использовать многофакторную проверку подлинности (MFA) для привилегированных учетных записей. Щелкните оповещение системы безопасности на панели мониторинга управления привилегированными пользователями. Отобразится новая колонка со списком учетных записей администраторов, для которых необходимо обязательно использовать MFA. Чтобы сделать использование MFA обязательным, выберите несколько ролей и нажмите кнопку **Исправить**. Вы также можете щелкнуть многоточия рядом с отдельными ролями, а затем нажать кнопку **Исправить**.

Еще можно щелкнуть конкретную роль в разделе "Роли" на панели мониторинга, а затем включить для нее использование MFA, щелкнув **Параметры** в колонке роли и выбрав пункт **Включить** рядом с обозначением многофакторной проверки подлинности.
