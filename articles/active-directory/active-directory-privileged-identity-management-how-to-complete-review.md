# Как выполнить проверку безопасности в компоненте управления привилегированными пользователями Azure AD


После [запуска функции проверки безопасности](active-directory-privileged-identity-management-how-to-start-security-review.md) администраторы безопасности могут проверить привилегированный доступ. Компонент управления привилегированными пользователями (PIM) Azure AD автоматически отправит пользователям электронное письмо с предложением проверить доступ. Если пользователь не получил электронное письмо, ему можно отправить [инструкции по выполнению проверки безопасности](active-directory-privileged-identity-management-how-to-perform-security-review.md).

После окончания периода проверки безопасности или после завершения всеми пользователями самостоятельной проверки выполните шаги, описанные в этой статье, для управления проверкой и просмотра результатов.

## Управление проверкой безопасности

1. Войдите на [портал Azure](https://portal.azure.com/) и выберите приложение **Управление привилегированными пользователями Azure AD** на панели мониторинга.
2. Выберите раздел **Проверка безопасности** на панели мониторинга. Отобразится колонка **Проверки безопасности**.
3. Выберите необходимую проверку безопасности. Отобразится колонка со сведениями о проверке безопасности.

В колонке со сведениями о проверке безопасности отображаются три варианта действий для этой проверки. Можно завершить проверку, экспортировать сведения о проверке или удалить проверку.

### Завершение или остановка проверки

Если доступ для пользователей запрещен, можно завершить проверку, чтобы удалить назначение роли для пользователя в каталоге. При нажатии на кнопку **Остановить проверку** проверка будет отправлена в архив.

### Экспорт проверки

Если вы хотите применить результаты проверки безопасности вручную, можно экспортировать проверку. При нажатии на кнопку **Экспорт** будет загружен CSV-файл. Открыть этот файл можно в Excel или в других программах, поддерживающих CSV-файлы.

### Удаление проверки

> [AZURE.IMPORTANT] Учтите, что при удалении проверки предупреждение не выдается. Поэтому убедитесь, что выбрана нужная проверка.

Если проверка вам больше не нужна, удалите ее. При нажатии на кнопку **Удаление** проверка будет удалена из приложения PIM.
