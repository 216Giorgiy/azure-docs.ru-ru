# Выделенные группы в Azure Active Directory

В Azure Active Directory выделенные группы создаются автоматически, и членство в выделенных группах также назначается автоматически. Нельзя добавлять или удалять участников выделенной группы с помощью портала Azure, командлетов Windows PowerShell или программно. Чтобы включить выделенные группы, на портале Azure на вкладке «Настройка» для переключателя **Включить выделенные группы** выберите значение «Да».

После включения выделенных групп, т. е. установки значения **Да**, можно дополнительно включить автоматическое создание выделенной группы «Все пользователи» в каталоге, задав для параметра **Включить группу «Все пользователи»** значение **Да**. Затем можно также изменить имя этой выделенной группы, введя его в поле **Отображаемое имя для группы «Все пользователи»**.

Выделенная группа «Все пользователи» может быть удобна, если требуется назначить одинаковые разрешения для всех пользователей в каталоге. Например, можно предоставить всем пользователям в каталоге доступ к приложению SaaS, назначив выделенной группе «Все пользователи» доступ к этому приложению.
