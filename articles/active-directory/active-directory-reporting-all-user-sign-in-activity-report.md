# Отчет Azure Active Directory "Все действия пользователей при входе"

*Данная документация является частью [руководства по отчетам Azure Active Directory](active-directory-reporting-guide.md).*

 >[AZURE.IMPORTANT]> Отчет "Все действия пользователей при входе" является расширенным отчетом в закрытой предварительной версии. Чтобы получить доступ к отчету, отправьте по электронной почте aadreportinghelp@microsoft.com с именем домена (например **contoso.onmicrosoft.com**).


Отчет "Все действия пользователей при входе" содержит все операции входа всех пользователей в каталоге, а также дату, время, UPN, IP-адрес и приложение, в которое выполнен вход.


 >[AZURE.IMPORTANT]> В настоящее время отчет "Все действия пользователей при входе" доступен только через API отчетов. Дополнительные сведения см. в статью [Приступая к работе с API отчетов](active-directory-reporting-api-getting-started.md).
