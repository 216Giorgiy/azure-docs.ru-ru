# Добавление фирменной символики на страницы входа и панели доступа

> [AZURE.NOTE]
> 
- Фирменная символика компании доступна только после обновления до выпуска Premium или Basic службы Azure Active Directory. Дополнительные сведения см. в статье [Выпуски Azure Active Directory](active-directory-editions.md).
- Выпуски Azure Active Directory Premium и Basic доступны для клиентов в Китае, использующих всемирный экземпляр Azure Active Directory. Выпуски Azure Active Directory Premium и Basic в текущий момент не поддерживаются Microsoft Azure, обслуживаемой компанией 21Vianet в Китае. Чтобы получить дополнительную информацию, свяжитесь с нами на [форуме Azure Active Directory](http://feedback.azure.com/forums/169401-azure-active-directory).

Многим компаниям требуется применить согласованное оформление для всех веб-сайтов и служб, которыми они управляют, чтобы пользователи не путались при работе с этими сайтами. Azure Active Directory предоставляет такую возможность, позволяя настраивать внешний вид следующих веб-страниц, доступных пользователям, добавляя на них логотип и цветовые схемы компании:

- **Страница входа** — пользователи перенаправляются на эту страницу, когда они входят в Office 365 или другие современные веб-приложения, использующие Azure AD в качестве поставщика удостоверений. Большинство пользователей взаимодействуют с этой страницей для обнаружения домашней области, что позволяет системе перенаправлять федеративных пользователей в локальную службу маркеров безопасности (AD FS), или для ввода учетных данных.

- **Страница панели доступа** — это веб-портал, который позволяет конечным пользователям с рабочей или школьной учетной записью в каталоге Azure AD просматривать и запускать облачные приложения, к которым администратор Azure AD предоставил доступ. Все пользователи организации могут открыть панель доступа на сайте myapps.microsoft.com.

## Настройка страницы входа

Страница входа обычно чаще всего применяется конечными пользователями, которым необходим доступ к облачным приложениям и службам вашей организации в браузере, поэтому очень важно, чтобы она выглядела соответствующим образом. Если вы хотите использовать страницу входа по умолчанию без фирменной символики, никаких действий не требуется.

### Когда изменения фирменной символики появятся на странице входа?

Для применения всех изменений фирменной символики страницы входа может потребоваться до одного часа.

### Когда пользователи увидят страницу входа с фирменной символикой?

Пользователи увидят страницу входа с фирменной символикой при посещении службы с URL-адресом определенного клиента, например https://outlook.com/**contoso**.com или https://mail.**contoso**.com (если вы создали запись CNAME).

При посещении службы без URL-адреса клиента (например, https://mail.office365.com) они увидят страницу входа без фирменной символики. Страница входа обновится, и на ней появится фирменная символика после ввода идентификатора пользователя или выбора плитки пользователя.

> [AZURE.NOTE]
> 
- Ваше доменное имя должно быть показано как "Активное" в разделе **Active Directory** > **Каталог** > **Домены** портала управления Azure, где была настроена фирменная символика.
- Фирменная символика страницы входа не применяется к странице входа потребителя корпорации Майкрософт. Это означает, что пользователи, входящие с помощью личной учетной записи Майкрософт (ранее Windows Live ID), могут увидеть список плиток пользователей, отображаемый Azure AD, но фирменная символика вашей организации не будет применяться к странице входа в учетную запись Майкрософт.

### Какие элементы на странице можно настраивать?

На странице входа можно настроить следующие элементы:

![][5]

 Элемент страницы | Расположение на странице
	------------- | -------------
Баннер с логотипом | Отображается в правой верхней части страницы. Заменяет логотип, который обычно отображается целевым сайтом, на который пользователи входят (например, Office 365 или Azure).
Большая иллюстрация и цвет фона | Отображается в левой части страницы. Заменяет изображение, которое обычно отображается целевым сайтом, на который пользователи входят. Вместо большой иллюстрации для соединений с низкой пропускной способностью или на очень узких экранах может отображаться цвет фона.
Текст страницы входа | Отображается над нижним колонтитулом страницы, если вам необходимо сообщить полезные сведения пользователям перед входом с помощью рабочей или школьной учетной записи. Например, вам может потребоваться указать номер телефона службы поддержки или юридическое заявление.

> [AZURE.NOTE]Все элементы необязательные. Например, если указать баннер с логотипом, но без большой иллюстрации, на странице входа будет отображаться логотип и иллюстрация для целевого сайта (т. е. изображение автострады Калифорнии из Office 365).

Вы также можете локализовать все элементы на этой странице. После настройки набора элементов по умолчанию можно настроить дополнительные версии для разных языков. Вы также можете смешивать разные элементы. Например, вы можете просматривать:

- Создайте большую иллюстрацию по умолчанию, подходящую для всех языков, а затем создайте отдельные версии для английского и русского языков. Пользователи с браузерами, в которых выбран один из этих двух языков, увидят соответствующее изображение, а все остальные будут видеть изображение по умолчанию.
- Настройте разные логотипы для организации (например, японскую и китайскую версии).

### Как будет отображаться иллюстрация после изменения размера браузера?

При изменении размера окна браузера большая иллюстрация, например показанная выше, практически всегда будет обрезана для адаптации к другим пропорциям экрана. С учетом этого следует попытаться сохранить основные визуальные элементы на иллюстрации, чтобы они всегда отображались в левом верхнем углу (в правом верхнем углу для языков с написанием справа налево). Это важно, поскольку изменение размера окна обычно происходит с правого нижнего угла по направлению вверх или влево или снизу вверх.

Если вы хотите использовать страницу панели доступа по умолчанию без фирменной символики, никаких действий не требуется.

## Настройка каталога с фирменной символики компании

На портале управления можно настроить один набор элементов по умолчанию для каждого каталога. После сохранения значений по умолчанию администратор также может добавить локализованные версии каждого элемента для различных языков и региональных стандартов. Все настраиваемые элементы необязательные.

Например, если настроить баннер с логотипом по умолчанию, но без большой иллюстрации, в правом верхнем углу страницы входа будет отображаться ваш логотип, но иллюстрация сайта по умолчанию. Если настроить баннер с логотипом и текст страницы входа на английском языке по умолчанию и текст страницы входа для немецкого языка, то пользователи с выбранным немецким языком будут видеть баннер с логотипом по умолчанию, но текст на немецком языке. Хотя технически вы можете настроить другой набор для каждого языка, поддерживаемого Azure AD, мы рекомендуем использовать небольшое количество вариантов в целях обслуживания и производительности.

Добавление фирменной символики компании в ваш каталог:

1. Войдите на [портал управления Azure](https://manage.windowsazure.com) как администратор каталога, который необходимо настроить.
2. Выберите нужный каталог.
3. Откройте вкладку **Настройка**, а затем щелкните **Настроить фирменную символику**.
4. Измените элементы, которые вы хотите настроить. Обратите внимание, что все поля необязательные.
5. Щелкните **Сохранить**.

Для применения всех изменений фирменной символики страницы входа может потребоваться до одного часа.

Добавление фирменной символики компании для конкретного языка:

1. На [портале управления Azure](https://manage.windowsazure.com) на вкладке **Настройка** выберите **Настроить фирменную символику**.
2. Выберите параметр **Добавить фирменную символику для определенного языка**, выберите язык, для которого необходимо настроить логотип, и нажмите кнопку **Далее**.
3. Измените только те элементы, для которых требуется настроить переопределение для другого языка. Обратите внимание, что все поля необязательные. Если поле остается пустым, то будет отображаться значение по умолчанию (или значение Майкрософт по умолчанию, если пользовательское значение не настроено).
4. Щелкните **Сохранить**.

Удаление фирменной символики компании из каталога

1. На [портале управления Azure](https://manage.windowsazure.com) на вкладке **Настройка** выберите **Настроить фирменную символику**.
2. На странице настройки фирменной символики выберите **Изменить существующие параметры фирменной символики** и перейдите к следующей странице.
3. В зависимости от того, какие элементы необходимо удалить, выполните одно или несколько из следующих действий.
	1. Для баннера с логотипом установите флажок **Удалить отправленный логотип**.
    2. Для логотипа плитки установите флажок **Удалить отправленный логотип**.
    3. Очистите весь текст метки имени пользователя на странице входа.
    4. Очистите весь текст страницы входа.
    5. Для иллюстрации страницы входа установите флажок, чтобы **Удалить иллюстрацию**.
    6. Очистите весь текст для цвета фона страницы входа.
4. Нажмите кнопку **Сохранить**, чтобы удалить элементы.
5. При необходимости щелкните **Настроить фирменную символику** еще раз и повторите эти действия для всех элементов фирменной символики, которые требуется удалить. Если нажать кнопку **Настроить фирменную символику** и просмотреть форму **Настройка фирменной символики по умолчанию** без настроенных параметров, все параметры фирменной символики будут удалены.

## Тестирование и примеры

Мы рекомендуем поэкспериментировать с тестовым клиентом перед внесением изменений в рабочую среду. Самый простой способ проверить, применена ли фирменная символика, — открыть сеанс браузера InPrivate или анонимный сеанс, а затем открыть адрес https://outlook.com/contoso.com, заменив contoso.com на настроенный вами домен. Обратите внимание, что это также работает с доменами, аналогичными contoso.onmicrosoft.com.

Чтобы помочь вам в создании эффективных наборов настройки, мы обработали следующие две фиктивные страницы входа:

- [http://aka.ms/aaddemo001](http://aka.ms/aaddemo001)
- [http://aka.ms/aaddemo002](http://aka.ms/aaddemo002)

Чтобы проверить параметры для определенного языка, необходимо выбрать в качестве языка по умолчанию в браузере язык, заданный в вашей настройке. В Internet Explorer эти параметры можно настроить в меню **Параметры браузера**.

Логотипы компании можно настроить один раз для каждого каталога и использовать их как для страницы входа, так и для панели доступа, но при этом некоторые настраиваемые элементы применяются только на странице входа.
