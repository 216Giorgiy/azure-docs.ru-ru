# <a name="how-to-review-your-access"></a>Как проверить доступ

Azure Active Directory позволяет предприятиям упростить управление доступом для приложений и участников групп в Azure AD, а также других служб Microsoft Online Services с помощью функции проверки доступа.  Вы могли получать от корпорации Майкрософт письмо, предлагающее вам проверить доступ для участников группы или пользователей, имеющих доступ к приложению. 

## <a name="opening-an-access-review"></a>Открытие проверки доступа

Вы увидите ожидающие проверки доступа, щелкнув ссылку в письме.  Если вам не пришло письмо, вы можете найти проверки доступа, выполнив следующие действия:

1. Войдите на [панель доступа Azure AD](https://myapps.microsoft.com).
2. Щелкните значок пользователя в правом верхнем углу страницы, где отображается ваше имя и название организации по умолчанию. Если в списке несколько организаций, выберите ту организацию, которая запросила проверку доступа.
3. Если в правой части страницы имеется элемент **Проверки доступа**, щелкните его. Если такой элемент не отображается, для этой организации нет ожидающих проверок доступа и в данный момент не требуется никаких действий.

## <a name="filling-out-an-access-review"></a>Заполнение проверки доступа

Выберите проверку доступа в списке, после чего отобразится доступ.  Можно щелкнуть запись и выбрать необходимое действие: утвердить или отклонить необходимость получения доступа.

Проверяющий может потребовать, чтобы вы указали обоснование для продолжения предоставления доступа.

## <a name="next-steps"></a>Дальнейшие действия

Обратите внимание, что отклоненный доступ не удаляется немедленно, поэтому если вы хотите изменить свой ответ и подтвердить, сбросьте его и выберите новый ответ.  Это можно сделать до завершения проверки доступа.
