# <a name="quickstart-add-company-branding-to-your-sign-in-page-in-azure-ad"></a>Краткое руководство по добавлению фирменной символики организации на страницу входа в Azure Active Directory
Чтобы избежать путаницы, многие компании используют единый стиль оформления всех веб-сайтов и служб, которыми они управляют. Azure Active Directory (Azure AD) предоставляет такую возможность, позволяя настроить внешний вид страницы входа, добавив на нее логотип организации и цветовые схемы. Страница входа отображается при входе в веб-приложения, например в Office 365, которые используют Azure AD в качестве поставщика удостоверений. На этой странице вводятся учетные данные.

> [!NOTE]
> * Фирменная символика компании доступна только после приобретения лицензий Premium или Basic для службы Azure AD, а также обладателям лицензий Office 365. Чтобы узнать, поддерживается ли функция вашим типом лицензии, перейдите на страницу [с ценами на Azure Active Directory](https://azure.microsoft.com/pricing/details/active-directory/).
> 
> * Выпуски Azure AD Premium и Basic доступны для клиентов в Китае, использующих доступный по всему миру экземпляр Azure Active Directory. Эти выпуски в настоящее время не поддерживаются службой Azure под управлением 21Vianet в Китае. Чтобы получить дополнительные сведения, обратитесь к нам через [форум Azure Active Directory](https://feedback.azure.com/forums/169401-azure-active-directory/).

## <a name="customizing-the-sign-in-page"></a>Настройка страницы входа

<!--You can customize the following elements on the sign-in page: <attach image>-->

Элементы фирменной символики компании появляются на странице входа в Azure AD, когда пользователи пытаются получить доступ к URL-адресу конкретного клиента, например [*https://outlook.com/contoso.com*](https://outlook.com/contoso.com).

Например, при посещении сайта www.office.com страница входа не отображает корпоративную символику, поскольку пользователь еще не ввел учетные данные. Символика появляется, когда пользователь введет идентификатор или нажмет плитку пользователя.

> [!NOTE]
> * Для доменного имени в части портала Azure **Домены**, в которой была настроена фирменная символика, должно отображаться состояние "Активно". Дополнительные сведения см. в статье [Добавление имени личного домена](add-custom-domain.md).
> * Фирменная символика страницы входа не применяется к странице входа для личных учетных записей Майкрософт. Если сотрудники или бизнес-гости выполнят вход с помощью личных учетных записей Майкрософт, фирменная символика компании на этих страницах входа отображаться не будет.

## <a name="add-company-branding-to-your-directory"></a>Добавление фирменной символики компании в ваш каталог

1. Войдите в [центр администрирования Azure AD](https://aad.portal.azure.com) с помощью учетной записи глобального администратора клиента.
2. Последовательно выберите **Пользователи и группы** > **Корпоративная фирменная символика** > **Изменить**.
  
  ![Открытие страницы корпоративной фирменной символики](./media/customize-branding/navigation-to-branding.png)
3. Измените элементы, которые нужно настроить. Все элементы необязательные.
  
  ![Изменение пользовательской фирменной символики](./media/customize-branding/edit-branding.png)
5. Когда все будет готово, нажмите кнопку **Сохранить**.

Применение каких-либо изменений фирменной символики страницы входа может занять до одного часа.

## <a name="add-language-specific-company-branding-to-your-directory"></a>Добавление языковой фирменной символики в каталог

1. Войдите в [центр администрирования Azure AD](https://aad.portal.azure.com) с помощью учетной записи глобального администратора каталога.
2. Последовательно выберите **Пользователи и группы** > **Корпоративная фирменная символика** > **Новый язык**.
  
  ![Добавление элементов фирменной символики для определенного языка](./media/customize-branding/add-language.png)
5. Измените элементы, которые нужно настроить. Все элементы необязательные.
6. Когда все будет готово, нажмите кнопку **Сохранить**.

Применение каких-либо изменений фирменной символики страницы входа может занять до одного часа.
