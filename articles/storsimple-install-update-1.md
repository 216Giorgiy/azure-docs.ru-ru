# Установка обновления 1 на устройство StorSimple

## Обзор

В этом учебнике описывается установка обновления 1 на устройство StorSimple, на котором используется программное обеспечение более ранней версии. На вашем устройстве может использоваться общедоступный выпуск программного обеспечения, обновление 0.1, 0.2 или 0.3. В учебнике также объясняется, какие действия предпринять, если шлюз на вашем устройстве StorSimple настроен на сетевом интерфейсе, отличном от DATA 0.

Если на вашем устройстве установлена более ранняя версия, то при установке обновления 1.0 выполняются проверки устройства. Эти проверки позволяют определить работоспособность устройства на основании состояния оборудования и качества сетевого подключения.

Вам будет предложено вручную выполнить предварительную проверку, чтобы убедиться в следующем.

- Фиксированные IP-адреса контроллеров поддерживают маршрутизацию и могут подключиться к Интернету. Эти IP-адреса используются для обновлений служб на устройстве StorSimple. Это можно проверить, выполнив следующий командлет на каждом контроллере:

    `Test-Connection -Source <fixed IP of your device controller> <Destination IP> `
 
- Перед обновлением устройства рекомендуем сделать облачный моментальный снимок данных на устройстве.

После того как вы перепроверите и подтвердите ручные проверки (см. выше), будет выполнен набор автоматических проверок перед обновлением. В частности, описаны такие возможности:

- **Проверки работоспособности контроллера** подтверждают, что оба контроллера устройства готовы к работе и подключены к сети.

- **Проверки работоспособности компонентов оборудования** подтверждают, что все компоненты оборудования устройства StorSimple готовы к работе.

- **Проверки DATA 0** подтверждают, что интерфейс DATA 0 включен на устройстве. Если этот интерфейс не включен, включите его и повторите проверку.

- **Проверки DATA 2 и DATA 3** подтверждают, что сетевые интерфейсы DATA 2 И DATA 3 не включены. Если эти интерфейсы включены, отключите их и попробуйте обновить устройство. Эта проверка выполняется только при обновлении на устройстве с общедоступной версией программного обеспечения. Устройствам, на которых используется версия 0.1, 0.2 или 0.3, эта проверка не требуется.

- **Проверка шлюза** на любом устройстве, использующем ПО версии до обновления 1. Эта проверка выполняется только на устройствах со шлюзом, настроенным для сетевого интерфейса, отличного от DATA 0.
 
Обновление 1 будет установлено лишь в случае успешного выполнения всех предварительных проверок. После установки обновления 1 на устройство StorSimple проверки DATA 2, DATA 3 и проверка шлюза не будут выполняться при последующих обновлениях. Остальные предварительные проверки будут выполняться.

## Установка обновления 1 на устройство со шлюзом, настроенным на сетевой интерфейс, отличный от DATA 0 

Эта процедура применима к устройствам StorSimple, на которых используется программное обеспечение версии до обновления 1.0, а один из шлюзов настроен на сетевой интерфейс, отличный от DATA 0.
 
Если на вашем устройстве нет шлюза на сетевом интерфейсе, отличном от DATA 0, вы можете обновить устройство непосредственно через портал управления. См. раздел [Установка обновления 1 с помощью портала управления](#use-the-management-portal-to-install-update-1).
 
> [AZURE.NOTE]Для установки обновления 1.0 эту процедуру требуется выполнить только один раз. Вы можете использовать портал управления Azure для установки всех последующих обновлений.
 
Если на вашем устройстве используется программное обеспечение версии до обновления 1.0, а один из шлюзов устройства настроен на сетевой интерфейс, отличный от DATA 0, вы можете установить обновление 1.0 следующими двумя способами.

- **Способ 1**. Загрузите обновление и установите его с помощью командлета [Start-HcsHotfix](https://technet.microsoft.com/library/dn688134.aspx) в интерфейсе Windows PowerShell на устройстве. Рекомендуем использовать этот метод.

- **Способ 2**. Установите обновление непосредственно через портал управления.
 
По каждому из этих способов в следующих разделах приведены подробные инструкции.

### Способ 1. Установка обновления 1 с помощью Windows PowerShell для StorSimple

Перед использованием этой процедуры для установки обновления убедитесь в следующем.

- Оба контроллера устройства подключены к сети.

- Интерфейсы DATA 2 и DATA 3 отключены. Это потребуется лишь в том случае, если на устройствах используется общедоступный выпуск ПО. Устройства, на которых используется обновление 0.2 или 0.3, не требуют отключения этих интерфейсов. После завершения обновления вы можете снова включить эти сетевые интерфейсы.

## Устранение ошибок при обновлении

**Что делать, если появляется уведомление об ошибке при выполнении предварительных проверок?**

Если предварительная проверка завершается с ошибкой, обратите внимание на подробную панель уведомлений внизу страницы. Она позволяет понять, какая предварительная проверка завершилась с ошибкой. Ниже показан пример вывода такого уведомления. В этом случае проверки работоспособности контроллера и компонентов оборудования завершились с ошибкой. В разделе **Состояние оборудования** видно, что требуется проверить компоненты контроллеров 0 и 1.
 
  ![Ошибка при предварительной проверке](./media/storsimple-install-update-1/HCS_PreUpdateCheckFailed-include.png)

Вам необходимо убедиться, что оба контроллера готовы к работе и подключены к сети. Вам также необходимо убедиться, что все компоненты оборудования устройства StorSimple готовы к работе, согласно информации на странице "Обслуживание". После этого попробуйте установить обновления. Если вам не удается устранить проблемы с компонентами оборудования, обратитесь в службу поддержки корпорации Майкрософт для получения дальнейших инструкций.

**Что делать, если появляется сообщение об ошибке "Не удается установить обновления" и рекомендуется обратиться к руководству по устранению ошибок при обновлении, чтобы определить причину возникновения ошибки?**

Одна из возможных причин — отсутствие подключения к серверам Центра обновления Майкрософт. Это необходимо проверить вручную. При потере подключения к серверу обновления задание обновления завершится с ошибкой. Состояние подключения можно проверить, выполнив следующий командлет через интерфейс Windows PowerShell на устройстве StorSimple:

 `Test-Connection -Source <Fixed IP of your device controller> <Destination IP>`

Выполните командлет на обоих контроллерах.
 
Если вы убедились в наличии подключения, но проблема не устранена, обратитесь в службу поддержки корпорации Майкрософт для получения дальнейших инструкций.
