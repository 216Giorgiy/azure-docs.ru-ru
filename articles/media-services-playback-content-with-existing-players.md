#Воспроизведение содержимого

Службы мультимедиа Azure поддерживают многие популярные форматы потоковой передачи, например Smooth Streaming, HTTP Live Streaming и MPEG-Dash. В этом подразделе описываются имеющиеся проигрыватели, которые можно использовать для тестирования потоков.  

##Воспроизведение содержимого с помощью существующих проигрывателей

В этом разделе показаны существующие проигрыватели, которые можно использовать для воспроизведения содержимого.

>[AZURE.NOTE]Для воспроизведения упакованного динамически или динамически зашифрованного содержимого, необходимо получить не менее одной единицы потоковой передачи для конечной точки потоковой передачи, из которой планируется доставка содержимого. Информацию о масштабировании единиц потоковой передачи см. в статье [Как масштабировать единицы потоковой передачи](../media-services-manage-origins#scale_streaming_endpoints).


###Проигрыватель содержимого служб мультимедиа портала управления Azure

**Портал управления Azure** предлагает проигрыватель содержимого, который можно использовать для тестирования видео.

Щелкните нужное видео (убедитесь, что он было [опубликовано](../media-services-manage-content#publish)) и нажмите кнопку **воспроизведение** в нижней части портала. 
 
Важные особенности

- **ПРОИГРЫВАТЕЛЬ СЛУЖБ МУЛЬТИМЕДИА** выполняет воспроизведение из конечной точки потоковой передачи по умолчанию. Если требуется воспроизвести из конечной точке потоковой передачи не по умолчанию, используйте другой проигрыватель. Например, [Проигрыватель служб мультимедиа Azure](http://amsplayer.azurewebsites.net/azuremediaplayer.html).
 

![AMSPlayer][AMSPlayer]

###Проигрыватель служб мультимедиа Azure

Используйте [Проигрыватель служб мультимедиа Azure](http://amsplayer.azurewebsites.net/azuremediaplayer.html) для воспроизведения содержимого.
