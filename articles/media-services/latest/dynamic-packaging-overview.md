# <a name="dynamic-packaging"></a>Динамическая упаковка

Службы мультимедиа Microsoft Azure можно использовать для передачи многих файлов форматов мультимедиа, форматов потоков мультимедиа и форматов защищенного содержимого в клиенты различных технологий (например, iOS и XBOX). Эти клиенты работают по разным протоколам, например, для iOS используется формат потоковой трансляции HTTP (HLS), а для технологии Xbox необходимо использовать формат Smooth Streaming. Если у вас есть набор файлов MP4 (ISO Base Media 14496-12) с адаптивной скоростью (несколько скоростей) или аналогичных файлов Smooth Streaming, которые вам нужно передать клиентам, принимающим форматы HLS, MPEG DASH, или Smooth Streaming, вы можете воспользоваться функциями динамической упаковки служб мультимедиа.

При использовании динамической упаковки достаточно создать ресурс, содержащий набор файлов MP4 с адаптивной скоростью. Затем с учетом формата, указанного в манифесте или запросе фрагмента, сервер потоковой передачи по запросу организует передачу содержимого по выбранному протоколу. В результате вы сможете хранить и оплачивать файлы только в одном формате, а службы мультимедиа выполнят сборку и будут обслуживать соответствующий ответ на основе запросов клиента.

## <a name="dynamic-packaging-workflow"></a>Рабочий процесс динамической упаковки

1. Отправьте входной файл (он также называется мезонинным). Например, это может быть файл в формате H.264, MP4 или WMV (список поддерживаемых форматов см. в статье [Форматы и кодеки стандартного кодировщика служб мультимедиа](media-encoder-standard-formats.md)).
2. Преобразуйте мезонинный файл в набор мультискоростных MP4-файлов в формате H.264.
3. Опубликуйте ресурс,который содержит набор MP4-файлов с адаптивной скоростью.
4. Создайте URL-адреса для доступа к содержимому и его потоковой передачи.

## <a name="audio-codecs-supported-by-dynamic-packaging"></a>Аудиокодеки, поддерживаемые для динамической упаковки

Для динамической упаковки поддерживаются MP4-файлы с аудиоданными, закодированными с помощью [AAC](https://en.wikipedia.org/wiki/Advanced_Audio_Coding) (AAC-LC, HE-AAC версии 1, HE-AAC версии 2), [Dolby Digital Plus](https://en.wikipedia.org/wiki/Dolby_Digital_Plus) (Enhanced AC-3 или E-AC3) либо [DTS](https://en.wikipedia.org/wiki/DTS_%28sound_system%29) (DTS Express, DTS LBR, DTS HD, DTS HD Lossless).

> [!NOTE]
> Для динамической упаковки не поддерживаются файлы с аудиоданными в формате [Dolby Digital](https://en.wikipedia.org/wiki/Dolby_Digital) (AC3) (это устаревший кодек).
