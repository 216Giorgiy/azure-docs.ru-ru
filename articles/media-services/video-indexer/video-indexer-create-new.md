# <a name="create-highlights-from-existing-videos"></a>Создание аналитических сведений из существующих видео

В этом разделе показано, как создавать и публиковать аналитические сведения о видео на основе некоторых других видео.

1. Откройте веб-сайт [Индексатора видео](https://www.videoindexer.ai/) и выполните вход.
2. Найдите видео, для которого вы хотите создать аналитические сведения.
3. Нажмите кнопку **Воспроизвести**.

    На странице отображаются сводные аналитические сведения о видео. 

    ![Аналитика](./media/video-indexer-create-new/video-indexer-summarized-insights.png)

3. Нажмите кнопку **Изменить**.

    На этой странице отображается полная разбивка видео. Разбивка видео состоит из блоков. Блоки предназначены для упрощения восприятия данных. Например, разделение на блоки происходит, когда меняется докладчик или возникает длинная пауза. Можно создать собственный список воспроизведения, который содержит только те строки, которые вам нужны. Чтобы отображались только определенные части исходного видео, можно настроить фильтрацию по разделам или ключевым словам, тональности, людям и докладчикам. Вы можете выбрать параметр, при котором будете просматривать только расшифровку видео или распознавание текста.    

    ![Аналитика](./media/video-indexer-create-new/video-indexer-create-new-playlist.png)

4. Создайте свой список воспроизведения.

    Чтобы добавить строки в список воспроизведения или удалить их из него, кнопку **+**/**-**.

5. Просмотрите свой список воспроизведения.

    Создав список воспроизведения, нажмите кнопку **Предварительный просмотр**.
6. Опубликуйте список воспроизведения.

    После предварительного просмотра списка воспроизведения вы можете его опубликовать.

    После публикации список воспроизведения добавляется в список аналитических сведений о видео.
