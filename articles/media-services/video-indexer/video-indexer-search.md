# <a name="find-exact-moments-within-videos"></a>Поиск конкретных кадров в видео

В этом разделе описываются параметры поиска, с помощью которых можно находить нужные кадры в видео.

1. Откройте веб-сайт [Индексатора видео](https://www.videoindexer.ai/) и выполните вход.
2. Выполните поиск среди всех видео в учетной записи.

    В следующем примере мы выполнили поиск всех видео со Скоттом Хансельманом (Scott Hanselman), созданных на веб-сайте Channel9.

    ![поиска](./media/video-indexer-search/video-indexer-search01.png)
3. Выполните поиск сводных аналитических сведений о видео.

    После этого можно осуществлять поиск по конкретному видео, нажав кнопку **воспроизведения** для нужного ролика. Затем можно выполнять поиск по видео, открыв вкладку **Search** (Поиск). Например, мы выполнили поиск всех кадров, где используется текст "identity protection" (защита удостоверений). 

    ![поиска](./media/video-indexer-search/video-indexer-search02.png)

    Если щелкнуть один из результатов, проигрыватель откроет соответствующий кадр видео. В приложении можно настроить представление проигрывателя или аналитических сведений, а также синхронизацию. Дополнительные сведения см. в руководстве по [внедрению мини-приложений Индексатора видео в приложение](video-indexer-embed-widgets.md). 
4. Поиск подробной разбивки видео.

    Чтобы создать собственную разбивку на основе найденного видео, нажмите кнопку **Изменить**. На этой странице отображается полная разбивка видео. Вы можете выполнять поиск по разбивке, чтобы отображались только те строки, которые вас интересуют. Дополнительные сведения см. в руководстве по [просмотру и редактированию аналитических сведений Индексатора видео](video-indexer-view-edit.md).

    В этом примере выполнен поиск текста "identity protection" (защита удостоверений). Кроме того, мы применили дополнительные фильтры, как показано на следующем рисунке.

    ![поиска](./media/video-indexer-search/video-indexer-search03.png)
