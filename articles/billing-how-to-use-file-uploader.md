# Инструкции по подаче новых запросов на техническую поддержку в утилите загрузки файлов Microsoft Azure

При [создании нового запроса в службу поддержки](https://portal.azure.com/#create/Microsoft.Support) служебная программа передачи файлов поможет легко передать файлы диагностики и любые другие файлы, которые могут быть связаны с инцидентом.

>[AZURE.NOTE] Для защиты вашей конфиденциальности не включайте в загружаемые файлы никакие личные данные.

При работе с утилитой загрузки файлов соблюдайте следующие инструкции:

- Все файлы должны иметь расширение имени файла, например .docx или .xlsx. Список допустимых расширений см. в таблице.
- Имя файла должно содержать не больше 110 знаков.
- Загрузить можно только один файл.
- Размер файла не должен превышать 4 МБ.
