# Как купить или оформить подписку на Azure

Эта статья поможет вам изучить следующие темы:

* оформление подписки на бесплатную пробную версию Azure;
* обновление подписки на бесплатную пробную версию Azure до подписки с оплатой по мере использования;
* активация подписки Azure с помощью ключа активации веб-служб (ключ 5x5).

## Оформление подписки на бесплатную пробную версию Azure
Чтобы оформить подписку на бесплатную пробную версию Azure, необходимо указать такие данные, как номер телефона, кредитную карту и [учетную запись Майкрософт](https://www.microsoft.com/ru-RU/account/faq.aspx) (ранее идентификатор Windows Live ID) или учебную либо рабочую учетную запись (или идентификатор организации). Номер телефона и кредитная карта используются только для проверки личности. Мы не будем выставлять счет на кредитную карту.

>[AZURE.NOTE]Чтобы провести предварительную авторизацию и убедиться, что ваша карта оплаты действительна, мы наложим на нее платеж в размере одного доллара США или подобной суммы в валюте вашего региона. Он будет возвращен в течение 3–5 дней в зависимости от политик вашего финансового учреждения. С карты не будет взиматься плата.

Вот как можно оформить подписку на бесплатную пробную версию Azure:

1. Откройте [страницу бесплатной пробной версии Azure](https://azure.microsoft.com/ru-RU/pricing/free-trial/), а затем выберите **Попробовать сейчас**.

2. Войдите в учетную запись Майкрософт или рабочую либо учебную учетную запись.

3. В разделе **О вас** введите личные данные. Обратите внимание, что если вы указали страну, ее нельзя изменить. ![Снимок экрана с личными данными](./media/billing-buy-sign-up-azure-subscription/AboutYou.png)

4. При проверке по телефону на телефон отправляется код предпочтительным для вас способом: с помощью телефонного звонка или SMS. Ввод полученного кода позволяет нам проверить вашу личность. </br></br> В разделе **Проверка по телефону** выберите международный телефонный код для вашей страны, введите свой номер телефона (не указывайте внутренний телефонный код), а затем выберите **Отправить текстовое сообщение** или **Позвонить мне**. После получения кода проверки введите его в поле **Проверить код**, а затем нажмите кнопку **Проверить код**, чтобы проверить номер телефона. Для проверки номера телефона нельзя использовать номер VoIP-телефона.</br> ![Снимок экрана, на котором показано, как проверяется номер телефона](./media/billing-buy-sign-up-azure-subscription/PhoneVerify.png)</br>Если вы не получили текстовое сообщение или вам не перезвонили, попробуйте использовать другой номер телефона или обратитесь за помощью в [службу поддержки Майкрософт](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade). </br>
5. После проверки номера телефона перейдите к разделу проверки карты.

 ![cardverify](./media/billing-buy-sign-up-azure-subscription/VardVerify.png)</br>

6. После проверки кредитной карты выберите раздел **Соглашение**, а затем выберите **Зарегистрироваться**. ![Регистрация](./media/billing-buy-sign-up-azure-subscription/Signup.png)</br> Настройка подписки может занять несколько минут. Если настройка подписки не завершается за 15 минут, обратитесь за помощью в [службу поддержки Майкрософт](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade). </br> При получении сообщения об ошибке следуйте указаниям в сообщении, чтобы устранить проблему. Стандартное сообщение об ошибке:

**Не удалось завершить операцию. Повторите попытку позже или перезагрузите веб-страницу.** </br>

В этом случае попробуйте завершить регистрацию с помощью другого браузера. Если проблема повторяется, обратитесь за помощью в [службу поддержки Майкрософт](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade).

![signuperror1](./media/billing-buy-sign-up-azure-subscription/signuperror1.png)</br>7. Если подписка готова, выберите **Начать управлять моей службой**. ![startservice](./media/billing-buy-sign-up-azure-subscription/startservice.png)

## Обновление подписки на бесплатную пробную версию Azure до подписки с оплатой по мере использования

1. Войдите на [портал учетной записи Azure](https://account.windowsazure.com/subscriptions) с использованием учетной записи Майкрософт или учебной либо рабочей учетной записи, с помощью которой вы оформили подписку на бесплатную пробную версию.
2. Выберите подписку, а затем выберите **Обновить**.</br>![billpage](./media/billing-buy-sign-up-azure-subscription/billpage.png)
3. Выберите **Да**, а затем выберите **Обновить сейчас**. </br>![upgrade](./media/billing-buy-sign-up-azure-subscription/Upgrade.png)

После обновления до подписки с оплатой по мере использования вы можете продолжать пользоваться преимуществами пробной версии (200 долларов за 30 дней), пока период их использования не будет исчерпан.

## Активация подписки Azure с помощью ключа активации веб-служб (ключ 5x5)

Если вы приобрели кредиты Azure с открытой лицензией у торгового посредника, выполните следующие действия, чтобы активировать подписку.

1. Откройте страницу [Лицензионный ключ Azure с открытой лицензией](http://azure.microsoft.com/offers/ms-azr-0111p/), а затем активируйте новую или существующую подписку.
2. Чтобы активировать подписку, введите ключ продукта и установите первый флажок в разделе «Соглашение» (и второй флажок, если вы хотите получать специальные предложения).</br>![КЛЮЧ OSA](./media/billing-buy-sign-up-azure-subscription/OSAkey.png)
3. Выберите **Активировать**.
