## <a name="sign-up-for-an-azure-free-trial-subscription"></a>Оформление подписки на бесплатную пробную версию Azure
Чтобы оформить подписку на бесплатную пробную версию Azure, необходимо указать номер телефона, кредитную карту и учетную запись Майкрософт. Учетная запись Майкрософт представляет собой адрес электронной почты, который вы используете вместе с паролем для входа в любую программу или службу Windows Live, например Outlook, Hotmail или OneDrive. Вы можете настроить учетную запись Майкрософт с использованием любого принадлежащего вам адреса электронной почты, включая корпоративную электронную почту. Дополнительные сведения см. в статье, содержащей [вопросы и ответы по учетной записи Майкрософт](https://www.microsoft.com/account/faq.aspx).

В случае возникновения неполадок ознакомьтесь со статьей [Устранение неполадок при регистрации в Azure](billing-troubleshoot-azure-sign-up-issues.md).

> [!NOTE]
> Номер телефона и кредитная карта используются только для проверки личности. Мы не будем списывать средства с вашей кредитной карты, но Майкрософт заблокирует на ней минимальную сумму для проверки действительности кредитной карты. Блокировка будет снята в течение 3-5 рабочих дней в зависимости от политики вашего банка.
> 
> 

1. Откройте [страницу бесплатной пробной версии Azure](https://azure.microsoft.com/pricing/free-trial/)и нажмите кнопку **Начните использовать бесплатно**.
2. Выполните вход с помощью своей учетной записи Майкрософт или учетной записи организации.
3. В разделе **сведений о пользователе** введите личные данные. Ознакомьтесь с [дополнительными сведениями](billing-troubleshoot-azure-sign-up-issues.md#i-am-getting-an-error-when-entering-my-information-to-sign-up) о правильном вводе личных данных.
   
    ![Введите здесь свои личные сведения](./media/billing-buy-sign-up-azure-subscription/AboutYou.png)
   
   > [!NOTE]
   > Указанные при регистрации адрес электронной почты и страна будут навсегда связаны с подпиской. Их нельзя будет изменить.
   > 
   > 
4. В разделе **Проверка по телефону** необходимо указать номер телефона для получения кода проверки, который нужно указать для завершения процесса проверки.
   
    Вы можете получить код проверки в текстовом сообщении или с помощью голосового звонка. Для этого выберите **Отправить текстовое сообщение** или **Позвонить мне**. Выберите международный телефонный код вашей страны в первом поле и введите номер телефона во втором поле. После получения кода проверки введите его в третье поле и нажмите кнопку **Проверить код**. Дополнительные сведения см. в [этой статье](billing-troubleshoot-azure-sign-up-issues.md#i-am-not-getting-text-messages-or-calls-during-account-verification-when-i-try-to-sign-up).
   
   > [!NOTE]
   > Для проверки номера телефона нельзя использовать номер VoIP-телефона.
   > 
   > 
   
    ![Введите рабочий номер телефона для проверки](./media/billing-buy-sign-up-azure-subscription/PhoneVerify.png)
   
    Если вы не получили текстовое сообщение или вам не перезвонили, попробуйте использовать другой номер телефона или обратитесь за помощью в [службу поддержки Azure](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade) .
5. В разделе **Проверка по кредитной карте** введите подробные сведения о действительной кредитной карте. Эти сведения собираются только для проверки вашей личности.
   
    Если после отправки данных кредитной карты произошла ошибка, ознакомьтесь со статьей [Я не могу воспользоваться своей кредитной картой при регистрации в Azure](billing-credit-card-fails-during-azure-sign-up.md).
   
    ![Также можно выполнить проверку с помощью карты](./media/billing-buy-sign-up-azure-subscription/VardVerify.png)
6. В разделе **Соглашение** установите флажок, если вы принимаете соглашение о подписке, сведения о предложении и заявление о конфиденциальности. Затем выберите **Зарегистрироваться**. Форма соглашения может изменяться в зависимости от страны.
   
    ![Английская версия соглашения. Форма соглашения может изменяться в зависимости от страны.](./media/billing-buy-sign-up-azure-subscription/Signup.png)
   
    Настройка подписки может занять несколько минут. Если настройка подписки не завершится за 15 минут, обратитесь за помощью в [службу поддержки Azure](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade) .
   
   > [!NOTE]
   > При получении сообщения об ошибке следуйте указаниям в сообщении, чтобы устранить проблему. Стандартное сообщение об ошибке:
   > 
   > **Не удалось завершить операцию. Повторите попытку позже или перезагрузите веб-страницу.**
   > 
   > В этом случае попробуйте завершить регистрацию с помощью другого браузера. Если проблема повторяется, обратитесь за помощью в [службу поддержки Azure](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade) .
   > 
   > 
7. Когда подписка будет готова, выберите **Начать управлять моей службой** .
   
    ![Добро пожаловать в подписку Azure](./media/billing-buy-sign-up-azure-subscription/startservice.png)

## <a name="upgrade-azure-free-trial-to-pay-as-you-go"></a>Обновление бесплатной пробной версии Azure до версии с оплатой по мере использования
1. Войдите на [портал учетной записи Azure](https://account.windowsazure.com/subscriptions) с помощью учетной записи Майкрософт или корпоративной учетной записи, которую вы указали при регистрации на бесплатную пробную версию.
2. Выберите подписку, а затем выберите **Обновить**
   
    ![Не забудьте обновить подписку с бесплатной пробной версии до версии с оплатой по мере использования](./media/billing-buy-sign-up-azure-subscription/billpage.png)
3. Выберите **Да**, а затем выберите **Обновить сейчас**.
   
    ![Подтверждение обновления подписки](./media/billing-buy-sign-up-azure-subscription/Upgrade.png)

При обновлении с бесплатной пробной версии подписки на подписку с оплатой по мере использования будут действовать все преимущества, не использованные в бесплатной пробной версии подписки, такие как период использования 30 дней или кредит 200 долларов на службы Azure, пока одно из них не будет исчерпано. После этого подписка обновится.

<a name="activateKey"></a>

## <a name="activate-using-an-online-service-activation-key"></a>Активация с помощью ключа активации служб OLS
Если вы приобрели кредит **Azure с открытой лицензией** у торгового посредника и получили ключ активации (ключ 5x5), выполните следующие действия, чтобы активировать подписку.

1. Перейдите на [страницу сводных сведений о подписке](https://account.windowsazure.com/subscriptions).
2. Выберите **Azure с открытой лицензией**. На странице [лицензионного ключа Azure с открытой лицензией](https://azure.microsoft.com/offers/ms-azr-0111p/) выберите **Activate a new subscription or existing subscription**(Активировать новую или существующую подписку).
3. Введите ключ продукта и затем установите флажок соглашения.

> Если у вас есть дополнительные вопросы, [обратитесь в службу поддержки](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade) , которая поможет быстро устранить проблему.
