# Доменные службы Azure AD *(предварительная версия)*

## Поддерживаемые регионы Azure
Доменные службы Azure Active Directory доступны в следующих регионах Azure:

- США
  - Центральный регион США
  - Восток США
  - Восток США 2
  - Южно-центральный регион США
  - Запад США

- Азия
  - Восточная Азия
  - Юго-Восточная Азия
