# Примеры самообучаемых приложений в каталоге Azure Marketplace

С помощью средств машинного обучения Azure специалисты по обработке и анализу данных могут выгодным образом использовать свои технологии и творческие способности, как это делают разработчики приложений для смартфонов. Публикуя свои службы машинного обучения в каталоге Azure Marketplace, вы позволяете людям со всего мира за определенную плату подписаться на опубликованные вами облачные API-интерфейсы и использовать их в своих приложениях.

Однако каталог Azure Marketplace может предложить намного больше, чем простейшие аналитические API-интерфейсы. Например, мы создали набор готовых полнофункциональных приложений машинного обучения на платформе Azure ML для решения определенных задач. Для их использования не нужно быть специалистом по обработке и анализу данных или обладать знаниями в области машинного обучения: все необходимое уже есть непосредственно в приложении. Пользователь может просто загрузить свои данные и приступить к работе. 

Вот лишь несколько примеров.

* [Рекомендации](http://datamarket.azure.com/dataset/amla/recommendations) - эта система, обслуживаемая пользователем самостоятельно, но при этом обладающая исключительным набором функциональных возможностей, позволяет легко добавить на веб-сайт или в службу электронной почтовой рассылки персонализированные рекомендации различных товаров. 
  
* [Товары, часто приобретаемые вместе](https://datamarket.azure.com/dataset/amla/mba) - этот API-интерфейс помогает покупателям находить в каталоге интернет-магазина товары, которые часто приобретаются вместе с уже выбранными ими покупками. Чтобы добавить рекомендации "Товары, часто покупаемые вместе" на свой веб-сайт и увеличить объемы продаж в своем интернет-магазине, используйте имеющуюся у вас историю покупок.

* [Обнаружение аномалий](https://datamarket.azure.com/dataset/aml_labs/anomalydetection) - этот API-интерфейс способен обнаруживать аномальные явления во временных рядах (например, данных серверной телеметрии или информации, поступающей с датчиков).

Эти API-интерфейсы позволяют использовать функциональные возможности машинного обучения на платформе Azure ML, не будучи специалистом в области обработки и анализа данных и не тратя время на обучение: для использования этих служб вам понадобятся только собственные данные.
