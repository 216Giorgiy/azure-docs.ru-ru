#<a name="azure-stack-documentation"></a>Документация по Azure Stack

Microsoft Azure Stack ― это новая гибридная облачная платформа, позволяющая предоставлять службы Azure из центра обработки данных организации. Выпуск Azure Stack Technical Preview 2 уже доступен в виде подтверждения концепции (POC). Подтверждение концепции — это среда для изучения и демонстрации возможностей Azure Stack. Она позволяет развернуть все необходимые компоненты на одном физическом компьютере, чтобы создать среду для оценки основных концепций и возможностей и проверки модели расширяемости API-интерфейсов.
