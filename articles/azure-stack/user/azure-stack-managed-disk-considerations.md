# <a name="azure-stack-managed-disks-differences-and-considerations"></a>Azure Stack Managed Disks: Differences and considerations (Управляемые диски Azure Stack. Отличия и рекомендации)
В этой статье перечислены известные различия между Управляемыми дисками Azure Stack и Управляемыми дисками для Azure. Чтобы узнать об общих различиях между Azure Stack и Azure, прочитайте статью [Важные аспекты использования служб и создания приложений в Azure Stack](azure-stack-considerations.md).

Эта служба упрощает управление дисками виртуальных машин IaaS. Они управляют [учетными записями хранения](/azure/azure-stack/azure-stack-manage-storage-accounts), связанными с этими дисками.
  

## <a name="cheat-sheet-managed-disk-differences"></a>Памятка: Различия между Управляемыми дисками

| Функция | Azure (глобальная) | Azure Stack |
| --- | --- | --- |
|Шифрование неактивных данных |Azure Storage Service Encryption (SSE), Azure Disk Encryption (ADE) (Шифрование службы хранилища Azure и шифрование дисков Azure)     |128-битное шифрование AES BitLocker      |
|Образ —          | Поддержка управляемого пользовательского образа |Еще не поддерживается|
|Варианты резервного копирования |Поддержка службы Azure Backup |Еще не поддерживается |
|Параметры аварийного восстановления |Поддержка Azure Site Recovery |Еще не поддерживается|
|Типы дисков     |SSD (цен. категория "Премиум"), SSD (цен. категория "Стандартный") (Предварительная версия) и HDD (цен. категория "Стандартный") |SSD (цен. категория "Премиум"), SSD (цен. категория "Стандартный") |
|Диски уровня "Премиум"  |Полностью поддерживается |Может быть подготовлено, но не имеет ограничений производительности или гарантий  |
|Операции ввода-вывода дисков уровня "Премиум"  |Зависит от размера диска  |2300 операций ввода-вывода в секунду на диск |
|Пропускная способность дисков уровня "Премиум" |Зависит от размера диска |145 МБ/с на диск |
|Размер диска  |Диск Azure ценовой категории "Премиум": от P4 (32 ГиБ) до P80 (32 ТиБ)<br>Диск SSD Azure ценовой категории "Стандартный": от E10 (128 ГиБ) до E80 (32 ТиБ)<br>Диск HDD Azure ценовой категории "Стандартный": от S4 (32 ГиБ) до S80 (32 ТиБ) |M4: 32 ГиБ<br>M6: 64 ГиБ<br>M10: 128 ГиБ<br>M15: 256 ГиБ<br>M20: 512 ГиБ<br>M30: 1024 ГиБ |
|Производительность аналитики дисков |Поддерживаются статистические и дисковые метрики |Еще не поддерживается |
|Миграция      |Представление средств для переноса из существующих неуправляемых виртуальных машин Azure Resource Manager без необходимости повторного создания виртуальной машины  |Еще не поддерживается |


## <a name="metrics"></a>Метрики
Кроме того, есть различия в метриках хранилища:
- С помощью Azure Stack данные транзакций в метриках хранилища не различают внутреннюю или внешнюю пропускную способность сети.
- Данные транзакции Azure Stack в метриках хранилища не включают доступ виртуальных машин к подключенным дискам.
