*Область применения: интегрированные системы Azure Stack и Пакет средств разработки Azure Stack*

## <a name="general-purpose"></a>Универсальные

В размерах виртуальных машин общего назначения предоставляется сбалансированное соотношение ресурсов ЦП и памяти. Они используются для тестирования и разработки, небольших и средних баз данных, а также веб-серверов с небольшим или средним объемом трафика.
