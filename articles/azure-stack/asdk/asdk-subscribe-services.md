# <a name="tutorial-create-and-test-a-subscription"></a>Руководство. Создание и тестирование подписки
В этом руководстве объясняется, как создать подписку, содержащую предложение, и протестировать ее. Для проверки войдите на [портал пользователя](https://portal.local.azurestack.external) как администратор облака, подпишитесь на предложение и создайте виртуальную машину.

> [!TIP]
> Для более подробной оценки [создайте подписку для конкретного пользователя](https://docs.microsoft.com/azure/azure-stack/azure-stack-subscribe-plan-provision-vm#create-a-subscription-as-a-cloud-operator), затем войдите от имени этого пользователя на портал пользователя. 

В этом руководстве объясняется, как подписаться на предложение, созданное по инструкциям [предыдущего руководства](asdk-offer-services.md).

Освещаются следующие темы:

> [!div class="checklist"]
> * Подписка на предложение 
> * Тестирование предложения.

## <a name="subscribe-to-an-offer"></a>Подписка на предложение
Чтобы подписаться на предложение от имени пользователя, необходимо войти на портал пользователя Azure Stack и найти службы, предложенные оператором Azure Stack.

1. Войдите на [портал пользователя](https://portal.local.azurestack.external) от имени пользователя и щелкните **Получить подписку**.

   ![Получение подписки](media/asdk-subscribe-services/get-subscription.png)

2. В поле **Отображаемое имя** введите имя для подписки. Затем щелкните **Предложение**, чтобы выбрать одно из доступных предложений в разделе **Выборе предложения**, и нажмите кнопку **Создать**.

   ![Создание предложения](media/asdk-subscribe-services/create-subscription.png)

   > [!TIP]
   > Теперь обновите портал пользователя, чтобы начать использовать подписку.

3. Чтобы просмотреть созданную подписку, щелкните **Больше служб** и **Подписки**, а затем выберите новую подписку. Подписавшись на предложение, обновите страницу портала, чтобы узнать, какие новые службы включены в новую подписку. В этом примере добавлена служба **Виртуальные машины**.

   ![Просмотр подписки](media/asdk-subscribe-services/view-subscription.png)


## <a name="test-the-offer"></a>Тестирование предложения.
Войдя на [портал пользователя](https://portal.local.azurestack.external), вы можете протестировать предложение. Для этого нужно подготовить виртуальную машину, используя новые возможности подписки. 

> [!NOTE]
> Для этого теста требуется образ виртуальной машины Windows Server 2016 Datacenter, который добавлен в Azure Stack Marketplace при работе [с предыдущим руководством](asdk-marketplace-item.md). 

1. Войдите на [портал пользователя](https://portal.local.azurestack.external).

2. На портале пользователя выберите **Виртуальные машины** > **Добавить** > **Windows Server 2016 Datacenter** и нажмите кнопку **Создать** .

3. В разделе **Основные сведения** введите **имя**, **имя пользователя** и **пароль**. Затем выберите **Подписка**, создайте **группу ресурсов** (или выберите существующую) и нажмите кнопку **ОК**.

4. В разделе **Выбор размера** выберите **A1 Standard** и щелкните **Выбрать**.  

5. В колонке "Параметры" оставьте значения по умолчанию и нажмите кнопку **ОК**.

6. В разделе **Сводка** щелкните **ОК**, чтобы создать виртуальную машину.  

7. Чтобы просмотреть новую виртуальную машину, щелкните **Виртуальные машины**, а затем найдите виртуальную машину и щелкните ее имя.

    ![All resources (Все ресурсы)](media/asdk-subscribe-services/view-vm.png)

> [!NOTE]
> Развертывание виртуальной машины займет несколько минут.
