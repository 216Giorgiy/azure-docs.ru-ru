# Квоты и ограничения пакетной службы Azure

В этой статье перечислены лимиты по умолчанию и максимальные лимиты определенных ресурсов, которые можно использовать с пакетной службой Azure. Большинство лимитов представляют собой квоты, применяемые Azure в отношении вашей подписки или учетных записей пакетной службы.

Если планируется выполнять рабочие нагрузки пакетной службы в производственной среде, имеет смысл увеличить одну или несколько квот по сравнению со значениями по умолчанию. Для увеличения квоты создайте (бесплатно) интернет-запрос в службу поддержки клиентов.

>[AZURE.NOTE]Квота — это кредитный лимит, а не гарантия наличия ресурсов. Если вам нужны ресурсы в очень большом объеме, обратитесь в службу поддержки Azure.

## Квоты для подписки
Ресурс|Ограничение по умолчанию|Максимальное ограничение
---|---|---
Учетных записей пакетной службы на регион на подписку|1|50

## Просмотр квот пакетной службы

Просмотрите квоты своей учетной записи пакетной службы на [портале предварительной версии Azure](https://portal.azure.com).

1. На портале предварительной версии щелкните **Учетные записи пакетной службы** и присвойте имя своей учетной записи пакетной службы.

2. В колонке учетной записи щелкните **Параметры** > **Свойства**.

	![Квоты для учетной записи пакетной службы][account_quotas]

3. В колонке **Свойства** изучите квоты, которые в настоящее время действуют в отношении учетной записи пакетной службы.

## Увеличение квоты

Выполните следующие шаги, чтобы отправить запрос на увеличение квоты на портале предварительной версии Azure (на [портале Azure](http://azure.microsoft.com/blog/2014/06/04/azure-limits-quotas-increase-requests/) также можно запросить увеличение квот).

1. На панели мониторинга портала предварительной версии щелкните **Справка + поддержка**.

2. Щелкните **Создать запрос в службу поддержки > Основы**.

3. В колонке **Основы** выполните следующие действия.

	а. В разделе **Тип проблемы** укажите **Квота**.

	b. Выберите свою подписку.

	c. В разделе **Служба** укажите **Пакетная служба**.

	г) В разделе **План поддержки** выберите **План поддержки Azure — разработчик**.

	Нажмите кнопку **Далее**.

4. В колонке **Проблема** выполните следующие действия.

	а. В разделе **Тип проблемы** выберите **Пакетная служба**.

	b. В разделе **Сведения** перечислите квоту или квоты, которые требуется изменить для определенной учетной записи, и укажите новые приемлемые для вас ограничения.

	Нажмите кнопку **Далее**.

5. В колонке **Контактные данные** введите свои контактные данные и нажмите кнопку **Далее**.

6. Щелкните **Создать**, чтобы отправить новый запрос в службу поддержки.

Служба поддержки Azure свяжется с вами. Выполнение запроса может занять до 2 рабочих дней.
