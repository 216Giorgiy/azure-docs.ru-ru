# Проверка службы Service Fabric
Testability — это набор инструментов, специально предназначенных для тестирования ваших служб, созданных с помощью Microsoft Azure Service Fabric. Эти инструменты позволяют вам без труда вызывать значимые ошибки и запускать сценарии тестирования, чтобы воспроизвести и проверить множество различных состояний и переходов, которые будут происходить со службами в течение их жизненного цикла, в контролируемых и безопасных условиях. Дополнительные сведения см. в следующих статьях.

- [Выполнение действия](service-fabric-testability-actions.md)  
- [Выполнение сценария](service-fabric-testability-scenarios.md)
