# <a name="change-the-credit-card-used-to-pay-for-an-azure-subscription"></a>Изменение кредитной карты, используемой для оплаты подписки Azure

В этой статье описывается, как добавить, изменить или удалить кредитную карту, связанную с вашей учетной записью Azure, а также объясняются методы оплаты.

<a id="addcard"></a>
## <a name="add-a-credit-card-as-a-payment-method"></a>Добавление кредитной карты в качестве метода оплаты

Чтобы добавить кредитную карту в учетную запись, выполните приведенные ниже действия.

1. Войдите в [Центр управления учетной записью Azure](https://account.windowsazure.com/Subscriptions) как администратор учетной записи.
2. Под элементом **Щелкните подписку для просмотра информации и сведений об использовании** выберите подписку, для которой требуется добавить метод оплаты.
3. В правой части страницы выберите **Manage payment methods**(Управление методами оплаты).
4. На странице **Управление методами оплаты** щелкните "+", чтобы добавить кредитную карту.
5. Выберите тип кредитной карты и укажите необходимые сведения.
6. Нажмите кнопку **Далее**, чтобы вернуться на страницу **Управление методами оплаты**. Если добавленная карта действительна, она будет добавлена в список действительных способов оплаты.

> [!NOTE]
>Если после отправки данных кредитной карты приходит сообщение об ошибке, ознакомьтесь со статьей [Что делать, если кредитная или банковская карта отклонена при регистрации](billing-credit-card-fails-during-azure-sign-up.md).
>
>

## <a name="edit-payment-information-for-an-existing-credit-card"></a>Изменение платежной информации для существующей кредитной карты

Чтобы проверить метод оплаты и обновить данные существующей кредитной карты, выполните приведенные ниже действия.

1. Войдите в [Центр управления учетной записью Azure](https://account.windowsazure.com/Subscriptions) как администратор учетной записи.

   > [!NOTE]
   > Доступ к Центру управления учетной записью предоставляется только администратору учетной записи. Дополнительные сведения об администраторе учетной записи см. в статье [Добавление или изменение ролей администратора Azure](billing-add-change-azure-subscription-administrator.md).
   >
   >
2. Под элементом **Щелкните подписку для просмотра информации и сведений об использовании** выберите подписку, привязанную к кредитной карте.</br> ![selectsub](./media/billing-how-to-change-credit-card/selectsub.png)
3. В правой части страницы выберите **Manage payment methods**(Управление методами оплаты).</br> ![changesub](./media/billing-how-to-change-credit-card/changesub_new.png)

  Вы увидите в списке текущий метод оплаты. Выберите его, чтобы проверить обязательные сведения.

  **Обновление данных существующей кредитной карты**

  Если существующая кредитная карта обновлена, а ее номер остается прежним, можно просто обновить данные такой карты, например дату окончания срока действия. Однако, если номер кредитной карты изменился из-за потери, кражи или истечения срока действия, то необходимо [добавить кредитную карту в учетную запись](#addcard). Это вызвано тем, что CVV-код безопасности меняется при изменении номера кредитной карты.

4. На странице **Управление методами оплаты** щелкните **Изменить** рядом с номером кредитной карты.</br> ![changesub](./media/billing-how-to-change-credit-card/editcard_new.png)
5. На странице **Изменить** убедитесь, что выбран правильный тип и номер кредитной карты.
6. Внесите необходимые изменения в данные карты и нажмите кнопку **Далее**, чтобы перейти на страницу **Выбрать метод оплаты**.

## <a name="change-the-credit-card-that-you-use-to-pay-an-azure-bill"></a>Изменение кредитной карты, используемой для оплаты счетов Azure

1. Войдите в [Центр управления учетной записью Azure](https://account.windowsazure.com/Subscriptions) как администратор учетной записи.
2. Под элементом **Щелкните подписку для просмотра информации и сведений об использовании**выберите подписку, для которой требуется обновить метод оплаты.
3. В правой части страницы выберите **Manage payment methods**(Управление методами оплаты).
4. На странице **Управление методами оплаты** щелкните **Использовать замену** и выберите кредитную карту, которую хотите использовать.

## <a name="remove-a-credit-card-from-the-account"></a>Удаление кредитной карты из учетной записи
1. Войдите в [Центр управления учетной записью Azure](https://account.windowsazure.com/Subscriptions) как администратор учетной записи.
2. Под элементом **Щелкните подписку для просмотра информации и сведений об использовании**выберите подписку, для которой требуется обновить метод оплаты.
3. В правой части страницы выберите **Manage payment methods**(Управление методами оплаты).
4. На странице **Управление методами оплаты** нажмите кнопку **Удалить**, чтобы выбрать кредитную карту, которую вы хотите удалить.

   > [!NOTE]
   > Вы не сможете удалить кредитную карту, если она связана с другими активными подписками Майкрософт. Если появится сообщение о том, что карта используется, отключите ее во всех активных подписках Майкрософт.
   >
   >

##  <a name="how-to-make-payments"></a>Выполнение оплаты

Если в качестве метода оплаты вы указали кредитную или платежную карту, то оплата производится автоматически.
Если вы оплачиваете услугу [по выставляемым счетам](https://azure.microsoft.com/pricing/invoicing/), отправьте платеж в местоположение, указанное внизу счета.
