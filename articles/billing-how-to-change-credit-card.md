# Изменение кредитной карты, используемой для оплаты счета Azure

Иногда может потребоваться обновить сведения о кредитной карте в учетной записи или изменить карту, используемую для оплаты счета.

Если существующая кредитная карта обновлена, а ее номер остается прежним, можно просто обновить сведения о такой карте, например дату окончания срока действия.

Если номер кредитной карты изменился из-за потери, кражи или истечения срока действия, вам нужно добавить в учетную запись новую кредитную карту. Это вызвано тем, что CVV-код безопасности меняется при изменении номера кредитной карты.

## Изменение платежной информации для существующей кредитной карты
1. Выполните вход в [центр учетных записей Azure](https://account.windowsazure.com/Subscriptions) в качестве администратора учетной записи. На [портале Azure](https://portal.azure.com) в центр учетных записей можно перейти, выбрав **Подписка** > *имя подписки* > **Управление**.

	**Примечание**. Доступ к центру учетных записей предоставляется только администратору учетной записи. Дополнительные сведения об администраторе учетной записи см. в разделе [Добавление или изменение ролей администратора Azure](billing-add-change-azure-subscription-administrator.md).

2. В разделе **Выбор подписки для просмотра сведений и использования** выберите подписку, привязанную к кредитной карте.</br> ![selectsub](./media/billing-how-to-change-credit-card/selectsub.png)
3. В правой части страницы выберите **Изменить способ оплаты**.</br> ![changesub](./media/billing-how-to-change-credit-card/changesub.png)
4. На странице **Выбор способа оплаты** щелкните **Изменить** рядом с номером кредитной карты.</br> ![changesub](./media/billing-how-to-change-credit-card/editcard.png)
5. На странице **Изменить** убедитесь, что выбран правильный тип и номер кредитной карты.
6. Внесите необходимые изменения в сведения о карте и выберите **Далее**, чтобы вернуться на страницу **Выбор способа оплаты**.

## Добавление новой кредитной карты в качестве метода оплаты
1. Выполните вход в [Центр учетных записей Azure](https://account.windowsazure.com/Subscriptions) в качестве администратора учетной записи.
2. В разделе **Выбор подписки для просмотра сведений и использования** выберите подписку, для которой требуется добавить новый способ оплаты.
3. В правой части страницы выберите **Изменить способ оплаты**.
4. На странице **Выберите способ оплаты** щелкните «+», чтобы добавить новую кредитную карту.
5. Выберите тип кредитной карты и укажите необходимые сведения.
6. Щелкните **Далее**, чтобы вернуться на страницу **Выбор способа оплаты**. Если добавленная карта действительна, она будет добавлена в список действительных способов оплаты. Для просмотра всех доступных кредитных карт щелкните стрелку раскрывающегося списка рядом с кредитной картой.

## Изменение кредитной карты, используемой для оплаты счета Azure
1. Выполните вход в [Центр учетных записей Azure](https://account.windowsazure.com/Subscriptions) в качестве администратора учетной записи.
2. В разделе **Выбор подписки для просмотра сведений и использования** выберите подписку, для которой требуется обновить способ оплаты.
3. В правой части страницы выберите **Изменить способ оплаты**.
4. На странице **Выбор способа оплаты** щелкните стрелку раскрывающегося списка рядом с кредитной картой, выберите необходимую кредитную карту и щелкните ![checkbutton](./media/billing-how-to-change-credit-card/checkbutton.png), чтобы закрыть окно.
