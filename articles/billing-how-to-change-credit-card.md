# Изменение кредитной карты, используемой для оплаты подписки Azure

Можно выполнить оплату с использованием кредитных или дебетовых карт либо получить одобрение на оплату по счету. Дополнительные сведения об оплате по счету см. в разделе [Выставление счетов Azure](http://azure.microsoft.com/pricing/invoicing/).

## Изменение платежной информации для существующей кредитной карты
1. Выполните вход в [Центр учетных записей Azure](https://account.windowsazure.com/Subscriptions) в качестве администратора учетной записи.
2. В разделе **Щелкните подписку, чтобы просмотреть сведения и использование** щелкните подписку, привязанную к кредитной карте.</br> ![selectsub](./media/billing-how-to-change-credit-card/selectsub.png)
3. В правой части страницы щелкните **Изменить метод оплаты**.</br> ![changesub](./media/billing-how-to-change-credit-card/changesub.png)
4. На странице **Выберите метод оплаты** щелкните **Изменить** рядом с номером кредитной карты.</br> ![changesub](./media/billing-how-to-change-credit-card/editcard.png)
5. На странице **Изменить** убедитесь, что выбран правильный тип и номер кредитной карты.
6. Внесите необходимые изменения в сведения о карте и щелкните **Далее**, чтобы вернуться на страницу «Выберите метод оплаты».

## Добавление новой кредитной карты в качестве метода оплаты
1. Выполните вход в [Центр учетных записей Azure](https://account.windowsazure.com/Subscriptions) в качестве администратора учетной записи.
2. В разделе **Щелкните подписку, чтобы просмотреть сведения и показатели использования** щелкните подписку, для которой требуется добавить новый способ оплаты.
3. В правой части страницы щелкните **Изменить способ оплаты**.
4. На странице **Выберите способ оплаты** щелкните «+», чтобы добавить новую кредитную карту.
5. Выберите тип кредитной карты и укажите необходимые сведения.
6. Щелкните **Далее**, чтобы вернуться на страницу «Выберите способ оплаты». Если добавленная карта действительна, она будет добавлена в список действительных способов оплаты. Для просмотра всех доступных кредитных карт щелкните стрелку раскрывающегося списка рядом с кредитной картой.

## Изменение кредитной карты, используемой для оплаты счета Azure
1. Выполните вход в [Центр учетных записей Azure](https://account.windowsazure.com/Subscriptions) в качестве администратора учетной записи.
2. В разделе **Щелкните подписку, чтобы просмотреть сведения и показатели использования** щелкните подписку, для которой требуется обновить способ оплаты.
3. В правой части страницы щелкните **Изменить способ оплаты**.
4. На странице «Выберите способ оплаты» щелкните стрелку раскрывающегося списка рядом с кредитной карты, выберите кредитную карту, которую требуется использовать, и щелкните ![checkbutton](./media/billing-how-to-change-credit-card/checkbutton.png). Окно закроется.
