# <a name="azure-application-offer"></a>Предложение приложения Azure

<table> <tr> <td>В этом разделе объясняется, как выполнить публикацию нового предложения управляемого приложения Azure в <a href="https://azuremarketplace.microsoft.com/marketplace/">Microsoft Azure Marketplace</a>.  Каждое управляемое приложение содержит шаблон Azure Resource Manager, в котором определены все технические ресурсы, используемые приложением. Сюда можно отнести одну или несколько виртуальных машин и другие вспомогательные службы Azure или веб-службы. </td> <td><img src="./media/cloud-partner-portal-publish-managed-app/azureapp-icon1.png"  alt="Azure application icon" /></td> </tr> </table>
