<a name="update-an-existing-offer-for-azure-marketplace"></a>Обновление существующего предложения в Azure Marketplace 
==============================================

Есть несколько типов обновлений, которые вы можете применить к своему предложению после публикации, например:

1.  Добавление нового \"пакета\" к существующему номеру SKU.
2.  Добавление нового номер SKU к существующему предложению.
3.  Обновление метаданных для предложения и номера SKU в Marketplace.

Вы будет использовать Портал Cloud Partner для реализации этих изменений, а затем повторно опубликуете предложение. В этой статье рассматриваются различные аспекты обновлений для предложений приложения Azure.

<a name="unpermitted-changes-to-azure-application-offer"></a>Недопустимые изменения предложения приложения Azure 
-----------------------------------------------

Некоторые атрибуты предложений приложений Azure и SKU нельзя изменять после публикации в Azure Marketplace.

* Идентификатор предложения и идентификатор издателя предложения.
* Идентификаторы существующих номеров SKU.
* Обновите опубликованный пакет.

<a name="adding-a-new-package-to-an-existing-sku"></a>Добавление нового пакета к существующему номеру SKU 
---------------------------------------

Издателю может потребоваться добавить новую версию пакета для обновления существующего. Это добавление можно сделать путем отправки нового пакета с другим номером версии.

1.  Войдите на [Портал Microsoft Cloud Partner](http://cloudpartner.azure.com).
2.  В разделе **Все предложения** найдите предложение, которое хотите обновить.
3.  В форме **номера SKU** щелкните тот номер SKU, пакет которого\' хотите обновить.
4.  Щелкните Новый пакет и предоставьте новую версию
5.  Отправьте новый ZIP-файл, содержащий обновленный файл шаблона.
6.  Щелкните "Опубликовать", чтобы запустить рабочий процесс публикации для предоставления нового номера SKU.

<a name="adding-a-new-sku-to-an-existing-offer"></a>Добавление нового номер SKU к существующему предложению
-------------------------------------

Вы можете предоставить новый номер SKU для существующего предложения, выполнив следующие действия.

1.  Войдите на [Портал Cloud Partner](http://cloudpartner.azure.com).
2.  В разделе **Все предложения** найдите предложение, которое хотите обновить.
3.  В форме **номера SKU** щелкните Add new SKU (Добавить новый номер SKU) и во всплывающем окне введите идентификатор номера SKU.
4.  Выполните остальные действия, описанные [здесь](./cloud-partner-portal-managed-app-publish.md).
5.  Щелкните **Опубликовать**, чтобы запустить рабочий процесс публикации для предоставления нового номера SKU.

<a name="updating-offer-marketplace-metadata"></a>Обновление метаданных для предложения в Marketplace. 
-----------------------------------

Возможны такие ситуации, когда вам потребуется обновить в Marketplace метаданные, связанные с вашим предложением, например, логотип компании и т. п. Для этого сделайте следующее.

1.  Войдите на Портал Cloud Partner.
2.  В разделе **Все предложения** найдите предложение, которое хотите обновить.
3.  Перейдите к форме Marketplace и выполните описанные [здесь](../cloud-partner-portal/azure-applications/cpp-marketplace-tab.md) инструкции.
4.  Щелкните "Опубликовать", чтобы запустить рабочий процесс публикации для применения внесенных изменений.
