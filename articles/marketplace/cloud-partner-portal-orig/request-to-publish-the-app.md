# <a name="request-to-publish-the-app"></a>Запрос на публикацию приложения

![Щелкните "Опубликовать"](./media/CRMScreenShot18.png)

1. Нажмите кнопку "Опубликовать", чтобы отправить предложение нашей команде сертификации. Можно просматривать состояние вашего предложения на вкладке "Состояние".

![Проверьте состояние публикации.](./media/CRMScreenShot19.png)

## <a name="republish-the-app"></a>Повторная публикация приложения

 Если необходимо сделать запрос на повторную публикацию приложения, нажмите кнопку "Отменить публикацию" под вкладкой "Состояние" (если запрос больше не активен, пропустите этот шаг). Внесите необходимые изменения в приложение, соответствующим образом обновить предложение, а затем нажмите кнопку "Опубликовать", чтобы представить на повторное рассмотрение. # специальных рекомендаций для Microsoft Dynamics 365 Business Edition.
