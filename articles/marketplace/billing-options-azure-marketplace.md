# <a name="billing-options-in-the-azure-marketplace"></a>Варианты выставления счетов за использование Azure Marketplace

В этой статье описаны варианты выставления счетов, доступные в [Azure Marketplace](https://azuremarketplace.microsoft.com).

## <a name="commercial-considerations-in-the-marketplace"></a>Рекомендации по коммерческим предложениям в Marketplace
Marketplace не поддерживает разделение прибыли для следующих типов предложений. 
*   список
*   Пробная версия
*   "Транзакция" с применением модели выставления счетов с использованием собственной лицензии (BYOL).

Дополнительная плата за участие в онлайн-магазинах Marketplace не взимается.

Дополнительные сведения см. в разделе [Политики участия в Microsoft Azure Marketplace](https://azure.microsoft.com/support/legal/marketplace/participation-policies).  

## <a name="pay-as-you-go-and-byol-billing-options"></a>Оплата по мере использования и оплата с использованием собственной лицензии (BYOL)
При использовании модели выставления счетов с оплатой по мере использования 80 % лицензионных выплат по модели на основе использования получаете вы, а 20 % — корпорация Майкрософт. Счет за одноразовое предложение можно выставить как с помощью оплаты по мере использования, так и с помощью модели выставления счетов BYOL. Эти две модели сосуществуют на уровне предложения как отдельные номера SKU. Модели выставления счетов для предложения настраиваются на портале Cloud Partner. 

Рассмотрим следующие примеры.
*   Если включить вариант "Оплата по мере использования", вы получите следующий результат.
    | Стоимость лицензии | 1 долл. США в час |
    |:--- |:--- |
    | Стоимость использования Azure (D1 с 1 ядром) | 0,14 долл. США в час |
    | *Счет выставляется корпорацией Майкрософт* | *1,14 долл. США в час* |

    В этом случае корпорация Майкрософт выставляет счет в размере 1,14 долл. США в час за использование опубликованного образа виртуальной машины.
    | Майкрософт выставляет счет | 1,14 долл. США в час |
    |:--- |:--- |
    | Корпорация Майкрософт оплачивает 80 % стоимости лицензии | 0,8 долл. США в час |
    | Корпорация Майкрософт удерживает 20 % стоимости лицензии | 0,2 долл. США в час |
    | Корпорация Майкрософт удерживает стоимость использования Azure | 0,14 долл. США в час |

*   Если выберете вариант BYOL, вы получите следующий результат.
    | Стоимость лицензии | Лицензионная плата, которую согласовываете и выставляете вы |
    |:--- |:--- |
    | Стоимость использования Azure (D1 с 1 ядром) | 0,14 долл. США в час |
    | *Счет выставляется корпорацией Майкрософт* | *0,14 долл. США в час* |

    В этом случае корпорация Майкрософт выставляет счет в размере 0,14 долл. США в час за использование опубликованного образа виртуальной машины. 
    | Майкрософт выставляет счет | 0,14 долл. США в час |
    |:--- |:--- |
    | Корпорация Майкрософт удерживает стоимость использования Azure | 0,14 долл. США в час |
    | Корпорация Майкрософт удерживает 0 % стоимости лицензии | 0 долл. США в час |

## <a name="single-billing-and-payment-methods"></a>Единый метод выставления счетов и оплаты
Важным преимуществом использования в Marketplace варианта предложения "Транзакция" является то, что стоимость лицензирования и использования Azure оплачивает непосредственно клиент.

В этом случае корпорация Майкрософт выставляет счета и получает оплату от вашего имени. Это устраняет необходимость создания собственных торговых отношений с клиентом. Единый метод выставления счетов экономит время и ресурсы. Он также может помочь сконцентрироваться на продажах, а не на сборе оплаты. 

## <a name="enterprise-agreement"></a>Соглашение Enterprise  
Если вы являетесь клиентом соглашения Microsoft Enterprise Agreement, можно использовать свое соглашение Enterprise для оплаты продуктов Майкрософт. Вам могут быть выставлены счета за продукты, в том числе за использование Azure. Оплата с помощью соглашения Enterprise предназначена для организаций, которые хотят лицензировать программное обеспечение и облачные службы в течение не менее трех лет. Вы распределяете платежи, а не выплачиваете одноразовый авансовый платеж. Если используется вариант оплаты по мере использования, выставление счетов за лицензирование на программное обеспечение соответствует ежеквартальному циклу выставления счетов по соглашению Enterprise.  

### <a name="monetary-commitment"></a>Денежное обязательство
Если вы являетесь клиентом соглашения Enterprise, к соглашению можно добавить Azure. При этом вы принимаете на себя предварительные денежные обязательства Azure. Ваши денежные обязательства действуют в течение года. Ваше предложение включает в себя любую комбинацию использования служб Azure.
