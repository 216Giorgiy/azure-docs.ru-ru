# <a name="billing-options"></a>Варианты выставления счетов  

## <a name="commercial-considerations-on-the-marketplace"></a>Рекомендации по коммерческим предложениям в Marketplace  
Marketplace не поддерживает разделение прибыли для следующих типов предложений:  
*   список  
*   Пробная версия  
*   "Транзакция" с применением модели выставления счетов с использованием собственной лицензии (BYOL).

Дополнительная плата за участие в онлайн-магазинах Marketplace не взимается.  
*   Дополнительные сведения см. на странице политик участия в Azure Marketplace по адресу [azure.microsoft.com/support/legal/marketplace/participation-policies](https://azure.microsoft.com/support/legal/marketplace/participation-policies).  

## <a name="pay-as-you-go-and-bring-your-own-license-billing-options"></a>Варианты выставления счетов "Оплата по мере использования" и "Использование собственной лицензии"  
При использовании модели выставления счетов с оплатой по мере использования 80/20 % лицензионных выплат по модели на основе использования получаете вы, а 20 % — корпорация Майкрософт. Для одного предложения могут использоваться модели выставления счетов "Оплата по мере использования" и "Использование собственной лицензии". Эти две модели сосуществуют на уровне предложения как отдельные номера SKU. Модели выставления счетов для предложения настраиваются на портале Cloud Partner. 

Рассмотрим следующие примеры.  
*   Если вы включите вариант "Оплата по мере использования", вы получите следующий результат.  
    | Стоимость лицензии | 1 долл. США в час |  
    |:--- |:--- |  
    | Стоимость использования Azure (D1 с 1 ядром) | 0,14 долл. США в час |  
    | *Счет выставляется корпорацией Майкрософт* | *1,14 долл. США в час* |  

    В этом случае корпорация Майкрософт выставляет счет в размере 1,14 долл. США в час за использование опубликованного образа виртуальной машины.  
    | Майкрософт выставляет счет | 1,14 долл. США в час |  
    |:--- |:--- |  
    | Корпорация Майкрософт оплачивает 80 % стоимости лицензии | 0,8 долл. США в час |  
    | Корпорация Майкрософт удерживает 20 % стоимости лицензии | 0,2 долл. США в час |  
    | Корпорация Майкрософт удерживает стоимость использования Azure | 0,14 долл. США в час |  

*   Если вы выберете вариант BYOL, вы получите следующий результат.  
    | Стоимость лицензии | Лицензионная плата, которую согласовываете и выставляете вы |  
    |:--- |:--- |  
    | Стоимость использования Azure (D1 с 1 ядром) | 0,14 долл. США в час |  
    | *Счет выставляется корпорацией Майкрософт* | *0,14 долл. США в час* |  

    В этом случае корпорация Майкрософт выставляет счет в размере 0,14 долл. США в час за использование опубликованного образа виртуальной машины.  
    | Майкрософт выставляет счет | 0,14 долл. США в час |  
    |:--- |:--- |  
    | Корпорация Майкрософт удерживает стоимость использования Azure | 0,14 долл. США в час |  
    | Корпорация Майкрософт удерживает 0 % стоимости лицензии | 0 долл. США в час |  

## <a name="single-billing-and-payment-methods"></a>Единый метод выставления счетов и оплаты  
Важным преимуществом использования варианта предложения "Транзакция" является то, что вашу стоимость лицензирования и использования Azure оплачивает непосредственно ваш клиент.  

В этом сценарии корпорация Майкрософт выставляет счета и получает оплату от вашего имени. Это устраняет необходимость создания собственных торговых отношений с клиентом. Единый метод выставления счетов экономит ваше время и ресурсы. Вы можете сконцентрироваться на продажах, а не на счетах. 

## <a name="enterprise-agreement"></a>Соглашение Enterprise  
Если вы клиент корпорации Майкрософт с соглашением Enterprise (EA), вы можете его использовать для оплаты продуктов Майкрософт. Вы можете оплачивать товары, включая использование Azure. Оплата с помощью EA предназначена для организаций, которые хотят лицензировать программное обеспечение и облачные службы в течение не менее трех лет. Вы можете распределить платежи, а не выплачивать одноразовый аванс. Если вы используете тип предложения "Транзакция" с оплатой по мере использования, выставление счетов за лицензию программного обеспечения издателя осуществляется в соответствии с ежеквартальным циклом EA за превышение объема потребления.  

### <a name="monetary-commitment"></a>Денежное обязательство  
Если вы являетесь клиентом EA, вы можете добавить Azure к своему соглашению. При этом вы принимаете на себя предварительные денежные обязательства Azure. Ваши денежные обязательства действуют в течение года. Ваше обязательство включает в себя любую комбинацию использования широкого спектра услуг в Azure.
