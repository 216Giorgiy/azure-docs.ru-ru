# <a name="enable-lead-sharing"></a>Возможность общего доступа к данным о потенциальных клиентах  
Убедитесь, что управление потенциальными клиентами включено для вашего предложения в Marketplace, чтобы вы могли получать контактную информацию для следующих категорий аудитории.

| Онлайн-магазин | audience |  
|:--- |:--- |  
| AppSource | Бизнес-клиенты |  
| Azure Marketplace | Клиенты |  

Потенциальные клиенты обеспечивают базу для последующих кампаний и рекламных акций ваших специалистов по продажам, а также позволяют получить сведения об эффективности своего предложения. 

| Онлайн-магазин | campaign |  
|:--- |:--- |  
| AppSource | Развитие |  
| Azure Marketplace | Создание спроса |  

Используйте следующие рекомендации по работе с потенциальными клиентами.  
*   Квалификация потенциальных клиентов и их оценка в качестве возможностей для продаж.
*   Развитие потенциальных клиентов для активирования канала продаж.
*   Совмещение инициатив с общей стратегией маркетинговой кампании. 

Потенциальные клиенты демонстрируют заинтересованность клиентов в следующем предложении. 

| Онлайн-магазин | Предложение |  
|:--- |:--- |  
| AppSource | <table> <tr><td>Приложение</td></tr> <tr><td>Консультационная услуга</td></tr> </table> |  
| Azure Marketplace | <table> <tr><td>ПРЕДЛОЖЕНИЕ</td></tr> <tr><td>Технология</td></tr> </table> |  

У вас должен быть налажен процесс для работы с потенциальными клиентами.  

| Онлайн-магазин | Потенциальные клиенты |  
|:--- |:--- |  
| AppSource | <table> <tr><td>Система управления отношениями с клиентами (CRM)</td></tr> <tr><td>Специалист по продажам Microsoft CRM</td></tr> </table> |  
| Azure Marketplace | <table> <tr><td>Azure Marketplace</td></tr> <tr><td>Специалист по продажам Microsoft CRM</td></tr> </table> |  

Потенциальные клиенты онлайн-магазина в Marketplace отличаются от определения высокоэффективных потенциальных клиентов, что является отличительной чертой программы совместных продаж.  
*   Дополнительные сведения о получении доступа к программе "Совместная продажа" см. на странице о развитии бизнеса с помощью решений Майкрософт [здесь](./promote-your-business-with-microsoft.md).
