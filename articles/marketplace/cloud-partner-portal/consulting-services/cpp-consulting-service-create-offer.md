# <a name="create-offer"></a>Создание предложения

В этой статье описывается, как создать и опубликовать предложение консультационных услуг.

## <a name="define-your-offer"></a>Описание своего предложения

Определите свое пакетное предложение консультационных услуг. Сосредоточьтесь на фиксированном объеме, фиксированной продолжительности, оценке, фиксированной цене (или бесплатном предложении) и в первую очередь на предложениях, ориентированных на предпродажи, для отдельного клиента. Выберите проверенные пакеты услуг, которые оказались наиболее популярными и эффективными для вовлечения клиентов для вашего бизнеса.

## <a name="create-a-new-offer"></a>Создание нового предложения

Чтобы создать предложение, сделайте следующее:
1. В главном меню на портале Cloud Partner выберите **New offer** (Создать предложение).
1. В меню **New Offer** (Новое предложение) выберите **Consulting Service** (Консультационная услуга).

    ![Создание нового предложения консультационных услуг](media/cppselectnewconsultingoffer.png)
