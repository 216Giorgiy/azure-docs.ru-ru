# <a name="publish-azure-application-offer"></a>Публикация предложения приложения Azure

Опубликовать предложение можно после создания нового предложения с информацией на странице **Новое предложение**. Выберите **Публиковать**, чтобы начать процесс публикации.

На следующей схеме представлены основные шаги в процессе публикации для запуска предложения.

  ![Шаги публикации предложения](./media/offer-publishing-steps.png)

## <a name="detailed-description-of-publishing-steps"></a>Подробное описание шагов публикации

В следующей таблице описывается каждый шаг публикации с оценкой времени (максимум) для завершения каждого шага.
В приведенной ниже таблице описан каждый этап публикации. Также здесь указано предполагаемое время завершения каждого этапа.


|  **Шаг публикации**           | **Время**    | **Описание**                                                            |
|  -------------------           | --------    | ---------------                                                            |
| Проверка необходимых компонентов         | 15 мин.   | Информация и параметры предложения — проверенные.                        |
| Сертификация                  | 1 неделя | Команда сертификации Azure анализирует предложение. Предложение проверяется на предмет вирусов, вредоносных программ, соответствия стандартам безопасности и проблем безопасности. Это позволяет определить, соответствует ли предложение всем критериям применимости. Дополнительные сведения см. в разделе [Предварительные требования](./cpp-prerequisites.md). При обнаружении проблем отправляется отзыв. |
| Упаковка | 1 час  | Технические ресурсы предложения упакованы для использования клиентами, а ведущие системы настроены и установлены. |
|  Утверждение издателем             |  -        | Просмотрите отзыв издателя и подтверждение активации предложения. Убедитесь, что предложение соответствует всем требованиям. Для этого разверните его в выбранной подписке (на шагах сведений о предложении).  Выберите **Go Live** (Запустить), чтобы сделать следующий шаг. |
| Упаковка                 | 1 час | Готовое предложение реплицируется в производственные системы и регионы Marketplace. | 
| Активная                           | 4 дня |Предложение выпущено, реплицировано в необходимые регионы и доступно для просмотра. |

Предоставляется до 10 рабочих дней для завершения процесса публикации и выпуска предложения. После публикации предложение будет добавлено в список в [Microsoft Azure Marketplace](https://azuremarketplace.microsoft.com/marketplace/apps/).
