## <a name="publishing-steps"></a>Этапы публикации

Ниже приведены общие инструкции по публикации предложения приложение Power BI.

![Приложение Power BI предлагают этапы публикации](media/publishing-steps.png)

Ниже приведен процесс публикации предложения приложение Power BI.

1. Создание приложения службы в Power BI. Это действие создает пакет установки URL-адрес, который представляет технические средства для предложения. Дополнительные сведения см. в разделе [что такое Power BI?](https://go.microsoft.com/fwlink/?linkid=2028636).
2. Соберите или создайте бизнес-ресурсы, включая:
    - Маркетинговые материалы.
    - Юридические и поддержки документов.
3.  Создайте предложение. Позволяет настроить сведения предложения, включая описание предложения, маркетинговые материалы, юридические сведения, сведения о поддержке и спецификации активов на портале Cloud Partner.
4.  Сертификация и публикация предложения. Запустится процесс публикации.
    - Команда по подключению AppSource тестирует, проверяет и сертифицирует ваше приложение. 
    - После предложения является сертифицированным, просмотрите ее в тестовую среду и освободите его.

После выполнения этих шагов, ваше приложение Power BI предложение станет активным в AppSource.
