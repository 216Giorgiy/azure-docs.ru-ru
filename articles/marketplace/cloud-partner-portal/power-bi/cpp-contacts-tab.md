# <a name="power-bi-apps-contacts-tab"></a>Вкладка контактов приложений Power BI

Используйте вкладку **Контакты** на странице **Новое предложение**, чтобы предоставить технические ресурсы и пользовательскую поддержку для предложения. Она разделена на две части: **Инженерные работы** и **Поддержка клиентов**.

![Вкладка "Контакты"](media/contacts-tab.png)


## <a name="contacts-fields"></a>Поля контактов 

В следующей таблице представлены названия и назначение этих полей. 

|   Поле               |   ОПИСАНИЕ                                                            |
|-----------------------|--------------------------------------------------------------------------|
| *Контакт по техническим вопросам* | Контактное лицо для связи по техническим вопросам между AppSource и вашей организацией.    |
| **Имя**              | Имя лица или группы, предоставляющих техническую поддержку. |
| **Электронная почта**             | Адрес электронной почты для связи по техническим вопросам.                                  |
| **Номер телефона**             | Номер телефона для получения технической поддержки.                                       |
| *Контакт поддержки*     | Лицо, принимающее запросы на поддержку, открытые клиентами в AppSource.            |
| **Имя**              | Имя лица или группы, отвечающих за поддержку клиентов.              |
| **Электронная почта**             | Адрес электронной почты службы поддержки клиентов                                        |
|  |  |


## <a name="next-steps"></a>Дополнительная информация

После заполнения предложения приложения Power BI необходимо подтвердить создание всех связанных [технических ресурсов](./cpp-create-technical-assets.md) для этого предложения.
