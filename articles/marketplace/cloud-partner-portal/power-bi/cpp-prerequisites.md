# <a name="power-bi-apps-prerequisites"></a>Предварительные требования для приложений Power BI

В этой статье перечислены технические и бизнес-требования, которые необходимо выполнить, прежде чем публиковать предложение приложения Power BI в Microsoft AppSource Marketplace.

## <a name="technical-requirements"></a>Технические требования

Основной технический ресурс, необходимый для этого типа предложения, — [приложение Power BI](https://go.microsoft.com/fwlink/?linkid=2028636). Дополнительные сведения см. в статье [Создание технических ресурсов приложения Power BI](./cpp-create-technical-assets.md).

После создания и проверки приложения службы в Power BI вам необходимо сохранить URL-адрес установки приложения, генерируемый службой Power BI. Он потребуется для [создания предложения приложения Power BI](./cpp-create-offer.md).

## <a name="business-requirements"></a>Бизнес-требования

Бизнес-требования включают процедурные, контрактные и юридические обязательства.

-   Необходимо зарегистрироваться в качестве издателя в облаке Marketplace. Если вы еще не зарегистрированы, выполните действия, описанные в статье [Регистрация в качестве издателя в облаке Marketplace](../../become-publisher.md).

    > [!NOTE] 
    > Следует использовать одну и ту же учетную запись регистрации Центра разработки для Майкрософт для входа на [портал Cloud Partner](https://cloudpartner.azure.com). У вас должна быть только одна учетная запись Майкрософт для публикации предложений в AppSource. Эта запись не должна быть привязана к конкретным службам или предложениям.
 
-   Вы должны предоставить клиентам техническую поддержку экономически обоснованным образом.
-   Необходимо предоставить маркетинговые материалы, отвечающие критериям вашего предложения, соблюдение которых требуется для добавления приложения в AppSource. Дополнительные сведения см. в записи блога [Have an app to list on AppSource? Here’s how](https://appsource.microsoft.com/blogs/have-an-app-to-list-on-appsource-here-s-how) (Есть приложение для публикации в AppSource? Вот как это можно сделать).
-   Вы должны следовать [заявлению о конфиденциальности корпорации Майкрософт](https://privacy.microsoft.com/privacystatement).


## <a name="next-steps"></a>Дополнительная информация

Когда все требования будут соблюдены, вы будете готовы [создать предложение приложения Power BI](./cpp-create-offer.md) на Портале Cloud Partner.
