# <a name="power-bi-apps-offer-settings-tab"></a>Вкладка параметров предложения для приложения Power BI

На странице **Новое предложение** приложений службы первой откроется вкладка **Параметры предложения**.  На этой вкладке нужно указать основные идентификаторы и имя для вашего предложения.  Символ звездочки (*) возле имени поля означает, что оно является обязательным.

![Вкладка "Параметры предложения"](./media/offer-settings-tab.png)


## <a name="offer-settings-fields"></a>Поля параметров предложения 

На вкладке **Параметры предложения** вы должны заполнить следующие обязательные поля.

|  Поле        |  ОПИСАНИЕ                                                               |
|---------------|----------------------------------------------------------------------------|
| **Идентификатор предложения**  | Уникальный идентификатор предложения (в профиле издателя). Этот идентификатор отображается в URL-адресах продукта, шаблонах Resource Manager и отчетах о выставлении счетов. Он должен состоять не более чем из 50 буквенно-цифровых символов нижнего регистра и дефисов, но не может заканчиваться дефисом. После активации предложения это поле нельзя изменить. Например, если Contoso опубликует предложение с идентификатором `sample-SvcApp`, в AppSource ему будет присвоен URL-адрес `https://appsource.microsoft.com/marketplace/apps/contoso.sample-SvcApp`.      |
| **Издатель** | Уникальный идентификатор организации в [AppSource](https://appsource.microsoft.com). Для всех предложений необходимо указывать свой идентификатор издателя. Это значение нельзя изменить после того, как предложение будет сохранено.                         |
| **Имя**      | Отображаемое имя для предложения. Это имя будет отображаться в AppSource и на Портале Cloud Partner. Его длина не должна превышать 50 символов. Рекомендуется указать для вашего продукта легко узнаваемое название марки. Не включайте в него название своей организации, если только это не требуется для продвижения продукта. Если вы продаете это предложение на других веб-сайтах и в ​​публикациях, убедитесь, что имя во всех публикациях одинаково.    <br/>Если требуется выпустить предложение в период действия предварительной версии приложений Power BI (режим предварительного просмотра), добавьте строку `(Preview)` к имени приложения, например `Sample Scv App (Preview)`. |
|     |     |
