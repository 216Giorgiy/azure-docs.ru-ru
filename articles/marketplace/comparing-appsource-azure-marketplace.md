# <a name="comparing-appsource-and-azure-marketplace"></a>Сравнение AppSource и Azure Marketplace  
Онлайн-магазины обслуживают уникальные требования заказчиков. В них включена возможность нацеленности на основе роли. Вы можете предложить подходящее решение или службу с учетом требований клиента.  

## <a name="understanding-the-differences-between-the-storefronts"></a>Различия между онлайн-магазинами  
Выбор онлайн-магазина начинается с определения целевой аудитории вашего предложения. Если оба онлайн-магазина подходят, можно опубликовать приложение один раз для продажи в обоих онлайн-магазинах.  
 
Рассмотрим дополнительные преимущества каждого онлайн-магазина.   

### <a name="appsource"></a>AppSource  

| Преимущества | Сведения | 
|:--- |:--- | 
| ***Гибкость выставления счетов*** | Подготавливает пробную версию. Сейчас не предоставляет вариант публикации для коммерческого использования. Вы можете использовать текущую систему заказа и выставления счетов без дополнительных инвестиций или изменений. | 
| ***Упрощение связи с другими партнерами*** | Связывает независимых поставщиков программного обеспечения, системных интеграторов и поставщиков управляемых служб в рамках конкретных сценариев реализации. Вы можете совместно продавать службы или предложения новым клиентам. | 
| ***Автоматизация*** | Связывает независимых поставщиков программного обеспечения, системных интеграторов и поставщиков управляемых служб в рамках конкретных сценариев реализации. Вы можете совместно продавать службы или предложения новым клиентам. |  
| ***Несколько типов облаков*** | Сейчас не поддерживает Azure Stack, Azure для государственных организаций или региональных облаков. | 
| ***Контекстная презентация для клиентов*** | Привлекайте большее количество клиентов через взаимодействие в приложении для продуктов Майкрософт, таких как Dynamics 365, Power BI и Office 365. |  

### <a name="azure-marketplace"></a>Azure Marketplace  

| Преимущества | Сведения | 
|:--- |:--- | 
| ***Гибкость выставления счетов*** | Для оплаты за решение "Виртуальные машины" по мере использования применяйте корпоративные соглашения Майкрософт или модели прямых веб-продаж. Варианты ценообразования включают в себя подписку на пробную версию, в рамках которой предложение предоставляется бесплатно на неограниченный срок. Варианты ценообразования также включают в себя подписку типа "Попробуйте сейчас". Подписка типа "Попробуйте сейчас" является бесплатной в течение ограниченного периода времени, по истечении которого виртуальная машина преобразуется в платную подписку. Активация с использованием собственной лицензии (BYOL) также поддерживается для клиентов.<br />Для обоих вариантов выставления счетов и следующих предложений виртуальной машины плата за использование всех подготовленных ресурсов Azure взимается непосредственно с клиента.<ul> <li>Приложения Azure. Управляемое приложение</li> <li>Приложения Azure. Шаблон решения</li> </ul> | 
| ***Упрощение связи с другими партнерами*** | Сейчас вы не можете связать поставщика службы или партнеров доставки с вашим предложением. Эта функция будет доступна в 2018 году. | 
| ***Автоматизация*** | Сейчас вы не можете связать поставщика службы или партнеров доставки с вашим предложением. Используйте автоматизированную подготовку SaaS с помощью надстройки, а также шаблоны решений для приложений Azure, позволяющие автоматизировать сценарии сбора данных и развертывания на основе SaaS. |  
| ***Несколько типов облаков*** | Публикуйте в решениях общедоступного облака и локальной среды, используя следующие типы.<ul> <li>Azure Stack</li> <li>Azure Government</li> <li>региональные облака, включая Китай и Германию.</li> </ul> | 
| ***Контекстная презентация для клиентов*** | Сделайте свое решение доступным для контекстного поиска на портале Azure, используя виртуальные машины и шаблоны решений для приложений Azure. |
