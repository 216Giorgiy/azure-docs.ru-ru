# <a name="become-a-publisher-in-the-storefronts-on-the-marketplace"></a>Как стать издателем в онлайн-магазинах в Marketplace  
Следующие шаги помогут вам зарегистрироваться в качестве издателя в онлайн-магазинах в Marketplace. В зависимости от вашего статуса взаимодействия с Майкрософт и выбранного типа предложения некоторые шаги выполнять не нужно.  

|  | Шаг регистрации | Время | Сведения |  
|:--- |:--- |:--- |:--- |  
| 1 | Регистрация в партнерской сети Microsoft Partner Network | 15 мин. | [Регистрация в партнерской сети Microsoft Partner Network](#register-in-microsoft-partner-network) |  
| 2 | Создание идентификатора Майкрософт | 15 минут | [Создание идентификатора Майкрософт](#create-a-microsoft-id) |  
| 3 | Подача формы предварительного утверждения Marketplace | От 1 до 3 дней | [Подача формы предварительного утверждения Marketplace](#submit-the-marketplace-nomination-form) |  
| 4. | Регистрация в Центре разработчиков Майкрософт | От 5 до 10 дней | [Регистрация в центре разработчиков](#register-in-dev-center) |  
| 5 | Вход в AppSource | 15 минут | [Вход в AppSource](#sign-into-appSource) |  
| 5 |  Вход на Портал Cloud Partner | 15 минут | [Вход на Портал Cloud Partner](#sign-into-cloud-partner-portal) |  

## <a name="register-in-microsoft-partner-network"></a>Регистрация в партнерской сети Microsoft Partner Network  
| 15 мин |  
Присоединитесь к сети Microsoft Partner Network (MPN), чтобы стать официальным партнером Майкрософт и получить дополнительные преимущества и поддержку в обмен на публикацию в Azure Marketplace. Зарегистрируйтесь в MPN и щелкните `Enroll`.  
*   Чтобы зарегистрироваться в MPN, посетите страницу членства по адресу [partner.microsoft.com/membership](https://partner.microsoft.com/membership).  

---  

Если у вашей организации есть членство, вы можете указать организацию во время регистрации. После регистрации запишите идентификатор MPN вашей организации. Корпорация Майкрософт запрашивает идентификатор MPN перед активацией профиля издателя на Портале Cloud Partner. 
*   Дополнительные сведения об активации профиля издателя можно найти в разделе [Регистрация в Центре разработчиков](#register-in-dev-center).  

---  

## <a name="create-a-microsoft-id"></a>Создание идентификатора Майкрософт  
| 15 минут |  
Идентификатор Майкрософт предоставляет доступ к Порталу Cloud Partner. В качестве идентификатора Майкрософт нужно зарегистрировать адрес электронной почты, который будет использоваться как для Портала Cloud Partner, так и для Центра разработчиков. Адрес электронной почты по возможности должен находиться в домене организации и управляться ИТ-отделом.  
*   Дополнительные сведения о регистрации в Центре разработчиков можно найти в разделе [Регистрация в Центре разработчиков](#register-in-dev-center).  
*   Дополнительные сведения о входе на Портал Cloud Partner см. в разделе [Вход на Портал Cloud Partner](#sign-into-cloud-partner-portal).  
*   Ознакомьтесь также разделом [Guidelines for creating a Microsoft ID to manage a marketplace account](./guidelines.md#guidelines-for-creating-a-microsoft-id-to-manage-a-marketplace-account) (Рекомендации по созданию идентификатора Майкрософт для управления учетной записью Marketplace).

## <a name="submit-the-marketplace-nomination-form"></a>Подача формы предварительного утверждения Marketplace  
| От 1 до 3 дней |  
Подайте заявку на регистрацию своей организации в качестве издателя в онлайн-магазинах в Marketplace. 
*   Чтобы отправить форму заявки на регистрацию в Azure Marketplace, откройте страницу "Форма предварительного утверждения Azure Marketplace" по адресу [aka.ms/listonazuremarketplace](http://aka.ms/listonazuremarketplace)  
*   Чтобы отправить форму заявки на регистрацию в AppSource, откройте страницу "Расскажите о своем приложении" по адресу [aka.ms/listonappsource](http://aka.ms/listonappsource)  

Включите в форму запроса следующие сведения:  
*   информация о вашей организации;  
*   сведения об исходном приложении или предложении;  
*   сведения о уровне поддержки, который вы предоставляете клиентам.  

Мы проверим и утвердим ваше приложение или предложение после отправки формы. Если заявка будет подтверждена, вы получите сообщение электронной почты с описанием, как зарегистрироваться в качестве официального партнера. Также в этом сообщении будет включен рекламный код, позволяющий зарегистрировать учетную запись Центра разработчиков без вступительного взноса. Кроме того, вы получите доступ к Порталу Cloud Partner или AppSource. Завершите создание первого приложения на AppSource или первого предложения на Портале Cloud Partner. Создайте дополнительные приложения на AppSource или предложения на Портале Cloud Partner.  
*   Дополнительные сведения о регистрации в Центре разработчиков можно найти в разделе [Регистрация в Центре разработчиков](#register-in-dev-center).

## <a name="register-in-dev-center"></a>Регистрация в Центре разработчиков  
| От 5 до 10 дней |  
Учетная запись Центра разработчиков является обязательным условием для публикации приложений с типом Transact, таких как виртуальные машины, шаблоны решений (приложения Azure) и управляемые приложения (приложения Azure). Это требование позволяет корпорации Майкрософт проверить юридическую, налоговую и банковскую информацию о вашей компании. Пользователь, выполняющий регистрацию, должен быть действующим представителем организации и обязан предоставить личные сведения для подтверждения своей личности. Для регистрации нужно использовать идентификатор Майкрософт. Идентификатор Майкрософт должен использоваться для всей компании. Нужно использовать тот же идентификатор Майкрософт на Портале Microsoft Cloud Partner.  
*   Дополнительные сведения о создании идентификатора Майкрософт есть в разделе [Создание идентификатора Майкрософт](#create-a-microsoft-id).  

>[!NOTE]
>Если вы не заполните форму предварительного утверждения в Marketplace, за регистрацию нужно заплатить 99 долл. США. Чтобы не платить этот сбор, заполните форму предварительного утверждения в Marketplace и получите рекламный код по электронной почте.  

>[!Important]
>Прежде чем создавать учетную запись Центра разработчиков, убедитесь, что ваша организация еще не имеет учетной записи Центра разработчиков. 
>*   Дополнительные сведения см. в разделе [Регистрация в Центре разработчиков](#how-to-register-in-dev-center).

## <a name="sign-into-appsource"></a>Вход в AppSource  
| 15 минут |  
После подтверждения заявки и регистрации в Центре разработки и MPN для вас создается учетная запись Marketplace с доступом к AppSource. Учетные данные для первого входа будут указаны в сообщении электронной почты с информацией об утверждении заявки. 
*   Чтобы получить доступ к MPN, посетите страницу членства по адресу [partner.microsoft.com/membership](https://partner.microsoft.com/membership).  
*   Чтобы получить доступ к Центру разработчиков, посетите страницу центра разработчиков, расположенную по адресу [dev.windows.com](https://dev.windows.com).  
*   Чтобы получить доступ к AppSource, посетите страницу AppSource по адресу [appsource.microsoft.com](https://appsource.microsoft.com).  
*   Чтобы получить доступ к Порталу Microsoft Cloud Partner, откройте страницу Портала по адресу [cloudpartner.azure.com](https://cloudpartner.azure.com).  
*   Дополнительные сведения об использовании Портала Cloud Partner можно получить в разделе знакомства по адресу [cloudpartner.azure.com/#Learn](https://cloudpartner.azure.com/#Learn).  
*   Документация по Порталу Cloud Partner размещена в разделе по адресу [cloudpartner.azure.com/#documentation](https://cloudpartner.azure.com/#documentation).

## <a name="sign-into-cloud-partner-portal"></a>Вход на Портал Cloud Partner
| 15 минут |  
После подтверждения заявки и регистрации в Центре разработки и MPN для вас создается учетная запись Marketplace с доступом к Порталу Cloud Partner. Учетные данные для первого входа будут указаны в сообщении электронной почты с информацией об утверждении заявки. 
*   Чтобы получить доступ к MPN, посетите страницу членства по адресу [partner.microsoft.com/membership](https://partner.microsoft.com/membership).  
*   Чтобы получить доступ к Центру разработчиков, посетите страницу центра разработчиков, расположенную по адресу [dev.windows.com](https://dev.windows.com).  
*   Чтобы получить доступ к Порталу Microsoft Cloud Partner, откройте страницу Портала по адресу [cloudpartner.azure.com](https://cloudpartner.azure.com).  
*   Дополнительные сведения об использовании Портала Cloud Partner можно получить в разделе знакомства по адресу [cloudpartner.azure.com/#Learn](https://cloudpartner.azure.com/#Learn).  
*   Документация по Порталу Cloud Partner размещена в разделе по адресу [cloudpartner.azure.com/#documentation](https://cloudpartner.azure.com/#documentation).
