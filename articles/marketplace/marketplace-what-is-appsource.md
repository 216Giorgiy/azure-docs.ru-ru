# <a name="what-is-appsource"></a>Что такое AppSource
AppSource позволяет партнерам Майкрософт делать свои технологии и услуги доступными для клиентов в интернет-магазине, поддерживаемом Майкрософт. Как издатель AppSource вы можете привлечь бизнес-пользователей с помощью AppSource, чтобы найти, протестировать и получить бизнес-приложения SaaS и службы внедрения для достижения бизнес-результатов и ускорения окупаемости инвестиций: 

| Потребность клиента       | AppSource          | 
|:---------------------------------------- |:----------------------------------------------------- |
| **Поиск бизнес-решений, которые работают с уже использующимися продуктами Майкрософт**    | Позволяет партнерам использовать сторонние приложения и службы для расширения возможностей приложений и технологий облачной платформы Майкрософт. |
| **Возможность легко находить подходящее решение или службу реализации**     | Предоставляет единый онлайн-магазин для поиска, тестирования и получения приложений и служб, надстроек и т. д. |
| **Бизнес-решение с учетом отраслевой специфики для решения конкретных бизнес-задач**   | Предоставляет готовое полное отраслевое решение для решения конкретных требований во многих отраслях. |
| **Приложения, помогающие улучшить производительность, эффективность и бизнес-аналитику**       | Предоставляет приложения для сферы деятельности, включая обслуживание клиентов, управление персоналом, операции и многое другое. |
| **Опытный партнер по реализации, позволяющий адаптировать приложения к уникальной ситуации**     | Предоставляет каталог предложений консультационных услуг для решений на основе Dynamics 365, Power BI, PowerApps и сторонних приложений, доступных в AppSource, чтобы помочь бизнес-пользователям найти консультационные услуги для достижения предсказуемых результатов. |

## <a name="appsource-publishing"></a>Публикация AppSource
В AppSource можно разместить предложение приложения или консультационной услуги, которое соответствуют требованиям пользователей, использующим продукты Майкрософт (например, Office 365, Dynamics 365, Power BI и Power Apps). В Azure Marketplace доступны следующие предложения. 
*   **Dynamics 365 for Finance and Operations**. Решение для планирования ресурсов предприятия (ERP). Дифференцированными решениями являются Enterprise Edition для средних и крупных компаний и Business Edition для малого и среднего бизнеса.
*   **Dynamics 365 for Customer Engagement**. Решение для управления отношениями с клиентами (CRM), включающее приложения Sales, Customer Service, Field Service и Project Service Automation.
*   **Управляемая служба Dynamics NAV**. Microsoft **Dynamics NAV** является комплектом программного обеспечения для планирования ресурсов предприятия (ERP) для организаций среднего размера. Эта служба обеспечивает специализированные функциональные возможности для производства, распространения, государственного управления, розничных продаж и других отраслей.
*   **Power BI**. Решение визуализации бизнес-аналитики.
*   **Консультационные услуги**. Службы, предоставляемые партнерами корпорации Майкрософт, чтобы помочь клиентам понять, протестировать и реализовать специализированные технологические решения.
*   **Cortana Intelligence**. Cortana Intelligence Suite — это коллекция независимых, но полностью интегрированных средств платформы данных и аналитики Azure.
*   **Office 365**. Современная рабочая область в облаке. Бесплатное взаимодействие с Microsoft Word, PowerPoint, Excel и OneNote.

Ознакомьтесь с [рекомендациями и узнайте, как отправить предложение](https://smp-cdn-prod.azureedge.net/documents/Microsoft%20AppSource%20Partner%20Listing%20Guidelines.pdf "Рекомендации по размещению в AppSource для партнеров").

## <a name="azure-active-directory-integration"></a>Интеграция с Azure Active Directory
Для того, чтобы опубликовать некоторые приложения в онлайн-магазине AppSource, их необходимо интегрировать с Azure Active Directory (Azure AD). Интеграция приложения с Azure AD описана во многих документах. Корпорация Майкрософт предоставляет множество дополнительных ресурсов и пакетов SDK для соответствия вашим требованиям. 

| Название предложения | Требуется интеграция с Azure AD | Заметки |
|:--- |:---:|:--- |
| Dynamics 365 for Operations | Нет  | Для предложения на основе лицензии не требуется интеграция с Azure AD |
| Dynamics 365 for Finance | Нет  | Для предложения на основе лицензии не требуется интеграция с Azure AD |
| Dynamics 365 for Customer Engagement | Нет  | Для предложения на основе лицензии не требуется интеграция с Azure AD |
| Управляемая служба Dynamics NAV | Нет  | Для предложения на основе лицензии не требуется интеграция с Azure AD |
| Power BI | Нет  | Для предложения на основе лицензии не требуется интеграция с Azure AD |
| Предложения консультационных услуг | Нет  | Службы координируются партнером, а не через веб-интерфейс |
| Cortana Intelligence | Yes | Azure AD рекомендуется использовать, чтобы обеспечить удобство работы пользователей и привлечь потенциальных клиентов |
| Приложения SaaS (прежнее название "Веб-приложения") | Предпочтительно | Azure AD рекомендуется использовать, чтобы обеспечить удобство работы пользователей и привлечь потенциальных клиентов |
