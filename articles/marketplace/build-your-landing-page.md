# <a name="build-your-landing-page"></a>Создание целевой страницы  
Целевая страница на веб-сайте вашей компании — это канал от инициатив по созданию спроса до вашего рейтинга в Marketplace.  

Начните с определения своих целей. Для своих решений выберите целевого клиента и действие, которое он должен выполнить.  Решение может включать следующие действия:  
*   тестирование решения;  
*   получение решения сейчас  
*   и многое другое...  

Вы можете ссылаться на свою целевую страницу из различных источников в соответствии с вашими маркетинговыми тактиками. Это могут быть:  
*   События  
*   веб-трансляции;  
*   социальные сети;  
*   Технические документы  
*   сеансы технического обучения;  
*   пресс-релизы.  

Чем яснее посыл и призыв к действию, тем проще вашим клиентам найти решение.  

При наличии готового плана кампании создайте целевую страницу. Ознакомьтесь со следующей таблицей, чтобы повысить эффективность целевой страницы.  

| Рекомендации | Чего следует избегать |  
|:--- |:--- |  
| ***Четко сформулируйте, какие проблемы клиентов поможет решить ваш продукт и как для этого используется Azure*** |  Избегайте нечеткой формулировки относительно интеграции вашего решения в Azure. |  
| ***Создайте короткий URL-адрес, который легко запомнить*** | Не используйте длинные URL-адреса. Длинные URL-адреса плохо запоминаются, и их трудно найти. |  
| ***Добавьте релевантное визуальное содержимое, например видеоролик с отзывами заказчиков или с кратким описанием архитектуры*** | Из-за использования слишком большого количества текста клиенты могут игнорировать ваше содержимое и прекращать изучение. |  
| ***Создайте четкий призыв к действию, чтобы направить посетителей на вашу страницу каталога Marketplace*** | Избегайте использования слишком большого количества ссылок или действий на странице. |  
| ***Поместите свой призыв к действию в заголовок или четко выделяющийся раздел*** |  Не встраивайте ссылки в текст предложения Marketplace. |  
| ***Изучите рекомендации по ключевым словам и оптимизируйте страницу для поиска*** | Не предполагайте, что название вашего продукта создаст более высокий рейтинг поиска само по себе. |  
| ***Используйте ключевые слова из своих рекламных кампаний*** |  Использование разных ключевых слов в ваших веб-ресурсах снизит эффективность ваших рекламных инвестиций. |  
| ***Поместите имена соответствующих продуктов и ключевые слова`above the fold`*** | Не заставляйте пользователей прокручивать страницу, чтобы увидеть, какой продукт или решение вы представляете. |  
| ***Используйте изображения торговой марки для проверки вашего решения в соответствии с рекомендациями по фирменной символике корпорации Майкрософт, например <br />"Сертифицировано для Azure"***. | Не используйте неутвержденные изображения фирменной символики корпорации Майкрософт. |  

*   Дополнительные сведения об эмблеме "Сертифицировано для Azure" см. по адресу [azure.microsoft.com/support/legal/marketplace/certified-guidelines](https://azure.microsoft.com/support/legal/marketplace/certified-guidelines).
