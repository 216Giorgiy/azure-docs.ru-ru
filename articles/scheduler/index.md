# <a name="scheduler-documentation"></a>Документация по планировщику

Планировщик Azure позволяет описывать действия, которые выполняются в облаке.  Затем он планирует и выполняет эти действия автоматически.  Планировщик хранит журнал результатов выполнения задания, который можно просмотреть.
