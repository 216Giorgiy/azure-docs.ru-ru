# Отслеживание работы для вашего веб-сайта. 

В этой теме вы узнаете, как использовать средства отслеживания рабочего элемента для управления работой на веб-сайте.

## Практическое руководство.Добавление рабочих элементов в список невыполненной работы
1. Перейдите к выноске командного проекта.

2. В области "Работа" в нижней части экрана щелкните по компоненту невыполненной работы. 

 Запускается выноска невыполненной работы. Невыполненная работа представлена в виде упорядоченного списка работ для веб-сайта.<br />
 ![Откройте список невыполненной работы][Откройте список невыполненной работы]
3. Добавьте элемент, введя заголовок в расположенном сверху текстовом поле и нажав клавишу ВВОД. 

 При использовании шаблона процесса Scrum можно добавить ошибку или элемент невыполненной работы по продукту, выбрав их из раскрывающегося списка справа.

4. Щелкните по рабочему элементу, чтобы открыть его и добавить дополнительные сведения.<br /> 
![Откройте рабочий элемент][Откройте рабочий элемент]

5. В форме рабочего элемента можно изменить заголовок, добавить описание, назначить рабочий элемент пользователю и пометить его как завершенный.

6. В нижней части формы рабочего элемента можно добавить комментарий для рабочего элемента и просмотреть его журнал. 

 Чтобы просмотреть более подробный журнал рабочего элемента, можно щелкнуть по изменению и открыть выноску с подетальным изложением того, что случилось. 

## Удалить работу после ее завершения
Управление списком невыполненной работыВедение хорошо упорядоченного списка невыполненной работы является ключом к успешному управлению работой над веб-сайтом. 
### Это можно осуществить двумя путями.
1. Откройте рабочий элемент, который нужно пометить как завершенный.
2. Отметьте флажком пункт с меткой completed.
3. Сохраните рабочий элемент.
4. Заглушка для данного элемента будет отображаться в списке невыполненной работы до тех пор, пока вы не обновите этот список либо не закроете и снова откроете его.<br />

![Завершенный рабочий элемент][Завершенный рабочий элемент]

## Поддержание списка невыполненной работы в упорядоченном состоянии
1. Наведите указатель мыши на рабочий элемент, приоритет которого вы хотите изменить.
2. Щелкните по полосе захвата слева от рабочего элемента.
3. Перетащите рабочий элемент в новое положение в списке невыполненной работы.<br />

![Перетаскивание рабочего элемента][Перетаскивание рабочего элемента]

## Практическое руководство. Просмотр недавно завершенной работы
1. На выноске командного проекта щелкните по выноске завершенных элементов.
2. Просмотрите список недавно завершенной работы.<br />
