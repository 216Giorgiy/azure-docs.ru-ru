#Как приобрести поддержку Azure?
Azure предоставляет гибкие варианты поддержки для компаний любых размеров — от разработчиков, начинающих работу в облаке, до предприятий, развертывающих критичные бизнес-приложения. Поддержка по вопросам выставления счетов и управления подпиской предоставляется бесплатно, однако на поддержку Azure по техническим вопросам потребуется приобрести подписку.

Дополнительные сведения о доступных для приобретения подписках Azure см. [здесь](https://azure.microsoft.com/ru-RU/support/legal/offer-details/). Чтобы приобрести техническую поддержку, посетите [страницу планов поддержки Azure](https://azure.microsoft.com/ru-RU/support/plans/).

Также доступны справочные ресурсы для самостоятельного изучения. [Часто задаваемые вопросы](https://azure.microsoft.com/ru-RU/support/faq/) можно просмотреть здесь. Статьи по поддержке доступны на вкладке "Документация" на каждой странице Azure.com.
