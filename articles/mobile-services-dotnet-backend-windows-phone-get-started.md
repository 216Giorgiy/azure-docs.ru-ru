Этот раздел больше не поддерживается. Проект быстрого начала работы с Windows Phone 8.0 уже не доступен для закачки с портала. Чтобы добавить мобильные службы в имеющееся приложение для Windows Phone 8.0 или Windows Phone 8.1, см. раздел [Добавление мобильных служб в существующее приложение](mobile-services-dotnet-backend-windows-phone-get-started-data.md). 

Скачать проект быстрого начала работы с приложениями для Магазина Windows Phone 8.1 можно в составе проекта универсальных приложений для Windows. Дополнительные сведения см. в разделе [Приступая к работе с мобильными службами](mobile-services-dotnet-backend-windows-store-dotnet-get-started.md).
