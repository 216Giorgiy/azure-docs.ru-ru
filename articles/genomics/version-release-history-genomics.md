# <a name="version-release-history"></a>Журнал выпуска версий
Команда Microsoft Genomics регулярно выпускает обновления для клиента Python Microsoft Genomics, связанные с исправлениями и новыми функциями. 

## <a name="latest-release"></a>Последний выпуск
0.7.4 — версия текущего клиента Python. Она выпущена 5 марта 2018 г. и использует GATK 3.3. 


## <a name="release-history"></a>Журнал выпуска версий 
Новые версии клиента Python Microsoft Genomics выпускаются два раза в год. По мере выпуска новых версий клиента Python Microsoft Genomics применяются обновления, связанные с исправлениями и новыми функциями. После выпуска новых версий использование предыдущих поддерживается как минимум 90 дней. Если предыдущие версии больше не поддерживаются, на этой странице появится уведомление. 

### <a name="version-074"></a>Версия 0.7.4
Версия 0.7.4 включает поддержку приема маркеров SAS вместо ключей учетной записи во входных данных `config.txt`. Дополнительные сведения см. в руководстве по [использованию входных маркеров SAS](quickstart-input-sas.md). 

### <a name="version-073"></a>Версия 0.7.3
Версия 0.7.3 включает исправления незначительных ошибок.

### <a name="version-072"></a>Версия 0.7.2
Версия 0.7.2 является исходной. Она выпущена 1 ноября 2017 г. и использует GATK 3.3.
