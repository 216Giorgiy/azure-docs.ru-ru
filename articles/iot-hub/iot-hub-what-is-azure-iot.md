## Дальнейшие действия

Центр IoT — это служба Azure, которая получает данные телеметрии в требуемом масштабе с ваших устройств и направляет их в обработчик потоковых событий. Вы можете использовать центр IoT для реализации собственной серверной части решения. Центр IoT также может отправлять команды "из облака на устройство" для конкретных устройств. Кроме того, центр IoT включает в себя реестр удостоверений устройств, с помощью которого можно подготавливать устройства и управлять их подключением к центру. Дополнительные сведения см. на следующих ресурсах:

- [Что такое центр IoT?][lnk-iot-hub]
- [Приступая к работе с центром IoT][lnk-getstarted]

Вы можете внедрить клиентские приложения на широком спектре аппаратных платформ и операционных систем с помощью пакетов SDK для устройств IoT. Пакеты SDK для устройств IoT включают библиотеки, упрощающие отправку телеметрии в центр IoT и получение команд "из облака на устройство". При использовании этих пакетов SDK для устройств IoT для взаимодействия с центром IoT доступен выбор из нескольких сетевых протоколов. Дополнительные сведения см. в разделе [Сведения о пакетах SDK для устройств][lnk-device-sdks].

Кроме того, вам может быть интересен набор [Azure IoT Suite][lnk-iot-suite] — коллекция предварительно настроенных решений, позволяющих быстро начать работу и масштабировать проекты IoT для решения распространенных сценариев IoT, например удаленного мониторинга, управления ресурсами-контейнерами и прогнозного обслуживания.
