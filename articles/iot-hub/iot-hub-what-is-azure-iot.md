## Содержание данной документации
Эти статьи об Azure и IoT уделяют основное внимание двум коллекциям ресурсов, которые помогут вам реализовать собственное решение IoT на основе платформы Microsoft IoT.

- Центр Azure IoT
- Пакеты SDK для устройств Azure IoT

Вам также может быть интересен набор [Azure IoT Suite][lnk-iot-suite] — коллекция предварительно настроенных решений, позволяющих быстро начать работу и масштабировать проекты IoT для решения распространенных сценариев IoT, например удаленного мониторинга, управления ресурс-контейнерами и прогнозного обслуживания.

### Центр Azure IoT
Центр IoT — это служба Azure, которая позволяет получать данные "из устройства в облако" в требуемом масштабе из ваших устройств и перенаправляет данные в обработчик потоковых событий. Центр IoT также может отправлять команды "из облака в устройство" на конкретные устройства с помощью очередей для этих устройств.

Кроме того, служба центра IoT включает реестр удостоверений устройств, который помогает подготавливать устройства и управлять подключением устройств к центру IoT.

### Пакеты SDK для устройств Azure IoT
Корпорация Майкрософт предоставляет пакеты SDK для устройств IoT, позволяющие реализовать запуск клиентских приложений на широком спектре аппаратных платформ и операционных систем устройств. Пакеты SDK для устройств IoT включают библиотеки, упрощающие отправку телеметрических данных "из устройства в облако" в центр IoT в и получение команд "из облака в устройство" из центра IoT. Эти пакеты SDK для устройств IoT позволяют выбирать среди нескольких различных сетевых протоколов для взаимодействия с центром Azure IoT.
