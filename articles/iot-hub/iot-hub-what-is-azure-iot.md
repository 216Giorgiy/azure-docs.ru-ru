## Содержание данной документации

Эти статьи об Azure и IoT уделяют основное внимание двум коллекциям ресурсов, которые помогут вам реализовать собственное решение IoT на основе платформы Microsoft IoT.

- Центр Azure IoT
- Пакеты SDK для устройств Azure IoT

Вам также может быть интересен набор [Azure IoT Suite][lnk-iot-suite] — коллекция предварительно настроенных решений, позволяющих быстро начать работу и масштабировать проекты IoT для решения распространенных сценариев IoT, например удаленного мониторинга, управления ресурс-контейнерами и прогнозного обслуживания.

### Центр Azure IoT

Центр IoT — это служба Azure, которая в требуемом масштабе получает данные, передаваемые из устройства в облако, из ваших устройств и перенаправляет их в обработчик потоковых событий. Центр IoT также может отправлять команды "из облака в устройство" на конкретные устройства с помощью очередей для этих устройств.

Кроме того, служба центра IoT включает в себя реестр удостоверений устройств, с помощью которого можно подготавливать устройства и управлять их подключением к центру IoT.

### Пакеты SDK для устройств Azure IoT

Корпорация Майкрософт предоставляет пакеты SDK для устройств IoT, с помощью которых можно реализовать запуск клиентских приложений на широком спектре аппаратных платформ и операционных систем. Пакеты SDK для устройств IoT включают библиотеки, упрощающие отправку телеметрических данных "из устройства в облако" в центр IoT в и получение команд "из облака в устройство" из центра IoT. При использовании этих пакетов SDK для устройств IoT для взаимодействия с центром IoT Azure доступен выбор из множества разных сетевых протоколов.
