# <a name="how-to-use-azure-iot-hub"></a>Использование Центра Интернета вещей в Azure

Вы можете обучиться разработке для служб Центра Интернета вещей разными способами. Например, можно читать тематические статьи, подробно описывающие функции Центра Интернета вещей, или работать с руководствами, в которых рассматриваются разные возможности Центра Интернета вещей.

## <a name="the-developer-guide"></a>Руководство для разработчиков

Разработчики могут ознакомиться с подробной документацией о Центре Интернета вещей из [руководства разработчика][lnk-devguide]. Это руководство содержит подробное описание всех функций Центра Интернета вещей. Оно поможет вам научиться не только использовать эти функции, но и выбирать нужную среди доступных вариантов.


## <a name="the-tutorials"></a>Руководства

Если вы предпочитаете изучать определенные функции Центра Интернета вещей с помощью практических упражнений, доступно несколько руководств. Некоторые руководства описывают работу с разными языками программирования. Ниже представлен список таких руководств.

- [Обработка сообщений Центра Интернета вещей, отправляемых с устройства в облако, с использованием маршрутов (.NET)][lnk-routes-tutorial]. Из этого руководства вы узнаете, как через настройки системы можно использовать правила маршрутизации Центра Интернета вещей для перенаправления сообщений, отправляемых с устройства в облако.

- [Отправка сообщений из облака на устройство с помощью Центра Интернета вещей (.NET)][lnk-c2d-tutorial]. В этом руководстве описано, как отправлять сообщения из облака на устройства с помощью Центра Интернета вещей и получать сообщения, передаваемые из облака на устройство.

- [Передача файлов из устройств в облако в центре IoT][lnk-upload-tutorial]. В этом руководстве описано, как использовать возможности передачи файлов Центра Интернета вещей.

- [Начало работы с двойниками устройств (Node)][lnk-twin-tutorial]. В этом руководстве описываются двойники устройств, сообщаемые свойства, требуемые свойства и теги. Двойники устройств позволяют синхронизировать значения с устройствами.

- [Использование прямых методов (Node)][lnk-methods-tutorial]. В этом руководстве описано использование прямых методов. Вы добавляете обработчик для прямого метода на виртуальном устройстве и вызываете прямой метод из Центра Интернета вещей.

- [Начало работы с управлением устройствами (Node)][lnk-dm-tutorial]. В этом руководстве описано, как с помощью ключевых функций управления устройствами (например, с помощью функции двойника устройства или прямого метода) можно удаленно перезагрузить виртуальное устройство.

- [Настройка устройств с помощью требуемых свойств][lnk-properties-tutorial]. В этом руководстве представлены сведения об удаленной настройке устройства с помощью требуемых и сообщаемых свойств двойника устройства.

- [Используйте управление устройствами, чтобы запустить обновление встроенного ПО устройства (Node/Node).][lnk-jobs-tutorial]. В этом руководстве описано, как с помощью ключевых функций управления устройствами (например, с помощью функции двойника устройства или прямого метода) можно удаленно обновить встроенное программное обеспечение устройства.

- [Учебник. Планирование и трансляция заданий][lnk-schedule-tutorial]. В этом руководстве показано, как использовать требуемые свойства и прямые методы для взаимодействия с несколькими устройствами в запланированное время.
