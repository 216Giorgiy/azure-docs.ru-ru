# Руководства по устранению неполадок Azure Mobile Engagement

## Введение
В следующих руководствах по устранению неполадок рассматриваются наиболее распространенные проблемы с Azure Mobile Engagement. Сюда не включены вопросы по [выставлению счетов][Link 11] и поддержка при регистрации для получения [предварительной версии][Link 7]. Если эти руководства по устранению неполадок не помогли решить вашу проблему, вы можете отправить запрос на обслуживание Azure (при наличии [соглашения о поддержке Azure](http://azure.microsoft.com/support/options/)) или оставить запрос о помощи на [форуме MSDN][Link 8]. При отправке запроса на обслуживание обязательно включите в него всю информацию об устранении неполадок со страницы <a href="#SR" title="SR">Информация о запросе на обслуживание</a>, которая поможет представителю технической поддержки изучить вашу проблему.
