# Установка обновления 0.1 на виртуальный массив StorSimple

## Обзор

В этой статье описываются шаги, необходимые для установки обновления 0.1 на виртуальный массив StorSimple. В ней описываются процедуры обновления через классический портал Azure и через локальный веб-интерфейс. Каждая процедура обновления занимает менее двух минут.

Обновления и исправления необходимы для поддержки виртуального массива StorSimple в актуальном состоянии. Как правило, обновления рекомендуется устанавливать с помощью классического портала Azure. Однако когда портал недоступен, вы можете использовать локальный веб-интерфейс для применения обновлений и исправлений.

Помните, что при установке обновления или исправления происходит перезагрузка устройства. Если виртуальный массив StorSimple — это устройство с одним узлом, то любые выполняемые операции ввода-вывода будут прерваны, а само устройство будет бездействовать в течение некоторого времени.

Перед применением обновления рекомендуется перевести в автономный режим тома или общие папки на узле, а затем перевести в автономный режим само устройство. Это уменьшит возможность повреждения данных.

## Использование классического портала Azure

Рекомендуется устанавливать обновления с помощью классического портала Azure. При использовании портала пользователю нужно проверить наличие обновлений, а затем скачать и установить обновления. Для установки обновления или исправления выполните следующие действия.

#### Установка обновлений с помощью классического портала Azure

1. На странице **Устройства** выберите устройство, на котором требуется установить обновления.

2. Перейдите в раздел **Устройства** > **Обслуживание** > **Обновления программного обеспечения**.

3. При наличии доступных обновлений вы увидите сообщение. Для проверки обновлений также можно нажать кнопку **Проверить наличие обновлений** в нижней части страницы.

	![Проверка наличия обновлений](./media/storsimple-ova-install-update-01/aupdate1m.png)

4. В нижней части страницы щелкните **Скачать обновления**. Появится диалоговое окно с сообщением о том, что обновление нарушит работу системы. Если виртуальный массив StorSimple является устройством с одним узлом, то после обновления он будет перезагружен. Это приведет к нарушению работы всех операций ввода-вывода. Щелкните значок галочки, чтобы запустить задание для скачивания доступных обновлений.

	![Подтверждение скачивания обновлений](./media/storsimple-ova-install-update-01/aupdate3m.png)

	После окончания скачивания вы получите уведомление.

	![Скачивание обновлений](./media/storsimple-ova-install-update-01/aupdate5m.png)

5. Щелкните **Установить обновления** в нижней части страницы, чтобы начать обновление устройства. На экране снова появится диалоговое окно. Щелкните значок галочки, чтобы запустить задание для установки обновлений.
 
 	![Подтверждение установки обновлений](./media/storsimple-ova-install-update-01/aupdate6m.png)
 
 
	После того как задание будет создано, вы получите уведомление.

	![Установка обновлений](./media/storsimple-ova-install-update-01/aupdate8m.png)
	
6. Щелкните ссылку **Просмотреть задание**, чтобы перейти на страницу **Задания** и просмотреть состояние установки. Чтобы получить подробные сведения о задании обновления, можно в любое время нажать кнопку **Сведения**.

	![Отслеживание обновлений](./media/storsimple-ova-install-update-01/aupdate9m.png)

6. После окончания установки (об этом свидетельствует состояние задания "100 %"), перейдите к разделу **Устройства** > **Обслуживание** > **Обновления программного обеспечения**. Отображаемая версия программного обеспечения должна быть следующей: **10.0.10279.0**.

	![Проверка обновлений](./media/storsimple-ova-install-update-01/aupdate13m.png)

## Использование локального веб-интерфейса 

Используйте локальный веб-интерфейс для установки обновлений, если классический портал Azure недоступен. При использовании локального веб-интерфейса установка обновлений включает два этапа:

- Загрузка обновления или исправления
- Установка обновления или исправления

### Загрузка обновления или исправления

Для загрузки обновления программного обеспечения из каталога обновления Майкрософт выполните следующие действия.

#### Загрузка обновления или исправления

1. Запустите Internet Explorer и откройте страницу [http://catalog.update.microsoft.com](http://catalog.update.microsoft.com).

2. Если вы впервые используете каталог Центра обновления Майкрософт на этом компьютере, нажмите кнопку **Установить**, когда будет предложено установить надстройку каталога Центра обновления Майкрософт.
   
	![Установка каталога](./media/storsimple-ova-install-update-01/install-catalog.png)

3. В поле поиска каталога Центра обновления Майкрософт введите номер необходимого исправления в базе знаний, например **3160441**, а затем нажмите кнопку **Найти**.

    Появится список исправлений, например **Обновление 0.1 для виртуального массива Microsoft Azure StorSimple**.

    ![Поиск в каталоге](./media/storsimple-ova-install-update-01/download1.png)

4. Щелкните **Добавить**. Обновление будет добавлено в корзину.

5. Щелкните **Просмотреть корзину**.

6. Щелкните элемент **Загрузить**. Укажите или **выберите** локальное расположение, в которое хотите скачать файлы. Обновления скачиваются в указанное расположение и помещаются во вложенную папку с тем же именем, что и имя обновления. Этот каталог также можно скопировать в сетевую папку, которая доступна с вашего устройства.


### Установка обновления или исправления

Перед установкой обновления или исправления убедитесь, что обновление или исправление загружено локально на вашем узле или доступно в сетевой папке. Для установки обновления или исправления выполните следующие действия.

#### Установка обновления или исправления

1. В локальном веб-интерфейсе перейдите в раздел **Обслуживание** > **Обновление программного обеспечения**.

2. В поле **Путь к файлу обновления** введите имя файла обновления или исправления. Также вы можете найти файл установки обновления или исправления, если он расположен в сетевой папке. Нажмите кнопку **Применить**.

	![обновление устройства](./media/storsimple-ova-install-update-01/update1m.png)

3.  Появится предупреждение. Если это устройство с одним узлом, то после обновления оно будет перезагружено, и появится время простоя. Щелкните значок галочки.

	![обновление устройства](./media/storsimple-ova-install-update-01/update4m.png)

4. Затем начнется обновление. После успешного обновления устройство будет перезапущено. В течение этого времени локальный пользовательский интерфейс будет недоступен.

    ![обновление устройства](./media/storsimple-ova-install-update-01/update6m.png)

4. После перезапуска откроется страница входа. Затем можно проверить версию программного обеспечения. Перейдите в раздел **Обслуживание** > **Обновление программного обеспечения**. Номер сборки должен быть следующим: **10.0.0.0.10279**.

	> [AZURE.NOTE] Версии программного обеспечения в локальном веб-интерфейсе и в классическом портале Azure отображаются немного по-разному. Для одной и той же версии в локальном веб-интерфейсе отображается версия **10.0.0.0.10279**, а в классическом портале Azure — **10.0.10279.0**.

	![обновление устройства](./media/storsimple-ova-install-update-01/update9m.png)
