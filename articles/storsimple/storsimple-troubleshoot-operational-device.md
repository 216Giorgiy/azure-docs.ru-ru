# Устранение неполадок в работающем устройстве StorSimple
## Обзор
В этой статье приведены полезные рекомендации по устранению неполадок с конфигурацией, которые могут возникнуть после развертывания и запуска устройства StorSimple. Здесь описаны типичные проблемы, возможные причины и рекомендуемые действия для устранения проблем, которые могут возникнуть при использовании Microsoft Azure StorSimple. Эти сведения актуальны как для локального физического устройства StorSimple, так и для виртуального устройства StorSimple.

В конце этой статьи вы найдете список кодов ошибок, которые могут возникнуть во время работы Microsoft Azure StorSimple, а также действия, которые необходимо предпринять для устранения ошибок.

## Процесс мастера установки для работающих устройств
Мастер установки ([Invoke-HcsSetupWizard][1]) используется для проверки конфигурации устройства и принятия корректирующих действий по необходимости.

Процесс мастера установки будет отличаться при запуске на ранее настроенном работающем устройстве. Можно изменить только следующие записи:

* IP-адрес, маску подсети и шлюз.
* Основной DNS-сервер.
* Основной NTP-сервер.
* Необязательная конфигурация веб-прокси-сервера.

Мастер установки не выполняет операции, связанные со сбором паролей и регистрацией устройства.

## Ошибки, возникающие при последующих запусках мастера установки
В следующей таблице перечислены ошибки, возникающие при запуске мастера установки на работающем устройстве, возможные причины ошибок и рекомендуемые действия по их устранению.

| № | Сообщение об ошибке или условие | Возможные причины | Рекомендуемое действие |
|:--- |:--- |:--- |:--- |
| 1 |Ошибка 350032: устройство уже деактивировано |Эта ошибка появляется, если запустить мастер настройки на устройстве, которое деактивировано. |[Обратитесь в службу поддержки Майкрософт](storsimple-contact-microsoft-support.md), чтобы узнать, что делать дальше. Деактивированное устройство нельзя ввести в эксплуатацию. Перед повторной активацией устройства может потребоваться сброс параметров к значениям по умолчанию. |
| 2 |Invoke-HcsSetupWizard: ERROR\_INVALID\_FUNCTION (исключение из HRESULT: 0x80070001) |Не удается обновить DNS-сервер. Параметры DNS являются глобальными и применяются ко всем включенным сетевым интерфейсам. |Включите интерфейс и повторно примените параметры DNS. Это может привести к нарушению работы сети для других интерфейсов, так как эти параметры являются глобальными. |
| 3 |Кажется, что устройство в сети (с портала службы диспетчера StorSimple), но при попытке выполнить минимальную установку и сохранить конфигурацию операция завершается ошибкой. |Во время первоначальной установки веб-прокси не был настроен, несмотря на то, что фактически был установлен прокси-сервер. |Используйте командлет [Test-HcsmConnection][2], чтобы найти ошибку. [Обратитесь в службу поддержки Майкрософт](storsimple-contact-microsoft-support.md) если не удается устранить проблему. |
| 4 |Invoke-HcsSetupWizard: значение не попадает в пределы ожидаемого диапазона. |Эта ошибка вызвана неправильной маской подсети. Возможные причины: <ul><li> Маска подсети отсутствует или пустая.</li><li>Неправильный формат префикса IPv6.</li><li>Интерфейс поддерживает работу с облаком, но отсутствует или неверен шлюз.</li></ul>Обратите внимание, что DATA 0 автоматически поддерживает работу с облаком, если настраивается с помощью мастера установки. |Чтобы определить проблему, используйте подсеть 0.0.0.0 или 256.256.256.256 и изучите выходные данные. При необходимости введите правильные значения для маски подсети, шлюза и префикса IPv6. |

|:--- |:--- |:--- |
| 10502 |Произошла ошибка при попытке доступа к учетной записи хранения. |Подождите несколько минут и повторите попытку. Если проблема возникнет снова, обратитесь в службу поддержки Майкрософт. |
| 40017 |Не удается разрешить диск в резервном наборе данных. |Если проблема возникнет снова, обратитесь в службу поддержки Майкрософт. |
| 40018 |Не удается разрешить ни один диск в резервном наборе данных. |Если проблема возникнет снова, обратитесь в службу поддержки Майкрософт. |
| 390061 |Система занята или недоступна. |Подождите несколько минут и повторите попытку. Если проблема возникнет снова, обратитесь в службу поддержки Майкрософт. |
| 390143 |Произошла ошибка с кодом ошибки 390143. (Неизвестная ошибка). |Если проблема возникнет снова, обратитесь в службу поддержки Майкрософт. |
