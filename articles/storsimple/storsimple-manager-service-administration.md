# <a name="use-the-storsimple-manager-service-to-administer-your-storsimple-device"></a>Использование службы StorSimple Manager для администрирования устройства StorSimple
> [!NOTE]
> Классический портал StorSimple устарел. Диспетчеры устройств StorSimple автоматически перейдут на новый портал Azure в соответствии с графиком устаревания. Вы получите сообщение электронной почты и уведомление с портала, касающиеся этого перехода. Этот документ скоро также перестанет использоваться. Чтобы просмотреть версию этой статьи для нового портала Azure, перейдите к статье [Использование службы диспетчера устройств StorSimple для администрирования устройства StorSimple](storsimple-8000-manager-service-administration.md). Сведения, связанные с переходом, см. в [ответах на вопросы о перемещении на портал Azure](storsimple-8000-move-azure-portal-faq.md).

## <a name="overview"></a>Обзор
В этой статье описывается интерфейс службы StorSimple Manager, включая способ подключения, разные доступные параметры и ссылки на определенные рабочие процессы, которые можно выполнить через этот пользовательский интерфейс. Это руководство применимо как к физическому, так и к виртуальному устройству StorSimple.

После прочтения этой статьи вы узнаете, как выполнить следующее.

* Подключиться к службе StorSimple Manager.
* работать с пользовательским интерфейсом StorSimple Manager;
* Администрировать устройство StorSimple с помощью службы StorSimple Manager.

## <a name="connect-to-storsimple-manager-service"></a>Подключиться к службе StorSimple Manager.
Служба StorSimple Manager выполняется в Microsoft Azure и подключается к нескольким устройствам StorSimple. Для управления этими устройствами используется центральный классический портал Microsoft Azure в браузере. Чтобы подключиться к службе StorSimple Manager, выполните следующие действия.

#### <a name="to-connect-to-the-service"></a>Подключение к службе
1. Откройте страницу [https://manage.windowsazure.com/](https://manage.windowsazure.com/).
2. С помощью учетной записи Майкрософт выполните вход на классический портал Microsoft Azure (расположен в правой верхней части панели).
3. Прокрутите вниз левую панель навигации для доступа к службе StorSimple Manager.

## <a name="navigate-storsimple-manager-service-ui"></a>Навигация по пользовательскому интерфейсу службы StorSimple Manager
