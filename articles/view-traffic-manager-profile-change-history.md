# Просмотр журнала изменений профиля диспетчера трафика

Журнал изменений профиля диспетчера трафика можно просмотреть на портале управления в разделе "Службы управления".

## Чтобы просмотреть журнал изменений диспетчера трафика, выполните следующее:


1. На левой панели портала управления щелкните **Службы управления**.

2. На странице служб управления щелкните **Журналы операций**.

3. На странице журналов операций примените фильтр, чтобы просмотреть журнал изменений для вашего профиля диспетчера трафика. Выбрав параметры фильтрации, установите флажок, чтобы просмотреть результаты.
	* Чтобы просмотреть изменения по всем профилям, выберите подписку и временной диапазон и щелкните **Диспетчер трафика** в раскрывающемся списке **Тип**.
	* Чтобы отфильтровать информацию по имени профиля, введите имя профиля в поле **Имя службы** или выберите его из раскрывающегося списка.
	* Чтобы просмотреть информацию о каждом отдельном изменении, выберите строку с необходимым изменением для просмотра и нажмите кнопку **Сведения** внизу страницы. В окне **Сведения об операции** можно просмотреть XML-представление объекта API, который был создан или обновлен в рамках операции, а также скопировать код XML в буфер обмена.
