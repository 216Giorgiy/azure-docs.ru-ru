# <a name="rate-limiting-for-sms-emails-and-webhooks"></a>Ограничение частоты отправки для SMS, сообщений электронной почты и вызовов Webhook
Ограничение частоты означает приостановку уведомлений, которые слишком часто отправляются на указанный номер телефона или адрес электронной почты. Ограничение скорости гарантирует управляемость и действенность оповещений.

Правила для сообщений SMS и электронной почты не отличаются. Порог частоты для
 - SMS — 10 сообщений в час
 - Сообщений электронной почты — 100 сообщений в час

## <a name="rate-limit-rules"></a>Правила ограничения частоты
- Нужный номер телефона или адрес электронной почты — если частота превысит пороговое значение.
- Номер телефона или адрес электронной почты может быть частью групп действий в нескольких подписках. Ограничение частоты применяется ко всем подпискам, то есть применяется, как только достигается пороговое значение, даже если уведомления отправлены из нескольких подписок.  
- Если для номера телефона или адреса электронной почты ограничена частота, отправляется уведомление, в котором сообщается об ограничении скорости. Уведомление сообщает также, когда истекает срок действия ограничения частоты.

## <a name="rate-limit-of-webhooks"></a>Ограничение частоты вызовов Webhook ##
Для вызовов Webhook сейчас нет ограничения частоты.
