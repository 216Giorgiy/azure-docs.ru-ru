# Управление контентом в службах мультимедиа #
Представление контента служб мультимедиа Azure позволяет управлять контентом мультимедиа для вашей учетной записи служб мультимедиа.

В настоящее время непосредственно на портале можно выполнять следующие операции с контентом:

- Просмотр таких сведений о контенте, как состояние публикации, опубликованный URL-адрес, размер, а также дата и время последнего обновления.
- Загрузка нового контента
- Кодирование контента
- Воспроизведение видеоконтента
- Публикация и отмена публикации контента
- Удаление контента


## Практическое руководство. Загрузка контента ##


1. На [портале управления](http://go.microsoft.com/fwlink/?LinkID=256666&clcid=0x409) выберите **Службы мультимедиа**, а затем щелкните имя учетной записи служб мультимедиа.
2. Выберите страницу "КОНТЕНТ". 
3. Нажмите кнопку кнопки **Загрузка** на странице или в нижней части портала. 
4. В диалоговом окне **Загрузка содержимого** перейдите к нужному файлу актива. Щелкните файл и нажмите кнопку **Открыть** либо нажмите клавишу **Ввод**.

	![UploadContentDialog][uploadcontent]

5. В диалоговом окне "Отправка контента" нажмите кнопку с флажком, чтобы принять значения полей "Файл" и "Имя контента".
6. Начнется загрузка, можно отслеживать ход выполнения в нижней части портала.  

	![JobStatus][status]

После завершения загрузки в списке "Контент" появится новый актив. По соглашению в конец имени будет добавлено "**-Источник**" для отслеживания нового контента как источника контента для задач кодирования.

Если значение размера файла не обновляется после остановки процесса загрузки, нажмите кнопку **Синхронизировать метаданные**. Это синхронизирует размер файла актива с фактическим размером файла в хранилище и обновляет значение на странице "Контент".	


## Практическое руководство. Кодирование контента

1. На [портале управления](http://go.microsoft.com/fwlink/?LinkID=256666&clcid=0x409) выберите **Службы мультимедиа**, а затем щелкните имя учетной записи служб мультимедиа. 
2. Щелкните страницу КОНТЕНТ в верхней части страницы.
3. Щелкните нужный источник видео для задания кодирования, а затем нажмите кнопку **Кодировать** в нижней части страницы.
4. В диалоговом окне "Средство кодирования мультимедиа Azure" выберите одну из общих или дополнительных предустановок кодирования.

	**Общие предустановки**

	+ Воспроизведение на ПК/Макинтоше (с помощью Flash или Silverlight)**. Эта предустановка позволяет получить актив с бесперебойной потоковой передачей, который имеет следующие характеристики: стереозвук CBR 44,1 кГц в разрядности 16 бит/сэмпл с кодировкой 96 Кбит/с в AAC, а также видео CBR с разрешением 720 пикселей и 6 скоростями — от 3400 Кбит/с до 400 Кбит/с — с использованием основного профиля H.264 и двумя вторичными группами GOP.
	+ Воспроизведение в HTML5 (IE/Chrome/Safari)** Эта предустановка позволяет получить отдельный MP4-файл, который имеет следующие характеристики: стереозвук CBR 44,1 кГц в разрядности 16 бит/сэмпл с кодировкой 128 Кбит/с в AAC, а также видео CBR с разрешением 720 пикселей и скоростью 4500 Кбит/с с использованием основного профиля H.264.
	+ Воспроизведение на устройствах iOS, а также ПК/Макинтошах**. Эта предустановка позволяет получить актив с теми же характеристиками, что и у актива с бесперебойной потоковой передачей (описан выше), но в формате, который может использоваться для доставки потоков Apple HLS на устройства iOS. 

	**Дополнительные предустановки**
	
	+ В разделе [Строки предустановок задач для кодировщика мультимедиа Azure](http://go.microsoft.com/fwlink/?LinkId=270865) объясняется, что означает каждая их предустановок в списке "Дополнительные предустановки". 


	![EncoderDialog][encoder]

	В настоящее время портал не поддерживает все форматы кодировки, поддерживаемые кодировщиком мультимедиа. Он также не поддерживает шифрование и дешифрование мультимедийных активов. Эти задачи можно выполнить программным образом. Дополнительные сведения см. в разделах [Создание приложений с помощью пакета SDK служб мультимедиа для .NET](http://go.microsoft.com/fwlink/?LinkId=270866) и [Строки предустановок задач для кодировщика мультимедиа Azure](http://go.microsoft.com/fwlink/?LinkId=270865)


5. В диалоговом окне кодировщика мультимедиа Azure введите нужное понятное имя выводимого контента или примите значение по умолчанию. Нажмите кнопку проверки, чтобы начать операцию кодирования, ход выполнения которой можно отслеживать в нижней части портала.

После завершения кодирования представление должно иметь вид, схожий с приведенным ниже снимком экрана. 

	![PortalViewUploadCompleted][portaloverview]


Если значение размера файла не обновляется после завершения кодирования, нажмите кнопку **Синхронизировать метаданные**. Это синхронизирует размер файла выводимого актива с фактическим размером файла в хранилище и обновляет значение на странице "Контент".	

## Практическое руководство. Публикация контента

1. На [портале управления](http://go.microsoft.com/fwlink/?LinkID=256666&clcid=0x409) выберите **Службы мультимедиа**, а затем щелкните имя учетной записи служб мультимедиа.
2. Щелкните страницу КОНТЕНТ в верхней части страницы.
3. Щелкните актив, который не был опубликован. Нажмите кнопку "Опубликовать", чтобы выполнить публикацию по открытому URL-адресу. После публикации контента на URL-адрес этот URL-адрес может быть открыт проигрывателем клиента, если тот может отображать контент в данной кодировке.

 ![PublishedContent][publishedcontent]

## Практическое руководство. Воспроизведение контента на портале


1. На [портале управления](http://go.microsoft.com/fwlink/?LinkID=256666&clcid=0x409) выберите **Службы мультимедиа**, а затем щелкните имя учетной записи служб мультимедиа.
2. Щелкните страницу КОНТЕНТ в верхней части страницы.
3. Выберите нужный видеоконтент и нажмите кнопку **Воспроизведение** в нижней части портала. На портале можно воспроизводить только опубликованный контент. Кроме того, данная кодировка должна поддерживаться вашим веб-браузером.
