# Воспроизведение содержимого

Службы мультимедиа Azure поддерживают многие популярные форматы потоковой передачи, например Smooth Streaming, HTTP Live Streaming и MPEG-Dash. В этом подразделе описываются имеющиеся проигрыватели, которые можно использовать для тестирования потоков. В разделе также приведены ссылки на пакеты SDK и платформы проигрывателей, которые можно использовать для разработки собственных клиентских приложений, способных использовать потоковое мультимедиа из служб мультимедиа.
