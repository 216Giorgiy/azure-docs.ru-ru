# Сочетания клавиш портала Azure 

## Открыть концентраторы

| Клавиша | Действие |
|--------|----------|
| Нет | Открыть концентратор уведомлений |
| S или / | Открыть концентратор обзора (служб) | 
| A | Выбрать активный сеанс | 
| C | Создание | 

## Навигация

| Клавиша | Действие |
|--------|----------|
| F | Перейти на первую открытую колонку | 
| H | Переключиться между главной страницей портала Azure и текущей колонкой | 
| J | Перейти к предыдущей открытой колонке | 
| K | Перейти к следующей открытой колонке | 
| L | Перейти к последней открытой колонке | 
| - | Свернуть колонку или восстановить из развернутого состояния | 
| + | Развернуть колонку или восстановить из свернутого состояния | 
| K | Перейти к следующей открытой колонке | 


## Управление
| Клавиша | Действие |
|--------|----------|
| ESC | Закрыть меню концентратора, диалоговое окно, меню учетной записи или любое открытое меню | 
| # | Закрыть все открытые колонки | 
| HOME | Прокрутить до верха текущей колонки | 
| END | Прокрутить до низа текущей колонки | 
| CTRL+M | Переключиться между переходом по элементам управления с помощью клавиши TAB и выбором вкладок внутри управления редактированием | 

# Дальнейшие действия

Познакомьтесь с порталом Azure поближе: http://azure.microsoft.com.
