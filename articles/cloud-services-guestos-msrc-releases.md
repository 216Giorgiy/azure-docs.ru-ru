# Гостевая ОС Azure 

В следующей таблице приведены обновления Microsoft Security Response Center (MSRC) для гостевой ОС Azure. Выполните поиск по этой статье, чтобы определить, применяется ли какое-либо конкретное обновление к используемой гостевой ОС. Обновления всегда пролонгируются для того [семейства версий][family-explain], в котором они были введны.
