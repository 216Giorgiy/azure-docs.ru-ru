# <a name="azure-sql-data-warehouse-gen2-support-for-lower-compute-tiers"></a>Хранилище данных SQL Azure 2-го поколения. Поддержка более низких уровней вычислений

Корпорация Майкрософт помогает снизить начальные расходы на хранилище данных, способное обрабатывать ресурсоемкие запросы, добавляя более низкие уровни вычислений для невероятно быстрого Хранилища данных SQL Azure 2-го поколения. Клиенты могут оценить лучшую производительность, гибкость и возможности безопасности Хранилища данных SQL Azure, начиная со 100 cDWU (вычислительные единицы использования хранилища данных), и увеличивать масштаб до 30 000 cDWU за считаные минуты. Клиенты могут извлечь выгоду из производительности и гибкости 2-го поколения благодаря более низким уровням вычислений. 

Обеспечив точку входа для хранилищ данных следующего поколения, корпорация Майкрософт открыла возможности для ориентированных на прибыль пользователей, которые хотят оценить все преимущества безопасного, высокопроизводительного хранилища данных, не тратя время на поиски наиболее подходящей для них пробной среды.  Пользователи смогут начать со 100 cDWU, что ниже текущей точки входа (500 cDWU).  Хранилище данных SQL 2-го поколения продолжает поддерживать операции остановки и возобновления и выходит за рамки простой гибкости при вычислениях.  2-е поколение также поддерживает неограниченную емкость хранилища столбцов, а еще выделяет в 2,5 раза больше памяти для каждого запроса, обеспечивает до 128 параллельных запросов и функции адаптивного кэширования. Это в среднем в 5 раз повышает производительность в сравнении с той же единицей использования хранилища данных 1-го поколения по той же цене.  Геоизбыточные резервные копии являются стандартными для 2-го поколения и обеспечивают встроенную гарантированную защиту данных. Хранилище данных SQL Azure 2-го поколения может выполнить масштабирование, когда вам нужно.

В настоящее время портал не поддерживает развертывание или масштабирование с переходом на экземпляры нижнего уровня 2-го поколения. Мы усердно работаем над созданием этой функциональности. Тем временем вы можете [отправить запрос](sql-data-warehouse-get-started-create-support-ticket.md), если хотите получить преимущества этого нового уровня.

## <a name="getting-started-with-azure-sql-data-warehouse-gen2"></a>Начало работы с Хранилищем данных SQL Azure 2-го поколения 

Клиенты могут развернуть новый экземпляр 2-го поколения или обновить существующий экземпляр хранилища данных 1-го поколения, чтобы получить гибкость и производительность хранилища данных следующего поколения. 

Испытайте [уровень Хранилища данных SQL Azure 2-го поколения, оптимизированный для вычислений.](https://azure.microsoft.com/services/sql-data-warehouse/?v=17.44)
Обновите [Хранилище данных SQL Azure 1-го поколения, оптимизированное для вычислений, до 2-го поколения](https://docs.microsoft.com/azure/sql-data-warehouse/upgrade-to-latest-generation). Вы можете увидеть Хранилище данных SQL Azure 2-го поколения в действии в этом [видео Microsoft Mechanics](https://www.youtube.com/watch?v=Ap8I3UZonzI&feature=youtu.be).
