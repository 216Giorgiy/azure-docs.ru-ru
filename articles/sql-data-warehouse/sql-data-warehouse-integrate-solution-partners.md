# Партнерские решения для хранилища данных SQL

Для создания собственного комплексного решение для хранилища данных можно воспользоваться одним из широкого спектра ведущих профессиональных средств. В этой статье перечислены партнерские компании, которые разработали решения для хранилища данных SQL.

| ------- | ----------- |------|

Решение упрощает и ускоряет загрузку пакетной службы, а также добавочную репликацию из разнородных универсальных баз данных, включая SQL Server, Oracle, DB2, Sybase и mySQL. | [ Подробнее ][attunity_more]|

Затем бизнес-пользователи в организации создают отчеты и анализируют сведения с помощью ведущих средств бизнес-аналитики.| [Подробнее][birst_more] |

Средство обеспечивает быструю совместную работу для бизнес-пользователей, которые смогут проводить более точный анализ.| [Подробнее][clearstory_data_more] |

Выбери, щелкни, создай отчет! | [Подробнее][coffing_data_warehousing_more] |

Dell Statistica упрощает анализ больших данных, ускоряет развертывание, создает отчеты из текста и дает новые возможности вашей компании. | [Подробнее][dell_statistica_more] |
| ![Dundas][7] |**Dundas Software**<br>Dundas Data Visualization — это ведущий глобальный поставщик программного обеспечения для бизнес-аналитики и визуализации данных. Панели мониторинга Dundas, отчеты и визуальный анализ данных обеспечивают гибкую интеграцию с приложениями компании, позволяя принимать более взвешенные решения. | [Подробнее][dundas_software_more] |
| ![Inbrein][8] |**Inbrein**<br>Inbrein MicroERD обеспечивает средства, которые потребуются вам, чтобы создавать точные модели данных, снижать избыточность данных, повышать эффективность и следовать стандартам. Используя графический интерфейс, разработанный с учетом опыта работы пользователей, разработчик может легко создавать модели баз данных. Вы сможете использовать новые функции MicroERD благодаря регулярным и быстрым обновлениям. | [Подробнее][inbrein_more] |

Кроме того, мы предоставляем программные решения для управления метаданными, данными и ресурсами, которые позволяют управлять метаданными и публиковать их из разных средств и технологий. | [Подробнее][infolibrarian_more] |
| ![Informatica][10] |**Informatica**<br>Informatica — один из наиболее узнаваемых независимых поставщиков программного обеспечения для интеграции данных в мире. Informatica Vibe, одна из первых внедряемых виртуальных машин, объединяет уникальные функции по технологии "Свяжи один раз. Развертывание в любом месте" на базе платформы Informatica. Более 5500 компаний во всем мире пользуются решениями от Informatica для управления ресурсами данных.| [Подробнее][informatica_more] |
| ![Jinfonet][11] |**Jinfonet**<br>JReport — встраиваемое решение бизнес-аналитики для компаний. Решение позволяет пользователям создавать отчеты, панели мониторинга и выполнять анализ данных в облаке, работать с большими объемами данных и источниками транзакционных данных. Визуализируя данные, пользователи могут создавать собственные отчеты и исследовать данные для принятия решений на ходу.| [Подробнее][jinfonet_more] |
| ![LogiAnalytics][12] |**LogiAnalytics**<br>Together, Logi Analytics и хранилище данных SQL Microsoft Azure отныне позволяют вашей организации собирать и анализировать большие и очень разнообразные наборы данных, а затем сразу же принимать решения на их основе, повышая эффективность. | [Подробнее][logianalytics_more] |

Компании смогли существенно изменить свою культуру благодаря средству Looker. | [Подробнее][looker_more] |
| ![Qlik][14] |**Qlik**<br>Qlik обеспечивает кросс-платформенный подход к визуальной аналитике во взаимодействии с хранилищем данных SQL Microsoft Azure. Используя функции корпоративного класса в рамках эластичного хранилища данных Майкрософт, Qlik позволяет целой организации уверенно принимать быстрые решения. Qlik обеспечивает прогнозы и ясность там, где это необходимо: в принятии решений. | [Подробнее][qlik_more] |

Используя преимущества хранилища данных SQL Microsoft Azure, пользователи RedPoint доводят до максимума ценность структурированных и неструктурированных данных и обеспечивают точные и контекстные взаимодействия, которые так необходимы для современного многоканального потребителя. Интерфейс обеспечивает перетаскивание данных и упрощает процесс управления данными. | [Подробнее][redpoint_global_more] |
| ![SiSense][16] |**SiSense**<br>SiSense — полноценное программное обеспечение бизнес-аналитики с инструментами, которые нужны компании для анализа и визуализации данных: эффективная база данных, возможность подключаться к разным источникам, простое извлечение данных и визуализация данных в браузере. Начните анализировать и визуализировать большие объемы данных с помощью SiSense BI and Analytics уже сегодня. | [Подробнее][sisense_more] |
| ![SnapLogic][17] |**SnapLogic**<br>Платформа SnapLogic позволяет выполнять быстрый перенос данных в хранилище данных SQL Microsoft Azure и из него. SnapLogic обеспечивает возможность интеграции с сотней разных приложений и служб в облаке, локально или в гибридной модели для более точного и эффективного анализа. | [Подробнее][snaplogic_more] |
| ![SQL Sentry][18] |**SQL Sentry**<br>SQL Sentry Performance Advisor обеспечивает актуальные и исторические представления наиболее важных данных из хранилища данных SQL Azure, позволяя увидеть тенденции изменения производительности и получить подробные сведения для более глубокого анализа и устранения неполадок. Performance Advisor обеспечивает просмотр данных производительности для любого момента времени или диапазона дат. | [Подробнее][sql_sentry_more] |
| ![Tableau][19] |**Tableau**<br>Средство анализа Tableau поможет увидеть и понять данные из разных источников, от неструктурированных файлов до баз данных. Tableau имеет собственное, оптимизированное средство подключения к хранилищу данных SQL Microsoft Azure, которое поддерживает работу с актуальными данными и анализ в памяти. | [Подробнее][tableau_more] |
| ![Targit][20] |**Targit**<br>Targit Decision Suite — это платформа бизнес-аналитики, которая объединяет актуальные панели мониторинга, аналитику, понятные отчеты, превосходные мобильные возможности и простые технологии работы с данными в одном, связанном решении. Targit дает компаниям мужество для действия. | [Подробнее][targit_more] |
| ![Yellowfin][21] |**Yellowfin**<br>По версии исследования в области бизнес-аналитики от BARC, Yellowfin — это ведущий поставщик облачной бизнес-аналитики для неструктурированных отчетов и панелей мониторинга. Подключитесь к хранилищу данных SQL Azure, создавайте и публикуйте красивые отчеты и панели мониторинга с помощью ведущей бизнес-аналитики и функций анализа. | [Подробнее][yellowfin_more] |
