# Все разделы о службе хранилища данных SQL Azure

В этом разделе перечислены все разделы, относящиеся к службе **хранилища данных SQL Azure**. На этой веб-странице с помощью клавиш **Ctrl+F** можно искать интересующие вас разделы по ключевым словам.

| 2 | overview-workload | [Рабочая нагрузка хранилища данных](sql-data-warehouse-overview-workload.md) | Эластичность хранилища данных SQL — это возможность повышать и понижать вычислительную мощность, а также приостанавливать вычислительную работу с помощью обычного ползунка изменения уровня производительности (количества DWU).
