#  Коллекция изображений государственных Microsoft Azure

<p> Коллекция изображения государственных Microsoft Azure - аналогичные возможности открытый портал Azure.  Клиенты можно развертывать образы, предварительно построенного корпорации Майкрософт и наших партнеров или передать их собственных виртуальных жестких дисков. Это обеспечивает гибкость в развертывании собственных стандартных образов при необходимости.

Ниже содержит список доступных образов в коллекции образов Azure правительства.  Некоторые из готовых образов включают повременную оплату лицензирования для конкретного программного обеспечения.  Ознакомьтесь с <a href="http://azure.microsoft.com/pricing/details/virtual-machines/">цен виртуальной машины</a> страницы Дополнительные рекомендации и работы с Microsoft учетной записи группы или через посредников для конкретного ценообразования Azure правительства.
