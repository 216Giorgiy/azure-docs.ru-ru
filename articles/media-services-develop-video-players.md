#Разработка приложений видеопроигрывателя

##Обзор

Службы мультимедиа Azure предоставляют средства, необходимые для создания динамических многофункциональных клиентских приложений проигрывателя для большинства платформ, включая: устройства iOS, устройства Android, Windows, Windows Phone, Xbox и абонентские приставки.

- [Клиентский пакет SDK Smooth Streaming](http://www.iis.net/downloads/microsoft/smooth-streaming) 
- [Платформа Microsoft Media - Платформа проигрывателя](http://playerframework.codeplex.com/) 
- [Документация по платформе проигрывателя HTML5](http://playerframework.codeplex.com/wikipage?title=HTML5%20Player&referringTitle=Documentation) 
- [Подключаемый модуль Smooth Streaming для OSMF](https://www.microsoft.com/download/details.aspx?id=36057) 
- [Платформа проигрывателя для iOS](https://github.com/Azure/azure-media-player-framework) 
- [Лицензирование пакета для портирования клиента бесперебойной потоковой передачи Microsoft(r) Smooth Streaming](https://www.microsoft.com/mediaplatform/sspk.aspx) 
- Создание приложений для воспроизведения видео в Windows 8 
- [Разработка приложений для воспроизведения видео на XBOX](http://xbox.create.msdn.com/)
