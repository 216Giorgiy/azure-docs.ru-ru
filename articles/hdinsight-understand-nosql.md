# Управление данными: общие сведения о технологиях NoSQL в Azure

**Сводка:** Azure предлагает широкий диапазон вариантов для рабочих нагрузок NoSQL, однако выбор правильного варианта может быть непростым. В данном документе рассматриваются технологии NoSQL в Azure и их отличия от реляционных моделей управления данными.

Реляционные технологии уже давно являются главным подходом для работы с данными. Однако при работе с большим объемом разнотипных данных (как это происходит сегодня) вы столкнетесь с некоторыми сложностями при управлении данными. Реляционные технологии в некоторых случаях являются наилучшим выбором для управления данными, а иногда - нет. Это руководство содержит пошаговые инструкции по вариантам с разъяснениями относительно возможностей каждого из них и потенциальных причин его выбора.


**Содержание:**

* Данные в Azure: Общая картина	
* Реляционные технологии Azure: Краткий обзор	
* Технологии Azure NoSQL
	* Хранилища документов: DocumentDB
	* Хранилища пар "ключ-значение": Таблицы
	* Хранилища семейств столбцов: HBase
	* Аналитика для данных большого размера: HDInsight
