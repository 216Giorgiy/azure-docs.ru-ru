## <a name="well-done"></a>Все готово!
Вы выполнили руководство по началу работы. Вы научились выполнять следующие задачи:

> [!div class="checklist"]
> * создание в Azure среды на основе Kubernetes, которая оптимизирована для разработки;
> * итеративная разработка кода в контейнерах с помощью VS Code и командной строки;
> * независимая разработка двух отдельных служб и вызов другой службы с помощью обнаружения службы DNS Kubernetes;
> * эффективная разработка и тестирование кода в среде командной работы.

Теперь, после ознакомления с Azure Dev Spaces, [предоставьте доступ к среде разработки своему сотруднику](../how-to/share-dev-spaces.md) и покажите ему, как просто работать вместе.
