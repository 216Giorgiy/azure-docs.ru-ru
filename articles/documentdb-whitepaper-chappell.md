# Технический документ, посвященный службе DocumentDB (автор: Дэвид Чэппел (David Chappell)) 

**Сводка:** Ознакомьтесь с информацией о DocumentDB, управляемой базе данных NoSQL в Microsoft Azure. DocumentDB - идеальная технология для построения базы данных в следующих случаях:

* для вашего приложения целесообразно использовать модель данных без схемы;
* база данных должна обладать возможностью масштабирования до больших объемов хранения и высокой пропускной способности;
* вы хотите использовать знакомый язык запросов и платформу разработки.

В этом техническом документе также рассматриваются следующие основные понятия:

* модель данных DocumentDB;
* работа приложений с данными;
* возможности для достижения баланса между производительностью и согласованностью в приложениях.
